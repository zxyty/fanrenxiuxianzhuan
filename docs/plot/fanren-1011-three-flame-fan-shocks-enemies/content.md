# 第六卷 通天灵宝 第一千一十一章 一扇惊敌

## 场景
虚天殿白玉广场，韩立面对天澜圣殿与阴罗宗联手的五人围攻。

## 对峙与索宝
- 天澜圣殿**徐姓青年**(白袍、元婴后期)冷漠质问韩立是否曾击杀阴罗宗一名元婴(婴)中期顶峰长老。
- 韩立承认：的确在天南灭杀了一名阴罗宗长老；并以元婴中期修为击杀同阶顶峰修士。
- 徐姓青年提出交易：
  - 交出当年用「古怪小鼎」(**虚天鼎**)收走的圣殿传承圣鼎与圣兽分身 → 可让圣殿与韩立化干戈为玉帛。
  - 交出**鬼罗幡** → 徐某尽力替韩立化解与阴罗宗的恩怨。
- 天澜**圣女林银屏**欲出言反对，被白袍青年抬手制止(「圣女不用多说，这事我自有主张」)。
- 韩立一口回绝索要虚天鼎，让对方动手。

## 交手
- 徐姓青年祭出**碧绿色玉如意**，绿光闪动，可化为蛇身兽首不知名怪兽护身。
- 韩立背现一对银白色翅膀(**风雷翅**)，取出早扣在手中的**三焰宝扇**(金银红三色灵芒)。
- 葛天豪看出三焰扇不凡，提议众人齐上擒拿；与两名黑衫老者各祭飞剑/磷火骨刀，林银屏取出绣银蚕锦帕。
- 韩立以雷遁瞬移至十余丈外石阶；徐姓青年身法诡异，瞬移不逊雷遁。
- 韩立挥三焰扇 → **三色火凤**扑出，化为不断涌现三色符文的神秘光晕，庞大灵压令人窒息，小半白玉广场寸寸碎裂融化。

## 结果
- **徐姓青年**：以玉如意护身遁出，人无损，但玉如意为抗光晕硬生生被毁成碎屑(未及发挥真正威力)。
- **葛天豪**：骨刀(与心神有联系)被融化，心神受创，一臂焦糊受伤不轻。
- **两名黑衫老者**：飞剑被毁，及时断联，仅损半片衣衫与眉毛发髻。
- **林银屏**：位置最远，以一面乌黑巨盾护身，波及较轻。
- **韩立**：不停留，服下**万年灵乳**(灵药)，借雷遁逃往最近石阶——恰是通往**昆吾殿**的方向(前方有乾老魔等修士)，欲借山上禁制地形周旋。

## 敌方判断与追击
- 徐姓青年判断三焰扇是某件**通天灵宝的仿制品**(威力单一、无其他变化)，若是真品葛天豪已死；并断言此扇耗法力极大，故韩立才转身逃走。
- 葛天豪主张趁韩立法力未复时追杀，放出**灵犀孔雀**(五色孔雀)。
- 白袍青年率先向石阶飞射，誓不放韩立逃脱，以绝后患。
