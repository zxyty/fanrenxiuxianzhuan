# 第四卷 风起海外 第484章｜化身符与补天丹

## 一、星宫搅局
- 半空中的**蛮胡子**察觉有人鬼鬼祟祟，一拳击出丈许金色巨手，逼出一道青色光罩——**星宫执法长老**现身。正魔双方惊呼「星宫？」。
- **极阴祖师**质问：星宫号称永不进内殿，为何偷偷摸摸。白衣长老不答，反称防御寒气时使了小手段被蛮胡子发现，并赞蛮胡子为此间修为第一人。
- 极阴心怒却忌惮星宫名头不敢主动出手；**万天明**等人亦心怀顾忌，空中一时鸦雀无声。

## 二、偷袭：斩血玉蜘蛛、击飞韩立
- **儒衫老者**识破对方含糊可疑，惊呼「还有一人！小心血玉蜘蛛和韩立！」。
- 祭坛另一侧低空迸射两道炙热白虹，分扑韩立与血玉蜘蛛，原地才现出**另一位白衣长老**。
- **韩立**反应不及，仅以正把玩的**玉如意**挡向胸前；一声闷响后双手巨震失去知觉，倒飞撞上巨型护罩贴壁滑落，虎口彻底震裂、血淋淋。
- 星宫长老以法宝**贯日剑**击中韩立，竟未将其斩为两截，判断其有**顶阶宝物护身**；但另一把飞剑已奏效。
- **血玉蜘蛛**被另一道白虹切成两截、鲜血遍地；已露小半截的**乾蓝冰焰**又无声下沉。仅凭两只异种火蟒无法拉起虚天鼎。

## 三、化身符遁去
- 蛮胡子怒而欲出手，星宫老者却在乳白莹光中化为点点星光消失，只留一张淡金色符箓飘落；光罩中另一长老同样化星光溃散，留下一模一样金色符箓。
- 蛮胡子认出此为**星宫的化身符**——并非真身亲至，事后亦无法秋后算账。符箓落地后自燃成灰，不留痕迹。

## 四、内殿第五层：真身议论补天丹
- 内殿第五层隐秘角落，两名盘膝白衣老者睁眼，庆幸此行阻止了虚天鼎被夺；遗憾化身被发现过早，否则可趁正魔纠缠暗中夺宝。
- 二人寿元将尽，决定留待下次虚天殿浮现再取虚天鼎。
- **补天丹议论**：
  - 传闻能弥补灵根不纯、精炼先天灵根，令人不可思议。
  - 另有传言称其可增寿元、突破元婴期瓶颈，引得众多元婴老怪信以为真；但星宫长老讥讽：若真如此，**六道**与**疯婆子**岂会不来。
  - 真实功效：对**结丹**期修士凝练**元婴**大有益处，唯炼化极难。
  - **星宫老宫主**早年曾千辛万苦取出过一枚，故星宫对此物念念不忘。
- 二人随即离去，以免几个老怪恼羞成怒寻晦气。

## 五、补天丹现世
- 正魔双方懊恼郁闷却很快恢复冷静，继续空中对峙。
- 韩立止住下落重新腾空，惊看震裂的双手，目光却落在右手紧握的玉如意上。
- **极阴祖师**的一对**异种火蟒**终于支持不住，哀鸣后缩回原状，**虚天鼎**急速下坠。
- 虚天鼎下落中受刺激，嗡鸣化为龙吟之啸，一声雷鸣后，一团拳头大小的**五色光团**从蓝色火焰中迸射冲天——正魔双方齐声惊呼「**补天丹**！」。

## 关键转折
- 第三方（星宫）以化身符介入，搅黄正魔取宝，韩立被殃及重伤。
- 补天丹真正现世，正魔双方注意力再被牵动，争夺将进入新阶段。
