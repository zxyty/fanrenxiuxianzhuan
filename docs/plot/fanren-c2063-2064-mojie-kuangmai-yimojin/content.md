# 第2063-2064章｜魔界之战：矿脉魔灵·再见异金

## 一、解决魔兽与魔灵（第2063章）
- **重创鹿首魔兽**：幻啸沙漠白家矿脉处，一头浑身赤血、伤痕累累、四爪已少一只的鹿首魔兽(魔尊后期)裹绿焰逃出。一道刺目银光(韩立飞剑)绕其一斩，魔兽负痛遁走，洒落赤血并坠下半截晶莹紫色怪角。
- **韩立收角**：韩立从坑中飞出，无形力量摄起沉重的半截紫角，神念扫过满意后即收起隐去。又以沙土掩埋兽血(紫发女子出手填坑)。
- **覆天仙子(紫发女子)**：白家紫发女子，黄发大汉之妹(五妹)。称韩立『韩兄』，言魔兽已被五人合力重创，数百年内难复原。
- **魔灵之患**：洞窟中韩立、覆天仙子与黄发大汉、栾龙天君、寒其子(三名魔尊)会合。覆天仙子言该魔灵乃矿脉成灵，片刻杀光白家六名炼虚弟子，等阶不低、神通诡异(银焰神通)。栾龙天君、寒其子以『非事先约定目标、危险更甚』为由本不愿续战。
- **加价续约**：黄发大汉将报酬翻倍；对栾龙天君，血牙米已无，改以血炼精金抵偿(栾龙天君勉强应下)。又密传音许韩立：除多赠一头六足魔蜥外，可在白家宝库另挑三件宝物——因韩立魔焰(紫罗极火)能克制魔灵并已击杀其一化身，是灭魔灵关键。
- **魔灵被灭**：兵分两路搜寻。韩立、覆天仙子持锁灵盘(原为探材料的魔器，可感应百丈内魔灵)搜索未果时，黄发大汉传讯：已寻得并联手灭杀魔灵，证实其十分虚弱。

## 二、返城与异魔金（第2064章）
- **魔灵根源**：众人最终找到魔灵根源——一颗已灰白、灵性尽失的巨大晶石(罕见炼器材料)，被黄发大汉收走。次日即踏上归程，顺利走出幻啸沙漠。
- **法力压制**：幻啸沙漠中虚空神秘力量压制韩立约九成法力；出沙漠瞬间法力恢复如初。韩立念及下次或要深入此沙漠数十年。
- **幻夜城·白家堡垒**：合体存在遁速极快，小半日抵幻夜城。栾龙天君、寒其子领取黑色储物镯报酬后相继告辞；韩立稳坐不动。黄发大汉独自自语，叹魔灵竟如此易解、报酬许得过厚，但矿脉之事总算彻底解决。
- **宝库三宝**：紫发女子以法决、紫玉佩解开藏宝阁楼禁制(外有十余白家精锐及八具青铜雕像守卫)。韩立凭强大神念已识大半宝物。先取走一件以符箓封印、透出银芒的玉盒；又在最后木架前盯住一块头颅大小、黑白相间、闪冷光的**异魔金**——正是他在血鸦城见过之物，体积大十倍以上——一卷收走。
- **意外之举**：异魔金在宝库中价值几乎最低，韩立这等神秘魔尊竟取此物，令紫发女子大感诧异。
