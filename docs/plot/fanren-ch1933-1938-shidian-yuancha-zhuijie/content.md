# 第1947-1949章 石殿 / 大敌联手 / 拦截

## 背景
- 地点：灵界（上界），魔界之战期间，魔族（圣族）大军降临灵界。
- 韩立现状：已进阶**合体中期**，刚刚到手**混沌之气**；因不久前与血光圣祖几具化身大战，临时布阵器具、符箓法器消耗七七八八，自感手段不足、状态不佳，不愿轻易生死一搏。

## 追杀者：元刹圣祖
- 一名身材窈窕的**蓝色宫装女子**，自称「本座」，手下石魔称其「**元刹大人**」，是魔族**圣祖**之一。
- 仅派**一缕化身**降临灵界协助「**六极**」，未统率大军、未让真身降临，自称不违协议。
- 散发的气息让韩立感到极其危险又略有熟悉——韩立终于忆起此女气息与当年在人界封印的**元刹圣祖分魂**相似；韩立当年曾灭杀其在下界的一缕分魂，故对方紧追不舍、誓要将其「抽魂炼魄」。
- **石殿**：由其手下两头巨大石人（**两大石魔，名石敢**）汇聚化成的数十丈灰白蛮荒石殿，殿顶可幻化粗糙巨脸说话。具**瞬移/传送神通**，单次可横跨百丈、亦可超远距离传送锁定目标，遁速不逊于韩立所化巨鹏；但耗法力极大，且首次传送被破灭法目击破后改将落点拉远（里许偏差）。

## 旧敌：血光圣祖
- 统率天渊城魔族大军的**血光圣祖**，外形为面容清秀的**血袍/血泡少年**。
- 此次让**三大化身借体降临灵界**（本体在圣界）。三化身可血光一闪联结一体共同催动法宝。
- 通过**黑色古镜**与元刹远程联络，得知元刹所追正是自己也想抓的人族修士（韩立），遂提议联手。
- 从**毒龙**处借来**乾坤盘**相助（毒龙一向视若性命，血光以足够代价借得）。

## 双方的交易（大敌联手）
- 约定：人（韩立）擒下后交元刹处理（元刹坚持，因韩立曾在下界大大得罪她、灭其分魂）；但血光提出身上宝物未答应全交元刹。
- 血光三化身私下打算：宝物至多分一半，但**镇魔锁**与**紫言鼎**必须先拿回（这两件原属血光、现在韩立身上）。
- 顾忌：元刹背后有**六极**撑腰，是个大麻烦，不宜此时招惹。
- 附加条件：此事了结后元刹须帮血光对付另一名人族**合体期**修士（血光化身借体降临、未带趁手宝物，奈何不了对方），事后元刹须立刻离开血光负责的区域。

## 韩立的逃遁与对抗
- **惊蛰决**变身术连番化形：四翅**银色巨鹏（雷鹏）**（遁速极快，连血光三化身联手一时难追）、头顶凤冠的**天凤化身**（具天赋破空之力，可直接撕裂空间瞬移，破开乾坤盘禁制逃走）、**金毛巨猿**（喷淡金色音波灭敌）。
- **破灭法目**：体内培育多年的黑色竖目，进阶合体后可借巨鹏之身强行逼出，喷漆黑光柱，曾击破石殿传送符阵。
- 其他手段：**辟邪神雷**（金灿雷球/金弧，识破并击碎乾坤盘的感应晶壁）、**青竹蜂云剑**之**青色剑气**、徒手抓出一青一黑两山峰攻敌；途中布下十几座法阵陷阱迟滞追兵（被对方迅速破除）。
- **乾坤盘**：百余丈巨大玉盘，放出无边白色光幕封路、并以禁空法阵困敌，但无法禁制凤类天赋破空，被天凤化身轻易破开。
- **豹麟兽（豹麟）**：韩立灵兽，曾吞噬暗兽望妖核，进阶合体后再次沉睡（积蓄力量冲关）。本片段中突然短暂苏醒、口吐稚气人言，示警「前边有敌人/敌意气息」后又一声哈欠沉睡——正是它令韩立提前停下、躲过血光乾坤盘的埋伏。

## 战况推进
- 元刹石殿与血光乾坤盘一前一后、轮番瞬移，对韩立穷追不舍，比当日血光三化身更难甩脱。
- 逃亡跨度：半个时辰出百万里，半日、两日、五日、十日、半月、两月……韩立沿途屠灭中阶魔族、低阶魔兽、超级巨兽及不死般的血色魔族大军，玉盘与石殿始终紧追。
- 局面：新仇（元刹）旧恨（血光）两大魔族对头同时找上门，韩立彻底死了正面交手之心，一心催动变身逃遁，并打算以对方瞬移耗法力巨大为机、不惜损耗元气与之硬耗拖垮对方。

## 关键悬念
- 韩立尚未弄清元刹确切来历与神通深浅，正苦思脱身之策。
- 血光三化身与元刹石殿已在湖面、热气沼泽等处多次前后包夹，围剿仍在持续。

## 章节
第十卷 魔界之战 1950《狼狈》、1951《脱壳》、1952《声东击西》。

## 背景
韩立已被魔族追杀约两月，一路击退击杀三四十波敌人(含两波有魔尊参与)，因身后有血光圣祖与元刹两大圣祖窥视，不敢纠缠，只能以霹雳手段一击而灭，导致法力(不足巅峰五成)与神念(仅剩小半)狂掉，全靠众多补灵丹药支撑。两大对头追踪隐匿之术皆在其之上，最多甩开半天又被追上；二者联手互补，单独脱身已无可能。

## 经过
### 1950 狼狈
- 韩立化巨鹏(风雷翅)被追近，放出数千只成熟体噬金虫，几个呼吸便将血光的『血光晶卫』近卫(修血身魔功、号称不死之身)吞噬殆尽，随即四翅化银弧遁走。
- 血光与元刹的化身赶到。血光因连失两件重宝(玄天残宝等)不肯放弃；元刹判断韩立噬金虫数量有限、神念不足以驱使十几万只，刚才已是极限。
- 二人定计：元刹向交情深厚的『六极』调精锐，血光封锁通往天渊城及各人族区域的通道，使韩立成惊弓之鸟，待其精疲力尽再合擒。血光另已布下七波人手沿途消耗。
- 韩立识破对方抓住其欲返天渊城的弱点，决意改变逃遁方向、给追兵摆下大阵以争喘息，灵机一动想出妙计——遁光一停，幻化出七八名一般无二的身影。

### 1951 脱壳(金蝉脱壳)
- 数个『韩立』四散激射。血光化身以背后魔影胸前竖目探查，判定『两实六虚』，两具实体改向两侧逃。元刹下令两实体各负责一个，找到本体即通报。上千魔族精锐分两路追去。
- 一具假『韩立』化灰狼(薄皮状物所变)继续奔天渊城；另一具引元刹石殿追入万丈翠山山腰，元刹以五色巨大光球轰击，将整座山峰洞穿崩塌、中心化为数十丈深巨坑，万物气化，却连化身残骸都未留，元刹以神念细查无果。
- 血光传讯告知其追的一具一连闯过三道封锁、击杀其大半人手，疑为真身，已被乾坤盘困住但无法找出，请元刹相助。元刹遂弃此处赶去会合。
- 两个时辰后，废墟枯木中现出韩立真灵躯(绿肤青年、纯木灵气)，拍开天灵盖放出器灵银月(白嫩女童)调息。银月赞主人未算错——以灵躯草木之体现原形藏于树石间可瞒天过海；任务完成后银月按计另寻地方藏身，盼主人逃过大劫。灵躯随后认准方向腾空而去。

### 1952 声东击西
- 血光三化身以玉盘化赤红骄阳笼罩山谷，放赤红光球四处搜寻被困『韩立』未果。元刹驾石殿赶到会合。
- 血光称韩立潜入地下欲地遁，反被乾坤盘封禁于山谷，只是隐匿之术使其无法寻出，请元刹施『魔元大法』逼出；并派一化身随元刹下谷『护法』(实为监视，防其先得手)。
- 元刹携两名手下石魔长老(可化石仑图腾巨柱)布黑旗法阵、催三头六臂魔相，凝聚魔云化上万黑鸦遍探山谷地下数千丈仍无所获。
- 此时另两名血光化身急传消息:另一『韩立』突现于后几道防线，动用玄天残宝、放出噬金虫群、连破数波拦截并击杀两名魔尊级得力助手，向天渊城逃去——确认其为本体，众人皆被金蝉脱壳之术骗过。血袍少年急令元刹收法离开，赶往追击。

## 关键判断
- 韩立以多重分身(假身)、噬金虫、灵躯木遁三重手段成功『声东击西/金蝉脱壳』，暂时甩开两大圣祖。
- 真身突破封锁但已暴露目标方向；两圣祖重新合拢追击，危机未解。

## 说明(据世界状态须知)
- 『血光圣祖』即以『阳刹』之名/身行事者(神念寄附/夺占阳刹躯体化身降临灵界，起于第1909章)，本片段为其化身行动。
- 『阴阳二刹美妇』已于第1903章死亡；本章女性追兵为『元刹』，另一活跃圣祖，勿混淆。
