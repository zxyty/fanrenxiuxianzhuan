# 第1598-1599章 云城拍卖会：瑕疵琉璃天火液与人面蛟异兽

## 背景
- 地点：云城（文中亦作『运城』）的拍卖大殿，主持人为萧布衣。会场分一、二、三层，三层石屋多为圣族存在。
- 韩立施幻术遮掩面容潜入，化身为面色苍白、身穿黝黑长袍的男子；只要变回本来面目无人能识。连合体顶阶的段天刃当面也只能看出他非本面示人。
- 韩立现境界：炼虚期（由化神初期一路靠丹药精进而来）。

## 琉璃天火液（瑕疵品）
- 该液是青元子交代韩立收集的主要材料之一，为炼制数种合体级丹药的必需物；凭空生服一滴即可省年许苦修，价值极大。天生于罕见天外之火，比同阶灵物更难寻觅，韩立原以为是青元子诸材料中最不可能集齐者。
- 本品在提炼时出现瑕疵，与正品相比效能大幅弱化，但主要效能仍在。底价仅一千八百万灵石，低得难以置信。
- 三层一名烈姓男子（炎灵之体）讥其虚高；萧布衣当场演示：飞剑触液团瞬间冒白气熔解消失；又喷一团青濛濛灵气引爆，火液如火凤重生般化为头颅大火球，散发炙热高温，证明其『熔金化石』本性仍在。

### 竞价过程
- 韩立开局直接喊三千万压住众人念头 → 黄袍符老（运城名人，焦黄头发、浓眉蓝目）出三千二百万 → 三层一名圣族女子出四千万（符老退出）→ 韩立四千五百万 → 圣族女子四千七百万 → 韩立五千万（几近其全部灵石）。圣族女子冷哼放弃。
- 韩立以五千万灵石拍得半瓶天火液，到偏殿用一袋极品灵石交割，得乳白色小瓶。即便是瑕疵品也势在必得，因放掉则以后基本无望得到正品。

## 腾龙丹
- 韩立拿出拍卖的腾龙丹为炼虚级丹药，可提升法力、对突破瓶颈大有用处；炼虚级丹药在拍卖会极罕见，引发激烈竞价。
- 最终以六千三百万天价成交，落入先前与韩立争天火液的那名圣族女子手中（多半留给门人后辈）。
- 韩立暗算自己前两年所服腾龙丹数量×此价，灵石数目骇人，且今后数百年还将持续，自嘲『败家之极』；并感叹若无神秘小瓶，他根本没信心凭资质走到纵横灵界一步——故视神秘小瓶价值甚至在另一件玄天之宝（所化宝剑）之上。

## 其它拍品
- 『万兽螺』：上古遗留异宝，未及通天灵宝，吹响可号令一定范围低阶灵兽听命。
- 一具深蓝色傀儡、一只上族四阶通灵傀儡等十几组宝物（通灵傀儡未列为压轴）。

## 压轴部分与人面蛟异兽
- 萧布衣宣布四件压轴：两件为四族珍品，两件为他人提供；同含灵兽、材料、灵宝等。
- 台上传送阵浮现四名合体中期存在——云城四大客卿：长须矮小老翁（蓝兄）、金色面具高大甲士、满面碧绿鳞片的丑妇、生有两颗同样头颅的黝黑怪人。
- 蓝兄取出巴掌大黑色皮袋，作法膨胀成丈许黑色铁笼；笼中一只通体紫色小兽被金银相间细索捆成毛球状。黑笼散黑光，可见却挡神念浸入。
- 此兽来历：继承灵界『真灵之下八大奇兽』之一『人面蛟』（文中亦写作人面狐）大半血脉的混血后代；出生不过百年，灵气已不下上族初阶，并展现人面狐数种天赋神通；四大客卿联手才生擒。
- 老翁虚空一抓解开金银细索；小兽却躺笼角不动，旋即体表爆发刺目紫色霞光，众人闭眼之际遁走，铁笼空空。韩立瞳孔蓝芒一闪（天眼术）未全闭眼。老翁哼称其『耍小花招』，抬手以粗糙手指冲笼虚空一弹。
