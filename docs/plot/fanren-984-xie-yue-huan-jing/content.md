# 第六卷 通天灵宝 第984章 邪月幻镜

## 场景
韩立与黑衣美妇（元婴中期女修）联手对阵**银翅夜叉**——一具修炼了不知多少万年、神通诡异的飞尸/炼尸。地点为一处布有巨型法阵、四周石柱绑缚干尸的广场。

## 经过

### 1. 困魔阵发动，三方被困
- 银翅夜叉单掌按地，激活了广场上的**困魔阵**。该阵当初正是专为围困它而布下，多年后其小半禁制反被它所用。
- 韩立察觉不妙，以雷遁（背后双翅，即风雷翅）瞬移避开，原地仅留噬金虫金云与啼魂兽；银翅夜叉亦以乌光遁走，二者相撞分开。
- 石柱干尸喷出黑气化黑色光柱冲天，凝成庞大黑色**光幕**罩住整座广场。黑衣美妇遁出二十余丈即被光幕封死，只得停下。
- 韩立点破：银翅夜叉受禁（四肢与脖颈浮现五只血色圆环禁锢法器），无法离开法阵范围；银翅夜叉得意承认，称此阵如今困住了它，也同样困住了入侵者。

### 2. 缠斗回合
- **啼魂兽**喷黄霞袭飞尸，本对鬼物无往不利，却被银翅夜叉双翅射出的银芒抵住，未能一击建功。银翅夜叉惧意大减。
- 银翅夜叉喷漆黑如墨的**尸气光柱**（污秽阴寒、附极厉害尸毒），击中扑下的**噬金虫**金云；噬金虫虽坚硬百毒不侵，仍如雨跌落，虫群顿损三分之一，余者四散。
- 韩立在霹雳声中现身，袖袍一拂放出众**飞剑**（青竹蜂云剑）化金光席卷，顶上**紫色镜子**喷紫色光柱紧随，并悄然食指一弹射出一道红线（暗手）；地上跌落的噬金虫又勉强腾空，韩立稍松一口气。

### 3. 邪月幻镜显威
- 面对剑光、紫光柱与黑衣美妇所化两条白色风蛟，银翅夜叉取出**邪月幻镜**与一团灰濛濛阴寒光球。
- 它将光球没入镜面，镜中乌光一闪，涌出无数同样光球融合，转眼成直径数丈、并持续狂涨的**巨型光球**护身。
- 飞剑将其斩成无数块、风蛟与光柱将其撕碎，但巨型光球破裂后碎片化为更多小光球，小光球重新聚合，银翅夜叉便完好无损地重现于光球中，宛如不死之身。光球体积已涨至十几丈宽。

### 4. 黑衣美妇破解之法
- 黑衣美妇（以两只**白玉蝴蝶**化为人大，再化两条白色**风蛟/风龙**）发动第二轮：风龙再破光球、灭杀飞尸幻影后不再离开，而是张口喷出无数白蒙蒙**风刃**四射，趁小光球未及凝聚便将其打成更碎的存在，使核心狂风肆虐。
- 韩立未催飞剑加入，而是两眼微眯凝望，瞳孔深处隐有蓝光闪动（似在以天眼术/秘术观察破绽）。

## 要点
- 多宝修士斗法并非宝物越多越占便宜：驱使宝物耗费神识，越厉害的宝物占神识越大，故修士通常只选几种大威力宝物对敌。韩立一次放出众多宝物且仍能施法，足见其神识强大。
- 本章悬念落在如何破解邪月幻镜的「光球不死」之能。

## 章节
第六卷 通天灵宝 第985章《破法一击》、随风掩形（第986章）、第987章《金遁与冰焰之精》

## 场景
虚天殿地下尸阵 / 光幕困阵之内。韩立与临时联手的黑衣美妇，对阵守阵的炼尸级强敌银翅夜叉。

## 经过
### 1. 破光球禁制
- 银翅夜叉以邪月幻境（邪月幻镜）配合幻术，将自身躲入可扭曲空间的巨型光球中，黑衣美妇的风龙/风刃只能将光球击成豆粒大小、旋即复原，飞剑难伤。韩立联想到当年乱星海与六道少主温天仁一战（紫云扭曲空间，终靠阴魔斩破之），判断需以霸道手法破术。
- 韩立拍出降灵符（炼三焰扇时顺手用八级赤火蛟精魂所炼，品质胜前次），蛟影附身，头生角、生火红鳞片，化半人半蛟，法力升至元婴中期顶峰，令美妇与夜叉皆惊（均未见过降灵符）。
- 数十口青竹蜂云剑各以一化六，近二百口金剑聚成十余丈金灿巨剑；韩立点出雪晶珠（化紫芒没入剑柄起紫焰）、再引辟邪神雷（手臂粗金弧缠剑），巨剑金弧紫焰交织。一声『破』，巨剑斩碎光球——夜叉不敢硬接，以银翅遁出，邪月幻镜停吐、功法被强行破除，光球无法复原。

### 2. 暗器毁镜
- 韩立暗放红线飞针（虚化火球佯攻后隐去），趁夜叉以煞魂丝大手挡巨剑之机，红线现于其胸前，洞穿并击碎邪月幻镜。夜叉因宝镜被毁而暴怒（似有特殊意义），扬言要亲手拿下韩立、抽魂慢吞。
- 夜叉踩尸狼施土遁遁入地下，法阵巨颤、冒黄光黑气，地底传出其凄厉大吼。

### 3. 黑衣美妇之死
- 黑衣美妇欲以剪刀状法宝救出灰茧中的富姓老者，贸然飞近。地下窜出绿色狼首人身的尸狼（被秘术化人形、体形骤缩、遁速极快）击飞金剪。
- 银翅夜叉形态大变：绿毛尽褪、换淡金鳞片、银翅泛青白异芒、瞳现淡金、尸气全敛如空灵之体，达『随风掩形』之境。它无视青色小盾、徒手穿透美妇护体灵光，一爪洞穿其心脏。美妇拼出元婴瞬移逃命，仍被灰色煞魂丝网罩住、捞成尺许灰茧擒获。
- 韩立以青色剑气救援争得片刻，但未贸然飞去（识破对方头顶灰巨网陷阱）。

### 4. 识破与对峙
- 夜叉冷笑称以为韩立会用雷遁来救，赞其明清灵目能看穿幻术。韩立指出其『随风掩形』几近无形，唯灵目可察。
- 啼魂兽（被韩立神念压制久已郁闷）对尸狼呲牙，尸狼畏缩；夜叉喂尸狼一灰珠提振尸气，令其化数道幻影缠斗啼魂（啼魂以鼻中黄霞护体、巨拳乱舞，二者僵持）。
- 夜叉与韩立在高空忽闪追逐，其风遁明显高于韩立雷遁，尾随渐紧。

### 5. 金遁现形与苦战
- 韩立设伏（雪晶珠紫焰+金剑剑光+辟邪神雷电蟒）反击，夜叉以金灿光罩借力打力弹开飞剑、溃散电蟒；唯紫焰将金罩连人冰封成淡紫冰块（未冻其本体，旋即金芒穿孔脱困）。
- 韩立点破：夜叉除土、风外竟会金遁术（人类修士罕有，唯传说金身月尸具此神通），疑其正进阶。夜叉承认万年前已开始进阶，气息暴涨、强过天南三大修士、堪比坠魔谷两只古魔，唯金遁对无金属性材料的法器无效。
- 夜叉抛金团化无数牛毛金针（系困此地期间以不知名金属性矿石经尸火数千年炼成）铺天盖地；四周石柱干尸挣脱锁链，喷煞魂丝织数十丈天罗地网。
- 韩立取三焰扇（金银红三色、符纹密布），一扇竟被吸走大半法力，旋出三色火凤扑向灰网；又喷精血催雪晶珠之冰焰之精（提炼自紫罗天火至寒之气，远胜普通紫罗天火）筑十余丈紫色冰壁护体，金针消其尺许仍未破。火凤撞入煞魂丝网，爆出金银红光晕符文——胜负未揭即断章。

## 要点/设定揭示
- 降灵符：可短时令修士附蛟影、法力暴涨一阶（此处达元婴中期顶峰）。
- 冰焰之精：韩立长期服雪魄丸、借雪晶珠之力从紫罗天火提炼，至寒，远胜普通紫罗天火，但法力不足时需精血强催。
- 银翅夜叉：炼尸/飞尸，会土、风、金三遁，随风掩形可至几近无形、尸气全敛，能徒手穿透防御宝物，正进阶金身月尸。
- 韩立修炼过明王决，本身煞气之体，故煞魂丝煞气对其无效。

## 归属说明
本片段中韩立以本名本身行事，无化名/冒名情形。

## 章节概要（第六卷 通天灵宝 第988-990章）

### 第988章 灵扇之威
- 韩立祭出**三焰扇**：扇上三色符文流转，三色火凤双翅化为金银红三色巨型光轮，瞬间将银翅夜叉凝聚毕生**尸煞之气**的灰色巨网（直径数十丈）尽数湮灭，光轮一呼一吸即消失。
- 银翅夜叉判断此扇一击蕴含**天地法则/天地之力**，是化神期才能接触的层次；其倚仗的**邪月幻镜**（多种神通的施展凭借）亦被击毁，令其暴怒又忌惮。
- 韩立服下一滴**万年灵乳**，法力尽复，准备彻底解决银翅夜叉（对方有禁制在身无法遁走）。
- 关键转折：地下法阵毫无征兆剧震，外围浮现镶嵌近百颗灵石的更大法阵，与原阵融合成**巨型传送阵**被激发。韩立欲以雷遁逃离已迟，与银翅夜叉一同被传走。同一时间，大晋另外三处也各有一只不知名怪物被同样巨阵传走。

### 第989章 天象巨变
- 幕后是**南疆无名小湖的叶家群修**。他们解除**昆吾山**封印，引发破禁天象：湖心一道直径三十丈乳白光柱冲天，四周另有六道环绕，共七柱，数千里修士皆见。
- 用以掩饰天象的法阵全部被毁，主持法阵的两名**外姓阵法弟子**（老五家、老十二家子侄）已失踪两日，被疑通敌或遭搜魂灭杀。
- **叶家大长老**：一名三十余岁、肌肤如玉的清秀白袍儒生，行事狠辣冷静，是大晋第一世家的大长老。其下有方脸修士（二哥）、白发阵法大师等。
- 叶家长老中混有一名相貌普通的**古魔化身**人类修士「韩长老」（叶家大长老亲请，自称一直与「大头怪人/七叔」同处，故脱嫌）——注意此「韩」非韩立。
- 叶家决意抢在外来修士赶到前进入昆吾山，夺取**两件通天灵宝**以自保。由老道姑放出精通土遁的**千足蜈蚣**载众人，自北方二十里、地下千丈处的封印裂缝潜入。
- 天象惊动数千里乃至邻近州郡，南疆风起云涌，各方修士涌向小湖。

### 第990章 擎天巨山
- 韩立落于一座单向**接收**传送阵的钟乳大洞，地处极品灵脉、灵气充沛。困住银翅夜叉躯体的血红禁锢圆环已碎裂，银翅夜叉趁韩立从传送眩晕中未及恢复时施风遁逃走。
- 传送瞬间，韩立腰间凭空浮现一张上古**绿玉传送符**（似乱星海所见纸符的玉质版），灵力仅够一次性传送、用后即溃散；他虽预取**大挪移令**防身亦无从闪避。
- 战果清点：紧挨韩立的**尸狼**被辟邪神雷击成重伤、再被**啼魂兽**喷黄霞吸尽尸气而毙；藏于尸狼体内的**阴芝马**（此行目标，半尺白嫩、碧绿双眼、善隐匿）被一并卷出，险被啼魂兽吞下，韩立喝止收入灵兽袋。两口**青竹蜂云剑**已破禁收回；**黑衣美妇元婴（元道友）**被银翅夜叉一口吞入腹中身亡；**晶化飞针**被银翅夜叉顺手夺走（非本命法宝，仍有夺回机会）。
- 韩立击碎两只灰茧，救出被煞魂丝困住、法力受损的**富姓老者**与**白瑶怡**。二人感激中藏忌惮。韩立放回阴芝马灵兽袋于富姓老者，只索要其日后多炼**培婴丹**时留两粒，二人应允；韩立体内仍有法力标记可追踪二人数日。
- 探索：洞外青石长阶极长，飞行数里方见出口，乃身处一座奇深山峡。出峡后韩立骇然——这是一座**擎天巨山**，规模不逊于（甚至远超）乱星海天星城那座可分八十一层的圣山，所谓山峡只是山脚一道微不足道裂缝。
- 整座巨山被一道自高空垂下、无边无际的**白濛濛上古禁制光幕**笼罩，连化神期也难以蛮力破除。山中灵气盎然却毫无鸟兽之声、死寂如死山，给韩立极强的危险预感。他最终脸色阴沉飞回传送阵山洞。

## 要点与悬念
- 三焰扇展现媲美化神期天地之力的恐怖威力，是本卷「通天灵宝」线的关键。
- 叶家图谋昆吾山封印下的通天灵宝，已百年布局，内部疑有叛徒导致掩饰法阵被毁、天象暴露。
- 韩立误打误撞被传入一座上古禁制中的擎天巨山，单向阵无法回返，前路凶险未明。
