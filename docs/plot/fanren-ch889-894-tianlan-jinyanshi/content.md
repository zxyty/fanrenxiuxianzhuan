# 第六卷 通天灵宝 第889-891章：天澜草原·金焰石

## 背景
韩立化名「寒仙师」，将修为压制到筑基期，伪装成押队仙师混入突兀人车队，潜行于大晋天澜草原，避开外族人（大仙师、圣女一方）的追踪。

## 第889章 巨幅
- 驻地的秃眉老者以隐秘神识搜寻韩立，韩立因修为骤降、收敛神识稍晚，险些被察觉；秃眉老者顾忌性命未敢追拦，韩立侥幸脱险。
- 韩立翻检鹰眼老者的储物袋，在一只绿色玉瓶中遇到紫色雾气（十绝毒「苦毒」残留），以青色光团强行包裹收入**神秘绿瓶**——十绝毒可遇不可求，孔家能有此毒非同一般。
- 韩立以神识探查同行的**枫岳**（冯家唯一嫡系血脉，筑基期）：枫岳以一颗能吸毒气的黑色异宝圆珠拖延苦毒，但毒气会重生，注定无解，仅余两三日寿命，黑发渐白、绝望。
- 韩立已于两日前参悟完**大衍神君**所给口诀，身份即便暴露也不惧元婴围攻。
- 车队抵一条数百丈宽大河（天水河支流）。韩立出车，与突兀少女**英珊**交谈，尝其烤肉，查其灵根，赐其一瓶洗髓易经丹药。
- 得知此河即**天水河**——天澜草原第一大河，上游天水河与大晋境内**舜江**为同一河流（上下游关系）。
- 河对岸天空现绿光，一只浑身绿光的巨大蝙蝠（巨蝠/巨幅）由小变大逼近车队。

## 第890章 金焰石
- 巨蝠载八人（四男四女），为大晋修士装束，非突兀人；为首两人**顾统领**与一名**白衣女子**为结丹期，余六人筑基期。
- 来者非冲韩立，而是追捕藏身车中的叛徒。众突兀仙师推韩立为首应对。
- 顾统领命韩立摘斗篷验容，并以密术辨认，确认韩立非「叛徒」、用的是真容。
- 藏身者**江剑英**（彪悍青年，筑基期，身具冰灵根）现身，手捧**雪晶珠**。九仙宫原派他携珠去请北寒山北玄老人，实为以其冰灵根为「北玄天妖」做血祭祭炼此珠，他遂携珠潜逃。
- 双方以大晋言语对话（韩立通晓，旁听）：九仙宫「宫主」「统领」之制，类似乱星海大小势力。江剑英以雪晶珠化寒雾顽抗，被六名筑基修士的赤红火旗合成的火云压制；顾统领以结丹修为入雾，一击擒下青年、夺得雪晶珠。
- 白衣女子顾忌天澜圣殿、突兀族护短，劝速离。临走顾统领（炼器部出身，熟读《天地奇石录》）瞥见地上经热气冲洗显形的半透明鹅卵石——**金焰石**。
- 大衍神君惊呼：金焰石乃炼制**仿制七焰扇**最关键的材料，天南灭迹多年，连他都不确定大晋能否寻得。

## 第891章 出手
- 韩立抢先以飞遁手段夺走金焰石收入储物袋。顾统领索要不成，威胁断臂留命；韩立拒绝。
- 白衣女子确认此石价值后，下令灭口（连数百凡人、突兀仙师一并不留，以免人证），当场斩杀突兀仙师数人。
- 顾统领喷本命飞剑斩韩立，被韩立祭出的黑色小幡光幕反弹；白衣女子飞刀亦无效。众修士布火炎阵欲炼化韩立。
- 韩立显**第二元婴**（黑绿色婴儿），抛出小幡狂涨为数丈，黑濛濛魔云吞没飞刀、巨蟒、火云，切断敌方与法宝的联系。
- 顾统领惊呼此为「**阴罗幡**」、韩立为「**阴罗宗执法长老**」（韩立并未自称，乃对方误认），弃本命法宝化白光逃走；白衣女子打入法决催动巨幅化绿虹反向遁逃。
- 韩立放出十几条雪白**六翼霜蚣**（幼虫，寒气可冰封结丹修士）追杀六名筑基期修士，尽数冰封撕碎；放出魔化**噬金虫**追上顾统领，咬破护罩钻体毙之；第二元婴驱魔云数十里外罩杀白衣女子。
- 战后韩立把玩雪晶珠，面露纳闷之色（江剑英被弃的某物）。

## 要点
- 韩立身份：全程化名「寒仙师」、伪装筑基期；战中暴露第二元婴实力。
- 收获：十绝毒苦毒残留、金焰石（仿制七焰扇关键料）、雪晶珠。
- 九仙宫为大晋一方势力（设宫主、统领、炼器部），行事狠辣、以血祭炼宝。

## 第八百九十二章 天澜圣女

### 一、灭口与焚尸
- 韩立欲救醒被擒的大晋青年修士以问来历，对方却已服剧毒自尽、一脸乌紫，且精魂消散极快（另动了手脚），韩立无从审魂。
- 韩立以一记小火球将青年尸体化为灰烬。
- 魔云飞遁而回，韩立的**第二元婴**手持**黑幡**现身，挥幡将所有魔气收入幡中，幡旗一闪没入韩立天灵盖。

### 二、押送枫岳
- 周围突兀族凡人远远畏惧观望。
- 韩立命某辆马车中的人随行，要在其**毒发前**交出**密窟地址**。
- **枫岳**头带斗篷走出，认出韩立为**阴罗宗长老**，并提及韩立『轻易灭杀冯枕老贼、又有昊元丹这种珍稀丹药』。
- 韩立冷笑，放飞剑御器升空，枫岳随之离去，二人遁光消失天际。

### 三、身份败露与圣殿围杀
- 数日后事发：一次死伤数名仙师（含与突兀族/圣殿做生意的大晋修士），引圣殿仙师赶来盘问。
- 经被灭九仙宫修士之口及鬼罗幡威名，凶手被认作**大晋十大魔宗中阴罗宗的长老**，且身怀镇宗之宝**鬼罗幡**（上次大战未尽全力、此至宝未用，推断其在阴罗宗长老中亦排名前列）。
- 突兀人推测：数十年前圣战中死对头**慕兰人**背后有阴罗宗影子，此人或为报复/图谋潜入草原。
- **天澜圣殿**应对：派大批修士巡视大晋与草原交界以防接应；集中遁速极快的元婴仙师待命绞杀，誓不让其活着离开天澜草原。

### 四、圣兽召唤仪式
- **开灵日**：某圣殿由**圣女**亲自主持的开灵仪式仓促两日结束。
- 一日后，圣殿后禁地：近千丈巨型广场，中心巨型阵图镶百余块中阶灵石；阵眼处有五六丈高灰白长方祭坛（普通白石、四壁刻不知名符文）。
- 百余名沐浴斋戒的突兀高阶仙师到场，元婴上阶仙师十几位。
- **三人入场**：
  - **银袍女子（天澜圣女）**：白纱蒙面、秀发乌云、肌肤如雪、美目清澈。
  - **俊秀青年（大仙师）**：貌似三十余岁却目光沧桑。
  - **紫发黄衣美妇（大仙师）**：紫缎诡异长发，外露双臂各有拳头大乌黑鬼头死咬，却不见痛楚。
- 圣女说明：此次召唤不光为追杀外族人，更是**首次尝试让圣兽分身永久停留圣殿**；两位大仙师耗十余年寻得两只合适祭品。

### 五、祭品与施法
- 各八只两丈高金光铜人抬来两座木架，上为两只如小山般、昏迷的妖兽：
  - **青风牛**（青光野牛）
  - **冰甲蟒**（雪白巨蟒）
  - 皆**七级顶阶、即将迈入八级**的蛮荒异种。
- 圣女以银丝化上百银索困住青牛，悬浮移上祭坛；青牛苏醒挣扎却被多重禁制压制，连嘶鸣都无声。
- 圣女自银袖放出拳头大青光——一只被**淡青色火焰**包裹的**迷你小鼎**，掐诀念咒后青焰高涨，小鼎化为丈许高古鼎。
- **关键伏笔**：此古鼎（除附带火焰外）造型与花鸟虫兽图案，与**虚天鼎**几分相似；若韩立在场必大惊。

# 第六卷 通天灵宝 第893章 圣兽铸身

## 一、圣兽铸身仪式
- **地点**：天澜分殿（天澜圣殿）禁地，祭坛与多处法阵环绕。
- **主持者**：天澜圣女（银袍女子，他人称『林道友』），额头浮现牛首蛟身古怪图案。
- **护法/助力**：两名大仙师——清秀青年『徐道友（徐某）』、紫发美妇『孙仙师』，两人各按一掌于圣女香肩，源源注入精纯灵力。

## 二、血祭与召唤过程
1. 圣女催动巨鼎，鼎中飞出亮晶晶青沙，幻化星光罩住祭坛。
2. 银链化细丝将妖兽分解：先后血祭**青风牛**、**白色巨蟒**两只七级妖兽，连其苦修元神也被青芒击落、青沙淹没吞噬。
3. 青沙吸食两兽精魂血肉，化为悬空血云（每粒泛血色）。
4. 巨鼎落坛，与法阵呼应；两大仙师灵力化光柱击鼎，鼎中凝出头颅大小光球，法阵射五色灵光注入。
5. 光球爆裂，现鸡蛋大小青濛光洞，一道赤芒射出扎入血云，血沙包裹成尺许血茧（内有蠕动物）。
6. 圣女以青丝将血茧卷入巨鼎，青焰高涨淹没鼎身，法阵注灵——圣兽分身于鼎中重铸灵体。

## 三、仪式结果与意图
- 圣女称仪式顺利，圣兽分身能否久留人界需一月见分晓，但**滞留七八日没问题**，足以借其神通找出外族人。
- 徐道友、孙仙师答应再护法两日，以防重铸生变（自言十余年心血，不愿功败垂成）。
- 鼎中传出雷鸣低吼，三人盘膝守护。

## 四、出动追猎
- 两日后天澜分殿钟声响起，十余道遁光向南破空而去。
- 三日后于天澜草原第一大河『天水河』南端，圣兽分身现身——**牛首蛟身赤红妖兽**，三四丈大、罩青濛霞光，圣女立于其牛首之上。
- 众人判断：外族人随身带大量**噬金虫**，纵入灵兽袋也瞒不过圣兽灵觉；其人狡诈，潜入河下遁走。
- 徐道友想验证其是否真为**阴罗宗长老**——中了**呼兄『大五行擒仙手』**竟安然无事。孙仙师好奇其『外表年轻』。

## 五、追逃对峙
- 圣兽察觉异动，韩立施秘术瞬移至数百里外，神识强大，颇为棘手。
- 圣女当机立断，与徐、孙三人留传音符给后队，全力先行堵截。
- 凭圣兽分身高深神通，三人死咬**韩立**尾巴，追逐一日一夜。
- 韩立数次以**血影遁**甩出三人千里之远，旋即被重新追上。
- 被迫**连用三次解封之法**仍无法摆脱，韩立心蓦然沉下。

## 备注
- 本章追猎方为天澜草原『突兀人』势力（仙师/大仙师/圣女体系）；明言圣兽分身『无法久待人界』，故事仍在人界。
- 『外族人』即韩立，作者视角点明。『阴罗宗长老』为追兵的怀疑/猜测，非韩立确凿身份。

# 第六卷 通天灵宝 第894章 诱敌

## 处境
- 韩立处于**解封**状态，化青虹疾遁。连续三次解封并多次施展**血影遁**，导致功法剧烈反噬，身体岌岌可危。
- 靠数种高阶秘术强压反噬、连服十余滴**万年灵乳**弥补精血亏损方撑住；预计收功解封后两三年法力难回巅峰（换作丹药稀少的修士甚至会被打落一层境界）。
- 仍被**元婴后期**修仙者追杀，且无法弄清对方精准跟踪之因前，血影遁不敢再轻用。

## 大衍神君的判断
- **大衍神君**（大衍决之灵）指出：追踪并非有人在千里外以神识锁定，而是韩立被一只**非人之物**盯上——即后方那只酷似突兀凡人口中**天澜圣兽**的妖兽。
- 推断为**圣殿**老家伙们因颜面受损，特意从**上界**召唤下界之物。
- 对比**慕兰人**的**圣禽**：圣禽是借宝物之力凭空化形的投影幻影；而此天澜兽经感应确系**实体**，疑似使用类似**古魔魔化借体**的神通，借他兽或媒介令自身分神降临此界——属真正的分身降临，神通不可小觑，给足时间将成此界最可怕的存在。此类降临存在此界早有数起，却似在惧怕什么而低调潜伏。
- 此兽当前法力波动仅**七级妖兽**水准，灵智需上百年修炼方能彻底开启，故神通有限。

## 对策
- 大衍神君临时传授韩立一种**假死之法**（彻底消除生气），但不保证对妖兽有效。
- 韩立决意诱敌脱身，两路布置：
  1. **第二元婴诱饵**：黑光中第二元婴浮现头顶，吐出**鬼罗幡**摇晃，汇聚百余丈黑云魔气；韩立又掷入**血魔剑**（血色小剑），嘱其若天澜圣女带圣兽来追便以此剑灭之。魔云发刺耳尖鸣朝一侧飞遁。韩立随即再服一滴万年灵乳，以更快遁速破空远去。
  2. **银月化形诱饵**：行出五六十里后，韩立放出银色小狼——本命飞剑器灵**银月**，化形成另一个“韩立”。银月自陈身为本命法宝器灵，只要飞剑不毁便是不灭之身，受伤至多在法宝中多沉眠些年月。韩立交给她一只装**噬金虫**的灵兽袋与一枚拇指大**雷珠（青珠）**，嘱其若天澜圣兽追来，只需灭杀或暂时重创此兽即可。银月化银光斜射而去。

## 追兵一方（突兀人）
- **紫发美妇**、**清秀青年（徐姓）**、**银袍蒙面女子**。后者为**天澜圣女**，元婴中期修为。
- 众人惊见韩立元婴单独出窍遁走而躯体仍能行动，推测可能是三尸元神术、化身之法、桂道友所修**六魂分元决**之类元婴分裂术，乃至传说中的**第二元婴**，一时难辨真假。
- 天澜圣女称圣兽仅一缕分神降世、修为只七级，无法施展过多神通，分辨不出蹊跷。
- 决议：**徐姓青年**独自追第二元婴；余者继续随圣兽追韩立本体。紫发美妇叮嘱徐——那元婴或为主元婴，**先别下死手，须先逼出噬金虫的培育之法**。

## 关注点
- 韩立不惜放弃第二元婴作饵的决心，凸显其绝境。
- 噬金虫的培育之法是突兀人追杀韩立的重要目的之一。
