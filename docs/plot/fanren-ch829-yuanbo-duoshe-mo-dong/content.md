# 第六卷 通天灵宝 第八百二十九章 魔动

## 背景
南陇侯与鲁姓老者（鲁卫英）此前结伴探宝，于一处供桌前面对一批灵药灵果及一件惹眼的圆钵古宝。两人虽有旧交，但在重宝前戒意尽生。鲁卫英自言修为本略逊南陇侯，但南陇侯昔年在**慕兰草原**大伤元气，如今二人修为扯平。

## 分赃约定
- 鲁卫英提议灵药平分、不动其他心思，以免同归于尽。
- 南陇侯只要供桌上的**圆钵**，其余宝物（三件法器+几张玉符）全归鲁卫英。鲁卫英声称只重修为寿元、不看重身外物，一口答应。
- 二人几乎同时动手：鲁卫英以白色霞光卷取三件宝物与玉符，南陇侯以金光光团罩取圆钵。

## 南陇侯的戒心
圆钵入手后，南陇侯突然连拍五六张各色符箓封死钵盖。他点破：鲁卫英所取的三件宝物与玉符，实为上古修士用来**镇压圆钵**的法器；圆钵乃顶尖古宝，内中封印之物非同小可，自己没有把握绝不会贸然开启。鲁卫英神色如常，只想尽快分药离去（『不知为何不想在此久待』）。

## 灵药皆为幻影
二人以光手分取**天元果**（可增寿元）与**紫色灵芝**时，灵药灵果绿光一闪、如气泡般溃散不见——原来全是幻象，二人皆抓空，惊得目瞪口呆。

## 圆钵崩裂·夺舍
- 与此同时，圆钵上所贴符箓在**黑色妖火**下自燃成灰，封印失效。
- 南陇侯急以一张残破金符补封，但已迟。钵盖爆射冲天，一道乌光射出，正中南陇侯面门、一闪即逝没入其体。
- 南陇侯惨叫、抱头半跪、面孔扭曲抽搐。鲁卫英以见识立判为「**夺舍**」，毫不留情射出两口**蓝色长戈**斩杀。
- 南陇侯猛然抬头，满脸黑气、双目尽白，徒手挡飞蓝戈；袖袍碎裂露出的手臂已乌黑发亮、遍布紫红粗筋、黑气缭绕如鬼爪，挡戈而毫发无损。

## 圣界·圣祖
夺舍成功者站起后端详己身，仰天怪笑：『这么多年了，本尊终于出来了……这一次看谁还能阻挡**圣祖**的降临，这一界注定要属于我们**圣界**。』鲁卫英从未听闻「圣祖」「圣界」，心中更寒。值得注意：以南陇侯**元婴中期**神识竟如此短时间被夺舍，那乌光绝非普通邪魂。

## 鲁卫英遁逃失败
鲁卫英祭出白色法旗与火红轻纱——**昊阳纱（元阳纱）**红光护身（自信足以抵挡任何妖鬼之道阴毒攻击），借遁术化白虹冲向水潭底部通道入口（早布有法阵）。眼看将入通道，被夺舍后的南陇侯瞬间堵住入口，一拳隔着护体灵光将其震飞、狠狠砸入对面石壁数尺，后背麻木失控。鲁卫英面无血色，南陇侯缓缓收回乌黑拳头冷笑相望。

## 要点
- 圆钵是封印「圣界」之物的上古古宝，三件法器+玉符为其镇压封印。
- 南陇侯（元婴中期）于本章被圣界之物夺舍，原主神识被取代。
- 鲁卫英虽有昊阳纱护身仍被重创，遁逃受阻。

# 第830章 吞婴 / 第831章 异兽

## 一、南陇侯吞鲁卫英元婴(坠魔谷秘境内)
- **困杀过程**：「南陇侯」以魔化躯体出手，身形带幻影残痕扑向藏身石壁的鲁姓老者(鲁卫英)。老者将全部法力灌入火红轻纱(昊阳纱古宝)所化红色光幕护身，挡下黑爪。
- **破防手段**：南陇侯一拳震散老者放出的护体灵光，随后连环重拳，每拳恰好击散老者注入法旗/秘术的半数灵气，使其无法祭出白色法旗、无法聚法，活活困死在石壁中，光幕渐薄。
- **吞婴**：老者催出酷似本人的元婴(死死抱一口蓝濛濛小剑)欲遁逃。南陇侯背脊长袍爆裂，现出**一张与血咒之门一模一样、银白色眼珠的鬼脸**，以拇指粗的紫色长舌穿透元婴眉心，将其卷入口中咀嚼吞下。
- **收尾**：南陇侯转身，反手掏出鲁卫英肉躯心脏捏为血雾，肉躯生机立断；随后化黑气遁走。飞至高空触动坠魔谷血红闪电，闪电近身丈许即自动绕开，安然无恙；辨明方向后破空消失。
  - 备注：本片段未明示「南陇侯」真实身份，仅知其鬼脸与「血咒之门」同貌、具银白眼珠，故作为独立条目记录，不臆断归属。

## 二、鬼灵门一行攀祭坛寻灵缈园
- 鬼灵门门主**魏无涯**、**王天古**及三名结丹弟子，在距祭坛十余里处先破一大范围禁制，耗去大半日方至。
- 祭坛宏伟巨大却古旧风化、阶梯遍布杂草。门主称登顶即可寻得**灵缈园**踪迹与入口(据残魂所言入口在此)。
- 阶梯另有增重等禁制，踏上者身沉如负千斤；魏无涯不动如常，结丹弟子则面白颤抖。一行缓缓上爬。

## 三、韩立、紫灵寻灵烛果·斩守护古兽
- **地点**：数万里大沙漠中一处被禁制(幻阵)遮掩的绿洲绿湖；湖中淤泥上生一株数尺翠草，顶端孤生四颗火红似烛、散药香的**灵烛果**。
- 韩立判断：灵草顶部有两处咀嚼残痕，应有守护古兽生吞过两枚灵果(灵烛果生服不催境界，但对突破瓶颈略有功效)。以明清灵眼透视湖底，发现潜藏古兽，令紫灵退避。
- **布阵**：放出三只灵兽袋中数万金色噬金虫(已近成熟体，个头倍增、外形狰狞)遮天；又出三十六口青竹蜂云剑分化上百道剑光，瞬间布下**大庚剑阵**。
- **诱敌**：双手凝出孩童头颅大的金色电球(辟邪神雷)轰击湖面，遍布金弧，逼出古兽。
- **古兽**：六七丈长如乌黑烂肉、遍布孔洞、中央一肉球独眼，黑白二气护体；喷碧绿腥臭液与绿色妖火皆奈何不了噬金虫。其真正天赋为**敛息变形**(化形隐匿，元婴后期都难觉察)，但已被明清灵眼看破，战力一般。
- **击杀**：古兽欲遁入水，韩立以核桃大蓝色火球(乾蓝冰焰)瞬冻整湖成冰块断其退路；大庚剑阵金丝随处削肉，噬金虫吞噬，转眼古兽身躯耗尽，坠冰而亡，仅余一颗黑白两色妖丹，被韩立摄取(此色妖丹他初见)。
- 紫灵赞韩立神通深不可测，提及外界传韩立为「三大修士之下最厉害之人」；韩立谦称是此兽不强。

## 第832章 得果
- 韩立将噬金虫收回三只灵兽袋，大庚剑阵还原为三十六口金色飞剑收入袖中，留下幻阵迷惑追来的鬼灵门修士。
- 韩立与**紫灵仙子**飞至灵草上空摘取**灵烛果**。韩立提醒：灵烛果需用木制容器盛放，否则药性几天即失。
- 紫灵用碧绿木盒取走一枚灵果，并主动表示此行未出力，剩余三枚灵果尽归韩立——其克制贪念令韩立印象改观。韩立用木匣收走三枚。
- 韩立尝试斩取一截灵草枝条移植，枝条触地即枯萎化灰，印证典籍所载：灵烛果灵根无法移植栽种。
- 韩立取出**青蚕袍中所得的坠魔谷地图**，发现出沙漠再飞两日即可达标注之地。决定先炼成造化丹、服丹炼化药力，再探地图秘宝。
- 二人驾**御风车**遁走千里。约一时辰后鬼灵门钟姓长老一行赶到绿洲，只见空草冰湖，扑空脸色铁青。
- 韩立深入丘陵群山数百里，以飞剑切山开辟简易洞府，洞口布小型法阵，并放出七八只结丹期傀儡守护。紫灵则另寻山头开辟密室。
- 韩立取出灵烛果木匣与**古色古香的青色小鼎**，喷出**青色婴火**包裹小鼎，开始炼制造化丹（生平第一次炼此丹，但备料充足、炼丹造诣近宗师级，胸有成竹）。

## 第833章 祭坛
- 韩立按丹方将块茎、灵草茎杆、灵烛果（剑光分四份取一）等依次投入小鼎，以青色婴火与连串法决催炼；鼎中雷鸣高低变幻，韩立以蓝芒灵目透视鼎壁观察丹情。
- 镜头转向盆地祭坛：鬼灵门一行攀登数百丈高的祭坛，阶梯禁制重力随高度骤增，结丹弟子几乎寸步挪移；唯**魏无涯**（元婴后期）始终不见吃力，令鬼灵门门主暗惊。
- 祭坛顶面铺白玉砖，中置碧玉巨大供桌，四角立十丈高青石柱，柱顶各蹲一只散发灵光的麒麟石像，柱面铭刻古老符文。
- **鬼灵门门主**抛出银白画轴，召出**苍坤上人化身精魂（儒生）**。此乃门主与苍坤上人化身交易的一环。
- 儒生指祭坛为坠魔谷中心、灵缈园入口所在。命四人向四根石柱注入灵力，麒麟石像双目亮起血红之光；儒生以四道灰法决击石像，喷出四道五色光柱，于高空形成五色光球并爆裂，掀起惊人灵气波动。
- 魏无涯怒视儒生质问：“这就是你所说的灵缈园入口？”

## 第834章 通道
- 祭坛上空百余丈处显露一道月牙状巨型**空间裂缝**（先前被禁制遮掩，元婴修士亦未察觉），即灵缈园入口。
- 儒生解释：**灵缈园**原为介于**灵界**与**人界**之间的存在，须借此空间裂缝进入，他有办法暂送众人进去，但只有六七成把握，结丹弟子不可去。
- 因儒生隐瞒风险，鬼灵门门主取出**血红令牌**施**惊魂咒**（鬼灵门密术，比禁神术更歹毒，可一念间令魂魄溃散）折磨儒生约一顿饭工夫，恩威并施令其带路。
- 儒生飞近裂缝，吐出一颗墨色**破界珠**（耗去其大半魂力），射入裂缝引动；裂缝中心化为墨绿色芒团。
- 魏无涯、鬼灵门门主、王天古三人射出灵柱注入光团（魏无涯灵柱碗口粗，修为最高），中心渐成黑洞通道；又加三名结丹弟子相助，耗去小半灵力方使通道稳固。
- 魏无涯以神识感应到通道另一头有广大所在，众人大喜。**王天古**放出黑色雾气（内含小鬼）冲入通道试探。

## 关键设定
- 灵烛果：天地灵果，灵根无法移植；可炼造化丹，生服提升修为；须木器盛放。
- 灵缈园：介于灵界与人界之间的秘境，入口隐于坠魔谷祭坛上空空间裂缝。
- 破界珠：开启界面/空间通道的秘宝，使用耗损施法者魂力。
- 惊魂咒：鬼灵门密咒，控制魂魄、可瞬间溃散对方魂魄。
