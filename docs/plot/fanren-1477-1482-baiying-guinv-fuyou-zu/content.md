# 第1480-1482章 白影 / 鬼女 / 蜉蝣族

## 一、冲出鬼雾
- 韩立与元瑶、妍丽带残余鬼兵驾遁光逃离鬼物纠缠之地。
- 元瑶、妍丽各喷一白一黑圆珠，融合为拳头大黑白眼球，闭目操纵其辨向——此为**鬼道秘术**，可走出迷雾但途中不可改向，否则前功尽弃，故仍会被鬼物缠住。
- 韩立一路以霹雳手段击杀挡路鬼物、冲散地渊妖物与鬼物的小型战团，开路而出。

## 二、二女身世与恢复人身
- 元瑶、妍丽修炼**阴阳轮回决**，已成半鬼之身，忧惧将来如鬼婆般彻底转化为鬼物之身、入天鬼之道。
- 韩立判断阴阳轮回决诡异但**并非不可逆**，可散尽阴气恢复人身，唯修为受损、要吃苦头，建议回人族境内详查。二女狂喜应允。
- 妍丽称当年初见韩立时他还只是炼气期，如今已可比炼虚级，恍如南柯一梦；韩立自言只要不碰炼虚后期便有一拼之力。

## 三、白袍鬼女（白影）
- 雾气边缘传来女子娇笑，二女半鬼之身竟被笑声抽空体内阴力、陷入困乏，韩立一声冷哼以神识相救。
- 韩立袖出金色剑光斩向虚空，逼出一道**淡淡白影**；白影以一化四、四化八无穷分裂，灰霞与五色光焰天罗地网亦无法尽灭。
- 韩立催动**破灭法目**（多年培育、乌黑眼珠，瞳孔喷拇指粗光柱），轰中虚空逼出白影真身。
- 真身为白袍女子，袍光不染、隔绝神念无法测其修为；精通**空间神通**：白骨爪手臂凭空消失后瞬移偷袭韩立咽喉。
- 韩立颈部自行浮现**金色鳞片**护体，反震开骨爪、连皮毛未损；其面容半边碧绿干瘪、半边唇红齿白貌美，身披白雾鬼影。
- 韩立正欲召**啼魂兽**辅以雷珠一击解决，雾深处忽传惊天巨吼；白袍鬼女闻吼弃战、卷雾朝吼声方向遁走。韩立疑六足等强敌仍困雾中，遂速速带二女出雾。

## 四、蜉蝣族傀儡大军
- 三人出雾后，高空一陌生男声讥其竟能走出所布禁制。
- 见**木青**与**白发美妇（蓝道友）**率十余高阶妖物、八名阴甲鬼王、数千鬼兵，与一支数倍于己的军队对峙。
- 对方约七八千血红战甲巨人，配刀枪弓弩、百余辆青色战车（狮身蛟首黑兽拉车）；为首三名武将。
- 韩立以神念辨出三武将及兵卒皆为**傀儡**，且三武将气息各异，应是**大神通之辈附身**，实力不在二女之下。
- 木青未追问韩立逃雾之事，反问失散的**金灵（金猿/金兄）**下落，韩立答迷雾中早已走失。
- 紫甲武将（背双金锤、紫色盔甲）自称此地为**蜉蝣族圣地**，外来者曾闯入打圣物主意；称包围空间的**冥河之水**由本族生成。白发美妇质疑蜉蝣族何时改修阴鬼之力，并轻视其仅来附身傀儡、非天蜉阶亲至。

## 五、开战
- 紫甲武将驱犀牛状怪兽喷青色光球，美妇召巨大鬼影喷乌球相抗，爆出蘑菇云团。
- 数千鬼兵在八名鬼王带领下化阴风冲杀；紫甲武将双金锤幻化上百锤影砸下，被鬼王黑色光幕网住后又催爆撕裂光幕；木青以五色巨花虚影托住金光。
- 血甲傀儡弓弩齐射，阴风扎入傀儡大军，鬼哭喊杀震天。三武将傀儡无视鬼兵，全神盯防美妇与木青。
- 另一**三目巨斧傀儡**开口：无需硬拼，只需拖住片刻；已通知其他人，待其助鬼物全力击杀完另一波外来者（六足、地血老怪等）后即来支援。

## 待续线索
- 六足、地血老怪疑仍困雾中未脱；金灵下落不明；蜉蝣族增援将至。

# 第1483-1485章 冥河之地：蜉蝣族傀儡伏击与天蜉虫影投影

## 背景
地点为灵界『冥河之地』，乃蜉蝣族圣地，特产『冥河神乳』。四大妖王一方（木青、白发美妇【蓝姐姐】、六足、地血老怪）为夺神乳深入此地，韩立（灵将修为）与元瑶、妍丽随行其中，另有金灵（金毛苍猿，木青属下）及十余名高阶妖物。对手为蜉蝣族常年驻守的三名合体级傀儡：紫甲武将（紫甲傀儡）、双首（双头）傀儡、三目傀儡——其本体为七名金蜉级存在，附身傀儡后只能发挥本身实力约一半。

## 第1483章 重聚
- 血甲傀儡（数量众多、擅配合）对阵阴甲鬼兵（白发美妇训练的阴甲玄鬼＋八大鬼王），鬼兵一方个体更强，一度占上风。三名武将傀儡对部下被灭无动于衷，与木青、美妇均按兵不动、互相忌惮。
- 雾海深处骤起异变：白色骄阳爆现、深不可测灵压肆虐，众人四散遁逃。六足亲自招来白色飓风（风属性神通），将整片雾海与未及逃离的傀儡、鬼兵尽数绞灭，仅八大鬼王勉力支撑。
- 八大鬼王合一化巨鬼冲出狂风。地血老怪的紫血傀儡现身，肩头立两名血袍人，傀儡头顶站着六足。六足抖袖滚落四颗血红头颅——四名傀儡敌手已被斩杀（『若你们本尊在此，七名金蜉联手我亦需退避；附身傀儡只剩一半实力』）。
- 六足下令围杀三名守地傀儡，三傀儡催座下怪兽（身怀诡异遁术）分散逃走，木青（足踩金花、类传送遁术）、白发美妇（化青烟＋合体巨鬼）、紫血傀儡分头追击。原地仅余韩立、元瑶、妍丽与十余妖物，妍丽传音劝韩立趁机逃走。

## 第1484章 虫影
- 韩立喝破隐匿者——金灵（金毛苍猿）现身，目光清冷。韩立自忖一人有七八分把握遁走，但带元瑶、妍丽则不足一半，遂决意暂不冒险，与金灵虚与委蛇闲聊。
- 木青、白发美妇先后返回；美妇法力消耗不小，八大鬼王因敌自爆受伤被收入阴云囊休养。六足、地血迟迟未归，追击方向传来巨响与刺耳尖鸣。
- 尖鸣无视护耳、直入神识，打断大衍决运行；韩立以精神刺自救，复以体表金弧＋五色光焰＋灰色霞光（元磁神光）双层光罩护身，并以元磁神光化晶丝救回急坠的元瑶、妍丽。十余妖物则坠地翻滚、护体灵光无效。
- 天边浮现赤红色巨大虫影——形如放大无数倍的苍蝇，半透明翅翼震动发出尖鸣，漆黑复眼喷无数赤红光柱织成红网。网下两物（缩小的紫血傀儡＋一只黑色巨虫）拼命躲避。

## 第1485章 蜉蝣族与虫海
- 黑色巨虫（六足所化）与紫血傀儡（地血）以残影避过光柱；攻击近虫影十余丈即被无形之力挡爆。
- 美妇惊呼『天蜉级存在』。木青判明：此乃天蜉级存在的『化身投影』，是被追杀的傀儡借秘术强召而来，仅约真身二三成威能、未借主体神念，威能耗尽即消。果然片刻后虫影碎裂、化赤红光霞溃散。
- 六足、地血归来。六足露真容：下半脸如常年轻男子，上半双目为一对翠绿复眼。六足自承『原本就出身蜉蝣族』，但当年修为低浅不知圣地具体位置，是在地渊困居多年后无意发现地渊与冥河之地互通，否认与守地傀儡有勾连；两名血袍人（地血）出言力挺六足。
- 众人确认：天蜉是蜉蝣族仅次于真灵的存在；对付他们只需一名天蜉即可。蜉蝣族『虫海大军』连真灵都要退避，却会令本族元气大伤过半，非生死关头不动用。圣地一般数千年才开一次，普通蜉蝣族人久留会修为大退乃至丧命。
- 六足称此地圣地约数月内难再开启，足够众人取神乳原路返回，遂决意抓紧上路，并询问尚余多少阴兵、傀儡可用。

## 关键判断/伏笔
- 木青、美妇虽看出六足之言『必有不实之地』却无从反驳，对六足蜉蝣族出身心存戒备。
- 韩立全程旁观不插手，暗自记下各方虚实，伺机脱身。
