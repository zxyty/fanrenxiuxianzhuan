# 第2035-2037章 泣灵秘藏（血鸦城外圣砖之争）

## 背景
- 地点：魔界·血鸦城外万余里的乱石堆上空。
- 韩立凭强大神通无视血鸦城禁制，跟踪两道遁光出城，意外卷入一场圣砖争夺。

## 核心设定：圣砖与泣灵秘藏
- **圣砖**：泣灵圣祖坐化之地遗物，共四块，内含口诀/神通。
- 四块圣砖凑齐 → 得泣灵圣祖全部神通 + 其亲绘的 **泣灵秘藏（泣灵秘图）**。
- 秘藏中宝物极多，尤以 **墨灵圣舟**（可排进圣界前三的飞行至宝）最为人垂涎；得之或可抗衡寻常圣祖。
- 当年雷海七煞与天房山洗劫泣灵圣祖坐化之地，血鸦城主趁乱偷走四块中的三块，引二者火拼，自己远走当城主。拍卖会上那块圣砖为羊老二故意流出（毁去部分口诀，但经鉴定为真）。

## 出场对峙
- **血鸦城主**：银目灰袍老者，姓炳（炳道友/炳兄），合体中期魔尊；持三块圣砖；有身外化身（第二化身，合体初期，皂袍、面容与本体一般无二）；护身有顶阶魔器血玉牌可自行激发挡致命一击。
- **羊老二**：蓝袍面目普通中年人，自称雷海散修，初现伪装成炼虚，实为合体中期；持第四块圣砖。
- **金葫魔尊**：羊老二暗中盟友，赤足丑陋汉子、背半人高金色葫芦，合体修为，谈吐斯文。

## 经过
1. 血鸦城主将羊老二引出城，欲杀之夺砖；羊老二提议平分秘藏。
2. 双方互试圣砖真伪，第一块『圣砖』实为陷阱，爆裂成银色漩涡狂扯神念，趁机偷袭血鸦城主（金光绕颈，被血玉牌挡下）；同时鬼手偷袭羊老二（被身外化身/异宝化险）。
3. 血鸦城主亮出第二化身，逼羊老二召出隐身的金葫魔尊。
4. 血鸦城主早有筹谋，放出 **七杀血煞**（血光按浓淡隐分七层）与 **黑骨魔虫**（拇指大、漆黑、背有骷髅白纹，坚硬如钢、巨力，魔界最负盛名魔虫之一），结成血煞+虫云圆环困住二魔。

## 韩立出手
- 趁羊老二再取真圣砖之际，韩立凭空伸金色巨手夺走圣砖，反震碎羊老二鬼头与金葫魔尊剑光。
- 现身为青袍普通青年，索要血鸦城主手中另三块。
- 一步横跨数十丈逼近；背现 **三头六臂金色法相**（十余丈高，梵圣真魔法相），凝出金色漩涡，将七杀血煞尽数吸收吞净（漩涡暴涨数倍、吸力增十倍）。
- 血鸦城主第二化身藏黑气偷袭，被韩立以 **辟邪神雷**（六团金色雷球→金色雷阵，金弧如金蛇狂舞）灭杀飞灰。
- 黑骨魔虫扑来，韩立放出漆黑圆环化出上千只 **成熟体噬金虫**，一接触即吞噬绞杀黑骨魔虫，地面铺满残骸。
- 血鸦城主心神受创、欲逃；韩立一声神识冷哼于其精魂深处爆发剧痛，将其击落，噬金虫蜂拥淹没将其擒杀。

## 结局/影响
- 韩立夺得至少一块（争夺中的真圣砖），并觊觎血鸦城主身上的另三块。
- 血鸦城主第二化身被灭、两大杀手锏（七杀血煞、黑骨魔虫）尽毁、本体被擒杀。
- 泣灵圣祖（已于第2011章死亡）的圣砖与秘藏线索落入韩立视野。

## 备注
- 羊老二自称『散修』属泛泛诈称，非冒用具体已命名者身份，不作冒名处理。
