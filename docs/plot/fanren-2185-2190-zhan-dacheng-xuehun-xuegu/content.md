# 第2204-2206章 大乘庆典斩黑枭王 · 血魂之讯 · 驱血蛊虫

## 第2204章 斩大乘
- 韩立大乘庆典之上，夜叉族隐藏的大乘巨汉（黑枭王）发难，施法以巨山般重压镇压广场数万修炼者，并以黑白天地、千丈黑白巨环困敌。
- 韩立背后三头六臂梵圣金身腾起，化为七八千丈高的三头六臂紫金巨猿，体表银白纹阵；硬撑碎黑白巨环、以银色拳影/银丝撕裂黑白虚空，再凝成百余丈银色巨手反压巨汉。
- 巨汉以七色玉扇（七色光霞）与漆黑巨砖（用七七四十九种材料、数百年凝成的黑色战甲一并）抵御，皆被击碎；身躯被银光一击飞出、恢复原形并喷血。
- 巨汉破空欲逃，韩立喝『念剑诀，斩！』，三颗头颅眉心裂开竖目，喷出三口晶莹小剑，于天边将巨汉拦腰斩断；巨汉以三四尺小人遁走，留言『姓韩的，今日之厚赐本座铭记在心，下一次定回报此恩』。
- 韩立放出三朵金花→三只丈许金色巨虫（**伪噬金虫王**，实力堪比合体后期、身躯几乎不损、耐力惊人）破空追杀重伤巨汉。
- 人妖两族沸腾，齐呼『韩前辈神通盖世，威震灵界』；异族使者忌惮，决意速报族中。

## 第2205章 血魂之讯
- 韩立彻底开放禁制，连续讲道七天七夜（自炼气期讲起），整座三色巨峰挤满听道者，多人入顿悟。庆典毕，三色巨峰凭空消失。
- 石塔大厅议事：在场有器灵子、海大少、银发老者（谷长老）等天渊城合体长老，及弟子白果儿（已化神中期，略高于海大少）、朱果儿、李蓉。重创黑枭王令众合体长老彻底心悦诚服。
- **魔界大劫情报（李蓉所述）**：傲啸前辈与莫简离大人等区域几族大乘、以及灵界众多超级大族大乘均已进入魔界，协助魔族始祖解决魔界大劫（因『螟虫之母』恐致魔界覆灭，下一个或轮到灵界）；魔族因此撤出已占灵族领地。代价是灵界几族大乘须随超级大族大乘一同入魔界。
- 黑枭王本是夜叉族故意隐瞒、欲作杀手锏的大乘，因诸族大乘入魔界、两族对峙未爆发大战；韩立意外进阶大乘恰解危局。
- 韩立计划赴**圣岛**：与长老会商议应对各族、探查傲啸下落，并对岛上藏秘术的**天书阁**与**混沌万灵榜**感兴趣；命李蓉陪同。安排带月天等离开天渊城以免引祸，令谷长老等照看弟子。
- **虚天鼎来由**：许芊羽奉冰魄先祖**血魂**之命，将本是许家重宝的虚天鼎当贺礼献韩立。血魂分身半年前身负重伤返回许家后即昏迷不醒，临前嘱速送鼎并请韩立赴许家。韩立应允数日后前往。

## 第2206章 血蛊虫
- 三日后韩立携银月、蟹道人、白果儿、李蓉赴许氏家族（魔族退后许家已返原驻地）。家主许蛟率许岩、许火等长老恭迎，对大乘修为的韩立诚惶诚恐。
- 韩立留银月、白果儿、李蓉于主殿，仅带蟹道人入许家地下石殿禁地。血魄（冰魄仙子血魂化身）躺于**万年玄冰棺**中，双目紧闭、满脸黑气、眉心现忽闪血纹（奇寒可减缓其气息削弱）。
- 韩立以神念化丝之术（眉心黑色妖目喷晶丝）从血魄体内逼出一只拇指大、似蜗牛而生七八触须的**血蛊虫**——血纹随之消失。
- 血蛊虫是灵界三大陆之一、最神秘的**血天大陆**独有奇虫，无形无色溶于血液，与寻常蛊虫不同；韩立当年游历**雷鸣大陆**时于典籍中见过其描述。该虫阴毒、合体修士亦极难驱除，但离体后不堪一击，被韩立晶丝绞碎。
- 韩立以青光通其经脉、喂一颗『补元丹』，将昏迷的血魄救醒；血魄脸色苍白但暂时无碍，向韩立道谢。

## 关键设定/伏笔
- 韩立正式立于大乘之境，成人妖两族数十万年来首位大乘。
- 螟虫之母 / 魔界大劫 牵动全灵界大乘格局，灵界几族大乘将被迫入魔界。
- 圣岛、天书阁、混沌万灵榜、血天大陆 为后续目标/线索。

## 一、第2207章 灵鼎之秘（许家）

- **补元丹与血蛊虫**：韩立以补元丹为血魂（白衣女子）补回小半精气。血魂自述在**血天大陆**遭人暗算中了血蛊虫，靠身躯特殊与秘术强压才返回；血蛊虫非风元大陆所有，在血天大陆亦是一等一的灵虫。
- **冰魄之失**：血魂恢复部分记忆后发现须赴血天；其本体『冰魄』最后在血天大陆失踪，故此前许家毫无线索；血魂势在必行要再赴血天寻回本体。
- **虚天鼎之秘（核心）**：血魂屏退许蛟等许家弟子及银月、李蓉等人，与韩立单独密谈，揭示**虚天鼎被炼制出的真正作用并非斗法争斗，而是一把『钥匙』**。此鼎乃血魂在人界依据**一页金阙玉书**的记载费心仿制之物。韩立听后由微笑转为凝重乃至震惊。
- **交易**：血魂以虚天鼎之秘与韩立做了一桩交易/约定。韩立离许家后对银月只称『做了个约定/勉强算交易』，未细说。
- **去向**：下一站为**圣岛**，由李蓉（李仙子/蓉仙子）带路。

## 二、第2208章 天书阁（圣岛）

- **圣岛**：人妖两族圣地，藏于人妖交界偏僻山脉雾海中；黑岛悬于低空，数百巨石阁楼绕之旋转，甲士驻守，飞舟巡逻，层层光幕禁制。长老会由十几名合体长老组成。
- **韩立威望**：圣岛长老（洛姓白发老翁、旬姓白净男子等）焦虑等候。传闻韩立稍一出手即重创夜叉族隐藏的大乘『黑枭王』，神通胜过莫、傲啸两位大人。长老此前因其弟子缘故得罪过韩立，但判断其大乘心胸不致计较，且莫简离、傲啸失陷魔界唯赖韩立相救，故须极力辅佐。
- **混沌万灵榜**：圣岛中心一高涯下方百丈晶壁，刻金色古文。排第三位者为最新上榜的**玄天之宝『玄天斩灵剑』**，出世时令风元大陆乃至各大陆超级大族疯狂寻找却始终无人得手。韩立触摸臂上图案有灼热刺痛之感。蟹道人从晶壁上感到极稀少却极纯的**仙灵气**，纯度不下于韩立所提供灵液——暗示灵界与真仙界虽失联多年仍有未断之事。
- **天书阁交换秘术**：藏秘术上万种，学一种需付天价灵石或以阁中未藏的同种秘术交换。韩立向守阁老妪一次拿出近百枚异族秘术玉简（消元分光术——飞羽族；九焱魔×——魔族；天芒青木功——木族等）。老妪核实，其中**六十一枚为天书阁未藏**，遂以之换走阁中六十一种秘术。规矩：秘术只许本人修炼，不得私传他人门徒，违者由执法使者制裁。

## 三、第2209章 再入魔界

- **伪虫王追黑枭王**：三头『伪虫王』（噬金虫）追杀黑枭王，黑枭王逃脱但近半身躯被吞噬，重创远超预料。
- **下半部祭雷术（此行真正目的）**：韩立先前从**飞羽族地渊妖王**处所得祭雷术仅残缺催动之法，催动**辟邪神雷**威力极大却施法迟缓、实战无用。经**黑雨上人**告知圣岛存有下半部口诀，遂于天书阁取得，凑成完整一套；其余数十种秘术只是附带，几种亦令其意动。
- **离岛入魔界**：玉舟载韩立、蟹道人、银月及圣岛洛姓老翁离岛。韩立令封锁其进入魔界之消息至少百余年。为救失陷魔界已久的**莫简离、傲啸**，决意速行；自言其功法大半以炼体进入大乘，无需久巩固境界。
- **空间节点破界**：目标为魔族大军撤退时遗留的空间通道（今为空间节点，界面障壁尚薄）。韩立以第三只竖目探查，布下阵旗阵盘、放出巨型金弧光阵，再祭出灵界大名鼎鼎的破界法器**逆星盘**（化为一轮银月悬空）。逆星盘喷五色光柱、光阵喷六道黑白光柱合击灰云，破开黑色巨洞。
- **随行安排**：银月（玲珑）挂念傲啸老祖且愿随行；蟹道人相陪；朱果儿因修为太低留圣岛，韩立传音告其需待日后寻找**小灵天**入口时方能随行自保。韩立卷起银月、蟹道人化金虹遁入魔界黑洞。

## 关键设定要点
- 虚天鼎=钥匙（仿金阙玉书）。
- 玄天斩灵剑=玄天之宝，居混沌万灵榜第三，含仙灵气。
- 韩立此时为新进大乘，以炼体功法为主。
