# 第1879-1880章 群魔初现·魔潮汹涌

## 背景
第十卷「魔界之战」。魔族借「圣祭」破界入侵灵界，人妖两族各处「魔斑」同时上演魔潮降临。圣族（飞升真魔界者）此次有机会彻底占据灵界部分地域，立大功者可申请留守灵界，故各魔族倾力而来。

## 第一波：魔兽群分散袭掠
滚滚魔气（魔海）中，各部魔兽在为首高阶魔族带领下接连冲出，共十三波，行动杂乱、似无固定目标：

- **雷牛群**：牛头巨汉吹金色角号（发蛮荒气息、后转金戈之音），驱动数以万计青色雷牛（双目血红、口喷白气），自身化银光率群离去。
- **血蝠群**：奇丑少年以尖鸣驱使，血蝠化为十几股血云四散；血蝠数量为雷牛十倍以上，每队上万，少年自身化血光没入最大血云。两队血蝠尾随雷牛而去。
- **三目魔蟒群**：约五六千条漆黑巨蟒、额生鲜红血目，气息远胜前两者。为首青甲魔族（乌黑面具、狰狞青甲，被称『卫兄』）立于三百余丈本命神蟒头顶。其本命蟒服珍稀灵药后进阶，额生两根血角、背撑青色肉翼，群蟒参拜如王；此蟒据称足以抵挡圣阶初期存在。青甲魔族令三目蟒钻入地下，自身化青光离去。
- **螳螂魔虫群**：魂珠（自称『本王』，三尺许七八岁幼童模样、头扎冲天小辫、淡绿道袍、肩插三口金色小剑，骑两丈淡紫巨螳螂）率七八十万头颅大小的紫翅螳螂魔虫，铺天盖地随青甲魔族同方向而去。两族（青甲/卫族与魂珠族）约定联手。

## 第二波：三角巨塔破界与筑城
半个时辰后，魔海雷鸣电闪、空间剧烈波动，魔海凭空扩展三分之一余。**数以百计的三角巨塔**（黑黝、银色巨大符文、底部百里之广、状如小型城市）徐徐破界降临，落地压平山峰湖泊。

巨塔放出：
- **黑色独木战舟**：十余丈、黑光笼罩、端部魔神雕像、两侧奇长利刃，各载五六名黑青甲衣魔族，绕塔盘旋警戒。
- **红甲双翅魔族甲士**：持长戈弓箭，腾空警戒。
- **各色魔族**（人形，男女服饰各异），有的一震身化双首四臂魔人（第二头颅狰狞如鬼），充满杀戮血腥之意。

魔族私语透露圣祭意图：红发鹰钩鼻老者『沙兄/三阳兄』感慨重见旧景；黑麻袍金带威严老者称立功后可留守灵界。

- **石魔**：通体灰白岩石、肩生四臂、十余丈高，每塔出上千只，性情温顺；可化身入土，并以四臂喷光柱「化土为石」，将拔地土墙瞬间化为坚石城墙。短短一个时辰内圈起方圆数万里石城。
- **黑气鬼影**：飞扑城墙铭印大小法阵后返回巨塔。
- 高阶魔族亲自布置玄妙禁制，层层笼罩巨城。

## 血光圣祖现身
巨城中心三角巨塔顶端裂开，升起血红巨大祭坛（上古文字、浓浓血腥）。祭坛中心立**血光圣祖**（血光笼罩高大人影、目如残月）。全城中高阶魔族半跪拜「血光圣祖」。

圣祖单手一招，抓下小半魔气化为百余丈乌杖（鬼头蛟身、诡异花纹），缩为两丈持于手中，念魔族咒语；乌杖鬼头喷黑亮符文化黑花飘满全城，再搓杖爆为无形灵波，黑花尽爆化魔气笼罩全城。魔气及身者法力体力以近半速度加快恢复。

**圣祖法旨**：「三日内，再分别建成另外三座圣城，再整顿七日后，等所有人手到齐后，立刻向附近最大的人类修士据点出发。」众魔欢呼「谨遵法旨」。

## 各处战场速写
- 同时在人妖两族各处魔斑上演相同降临。最大魔斑处：以直耸如天巨树为依靠的人族巨城，数名合体修士登高远望魔海，脸色阴沉。
- 离**天源圣城**万里外：一僧一道（铜镜+圆钵）击杀蛇头马身魔物，见高阶魔族将至，遁向**天元皇城**方向。
- 妖族枯黄平原：数十万灰濛巨狼群 vs 上万淡蓝色魔狮群（生乌黑短角、喷火球冰锥）厮杀；空中数千雪白巨鹰（钢爪缠银弧带雷鸣）vs 双首飞蛇（喷毒液）惨烈争斗。
