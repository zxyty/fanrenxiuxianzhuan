# 第1009章 阴魄凝形

第六卷《通天灵宝》。

## 一、韩立：搜完铸灵堂，决意下山续探

- 韩立搜完铸灵堂最后一处建筑，回到碑楼前那层白濛濛光幕处踌躇。
- 权衡风险：怕碰到第一波进入昆吾山的修士（多半会立即动手）；乾老魔、毒圣门、富姓老者等他都不惧。
- 反思后哑然失笑：困守也未必安全，别人搜完仍会找上铸灵堂；估计乾老魔等去的昆吾山重地多半已与第一波修士撞上甚至交手。
- 决断：趁人无暇旁顾时浑水摸鱼多搜几处。连小小铸灵堂都能寻得**太阴真火**，他处应也有收获。
- 自评神通：即便同时被数名元婴后期修士围攻，纵不能胜，逃遁有把握。
- 行动：取出**三焰扇**，三色火焰闪动轻易破开光幕，直接下山。

## 二、镇魔塔第七层：鬼王分身困局

同一时刻，昆吾山地下幽蓝大厅中：

- **大头怪人（叶家七叔／叶月圣）**、**古魔**、一名**叶家元婴初期修士（叶家老者）**被七八只红发赤身恶鬼团团围住。
- 恶鬼青面獠牙、口吐阴气、爪芒犀利，三人以多层防御光罩抵挡。
- 三人法宝（黄色飞剑、黑色小剑、白色玉尺、赤红火球）虽能斩碎恶鬼，但恶鬼一经阴气一喷即复原如初，宛如不死。

### 关键设定揭示
- 叶家老者指出：这些恶鬼受此地阴气万年滋养，已修成**阴魄凝形**境界，普通法术法宝无法斩杀，只有至阳功法宝物才有效。
- 大头怪人点破：恶鬼根本是**同一只鬼王的分身**，不同时尽杀就会在阴气中复活；他未施霹雳手段，是一直在找鬼王真身。
- 古魔补充：若是寻常阴魄凝形恶鬼，被连斩多次也该元气大损，但它们毫无疲态——鬼王意图耗死众人。
- 可惜『十五弟』修至阳神通者（**明礼**）在上一层被**乌影蛇**的暗影喷砂从自己影子中暴起偷袭，肉躯瞬间坏死，只逃出元婴，无法对付恶鬼。

### 镇魔塔设定
- 塔倒过来修建：入口在上、塔尖朝下；一层比一层妖鬼难缠。
- 据韩长老所言，此塔共**九层**，三人花长时间才闯到第七层。
- 大头怪人坚持继续：下面镇压的越厉害，**通天灵宝**出现在下两层几率越大——只有上古修士都忌惮的妖魔才需通天灵宝镇压；事关叶家数百年图谋。

## 三、毒圣门四老闯入，三方对峙

- 古魔忽称找到鬼王真身。话音未落，大厅入口飞入绿濛濛光团，现出四个人影——**毒圣门四老**，为首脸带碧纹者为**花天奇**。
- 花天奇认出大头怪人，惊呼『叶月圣！你不是早就陨落了吗？原来是叶家揭开的封印？』，并道破其假死后已暗进**元婴后期**，正魔几方若知必坐卧不宁。
- 毒圣门来历：开派祖师遗言提及镇魔塔藏有奇宝（具体不明），四老遂首来此塔；先前已发现有修士抢先，原以为四人之力足以应付，未料撞上一名元婴后期＋两名元婴初期大修士。
- 花天奇本欲掉头就走（不敌），但见叶家三人正被群鬼困住，遂神色阴晴不定、按兵观望。
- 大头怪人露凶光，张口射出银芒——大厅一角逼出鬼王真身（灰色模糊鬼影，口喷拇指大灰色圆珠），被**银色飞叉**缠住节节后退。
- 古魔趁机取出一面漆黑如墨的**黑镜**，释放黑蒙蒙光霞如刀山剑林，将众恶鬼一一绞碎。
- 但鬼影一声厉啸，溃散阴气化为浓厚阴气团，将古魔等人死死包住；黑镜黑霞一时也无法破除。花天奇终于脸色大变。

## 备注
- 本章『韩长老』指随叶家行动的古魔（化形冒充韩立形貌的古魔分魂，自第955章起；第985章起以叶家客卿『韩长老』身份出现）。据世界状态须知，其言行归属【古魔分魂】，非真韩立。
