# 第1329-1331章 天渊坊市

## 一、收尾巡逻（第1329章）
- 碧眼大汉以葫芦喷银霞将透明巨兽缩至拳头大小并封装，元磁山随之缩小不让其逃脱；韩立以黑色小山（元磁山）配合，巨兽化白光被吸入葫芦，再贴金色符箓封住。
- 众队员对韩立这位「新任队长」恭敬敬畏，认定其绝非寻常化神初期。
- 巡逻完毕，乘金庭舟传送回天渊城；擒获蛮兽移交「异灵堂」获不菲赏赐。
- 规则：每次巡逻后半年歇息；十年一轮巡逻结束后巡逻区随机变动。韩立首次巡视完成，领并当场服下一粒「灭尘丹」。

## 二、许仙子与冰魄仙子渊源
- 许仙子（许姓女修，元婴）持飞剑「冰晶剑」，大半以万年玄玉炼制，功法冰属性。
- 韩立认出此女星眸与人界小极宫开启虚灵殿时召出的创派祖师「冰魄仙子」光影几乎一般无二。冰魄仙子乃许家上古先祖、飞升修士、修极寒功法。
- 韩立自言在下界得冰魄仙子遗留好处（虚天鼎、寒髓、极寒之焰等皆与其相关），遂赠许仙子四枚拳头大万年玄玉以还人情、消未来心魔劫破绽。
- 许仙子告知：两日后本城拍卖行将办三年一次蛮荒物品拍卖，需化神修士方可参加，每件近天价。

## 三、采办化神丹方灵药（第1330章）
- 韩立遍访坊市药铺，得「玉清丹」「真蟾液」等丹方共七八种：
  - 玉清丹：主料为两三千年药龄珍稀灵草，主利化神初期，对中期效用大减；可降年限炼制，灵石足则可凑齐。
  - 真蟾液：主料为蛮荒古兽「碧眼真蟾」灵血，辅料普通；药性霸道，适合化神后期，中期服用有大风险；灵血极罕见，拍卖会一般有售。
- 韩立大肆收购可炼制各丹的灵草种子/幼草（而非昂贵成材），持神秘小瓶可无限尝试炼丹，自信短期内进阶无忧。
- 灵石不足，韩立换形为「紫脸大汉」入「回生堂」，售出数株七八千年火候灵草换得海量灵石，得手即走以防被盯。
- 灵界灵石丰足，物价远超人界数倍；但人界即便灵石再多也买不到化神丹方与材料。

## 四、傀儡符（第1330章）
- 坊市符箓远胜人界，有银蝌文符箓数种（多为辅助防御型）。其中「傀儡符」自天灵境流出，可召唤封印其中的影傀儡，威能视炼制与操纵者修为而定，最完美者可完全复制驱使者功法与修为。
- 韩立联想到自身残缺玉书亦载有傀儡符炼制法，遂收购足量银蝌文符箓材料以备参悟。

## 五、太玄殿暗市与妖族女子（第1331章）
- 太玄殿：天渊坊市中央贯通人妖两区的巨大石殿，供人妖私下交换己族无用而他族珍稀之物。入殿人族领遮光佩（白霞遮形并限修为）、妖族持弥云幡（黑气隐容），殿内妄动法力即被传送出殿；另有四名炼虚金卫、十名青冥卫维持秩序。
- 一名化形妖族女子（黑气、苗条、声悦耳）主动接洽，言真正好货皆私下一对一暗换。韩立索求碧眼真蟾灵血（对方称妖族亦缺、不会交易）及多种蛮荒丹药材料。
- 韩立退求其种子/幼苗，妖女应允「给我数年」代寻两三种；韩立以一株三千年灵药为定金、传音许诺万年灵药为酬，约定下次来此交换。
- 离去前韩立复看摊上「天雷成木」：天雷击枯死灵木后发芽所成的翠绿枝条，木灵气稀薄却充斥极霸道雷属性力，令其想起天雷竹/金雷竹（辟邪神雷平常深敛）；摊主老者称此物雷力过霸，无法吸收炼化亦无法炼器。

## 关键决策
- 决意参加蛮荒物品拍卖会以求碧眼真蟾灵血与真蟾液。
- 以低价灵药换取化形妖女信任，长期代购蛮荒灵草种苗，避免在人族这边过度引人注意。

第八卷 初入灵界 第一千三百三十二章 神秘雷纹

## 一、太玄殿换雷木
- 韩立对白霞中老者手中的雷木（柳条状灵木）感兴趣，自称‘也修炼些许雷电之力’。
- 老者称此雷木为天雷凝成、价值难估，开价：一株不限种类的五千年灵草（其正炼制一炉‘乾元丹’尚缺此料），或顶阶灵石/同价之物。
- 老者修有强化五识的小神通（嗅觉远超常人），凭韩立此前与妖族女子交易时木匣流露的药香，断出韩立身怀三千年灵药，推测亦有五千年以上者。
- 韩立心中一凛后释然（联想到佛门强化五识的他心通、千里眼、顺风耳之类神通，其明清灵目某种意义上亦属此类）。
- 韩立不讨价还价，直接抛出一只长方木盒，内为五千年灵药；老者验看大喜收下，韩立取走雷木抱拳离去，深知在老狐狸面前言多必失。
- 韩立又逛遍坊市，因身怀通天灵宝与无数灵药而无所动心，数个时辰后返回青石巨塔。

## 二、静室参悟雷木
- 韩立回小队所在殿堂静室，以法决激发整间静室禁制、布下数道彩光与白蒙蒙光幕，彻底遮蔽窥视后才盘膝研究。
- 雷木拇指粗、半尺长、翠绿异常，表体有漩涡状细纹，令人目光深陷。
- 注入灵力：方注入即被反弹，青芒与枝条白光交织难容、隐传嗡鸣；强灌可行但枝条会自爆，遂作罢。
- 金弧灌罩：一连串金色电丝（辟邪神雷）罩住枝条，枝条纹丝不动、毫无异常。
- 切割观察：以金色剑气一斩两段，断口呈银白色；以明清灵目（双目蓝芒）凝望，发现银白实为无数纤细电丝极规律交织成的复杂精细古怪花纹，无灵目绝难看出；切口旋即被翠绿掩覆，残枝化为两根独立枝条。
- 韩立大为吃惊：他本擅控电弧、凝雷成形，但将雷电细化至此并组成此状态远超其当年以辟邪神雷化电网、凝电球的手段，对神念操纵要求奇高。
- 反复切割：发现每次切口雷纹各不相同；遂将枝条一截截斩开（重复雷纹一瞥即过、新雷纹凝神细望），半截切成无数小块，另半截剖成薄如白纸的薄片；雷木残骸蕴含可怕雷电之力却始终稳定不爆。
- 记录：取白玉简，将所有切口景象一丝不差复制入内（以其记忆力亦颇吃力）。

## 三、参悟与复刻尝试
- 韩立直觉七八成把握认定这些雷纹是雷木形成的关键，静坐参悟一天一夜如木雕。
- 随后十指弹出金色电弧，碎为发丝般电丝，口中念念有词指点虚空，电丝凝聚交织成与所记雷纹几乎无二的金色布匹状存在。
- 将成形瞬间，韩立明显感到一种莫名力量，但一根电丝交织出错致整块布匹‘噗嗤’爆裂溃散、前功尽弃；韩立非但不失望反而双目兴奋。
- 韩立再搓出更粗大金弧、碎为远多于上次数倍的电丝，再次重试。

## 第八卷 初入灵界 · 第1333–1334章

### 一、雷纹符文的凝练（第1333章）
- 韩立以神念催动空中电丝，按雷木结构、辅以明清灵目微操，反复用辟邪神雷（金弧）凝练雷纹之物，七八次爆裂失败后成功。
- **锦帕**：巴掌大小、金灿灿，中铭一道神秘雷纹；柔软光滑胜真丝。激发后浮现头颅大小的淡金半透明雷纹，灵压冲天，蕴含灵力为组成锦帕之辟邪神雷的十倍以上。
- **金珠**：第二次凝出，为金色圆珠，表面雷纹淡闪。
- 韩立用噬灵天火（银色火焰，可化火丝/火鸟）与自身青色灵丝尝试编织操纵，均在半途溃散失败。
- **判断**：两物雷力同源而细微不同——锦帕雷力稳定纯净，金珠雷力危险蠢动、一触即发；推测雷纹乃天然自成的奇妙符文，类似银蝌文等仙家真文，各有独特效用，非一时可解。两物收入玉盒贴禁制符箓，与玉简同存储物镯。
- 之后韩立入定休息至第三日。

### 二、恶客逼婚（第1333–1334章）
- 第三日出关，韩立于五十六小队驻地殿堂见对峙四人：许仙子（许姓女修，面色煞白）、碧眼大汉卓冲，对面为黑袍黄鲍（三角眼老者，化神中期）及其徒（麻子矮子，元婴修士）。
- **赌约缘由**：矮子向许仙子求亲，许仙子自恃冰晶剑奇寒，曾许诺只要对方本命法宝能硬接冰晶剑一击不被冰封即应亲；如今绿波刃被黄鲍以新得灵符花半月祭炼加持，确能接住，许仙子欲反悔避居于此，黄鲍以许家见证人相要挟。
- 韩立以明清灵目（瞳孔蓝芒）看穿绿波刃内同时凝聚强弱两种灵力，证实作弊；要求二人当面重演，否则逐出五十六小队驻地。

### 三、噬灵天火破敌（第1334章）
- 韩立虚空夺过许仙子本命法宝冰晶剑，以银色噬灵天火强行灌注：剑身化银白，浮现栩栩如生的展翅火鸟图案。噬灵天火本由太阴真火与太阳精火融合，能随意转化至寒至热，故可轻易融入飞剑。
- 许仙子重掌飞剑，催动化丈许银色火鸟，殿堂温度骤降、地面结雪白寒霜。
- 矮子绿波刃化独角绿色蛟龙迎战；火鸟一口吞去绿蛟大半，复化银濛冰雾将其冻成十余丈巨冰，绿蛟溃散现出黯淡飞刀。
- **心寒入体**：飞刀被冰封瞬间，一缕纤银芒（噬灵天火极寒之力）悄入飞刀，经心神联系直侵矮子体内；矮子虽距法宝二十余丈仍如僵尸倒地昏迷，身现乳白霜、面色发青将冻毙。此为韩立近期凭噬灵天火新能施展的诡异神通，传说唯灵界数种极寒之力可为。
- 黄鲍（化神）以红光火云融冰、塞火红药丸救徒，青色渐退；为掩饰噬灵天火真正神通，冰封实由冰晶剑本身寒力所致（已先削减飞刀加持威能）。
- 黄鲍忌惮韩立飞升修士身份，拱手认输、作罢求亲，抱徒离去。

### 四、后续
- 韩立向卓冲问出黄鲍来历：流沙谷小宗门修为最高长老、第二十七小队领队（非上古大宗）。
- 韩立警告许仙子：本队任务期间出手相助仅此一次（因念其为冰魄仙子后裔的情分），下不为例。
- 卓冲（镯兄）追问银焰寒火来历，许仙子点明韩立不喜被打听底细，劝其少问。
- 韩立化青虹赶往天渊城坊市，参加将于中午举行的拍卖会。
