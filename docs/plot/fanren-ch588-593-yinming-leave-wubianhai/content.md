# 凡人修仙传 第588-593章 剧情梳理

卷次:第四卷《风起海外》第588-592章 + 第五卷《名震一方》第593章。地点主线:阴冥之地 → 暴风山 → 空间裂缝 → 无边海 → 天南溪国。

## 第588章 降灵符
- 天符门第五十七代掌门(红脸老者『云某』,因门派衰微以筑基期修为忝居掌门之位)将镇派三灵符之一**降灵符**的制符口诀(刻满小字的数块骨片,装于骨盒)交予韩立,只求他若有机会顺路将口诀送回晋国华云州本门,免使绝学失传。
- 此符历来只准掌门修习;韩立查看骨片后发现其**主材料需化形期妖兽精魄**,极难炼制,故天符门虽世代相传却几乎无人真炼。降灵符属修仙界『密符』一类(类比虚天殿星宫的化身符、传送符)。
- 梅凝在场,首次听闻『降灵符』之名。随后二人离村,继续在阴冥之地各村落寻找梅凝兄长。

## 第589章 啖魂
- 阴冥之地红土村:村民猎杀可食用阴兽『碧蟾兽』,韩立以青竹蜂云剑(寸许青色小剑、剑柄缠兽筋)击毙两只逃兽,借机打听新人下落。
- 梅凝在村中仓库一眼认出兄长遗物(她亲手缝制的衣衫、带兄妹独门标记的储物袋),确认其兄已遭阴兽啃噬身亡,悲痛后强忍。
- 韩立决意收集**阴冥兽晶**(言出此地后再难寻得),放出**啼魂兽**(小猴)。啼魂兽幻化黄霞吞噬巨爪兽精魂,韩立以金蚨子母刃(银光短刀)剖取绿色兽晶。

## 第590章 暴风山
- 暴风山:阴冥之地东北角的通天巨山,光秃黑石、阴冥之风刺骨、电弧不绝,为强大阴兽至爱之地。
- 啼魂兽吸食大量阴兽精魂后**突变进化**:可幻化十丈银色巨猿,皮毛转为银白(按元瑶所赠祭炼玉简,真正啼魂兽并无此幻化之能、进阶应转漆黑)。韩立判断此乃半成品啼魂兽与阴冥精魂巧合所致的独特进化,乐见其继续(因真正祭炼之法过于血腥)。
- 韩立携梅凝凭**避风珠**抵御阴风、凭**婆罗珠**(青光护体)穿过怪雾,赶在下次裂缝开启前攀山。

## 第591章 阴阳轮回决
- 怪雾中遭遇**温天仁**(六道传人,自称结丹第一人、温少主)。韩立连发青竹蜂云剑,先被其碧绿内甲弹开,继以缠兽筋之飞剑自颈后洞穿将其斩杀,搜得储物袋、内甲及**四象蟠龙带**(镶避风、辟火、避水、避尘四珠,兼具安神定魂之效)。
- **紫灵仙子**(显露艳冠乱星海的绝色真容,平日伪装清秀)随后现身,曾因当日金焰中未助温天仁对付韩立而获其谅解,遂加入韩立、梅凝一行。
- 另一线:石洞中,**元瑶**因施展**还魂术**欲救**妍丽**而震碎金丹,还魂术又被鬼雾中断,妍丽仅余凝固精魂(化为黑气鬼影)。二人决意借阴冥浓郁阴气合修青阳门秘传**阴阳轮回决**(罕见人鬼合修之术,大成后将丧失重入轮回之资格);元瑶望借此重回结丹乃至凝结元婴。

## 第592章 脱离
- 暴风山顶平岩被上千只飞行阴兽围困。韩立分予二女红色小驽与兽皮软盾,自身同时操纵**十口青竹蜂云剑**结青色剑网;啼魂兽巨猿喷吐银光横扫阴兽。数番恶战后击杀全部飞兽、登顶。
- 二女在山巅静候裂缝期间义结金兰。空间裂缝(十余丈)于山顶上空数百丈处开启,三人神通骤复飞出,却遭巨大吸力欲拉回;韩立眼疾手快展**风雷翅**、发**雷遁之术**,一手一个抱住二女冲出黑雾,飞遁千里脱险。
- 落点为一片浑浊死寂、无风无浪的海面。

## 第593章 委身之举(第五卷)
- 韩立剑斩海兽,辨认出**馨牙兽**——乱星海所无、此地独有,据此判定此海为天南北端的**无边海**(几无人探得边际,鱼虾稀少、海岛罕有,无法定位)。三人决定一路向南。
- 飞行三四月终见陆地:抵达天南最北国家**溪国**(较越国小,辖七州),近海的**闵州**为最大州,其州府**封日城**。三人歇于闵州沿海小城酒楼。
- 韩立透露自身出身天南(原属越国)、当年被追杀逃入乱星海、与师门失联多年;赠二女自制的天南语言文字翻译玉简。
- 韩立自觉已达结丹后期顶峰,欲探明天南百余年变化后即觅洞府闭关,准备**凝结元婴**。
- 梅凝(筑基中期)自陈资质难成金丹,有意借双修给人作妾;紫灵(才结丹不久)反对其随意委身,转而撮合梅凝委身于稳妥可靠的韩立,雅间气氛暧昧。韩立未表态,梅凝默然不拒。
- 韩立于无边海中再忆与之有合体之缘的掩月宗女子**南宫婉**,心绪难明。

## 关键转折与状态
- 梅凝兄长确认身亡,梅凝彻底失去依靠,埋下委身韩立的伏笔。
- 啼魂兽完成独特进化(银猿、吞精魂),成为韩立横扫阴兽的主力。
- 温天仁(六道传人)死于本卷;韩立缴获四象蟠龙带、内甲等宝物。
- 韩立脱离阴冥之地、意外重返天南故土,修为逼近凝结元婴关口。
