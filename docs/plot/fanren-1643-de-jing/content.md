# 第1643章 得镜

第九卷 灵界百族。

## 战斗经过
- 承接前文：青色巨峰（魔蜂/巨蜂）元神别灭，远处魔蜂体表光芒一闪、噗噗炸响；魔鹰亦魂飞魄散坠落。三只高阶魔兽中只剩**双角魔**一只。
- 双角魔目睹同伴被韩立一个照面间干净利落击杀（连元神都未逃出），通体发寒。它心机了得，盘算独自上前必陨落，欲等援兵、或冒违反『山脉与外界约定』之险埋伏入口击杀韩立。
- 双角魔收回两柄铁锤，喷出**三角铜镜**，镜面射出寒光将自身一罩，连人带镜化作残影闪入**虚空**遁走。
- 韩立**瞳孔蓝芒一闪**（天眼术）现漩涡奇景，看出端倪；掐诀于眉宇间点出一团黑气，凝成漆黑如墨的眼珠——**破灭法目**。妖目射出一道手指粗的乌光没入虚空。
- 百余丈外虚空巨响，空间剧烈波动，双角魔被乌光逼出、惊怒现身。

## 击杀
- 韩立头顶青白电弧浮现人影，施**风雷翅**：电光化为银灿灿光弧，体形狂涨化为数丈青色大鹏（雷鹏），双翅带碗口粗银弧扑下。
- 巨爪将双角魔两只黑色巨锤纸糊般抓碎，又抓向其蓝色幡旗化成的水晶光罩（电弧巨力一度泥牛入海般失效）。
- 青鹏钩嘴化青芒连啄，光罩从同一点寸寸碎裂溃散；再啄破其黑色爪网，青芒穿透双角魔胸膛，留下碗口大血洞，内有银芒闪动。
- 血洞中银弧爆裂，将双角魔肉身连元神彻底淹没，化为飞灰。原地只留下那面三角形怪异铜镜。

## 得宝
- 韩立化回人形，以吸力收取无主铜镜，匆匆未看出名堂，但颇感兴趣。
- 评估：此镜能将多人法力联结一气，还能带人遁入虚空中逃走，绝非普通宝物，不知双角魔从何得来。
- 韩立为速杀对手曾一口气祭出太一化清符与两道甲元符，得此宝不算吃亏；随即化青虹遁入雾海深处。

## 后续：魔兽追兵
- 数刻钟后，黑濛濛魔风带来五头奇形魔兽（含琉璃小兽、蟒首魔兽、虎形魔兽、生白色独角的鸭形怪兽等）。
- 由琉璃小兽统领，确认『五泣大人他们三人』最后消失于此。鸭形魔兽潜入雾海下方，叼出两截拳头大小的残缺牛角——正是五泣大人头上灵角，证实五泣三人确已陨落。
- 小兽冷哼：对方能杀五泣三人，己方追上去多半也不是对手，下令原路退回；有魔兽顾忌回去后『主人』必然震怒、不好交代。

## 要点
- 三角铜镜：联结多人法力 + 携人遁入虚空，来历不明，现归韩立。
- 韩立本章使用法宝/神通：天眼术、破灭法目、风雷翅（雷鹏）、太一化清符、甲元符。

## 第1644章 巧获 / 第1645章 大大出（节选梳理）

### 一、脱困与独行
- 琉璃小兽带领蟒首魔兽等五魔决定回到魔金山脉出口处埋伏等韩立，魔风一起原路返回。
- 此时韩立已在百万里之外飞遁，成功摆脱追兵但神色不轻松：大半日逃遁已完全偏离原路线，回出口处只能另走生路（道路不熟、恐遇新魔兽）。
- 韩立判断：只要没有圣阶魔兽，凭【太一化清符】或自身诡异遁速都能轻易脱身，魔兽堵口之事不放在心上。
- 牵挂之事：不知【纤纤】能否顺利脱身——若此女陨落，【天外魔甲】便一时无法修补。对大敌【圭姓男子】，韩立则巴不得其陨落于魔兽群中。
- 韩立独自飞行一日一夜，途中数头低阶魔兽被大手虚空一抓击杀。

### 二、截获芝仙
- 飞至一片被黑色魔气淹没的丘陵上空时，一侧天空惊天巨响，赤红火云中白炙火球爆裂、又有黑色水浪冲击火云，两者属性相克、声势惊人，绝非普通炼虚所为。韩立本不想节外生枝，加速前行。
- 火云方向忽射来四道遁光：最前一团紫光直奔韩立瞬移而来，后随绿、银、血三道遁光。
- 韩立以灰光护身、青色小剑化青莲，并以天眼（蓝芒）认出紫光中是【芝仙】——脸孔翠绿、紫色根须四肢、套枯木黄甲、体表紫色符文、带药香；其四肢尚未幻化成功，显然负伤未愈被人发现而逃窜。
- 韩立单手虚空一抓，以【元磁神光】灰色大手擒住芝仙，对方毫无反抗之力；随即十几道符箓封禁，再以一只【青色玉瓶】放出青霞，将芝仙缩成半尺收入瓶中。

### 三、三炼虚围攻
- 后三道遁光现出三人，距三十丈将韩立团团围住。三人此前曾在【雷云阁】见过韩立。
  - 【彦姓老者】：脸色苍白、灰白头发。持银色印玺（发风雷之音），后吐乳白色轻纱化白云。
  - 【绿发异族】：满头绿发、凶恶；持古色古香小鼎，放出阴黄风龙。
  - 【血色宫装女子】：黛眉樱口、血色宫装；持白玉雕成、铭血色符文的【血扇】，并使两口血色飞剑施展『剑灵化虚』（被缚后自爆、于他处重新凝聚）。
- 对话：彦姓老者询问同行的『越贤侄』，韩立告知【越宗】已遭人毒手陨落、遇敌时身边无人故无法说清；老者脸色阴沉、疑心韩立。三人最终决定联手击杀韩立、再平分芝仙（女子言魔鳄与铁翅手下将追来，需速战）。

### 四、混战
- 韩立先发制人，召出黑色小山虚影（【元磁神山】，经二次炼制）涨至数十丈砸向老者，压碎其银色印玺；老者以乳白轻纱化白云、卸去大半力道托住山峰。
- 绿发异族小鼎放出黄色风龙；韩立祭出【虚天宝鼎】（青色小鼎），放出巨兽怪鸟虚影迎战风龙，撕咬难分。
- 血色女子血扇喷出无数血焰火球；韩立放出一只尺许【银色火鸟】（即【噬灵火鸟】，吞噬【金乌真火】后进化）。火鸟骤涨至三十丈，银羽带白、似银色凤凰，激射银羽弩矢洞穿火球；女子以血色光柱催扇令血焰重聚，但洞穿的银羽燃成银色火团将血焰尽数吞噬，漫天血焰消失大半。女子脸色苍白无血。

### 备注
- 据【世界状态须知】，越宗已于第1615章死亡；本章为韩立向他人转述其陨落。

## 背景
灵界百族篇，三名炼虚顶阶修士（绿发异族人、血色宫装女子『血瑛』、彦姓老者）联手围攻韩立，争夺其手中『芝仙』灵物。

## 第1646章 击杀绿发
- **血瑛血扇/血葫芦**：血色宫装女子放出血焰（歹毒阴火），被韩立的乾蓝冰焰（银色火鸟/银焰）一举吞噬，玉血扇心神受创碎裂，女子喷血。她再祭血红葫芦放出阴寒血水（含黑气冰寒），与韩立的金乌真火所化乳白白霞（噬灵火鸟）相持。
- **韩立选定目标**：三人皆炼虚顶阶，彦姓老者气息最弱、血瑛最强、绿发异族居中。韩立决定先杀居中的绿发异族以破联手之势。
- **击杀手段**：韩立祭风雷翅化青白电弧瞬移，足下青莲化百道青光（青竹蜂云剑剑光）分袭老者与女子使其自顾不暇；以五色光焰凝『五色大手』擒住绿发异族的上古破法铜戈（专擅破法灭功，却只破开大手表皮一分），再合指抓死铜戈；虚天鼎放出青丝缠住绿发异族（绿火亦烧不断），韩立弹指鸣鼎令青丝勒紧数倍；五色大手下压拍爆其头颅。
- **斩尽残魂**：绿发头颅中冲出绿火（小人模样元神）瞬移逃逸，被韩立飞剑一斩两半，化绿光消散。绿发异族彻底死亡。
- 老者与女子见状惊惧，不敢妄动。远处黑水火云（铁翅魔、魔鳄）分两半逼近。韩立收宝。

## 第1647章 大敌追至
- 韩立收回飞剑、剑光、虚天鼎及绿发异族遗落的黄色小鼎（铜戈所属），驾风雷翅化电弧远遁。
- 彦姓老者驾白云霞光向另一方向逃走（遁速不逊雷遁）。
- **黑水火云的真相**：火云中巨蜍是『铁翅魔』（圣族圣阶存在）附身的一具化身，肌肤赤红、双目紧闭、脑门灰色拳头大圆珠（魔核）。黑水中是『魔鳄』，刚进阶圣阶、境界未稳。
- **二魔对话**：铁翅魔逼问血瑛是否得『芝仙』，血瑛恭敬否认（自称侄女，称对方『铁伯父』，并提『血臂老儿』为同党）。魔鳄索要先前约定的『璃水珠』（助夺芝仙的报酬）。二魔一言不合欲火拼，铁翅魔强压怒气。最终议定分头行动：魔鳄追彦姓老者，铁翅魔追韩立。
- 血瑛独留，自语魔鳄竟也进阶圣阶，须速禀父亲，盼『姓韩的外来人』带走芝仙脱身，化血虹离去。

## 第1648章 元剑再现
- **巨蜍逼近**：吞尽漫天火云后化作赤红烙铁/火红骄阳，数里高温。韩立拍天灵盖放乾蓝冰焰（银焰）护体，方圆十余丈转凉。
- **祭雷术虚晃**：韩立布金色电弧圆网，投入金光符文凝成数丈金色光晕，喝『辟邪神雷，祭雷术！』凝出黯淡金球。巨蜍认出辟邪神雷，惊惧弃术化火柱扑下；韩立金球虚晃即收，退入青莲（春黎剑阵）消失。
- **诱入剑阵**：火柱撞溃青莲后赤焰被吞，巨蜍现身火海。韩立宣布启动『春黎剑阵』，百余丈青濛光幕罩下。巨蜍红光柱、赤红巨爪（带红芒，能抓破光幕）猛攻，撕开光幕现深黑虚空；巨爪裂出十余丈红晕灭青莲，被韩立乾蓝冰焰银色火鸟扎入吞噬缩灭。
- **巨蜍蓄秘术**：背后黑色符文翅翼扇出黑魔风，与身上火焰相融成黑红风火之柱；口喷紫濛飞剑化无数薄刃没入风柱，风火柱浮现紫芒飞速变粗。
- **元气之剑**：韩立另一边以神念调集附近天地元气（五色光点遍布天空）注入春黎剑阵，光幕中倒吊生出藤蔓植物，开花结翠绿小葫芦，青莲喷剑影没入凝成『元气之剑』（春黎剑阵杀手锏，需海量天地元气）。对方专注蓄术未夺元气控制权，元气之剑顺利凝成并持续增威。
- **斩**：韩立一声『斩』，葫芦倒转喷出尺许长、五色灵光闪动的光剑。（章末，胜负未揭）
