# 第1544-1545章 灵漩邪光 / 斩杀

## 背景
- 地点：火阳族大殿。韩立以临时请来的外援「韩生生（被称韩前辈/韩兄）」身份出手，对付来犯的乌罗族。
- 据世界状态须知：本章「韩立」即韩立本人（非冒名片段，啼魂化身打坐在别处），正常归属于韩立。

## 灵漩邪光（漩灵邪光）登场
- 火阳族众人（妇人手持火蛟宝物、四名白袍蛇人祭祀的银色刀芒、密集光矢）联手攻击两名乌罗人，被其黑色光晕「化二为一」尽数吸入。
- 火阳族妇人惊呼对方「修炼成了此神通」，自言其余两族高层正因此邪光被轻易灭杀。
- **邪光三大特性**：① 能反弹攻击；② 无视大多数神通禁制；③ 奇毒无比，中者邪光溃散到肉身各处无法驱除。

## 韩立救白珠儿
- 乌罗人射出两根金银细丝追杀妇人与少女白珠儿。
- 白珠儿（金丹期）以黄色大弓发骨箭化黄白光柱迎击，反被细丝反弹倒射，大弓被击飞。
- 韩立连续两次青光瞬移、抓其后颈衣领救走。
- 元磁神光（灰濛霞光）、五色寒焰（蓝色乾蓝冰焰系）皆无法困住金银细丝；细丝甚至瞬移到韩立面前。
- 韩立喷出银色火球（噬灵天火），化为**银色火鸟**，将金银细丝缩成鸡蛋大并吞噬消化，再瞬移至大殿另一侧吞掉追杀妇人的另一根细丝，随即自爆消失。
- 妇人提醒邪光奇毒，担心火鸟吸毒留后遗症；韩立心知噬灵天火能将此毒化为己用，不以为意。

## 斩杀两名乌罗王族（第1545章）
- 韩立显露金色鳞甲，头顶浮现金色光晕，中心盘坐**三头六臂金色法相**（三面中两面与韩立面容一般无二，近似实体）。
- 黑色光晕化为七八丈黑色巨刃斩来 → 法相伸金色大手硬架住。
- **金瞳乌罗人**化巨大身躯、尾部生金色巨钩扎来 → 韩立放数十口青色小剑（青竹蜂云剑）化青丝斩碎金钩。
- **银瞳乌罗人**召出人面蜈蚣虚影喷银霞 → 韩立以灰色光幕（元磁神光）挡下，掷出黑色小山（**元磁神山**，溶入古怪石墩、分量比拟万丈巨山），瞬移压住蜈蚣；银瞳乌罗人喷淡银精血增援，韩立念「重」字使山再增重十倍，将蜈蚣压入大殿三十丈大坑、塌陷丈许，蜈蚣溃灭，银瞳乌罗人受反噬喷黑红血。
- 二人欲以黑色光晕遮护逃遁 → 韩立催动「**剑灵化虚**」神通（提炼飞剑后所得），使青色剑影虚化无声穿透黑色光晕禁制，将两名乌罗人洞穿成千疮百孔的残尸。
- 韩立瞳孔蓝芒一闪（天眼术）识破二人元神逃逸，双手探虚空抓出**金、银两条数寸人面蜈蚣**（头生女性面孔，即二人元神），以银色火焰（噬灵天火）困住搓灭。

## 战后
- 韩立判定二人仅化神初期修为，因灵漩邪光古怪才未被弹指灭杀；在意「无法以神念直接探查其修为」。
- 从两具残尸取走**两块淡黑色纱巾**，低首看后神色一动收起（未对火阳族提及）。
- 妇人、白珠儿、蛇人祭祀皆惊喜恭敬称谢，誓世代不忘救命之恩。
- 韩立答应一并帮火阳族打发大殿外其他乌罗人。

## 实体归属备注
- 「韩生生」为韩立临时使用的化名/泛称，无具体被冒名对象，不抽 impersonates。
- 「韩前辈/韩兄/韩生生」言行均归韩立。
