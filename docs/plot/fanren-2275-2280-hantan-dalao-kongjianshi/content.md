# 凡人修仙传 第2294-2296章｜第十一卷 真仙降世
（第2294章 打捞 / 第2295章 寒潭之宝 / 第2296章 重回）

## 背景
地点为**小修罗界**——一处物产与灵界颇为相异、容不下太多大乘存在的小界面。修罗蛛族欲取出深藏**寒潭**底部的宝物。

## 寒潭与法阵
- 修罗蛛族母命血裙少女（樱儿）指挥族人修改法阵，圆柱状冰台寸寸碎裂，露出方圆十余丈的迷你水潭，潭口白濛濛光柱冲天，被淡蓝色光幕罩住。
- 潭中奇寒**每深入一丈增强一分**，深不可测；需四件**纯阳之宝**（如意、镜子、晶球、圆环，散发炙热纯阳气息）压制寒气。
- 此族阵法师耗数百年构思的法阵，需**一瞬间六名以上大乘**同时注入法力方能催动，将潭底宝物直接摄出。此前因凑不齐大乘而搁置。

## 凑齐六大乘
- 在场外来大乘：韩立、血燃、黑鳞、莫简离四人，加修罗蛛族母仅五人。
- 血裙少女自陈有一种**以时空之力为基础**的秘术，可临时提升境界一截，勉强算作一位大乘。韩立由此察觉其涉及时空法则／时空神通。
- 六人分立六根石柱注入法力，族人于阵缘催动法器、注力，凝出符文之球。

## 打捞过程
- 修罗蛛族母喷出青色丝网化入符文球，再以五根晶丝下探寒潭。下沉至千丈后极缓，每息仅数寸。
- 危急时她拍出**本命晶核**（一尺许碧玉蜘蛛，两侧复眼紫焰闪动，吐出包裹白色晶核的青光），血裙少女亦喷银光相助，晶丝终触底。
- 水面形成漆黑漩涡，硬生生拉出一颗人头大小的**淡蓝色石头**，散发强烈**空间之力／空间法则**波动。黑鳞贪婪，血燃亦目露火热。
- 韩立因自身可幻化擅长空间之力的**天凤**，对此石并不在意。血裙少女抢先以晶丝裹走石头；修罗蛛族母以阴冷目光警告血燃，血燃遂罢手。

## 报酬与交易
- 血裙少女履约抛出白玉盒，内含**四枚晶核**分予四人；另有一张**可消减雷劫之力的雷符**（血燃认为比空间之石更重要）。
- 事后众人于邻近小岛办**临时小交易会**，互换修罗界特有珍稀资源，韩立、莫简离、血燃、黑鳞均有收获。
- 血燃以一皮袋换取修罗蛛族母标注**车泣子**（传说上古凶兽，莫简离曾险些栽其手中）巢穴位置的地图玉简。

## 散场与韩立暗返
- 修罗蛛族母提醒此界尚有两三个难缠存在，众人勿大意；族人乘黑色巨鹰离去。
- 血燃曾试探韩立是否要中途反悔截下修罗蛛与空间之石，韩立明确表示对空间之力不感兴趣、不愿冒险，血燃黑鳞遂破空离去。
- 莫简离仅得三枚晶核已属侥幸，不再冒险，欲寻隐秘处打坐静待离界，与韩立约定数日后灵界再见后离去。
- 韩立目光暗扫寒潭方向（举动隐秘），待众人远去后独自化青光重返寒潭巨岛。潭口因法阵撤去，已再次被一团玄冰冰封。
- 韩立放出一团银焰化为**银色火鸟**，催动后涨至十几丈，但神态萎靡——因先前与某盏古灯召出的**紫色孔雀**一战消耗甚大。韩立喷出数团**青濛濛精气**喂入火鸟身躯恢复之。

## 关键人物动向
- 韩立：化名/真实身份均为灵界外来大乘；隐藏对寒潭的真正图谋，暗中折返。
- 修罗蛛族母（罗道友）与女儿血裙少女（樱儿）：达成目的、取得空间之石，主导交易。
- 血燃、黑鳞：得晶核与雷符、车泣子巢穴地图后离去。
- 莫简离：得三枚晶核，谨慎避战。

## 章节梗概（第2297-2299章「真仙降世」卷）

### 一、噬灵真火吞寒潭，意外得山海珠（2297）
- 韩立驱使噬灵火鸟（噬灵真火所化）喷银焰，一个时辰内将寒潭奇寒之气吞噬殆尽，寒潭复为普通水潭。
- 噬灵真火本为阴阳合一的变异真焰，经多次借外物炼化，可吞噬至寒至阳之物以增威。吸取寒潭惊人寒气后，噬灵火鸟趋于晶莹实体形态，产生新一轮异变（此变化仅是开端，寒气需日后慢慢炼化）。
- 火鸟从潭底带出一颗鸡蛋大、奇寒无比的晶球，神念无法探入。韩立以晶化神念探查、再以金弧（辟邪神雷）击碎外壳，露出内裹拇指大小、蕴含强烈空间之力的淡蓝色圆珠。
- 韩立推断寒潭圣物实为一大一小两件：大者为修罗蛛族母带走的淡蓝色石头，小者即此晶珠；后者特意以宝物遮掩、能瞒过大乘神念，在空鱼族眼中应更重要。
- 韩立以贴满禁制符箓的玉盒封存晶珠，遁向空鱼族所在的乾火之地。

### 二、修罗蛛母女的盘算
- 修罗蛛族母担心空鱼族存在已被外人知晓。血裙少女表示已派人监视、任空鱼族自生自灭即可。
- 空鱼族晶核虽可提炼空间之力，但该支已退化，成年体不过炼虚，无法提炼真正空间法则。血裙少女称自己参悟空间/时空法则走的是「另一条打破常规的路」，并非靠吞噬晶核；现有储备的晶核已够。
- 计划：回去后闭关炼化「那人的元婴」，深参时空法则，出关后分化一丝时空法则种子交予族母，使母女皆有掌控顶尖法则之力的可能。族母称少女若进阶大乘，配时空之力可跻身真灵顶阶，除真龙天凤一类难逢敌手。

### 三、乾火之地见空鱼族（2298）
- 韩立返回熔岩湖，见数具似狼怪兽残尸被利刃切成七八截（修罗蛛探子）。预留于此的噬金虫王「金童」（紫金小人形态）回报详情，韩立以火球焚尸灭迹。
- 韩立催动山海珠的空间法则波动，引出数名半人半鱼的空鱼族人，其中修为最高的壮汉首领仅炼虚境界。韩立以「避火术」分开滚滚熔岩、下沉三四百丈，穿过蔚蓝光幕进入空鱼族地下聚居空间（方圆十几里、上百座三角形建筑、刻原始弧形文字），全族顶多五六百人，老幼妇孺居多，族势衰微。
- 壮汉奉族长令引韩立至天域厅。

### 四、族长流烁认主之约（2299）
- 空鱼族族长流烁，灰白头发、下半身人形、合体初期修为。一见韩立即跪求收留，愿全族世代侍奉，只求脱离修罗蛛魔爪。
- 韩立出示淡蓝晶珠，流烁认出此乃本族圣物「山海珠」，并亮出手腕「圣痕」——历代族长信物，可感应百里内的山海珠（故一见韩立便愿认主）。
- **山海珠**：除蕴含空间法则外，本身是一件罕见的洞天之宝，内部空间据说可装巨山大海，面积在顶尖洞天之宝中名列前茅。多年前族势衰弱时被先祖以秘术封印于寒潭，遗训「族不再强盛便不可取出」，以免重宝招祸。
- **镂灵石**：修罗蛛族母所得的另一件圣物，仅可辅助强大存在参悟空间之力，别无他用。
- 韩立顾虑难带众多族人垮界返回；流烁献策：只要炼化山海珠，全族可躲入珠内随其离去。流烁承诺献全族精血助抹去先祖痕迹、传授专门祭炼法诀（需炼化者为大乘强者方能压下反噬），两三天即可炼化成功。修罗蛛因故难离此界，不足为虑。
- 韩立应允：只要能炼化此洞天之宝，有生之年永远庇护空鱼一族。流烁狂喜，即动员全族割腕滴精血入青铜巨鼎、布银光法阵开始祭炼准备。

### 五、尾声伏笔
- 灵界血天大陆某地下上古祭坛，五色光阵浮现，一道通体晶莹的身影自仙界降临，叹「灵气好稀薄，果然无法和仙界相比」。

## 关键设定/物品
- 噬灵真火：阴阳合一的变异真焰，可炼化吞噬至寒至阳之物以增威，化形为噬灵火鸟。
- 山海珠：空鱼族圣物，洞天之宝，内含强烈空间法则。
- 镂灵石：空鱼族另一圣物，辅助参悟空间之力。
