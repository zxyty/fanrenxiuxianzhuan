# 第757章 冰雪蚕与庚精

## 场景
黑袍大汉单独唤走云露老魔；至阳上人、魏无涯留下与韩立、白姓妇人在大殿中谈交易与赌战。

## 白姓妇人与青王蚕
- 白姓妇人态度冷淡，直言此行若非至阳上人说手里有'那东西'，绝不出山；与至阳上人有旧日纠葛（当年之事）。
- 至阳上人取出白玉盒，盒现时整间大厅温度骤降如严寒；盒中为一对通体晶莹半透明的蚕蛹，散发白气。
- 韩立辨出此为奇虫榜排名靠后的'冰雪蚕'之蛹；此奇虫对敌威力不大，却有不可思议神通，极难寻觅。
- 至阳上人称这并非普通冰雪蚕，而是雪蚕极品'青王蚕'，得自寒冰洞百丈冰层之下；蛹内有异常，孵化后即便幼虫也够白姓妇人所用。
- 白姓妇人以神识查验后收下玉盒，答应参加赌战。至阳上人称以其'云雪决'神通对付区区法士必旗开得胜。妇人随即冷漠告辞离去。

## 韩立换庚精
- 韩立向至阳上人索取庚精，称越多越好，愿以等价之物相换。
- 魏无涯点明韩立以元婴初期修为竟能从慕兰神师手中安然脱逃，非寻常修士可比。
- 至阳上人先取出三块淡金色庚精（一大两小，大者核桃般大、小者其半），约够掺入十几口飞剑；韩立面露失望。
- 魏无涯再取出一块拳头大的庚精。韩立大喜：凑齐后大有希望够炼制三十六口飞剑，再配合剑影分光术，便可勉强布下小型大庚剑阵（七十二口仍不足）。

## 交换之物
- 韩立给出两玉盒：一为众多鲜红指甲大鳞片——**八级妖兽毒蛟的鳞片**（赠魏无涯）；一为拳头大乌黑龟壳——**八级龟妖的龟壳**（赠至阳上人）。
- 八级妖兽在乱星海亦极罕见，天南更几近绝迹；化形妖物本体材料在天南难觅。
- 魏无涯修炼毒功神通，毒蛟奇毒无比，其鳞片对他另有妙用，故失声惊喜、患得患失地观察血鳞。
- 至阳上人称龟妖龟壳是炼制战甲、盾牌等法宝的最佳材料，但仍将信将疑，需先鉴别真伪——难信一元婴初期修士一出手便是八级妖兽材料。

## 背景
承接换宝之局：韩立（化名行走外海期间之外，本章以本名/落云宗长老身份示人）以八级妖兽材料与两位元婴老怪交易，并被推上七日后的天南赌战。

## 第758章 玄天仙藤（交易收尾）
- **毒蛟鳞片换庚精**：魏无涯确认收下的是化形毒蛟蜕换的血红鳞片，正好用于增强其法宝；作为回礼，他额外掏出最后一块鸡蛋大小的庚精抛给韩立。
  - 韩立心喜：有此庚精，三十六口飞剑可同时炼制且绰绰有余，先前拿出稀有八级材料的用意大半实现。
  - 韩立深意（无需明言）：以重材逼对方再掏庚精，并换取参加赌战的合理性。
- **玄龟龟壳换玄天仙藤根茎**：至阳上人鉴别龟壳满意后，见韩立木灵气旺盛、主修木属性功法，遂赠一截拇指粗、五六寸长的淡黄根茎，称是玄天仙藤根茎的四分之一。
  - 魏无涯认出此物并愕然——因他也持有一截（仙藤枝干部分），系九国盟某修士从正魔手中换得后落入其手。
- **玄天仙藤设定**：
  - “仙藤”泛指有大用的藤蔓类灵根异种；唯最古老的几种才配冠以“玄天”二字。
  - 玄天仙藤为开天辟地、混沌初生时出现的上古灵根，其灵花灵果具无视此界天地法则的神通，乃逆天级存在；具体种类与神通早已失传。
  - 正魔两道是从上古修士洞窟遗言中辨出这看似普通的枯死藤蔓即玄天仙藤，曾为争抢火拼数场。
  - 高层决定先救活：尝试奇功秘术、灵水灵液浇灌，乃至打落云宗等三派圣树醇液的主意，均无复活迹象；遂将其连根带枝分成十几截平分。
  - 死藤顶多作顶阶炼器材料；木属性对修火功的至阳上人鸡肋，故换予韩立。
- 韩立本只缺庚精（不缺顶阶材料，本命法宝青竹蜂云剑都无暇温养），初有失望；但打定主意回去用**神秘小瓶绿液**催化此根茎，盼能救活（绿液曾催熟三大神木）。
- 韩立应下七日后赌战后告辞。临行魏无涯传音：看在毒蛟鳞片面子上，**不再追究韩立带走南宫道友之事**，其侄儿与南宫仙子之事作罢——韩立暗喜。
- 二老议韩立：疑其当年经古传送阵消失期间得上古修士完整衣钵，方能短期进阶元婴并多宝。至阳上人忌惮其能从**慕兰神师**手中逃脱的神通，且其为**天道盟**之人（龙晗夫妇不好惹），故不愿杀人夺宝。

## 第759章 灵符与蛟魂（备战）
- **遇董璇儿**：殿门外韩立撞见董璇儿，对方神情古怪、略带嫉恨；韩立视若不见离去。银月在脑海打趣此女与韩立关系不一般，韩立称只是当年关系不好、曾敌对过的旧识。
  - 自韩立救下银月并未追问其紫铖兜、花篮古宝被夺之事后，银月对韩立亲近、敞开部分心扉，二人互信改善。
- **告知吕洛**：韩立向吕洛说明赌战之事；吕洛劝阻，韩立称有八成把握全身而退，并托吕洛照应大战时门下弟子安危。
- **备降灵符**：韩立交吕洛一玉简，列出炼制降灵符的辅助材料；落云宗弟子两日内凑齐。
  - 降灵符是数日内唯一能成的厉害杀招（重炼飞剑、修大庚剑阵皆来不及）；顾虑在成功率（缺满意灵符笔与练习），仍决定一试。
- **炼制过程**：
  - 入密室，开墨绿色小瓶，射出血红光团；以青光大手擒住，现出数寸血色小蛟（毒蛟精魂/蛟魂）。
  - 以明清灵眼（瞳孔蓝芒）查看：蛟魂魂力依旧强大，但蛟目神光散而不聚、三分痴呆，灵识已消失、神智退化如野兽（落入小瓶数年所致）。韩立由此警惕修仙路坎坷，恐己身殒后元神落敌手下场更惨。
  - 取一块翠绿稀有灵皮，喷青色婴火裹之而不燃，掐诀打入法决；引入银粉，灵皮化银色光莲转动；陆续投入各种材料，火色由银→黑→绿→血红。
  - 念咒、射青丝牵引灵皮诡异晃动，逼出黑气（黑气遇血红婴火即灭、留腥臭），灵皮缩小；持续一个多时辰炼尽黑气，灵皮缩成巴掌大。
  - 添料再炼后吹熄火焰，灵皮变血红色。
  - 最后将头顶蛟魂引下，蛟魂没入血红灵皮表面——降灵符核心成形。

## 章节概要（第五卷 名震一方 第760章 大战将至）

### 一、韩立密室炼制降灵符
- 韩立取出蓝濛濛、六七寸长的小巧**灵符笔（制符笔）**，蘸取绿色玉盒中的金色液体，对准漂浮的**血红色灵皮**虚空点画。
- 金色符文没入灵皮，灵皮浮现整齐符文；血红**蛟魂（毒蛟精魂）**欲冲出逃匿，被活过来的金符紧紧包住拉回灵皮中。
- 韩立随后不断更换紫色等各色丹砂似的灵液，画出灵纹、皱文等古怪符号，巴掌大灵皮上闪着密密麻麻各色光泽；他神色凝重，催使**明清灵眼（天眼术）**至极致。
- 一红色符文画完后，半成灵符灵气波动狂暴，韩立面色大变，急以青霞收起材料，并放出**蓝色小盾**挡身前。灵皮在龙吟中爆裂，五色光团扑向蓝盾，大盾纹丝不动轻松挡下——韩立大出意外（按降灵符威力，失败爆炸本应破坏力不小）。

### 二、失败后的关键发现
- 爆裂后一道血光欲遁出密室，被屋顶白色光幕反弹现形，正是毒蛟精魂；韩立以青色大手将其擒回。
- 再用明清灵眼查看：精魂尚在，但魂力明显衰弱、萎靡不振。
- **结论**：降灵符炼制失败时妖魂不立刻崩溃，只损部分魂力，故可在魂力耗尽前反复炼制、大有成功机会；但重复过多则魂力大减、符威大打折扣。韩立视为不幸中的大幸，换上另一块灵兽皮重新开炼。

### 三、天一城总动员·天南大战将至
- 两三日内全城修士与大小宗门皆知数日后将与法士（慕兰人）生死大战，组成法士大军开往边界。此战决定慕兰兵败而退、还是天南落入法士之手致无数宗门失传，士气高昂。
- **十场赌战**谣言满天飞：为防慕兰奸细泄露应战名单，十名元婴修士身份保密（除天殿议老怪外几无人知），韩立等名字仍在保密中；上千名修士生还系于此。
- 落云宗弟子随队离城，**吕洛**不得不随行，吩咐**宋姓女子**留驻地等韩立出关。

### 四、韩立出关赴战
- 韩立闭关处隐传龙吟、爆裂之声，宋姓女子又惊又忧，至第五日决意若再不出关便强行叩关。傍晚钟声响，韩立于青光中现身出关。
- 韩立得知是第五日，称时间绰绰有余，仍宜早到。放出**御风车**（白色带翅四方车、专门飞行法器，耗灵力少、遁速远超常人），携宋姓女子破空飞向边界。
- 途中问及**紫灵**，宋姓女子答其已暂时离开天一城，待大战后再定打算；韩立淡笑赞其机灵明智，催车遁速骤增消失天际。

## 第761章 萧师侄

### 战前布阵
- 虞国、北凉国某段边界，天南修士与慕兰法士分从两侧扎下大营，正奇大阵、御敌禁制遍布两侧，杀气腾腾。
- 边界中心巡视双方虽冷冷相望但暂未动手（动手于大局无用）。第六日大阵布置将毕，众人深入简出蓄锐，高阶修仙者频繁出现于巡视队伍。

### 韩立重逢萧翠儿
- 一队黄枫谷修士在大营外巡视，为首为结丹初期美貌少妇——萧翠儿。她感应到一道极快遁光（韩立所乘御风车/白色飞车，同行有宋姓女子）。
- 萧翠儿自报『黄枫谷萧翠儿』验问身份，认出青袍男子为『韩师叔』韩立，惊喜不已（聂盈、雷万鹤等曾告知韩立进阶元婴之事）。
- 旧事回顾：萧翠儿当年由韩立一手引入黄枫谷，是马师兄唯一亲传弟子；与小老头惊险逃脱魔道追杀返谷，马师兄已坐化，她机缘巧合结成金丹。
- 萧翠儿代雷万鹤、聂盈等游说韩立重返黄枫谷（黄枫谷仅元婴中期令狐老祖坐镇，后继无人，地位堪忧）；韩立婉拒，称落云宗待其不薄。
- 韩立宽慰：曾与令狐老怪（令狐老祖）做过交易，黄枫谷真出现危机会在一定范围内出手相助。
- 韩立念昔日香火情，赠萧翠儿：①一枚碧绿指环（昔日灭杀其他结丹修士所得法宝，非本命法宝但炼化后对敌有妙用）；②一瓶丹药（助修为精进、突破瓶颈）。
- 韩立告辞，赴大营见至阳上人等三大修士，听其介绍大战安排后告退，分得独立石屋打坐养锐。

## 第762章 敌谋

### 韩立修炼近况
- 青元剑诀修至第十层后灵力回复大增。
- 取出黑木匣，内有一张血色小蛟游动的『降灵符』：花数日、连失败三次方炼成，威力较炼制之法所述降低不少，所用八级蛟魂已损大半魂力，真正效力待实战测试。

### 法士与魔宗地下密谋
地点：离天南大营数十里、法士驻地地下二三十丈石室。与会七人（五男两女，一男一女黑袍）：
- **慕兰三神师**：祝神师（枯瘦干瘦老者，主事）、仲神师（仲姓中年儒生，研究数百年灵术法阵、新改进数种大阵）、乐上师（乐姓绿衫女子，掌圣灯灯油）；另有毕姓矮子（身高不到四尺）、锦衣大汉、矮子房宗主等。
- **阴罗宗**：黑袍男子房宗主、声音粗哑的黑袍女子——大晋十大魔宗之一。

密谋要点：
- 乐上师携慕兰族储存千余年的全部灯油参战；圣灯点燃时长关系圣禽在此界出现时间长短。
- 法士法器数量质量逊于天南修士，靠灵术大阵压制；混战开始先令高阶法士召虚灵兽、配合巨兽吸引消耗修士法力。
- 阴罗宗因镇宗之宝『鬼罗幡』被正道击毁数杆、急需大量生魂修复，方掺和此战。交易：战后所有战死修仙者生魂归阴罗宗（法士生魂须战后秘密拘走，不得被人看见）。阴罗宗将带十几头巨兽及铜甲炼尸（铜甲尸兵）助阵。
- 慕兰承诺夺得天南后，几个凡人国家（皆燕族人）交阴罗宗管理。

### 盯上韩立
- 毕姓矮子提及阴罗宗一长老被同阶天南修士瞬间灭杀，警告勿轻视。
- 黑袍女子提出索要那名『会放金弧的修士』交阴罗宗处理。祝神师识破其牵涉『金雷竹法宝』与『辟邪神雷』（修仙界三大神木相关异宝），不肯轻易答应。
- 房宗主明言魔宗决不能让辟邪神雷落入他人之手，要价：慕兰若要阴罗宗关键时放出铜甲尸兵相助，则那名驱使辟邪神雷的天南修士须由慕兰活捉交予阴罗宗。
