# 第1317章 莫名麻烦

## 一、韩立激怒翁姓汉子
- 韩立以化神初期修为、且一直示弱，却突然对吊眉汉子说出强硬之语，连面无表情的紫袍大汉都为之诧异。
- 翁姓吊眉汉子煞气一闪，约定『三日后的灵地争夺战，你我不必留手』。韩立轻描淡写应下，向紫袍大汉躬身后飘然离开玉阙阁二层。

## 二、金胖子的挑拨与点破
- 玉阙阁附近高空，金胖子(金姓胖子)与翁姓汉子以传音交谈。
- 翁姓汉子坚称改换洞府、特意寻灵气稀薄之地，是为修炼一种秘术，金胖子不信。
- 金胖子点破：韩立是『刚刚到天渊城的飞升修士』。翁姓汉子不以为意——天渊城每两三年都有飞升修士加入，且韩立只是化神初期，自认稳胜。
- 金胖子劝其『教训即可、最好别真伤性命』，因城中飞升修士颇护短，若惊动他们(即便有『鸣前辈』依仗)仍会生风波。翁姓汉子表面轻飘应对，暗指金胖子新得一件威能不在灵宝之下的上古宝物，金胖子矢口否认。
- 金胖子化红光离去后，翁姓汉子识破其想激怒自己、令其与飞升修士结仇的算计。

## 三、翡翠蛟龙(师傅)现身
- 一只尺许大、通体翠绿如翡翠的蛟龙现身——乃翁姓汉子之师的分身。师本体正祭炼灵宝到关键处，故只派分身。
- 真实目的：该灵地或为某高人最后坐化之地，纵只一丝可能也值得一探；若得其遗宝与功法，『为师度劫再无忧虑，徒儿进阶炼虚也大有可能』。故三日后争夺只能成功，必要时可直接击毙韩立。
- 翡翠蛟龙暂借『千魂铃』(寸许乌铃，黑气缠绕、铭印密密符文咒语)给吊眉汉子，三天临时祭炼即可暂时驱使，以求万无一失。随后化翠光遁入其袖。
- 翡翠蛟龙提醒：金胖子已起疑、需谨慎；金胖子本身不足虑，其祖父却大为棘手。

## 四、天渊城设定(韩立巡城所见)
- 天渊城又称『渊国』，所辖面积近人界一小国；真正城区呈梯形，数万里广，巨墙围护。
- 面向蛮荒世界的一面城墙最窄(百余里)却高达数千丈、法阵符咒日夜开启，人妖两族大量警卫巡视，城墙后常驻百万以上人妖大军，故建城至今从未被异族偷袭成功。
- 另一面城墙绵延上千里、相对宽松，三境七地人妖两族进出频繁(多去蛮荒世界冒险或来贸易)。城外为连绵山脉与少量凡人村镇，为天渊城提供物资与低阶修士/炼体士补充。
- 人妖两族联手驻守却以一道巨大光幕将城区一分为二、各管一面；城外亦以无形无人带为界。
- 唯一可共处之地为城区中心的『长老塔』，比其他巨塔粗大倍许，是天渊城最终战力——十余名合体期两族长老共议决断之所，神秘异常。

## 关键人物/事件
- 韩立：化神初期、新到天渊城的飞升修士，尚不知有人动了杀心。
- 翁姓吊眉汉子：化神初期，欲进阶炼虚，奉师命夺灵地。
- 金胖子：飞升修士，新得上古宝物，祖父为鸣前辈。
- 翡翠蛟龙：吊眉汉子之师，仅以分身现身。

## 第1318章 豹麟兽

### 天渊城见闻
- 韩立所处位置离长老塔约二十余里，远观高耸入云的青色石塔。
- 据其所知，长老塔无论何时，人妖各族都派一名【合体期】长老驻扎其中。
- 韩立感慨：人界时化神修士已是神龙见首不见尾；天渊城中合体期修士近在咫尺，日后还有机会请教。判断在洗净身上异界气息前，留在此处不错。
- 对三日后与吊眉男子争夺灵地之事不甚在意（其神通曾与炼虚期修士交手）。随后化青虹返回飞灵殿自己屋中，打开禁制入密室打坐。

### 收服变异豹麟兽
- 韩立取出【灵兽环】：灵界与储物镯齐名之物，内分隔数个空间，可同时容纳不同灵兽灵虫。落日之墓之行后，他将噬金虫、啼魂兽等收入其中，弃用原灵兽袋。
- 放出当日在落日之墓被灵族擒下的变异豹麟兽（尺许大小，被青索五花大绑）。
- 据【噬炎的记忆】（注：噬炎已于第1297章死亡，此为回忆），此兽外形似豹，实为带麒麟些许血脉的妖兽，又属变异，未成年实力已堪比元婴修士。
- 此兽野性极大、已有灵智，初时宁死不从；被关数十年后发来屈服消息。
- 韩立威胁后种禁制：先以十几根银针没入兽身；再喷精血于鲜红符箓化为血雾；拍天灵盖现出足踩小鼎的数寸元婴，元婴喷绿光与血雾融为绿红色光焰，掐诀使其化为拳头大鬼脸，钻入小兽头颅。小兽昏迷，身缠青索寸寸断裂，降服完成。

## 第1318-1319章 广武殿争夺灵地之战

### 入场
- 闭关三日后，玉阙阁发来争夺灵地的传音符（火光），指明地点『广武殿』。韩立此时肩头趴一只狸猫般小兽（即收服的豹麟兽）。
- 战场为广武殿露天广场上空的白色巨大光幕，由身穿金色战甲的疤面大汉（监战，后文显为炼虚期）主持。
- 翁姓汉子（吊眉汉子）以祭炼法宝为由迟到，险被取消资格，在最后时刻赶到。
- 规则：离开禁制范围内任何手段皆可用，除非一方认输，监战不出手；常有人因重伤被打落一个境界。
- 吊眉汉子讥讽韩立以元婴初期境界迎战，韩立懒洋洋淡笑不语，激其杀意。

### 交手经过
- 韩立出手：袖中涌出七十二口寸许金色飞剑，化朵朵金莲盘旋（大庚剑阵）；又喷出小鼎化丈许巨鼎——【虚天宝鼎】，灵压远超普通宝物，令吊眉汉子心生警惕。
- 吊眉汉子：身周冒出米粒大蓝色光点（蓝晶沙/蓝沙）；祭出巴掌大银灿法轮（断金法轮，铭印符咒），抛空化三四丈银轮如银日。
- 第一回合：蓝色沙云压下，七十二朵金莲（每朵涨至丈许如利刃）硬接破云。断金法轮（黎金沙祭炼，斩切神通胜飞刀飞剑）斩向金莲相持不下，终被金莲张瓣包裹击碎。
- 韩立点动虚天鼎，鼎盖飞射、青霞卷出收摄蓝沙，断其与主人心神联系。吊眉汉子惊呼『灵宝』（蓝沙为其亲手祭炼）。
- 吊眉汉子以精血划血色符文，化双角青面獠牙巨大鬼头扑韩立，欲打断其收宝。韩立喷出一道金色电弧——【辟邪神雷】，一击灭鬼头（连古魔界始祖分身都不敢硬挨）。
- 虚天鼎青霞已摄走半数沙云。吊眉汉子掐奇异法决、发梵唱、动真元，将沙粒涨成头颅大蓝色晶岩（蓝晶沙）压下，金莲凝滞、青霞无法卷动。
- 韩立摸头顶放出灰蒙蒙霞光——【元磁神光】，使重逾巨山的晶岩瞬间轻飘还原为沙粒，再一卷收空全部沙云。此神通令吊眉汉子脸色发青，连炼虚期金甲修士也首次轻咦惊讶。
- 韩立心中自评：同具元磁神光、噬灵天火、大庚剑阵，面对化神后期修士亦无惧；因尚未摸清对方底细、且对方有炼虚期师傅，忌惮之下暂未真动杀机。

### 吊眉汉子的重宝（第1319章末）
- 取出黄色铜牌（白雾缭绕）与淡银符箓（密铭银蝌文）：
  - 铜牌投射出身高两丈、赤手空拳、肌肤翠绿晶莹、双目灵芒的【人形傀儡】（注：此为吊眉汉子所持的翠绿傀儡，非韩立之人形傀儡）；行走间体形每步涨尺许。
  - 银符化银雾，幻出狼目阴森碧绿的黑色巨狼，旋即化黑雾隐匿。
- 又拍天灵盖现蓝色元婴，双手捧寸许黑铃——【千魂铃】。监战金甲修士点破：此乃『鸣老怪』（其家师）之重宝，纵霸道却需借秘术勉强驱使，战后元气大伤；蓝色元婴称已深思熟虑，将黑铃抛出。
- 韩立将七十二朵金莲聚为直径超十丈的超大金莲飘向对方，并以元磁神光灰霞回卷护身。战斗进入新阶段。

## 章节
第八卷 初入灵界 第一千三百二十章 炼体之威

## 场景与对阵
- 地点：灵界，光幕之内的争斗，光幕外有一名疤面金甲修士旁观。
- 对阵：韩立（元婴，顶阶炼体士、法体双修） vs 翁姓吊眉汉子（元婴，灵宝千魂铃持有者），辅以翁方的翠绿傀儡与黑色巨狼魔物。

## 经过
1. **千魂铃摄魂**：翁姓汉子祭出黑铃（千魂铃），一声入耳便令韩立神识剧震、险些坠落；体内大衍决自行运转，一股清凉冲脑使其恢复。韩立确认此为灵宝。
2. **惊神刺反击**：韩立凝神念成团狂放，正催铃的翁姓汉子神识被惊神刺如尖锥扎入，凄厉惨叫、躬身如虾，元婴没入体中，暂失对千魂铃的控制。
3. **金莲擒铃**：韩立遥指空中巨大金莲压下，将黑铃摄入莲中金光大放；但千魂铃化丈许黑铃虚影，莲瓣切割无损，反放黑丝缠绕，使金莲旋转呆滞。
4. **黑狼偷袭**：黑色巨狼幽灵般闪现，利爪无视元磁神光所化灰色光罩探入（元磁神光对与五行无关之物力有不及）。韩立浑身金光、身后凭空探出金灿灿拳头硬砸，狼爪被砸烂、狼身倒射。
5. **撕狼吞魂**：韩立以罗烟步在虚空辗转闪至狼头顶，金掌拍飞其黑色光球，双目灌力暴射蓝芒（天眼术）看透此狼，闪至其腹，两手如铁钳擒头与后腿，硬生生摘下狼首；狼身化黑雾溃散，狼首仍活。韩立召出一团银色火焰化银色火鸟扑下，将狼首神魂吞噬，火鸟吐出一张光芒黯淡的银符（先前翁姓汉子所祭），被韩立收起。
6. **毁傀儡**：翠绿傀儡涨至六七丈、连击虚天鼎所化巨鼎（鼎被击退缩小但不散，此宝当年能挡魔界古魔始祖分神一击）。傀儡无灵智只知蛮击。韩立以天眼术看出弱点，闪至其身后，体内爆竹脆响，对准肋下一拳——拳风划出白痕似撕裂空间，将傀儡击趴在地；连击同一处碎开黑洞，再搓出一道金色闪电（辟邪神雷）灌入，傀儡爆裂、绿芒大放后急剧缩为尺许大小，瘫地不动。

## 结果与悬念
- 翁姓汉子从神识剧痛中恢复，惊呼“金刚诀大成，你还是顶阶炼体士”，疤面金甲修士亦震惊。
- 韩立平静反问“翁道友，还要争斗下去吗”。
- 翁姓汉子大怒，称韩立法体双修神通远超预料，但不认输，誓拼百年苦修也要展现千魂铃真正威力；重新催出元婴，以蓝色光柱与玄奥咒语唤出千魂铃本体，本体将光柱化灵光吸入。
- 吊眉汉子趁机弹指，光幕中空间剧烈波动，一道纤细黑色裂缝无征兆浮现于上空（章节悬念）。

## 韩立此战所用手段一览
- 大衍决（护神识）、惊神刺（神念攻击）、金莲（擒铃）、元磁神光（防御光罩）、金刚诀（炼体金身/金色拳脚）、罗烟步（虚空身法）、天眼术（瞳孔蓝芒看破）、银色火焰所化火鸟（焚吞神魂）、辟邪神雷（金色闪电毁傀儡）、虚天鼎（挡傀儡）。

## 第1321章 夺得灵地

### 千魂铃·阴魂空间
- 吊眉汉子(翁姓汉子)催动黑色千魂铃，开辟出黑色裂缝(阴魂空间)，将韩立连同巨鼎(虚天鼎)、金莲一并摄入。
- 千魂铃为灵宝级法宝，内有上万阴魂(皆穷凶极恶妖兽精魂祭炼)，主魂生前为化神中期天地灵兽，足以困住化神后期修士；空间内非阴魂之体法力大受限。
- 韩立反笑不惊。约一盏茶后铃声异变，主魂被灭、其余凶魂去十之六七，韩立破禁从虚空孔洞而出。

### 韩立反击
- 韩立瞬移至吊眉汉子背后，金刚诀金色拳头连出幻影破其分身。
- 吊眉汉子以银盾、青冥甲(清鸣甲/青甲)护身，泥鳅般卸去大半拳力，仍被击飞、口吐鲜血负内伤。
- 韩立祭青竹蜂云剑欲斩杀；吊眉汉子声称认输；疤面金甲修士以黄霞将其救走。

### 裁定归属
- 金甲修士(天卫)判翁姓汉子认输、韩立为胜者，灵地归韩立。
- 取韩立青冥佩，以翠绿晶石注入灵地标记；非主动放弃则保有此灵地，可继续参加百年一次的灵地争夺。
- 二人离场。吊眉汉子去寻其师翡翠蛟龙(尺许翡翠蛟龙)。

## 第1322章 洞府

### 翡翠蛟龙的盘算
- 翁姓汉子向师父翡翠蛟龙复命，称对方仅化神初期却神通深不可测、具鼎状灵宝、为金刚诀大成炼体士。
- 翡翠蛟龙论法体双修：法力与炼体术任一皆需穷尽岁月，前期超群但深处远逊；古来唯天元圣皇等寥寥数人法体双修大成、进阶合体。
- 金刚诀大成已是凡人炼体术最高境界，欲再炼体须修古魔界魔功或妖族功法(异族功法，凶险)。
- 翡翠蛟龙惊于韩立破其千魂铃，疑其有专克阴魂至宝或至阳至刚大神通；决定让翁姓汉子持信物去向好友公羊度(公孙/公羊老魔)借隐匿至宝‘迷天五色幡’(颠倒五行、化物无形)，伺机潜入灵地寻宝。

### 韩立勘察灵地
- 韩立遁出天渊城高墙，三四日抵达琼籁山——玉阙阁所示三片山脉中最次者，仍远胜人界圣地，划予化神修士修炼。
- 横跨大半山脉抵达自己两万余里灵地(再无他人)，沿标记范围撒下白濛濛迷幻法旗以示有主。
- 勘得数座低阶小灵矿与几处天然灵药地(皆剩小半)。疑吊眉汉子拼死相争另有缘故，决意日后细查。

### 建立洞府
- 选两山间五色斑斓瘴气沼泽：瘴气百余丈高、含奇毒、限制神念。韩立凭明清灵目(天眼术)透视，于瘴气深处寻得天然石谷连通两山。
- 在石壁开洞、以飞剑挖通道直入两山山腹，开辟两座洞府(一座迷惑外人，一座真正修炼/存放灵虫灵兽/培育灵药)。天机府体型过小暂不动用。
- 移植人界灵草灵木入药园；将多年催培、体积惊人将近成熟的噬金虫放入虫室；豹麟兽入兽巢自行修炼。
- 走过一间阴森空荡兽室时，韩立神色凝重，以灵兽环放出一只黑乎乎、丈许高之物。
