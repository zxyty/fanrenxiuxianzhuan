## 背景设定：东裕国与坠魔之地

- **东裕国**：天南一国，国土与修炼资源足以排进天南前十。因同时紧邻三大势力（正魔两道、天道盟）国家、被卡在中间，且本土无宗门可压制他派，三大势力遂约定互不插手，使其成为**中立之国**。与九国盟相隔太远，无关联。本土宗门正魔混杂、闲散小派众多，不少已暗投三大势力。
- **万岭山脉 / 坠魔之地**：东裕国北部昌州盆地中心、连绵百万里的巨大山脉，灵兽珍稀灵药丰富。常年被五颜六色剧毒瘴气封锁（粘之立毙、浓密迷向）。其隐秘处藏有自蛮荒时期即存在的秘谷**坠魔谷**，号称天南第一凶地，谷内空间裂缝密布。
- **入山寻宝期**：每约五十年瘴气散去一次、坠魔谷裂缝最少最稳定，仅年许时间，各地淘宝者（数以千计）涌入；周边家族商家临时设坊市收购灵药灵兽。本年瘴气将散，他国修士与高阶修士异常之多。

## 鬼灵门与坠魔令

- 半年前传出**鬼灵门**得到入谷之法、欲瞒众宗独吞谷宝。消息轰动修仙界，各宗（含魔道六宗五关）派人质问，鬼灵门被迫承认。
- 各宗要求共享入谷手段，鬼灵门强硬拒绝，折中：发放**坠魔令**（青铜令牌，刻『坠魔』红字与金色令子），持令者可在裂缝平稳期随鬼灵门修士一齐入谷，需付灵石。共放数百块，大半落入各宗，另放百余块给散修小派。
- 入谷需借众多鬼灵门弟子之力并耗大量灵石，于特定时刻进入，故限制人数。一块令牌数万灵石（书中一对结丹夫妇三块花十万灵石）。
- 传言**慕兰法士**也会派人参加。

## 韩立回归（落云宗一行）

- 紫灵、梅凝、宋姓女子（三名落云宗女修，与韩立交好）在万岭山附近城镇客栈等候韩立。紫灵施秘术隐去娇媚真容。
- 韩立两年多前对外宣称闭关疗伤，实赴**极西之地**（来回赶路一年多，遇『小麻烦』耽搁，否则半年前即归）。现身为二十余岁青袍青年，气质深不可测；**天南仅次于三大修士的存在**，曾灭杀元婴中期顶阶修士。背负一尺许黄竹筒（未收入储物袋）。
- 紫灵欲入坠魔谷取**灵烛果**炼**造化丹**助三人冲元婴；宋、梅二女因入谷修士激增、风险过大而放弃，仅紫灵坚持随韩立同入。
- 韩立入谷前确认事项：灵烛果确有其事（鬼灵门生还长老亲述，碧绿玉简记其位置，位于谷深处且描述模糊但有显著特征）；造化丹须在谷内炼制服下方不减效，紫灵已备十几份辅助材料。约定外谷紫灵自理、内谷由韩立庇护，遇不可抵御之危需各自保命（韩立以传音术细嘱）。
- 落云宗分得三块坠魔令，韩立持一块。多数修士只在外谷探宝，敢入内谷者多为元婴修士。**程师兄（银发老者，程师伯）**亦到此地，欲在大限前入谷一搏。

## 大衍神君真相

- 韩立随身黄竹筒中藏有一名老怪物——**千竹教创教始祖、大衍决（傀儡术）的创立者『大衍神君』**，活了上万年，附身于一台傀儡存活至今，神识强大不可思议。
- 大衍神君当年施展**七情决**欲对付韩立主元神，却因当时『区区元婴初期』的韩立修炼了**第二元婴**而被挡，反落到如今下场。七情决将其喜怒哀乐与韩立的第二元婴相连——本章他因觊觎紫灵姿色『发情』，连累并扰动韩立心境，遭韩立呵斥。
- 其精魂靠韩立**引魂木**所制珠子维持，仅余数十年即散；因其最后一种傀儡尚未完成而暂不投胎。
- 韩立曾对其施搜魂术毫无效果、反差点被其操纵神识，遂不敢再用。韩立留其性命，是因对方答应传授**最后三层大衍决**及数种从未面世的大威力傀儡炼制法。

## 要点
- 灵烛果→造化丹→冲击元婴，是紫灵一行的核心目标。
- 韩立持第二元婴、与大衍神君命运相牵，是本卷重要伏线。

## 概览
本条覆盖第796-798章。前两章为韩立与寄附于傀儡的『大衍神君』长谈，揭示其生平、寄神术与上古魔界设定；第三章韩立离城备战坠魔谷，途中救下黄枫谷陈巧天，得知陈巧倩死讯。

## 第796章 寄神术
- **大衍神君生平自述**：自恃才智过人，自创可强化神识的大衍决、创立千竹教、复原天南失传的傀儡术，又兼修功法进阶至元婴后期。
- **极西之地来历**：当年打败号称天南正魔两道第一的两名老修士，逼其将极西之地（约两国之地）让出作千竹教立教根本，并发毒誓万年内徒子徒孙不得入极西之地，否则尽杀；他炼有上千傀儡为凭。
- **致命错误**：因分心创功法与傀儡术耗去大半寿元，剩二百年无法由元婴后期突破化神期；不甘转世失却记忆，遂在大衍决基础上创出『寄神术』。
- **寄神术真相**：仅完成一半便被迫使用，寄附特制傀儡后修为未尽失、仅神识神通可勉强用，但无法令元神出窍；密室有厉害阵法与百余傀儡守卫、且从未告人，致其被困上万年、近魂飞魄散。
- **韩立相救与交锋**：韩立闯密室发现奄奄一息的他，看出其与众傀儡不同抢先下手；大衍神君以『七情决』偷袭，欲令韩立第二元婴狂性大发，韩立则可令主元婴操第二元婴使用同化之术同归于尽，遂成投鼠忌器。
- **交易**：韩立借出至宝『养魂木』滋养其元神续命，换大衍决后三层与傀儡术；大衍神君赞其不口是心非，称若非韩立资质太差欲收为徒。当年门人数百年无一进阶元婴，故只传下似是而非的假后三层口诀。

## 第797章 上古魔界
- **封魂咒**：大衍神君称非不能破，而是寿元有限无暇；韩立遂决意亲赴坠魔谷。
- **坠魔谷之险**：上古禁制与空间裂缝可怕，裂缝偶尔巧合打通其它界面通道，放出妖魔鬼怪。
- **界面与古魔设定**：除灵界（上界）、阴司之界、仙界外，尚有与人界最相似的『上古魔界』，其中古魔最爱来人界；上位古魔即便分身亦极强。上古时期魔界曾开通道，一批上位古魔肆虐人间，上古修士凭人数苦战无数年月将其灭尽，但死伤惨重、元气大伤，诸多上古功法灵兽就此灭绝——上古修士没落与此大有干系。
- **魔功渊源**：魔道部分上古魔功即模仿古魔神通而创，有的创立者可隔界借有名古魔魔气上身令神通大涨；传闻此类魔功修至极致飞升上古魔界、成古魔一员（未经证实）。
- **跨界限制**：空间裂缝非真正界面通道，过来者只能是化身或低阶存在，且因修为不足受天地法则排斥难久留，多自行暴体或原路逃回，故天南从未因此大乱。韩立联想到慕兰法士古灯所召妖禽情形类似。
- **来源**：大衍神君未入谷，但当年交好的几位修士联手入坠魔谷，刚入谷即撞上跨界古魔，大战后仅一人以分神逃脱并投奔于他。
- **报酬交付**：大衍神君交出大衍决第五层功法口诀与两种结丹傀儡炼制法的玉简，随后入『洞福天』（即韩立背后竹筒，空间宝物，藏其傀儡真身）继续研究傀儡术。
- **天煞魔尸**：韩立出城至林中，一颗埋于土中、绿发遮面的硕大人头已培养出简单神识，可在坠魔谷派用场；他设禁制后把玩刻『天极门』字样的黄色令牌。

## 第798章 陈巧天
- **备战安排**：韩立收起天煞魔尸；此前在城中未见天极门弟子，持长老所赠令牌一时无从联系南陇侯。准备到万灵山附近安置、炼制结丹傀儡，并去坊市收购常见傀儡材料（已凭边界之战赌斗及慕兰古灯所得近千万灵石的珍稀材料）。欲炼上古元婴级傀儡尚缺『血凤木』等材料（已嘱落云宗弟子代为留意，入谷前难凑齐）。
- **北极元光推断**：韩立持两仪环，料南陇侯会主动找上门；北极元光多半在进内谷通道而非外谷入口，否则鬼灵门无法令数百修士同时避光入谷。
- **路遇斗法**：途经四男一女结丹期修士对战，一方两名修士以数百毒蛾（喷五彩毒粉）压制另一方两男一女；后者中有锦袍儒生祭银色巨笔与金色书卷。儒生袖口绣黄枫叶标记。
- **出手与身份**：两名驱蛾者认出韩立为落云宗韩前辈（曾见其边界之战神通），自称御灵宗弟子（魔宗）；韩立以与黄枫谷有旧交为由令其收虫罢手，二人惶恐告退。
- **陈巧天**：获救儒生自称黄枫谷陈巧天、陈氏家族（黄枫谷三大修仙家族之一），结义兄妹为化刀坞弟子，疑韩立与令狐老祖有交情。
- **陈巧倩死讯**：韩立追问陈巧倩，陈巧天答其妹非结丹修士，百余年前得怪病已去世。韩立怔然，忆及与陈巧倩最后游山表露心声一幕，称当年最后一面时听闻她将嫁人，后逢魔道入侵不知婚期是否如期。
