# 第六卷 通天灵宝 第八百四十一章 联手抗魔

> 世界状态须知：本章『南陇侯』为**圣界夺舍者/古魔魔魂**夺舍占据南陇侯躯体（起于第829章），真『南陇侯』已于第835章死亡。本章『南陇侯』的一切言行归属于夺舍者，并以 note 标注；勿记到真正的南陇侯名下。

## 战况梳理

### 1. 古魔强攻，傀儡受挫
- 十余只虎妖傀儡（韩立精心炼制）齐动围攻『南陇侯』，数十道爪芒罩下。
- 古魔以平魔化之躯的蛮力，赤手迎击：黑气一涨即将傀儡利爪寸寸碎裂，并径直从两具独臂虎妖傀儡胸腔抽出手臂，使其失控坠地。傀儡虽无心脏、不畏生死，却挡不住其拳力分毫。
- 古魔真正目标是法力尚未恢复的韩立，一闪近身出拳。

### 2. 天绝魔尸偷袭
- 韩立面露诡异之色，脑后突起尖锐爆鸣——他藉众傀儡掩护，悄然唤出隐匿一旁的**天绝魔尸**（浑身绿毛、脸似骷髅，尸魈所化炼尸）。
- 古魔只顾防备傀儡，险被偷袭，急以无数残影遁走，残影被绿色爪芒撕碎；看清后意外叫出『炼尸？』。古魔评估此炼尸气息古怪，约相当再加一名元婴初期对手，不足为惧。

### 3. 韩立亮多重手段
- 韩立双手齐扬：一手金弧弹跳、雷鸣大响（**辟邪神雷/金弧**）；一手浮现一小团**紫色火焰（紫罗极火）**。
- 古魔忌惮：① 辟邪神雷对古魔伤害极大，且寻常拥有者顶多放一两次，眼前人却似源源不断、毫无顾忌；② 紫色火焰来历不明却本能觉其危险。
- 古魔灰白火焰半空爆裂，使韩立的**金色虫云（噬金虫）**安然脱困直奔韩立而回，同时令狐一方的**碧磷鬼爪**消失无踪。

### 4. 三方夹击与魔魂遁逃
- 古魔放话：普通手段难掏出韩立元婴，索性一次灭杀永绝后患，且这具躯体只是临时借用、坏了无妨；随即魔气暴涨、骨节怪响、身形拔高，后颈双肩凸鼓，似有物欲蹿出。
- 韩立以金弧幻化金色巨蟒、紫色火球、金色虫云齐攻；虎妖傀儡与天绝魔尸留守原地防偷袭。
- 令狐老祖与白衣女子（先前曾被古魔以蛮力杀得抱头鼠窜，忌惮其不知名魔功）同掐诀，五色光柱率先照下、白濛濛圆环狂涨为丈许巨环下套。古魔黑气吞噬五色光柱后倒射遁走，遁速奇快、路线诡异，金弧巨蟒、巨环、虫云、五色光柱皆追之不及、一一落空。

### 5. 大衍神君劝退与大庚剑阵
- **大衍神君**传声：古魔凶残善斗、实战丰富，纵非高阶古魔一旦认真韩立也非对手；自言元婴傀儡尚未研制完成，不愿同死，劝韩立逃命。
- 韩立不答，厉色一闪，抖袖放出数十口**金色小剑**（青竹蜂云剑系飞剑），化数十道尺许金光盘旋，连打数道法决分化出上百道剑光，布下**大庚剑阵**。
- 韩立传声：若古魔本体降临剑阵未必能伤；但既是附身人类修士的躯体，入阵『不死也得脱层皮』。大衍神君冷笑，称古魔至今仅以蛮力相拼，上古魔功尚未现身。

## 关键判定
- 古魔评估机关傀儡真打斗起来作用有限；炼尸顶多添一元婴初期对手；最忌惮辟邪神雷与紫色火焰。
- 战局僵持：古魔以遁速优势规避围攻，韩立以剑阵谋求困杀。

# 凡人修仙传 第842-843章：双头四臂魔魂弃战 · 飞蛾扑火

## 一、魔魂现形：双头四臂(第842章)
- 大衍神君透露：当年入坠魔谷的几位好友联手仍只一人垂死逃脱，古魔魔功极厉害。
- 逃窜黑气中，夺舍'南陇侯'者(据世界状态须知=【圣界夺舍者】，自第829章夺舍南陇侯躯体)彻底化为**双头四臂**怪物：
  - 前头颅紫鳞、额生白角，仍是原貌；脑后挤出一颗青面獠牙、吐紫红长舌、瞳孔似缝闪银白寒芒的妖鬼头颅。
  - 周身及四只过膝长臂遍布深紫鳞片，指尖生银白骨刺指甲。
- 令狐老祖与白衣女子(元婴修士)指挥宝物攻击：五色光柱罩住、白色巨环(铭阳环)落下，皆被魔魂四臂轻松抓住。

## 二、污秽法宝据为己用
- 魔魂后头颅喷出**漆黑如墨的液体**击中铭阳环——古宝哀鸣灵光尽散、变黑泛乌光，令狐老祖当即断了与古宝的心神联系，面色苍白。
- 魔魂收为己用，并又从身上'魔化'出**风旗**(鲁姓老者驱风古宝)与一口**黑芒小剑**(古修遗骸古宝之一)，自言宝物太少正好合用。

## 三、噬金虫畏惧 · 妖魔关联之疑
- 韩立因分心布阵，金弧、火球落后；金色噬金虫群一见双头四臂竟本能畏惧、盘旋不前。
- 韩立由此联想到**梵圣真片**图像与**苍坤上人**洞府所供**三头六臂**妖魔之像，疑三者有关联，但无暇深究——本欲激怒此魔引入**大庚剑阵**。

## 四、魔魂弃战飞遁
- 正欲再战，远处传来惊雷般狂暴魔吼。魔魂两头颅同时色变，收宝化黑芒、发长啸应和，急速向吼声(坠魔谷地图标注的气浪爆发点)飞遁而去。
- 韩立暂停金弧、紫罗极火、虫云，疑该处另有一只更强古魔。

## 五、救人与人情(第842章末)
- 令狐老祖、白衣女子(南宫婉师姐)死里逃生，向韩立道谢。同行的另一名交好修士先前已被魔魂掏出元婴而亡。
- 令狐老祖暗忖与韩立先前定下援手黄枫谷之约甚妙，韩立出手则黄枫谷千年可保。白衣女子(此前与韩立有不愉快)也熄了报仇心思。

## 六、飞蛾扑火：聚向战团(第843章)
- 远天黑紫两色异芒下升起数亩翠绿'**腹尸毒云**'——白衣女子认出是九国盟**魏无涯**的毒功所化，普通修士沾之立毙。
- 但绿云被黑紫魔气死死压制：与魏无涯交手的古魔比'南陇侯'更强，连元婴后期都难敌；三人皆被锁定，魔气逼近。
- 韩立以利害(魏无涯日后追究'见死不救'之罪)半警告半威胁令狐老祖与白衣女子同往。

## 七、多路修士被吸引
- **天晶真人**(背后两只伤痕累累、一只仅余单臂的恶鬼傀儡)与**落云宗程师兄**(银发老者)及一陌生老者三人飞向战团；同行另一老者已失踪。
- 韩立念程师兄在落云宗待他不薄、恩怨分明，决意前往,危急时以**血影遁**逃命(慎用，恐遁入空间裂缝)。
- 韩立化青虹先行；令狐老祖、白衣女子怀侥幸之心硬着头皮跟上。
- **御灵宗大长老东门图**带三名绿袍人(五行灵婴元婴修士仅余三人)自百里外赶来，欲混水摸鱼。
- **鬼灵门钟姓长老**(因费心寻得的**灵烛果**——进阶元婴后期的关键——被人抢先摘走，又遭气浪)带弟子自三百里外赶来。

# 第844-846章 双魔会 / 魔幕 / 再战魔魂

## 场景
坠魔谷上空。韩立携天煞魔尸与数十傀儡(虎妖、巨龟等)赶至战团，与令狐老祖、白衣女子同来。下方毒云中为重伤的魏无涯，空中是两具圣界古魔。另有银发老者(程姓老者/『程师兄』)、天晶真人及一名隐修元婴老者在旁。

## 两魔的来历(关键设定)
- 坠魔谷封印着一具**上古入侵人界的圣界古魔**，被还王门主等人无意放出；解封时空间裂缝爆炸，**王门主、王天古道友当场身亡**，魏无涯施尽神通侥幸活命但元气大伤。
- 场中两魔实为同一古魔的**主元神(本尊)**与**第二魔魂(分魂)**：
  - **魔首人身古魔**(额生独角、肌肤晶莹如玉、腹中发雷鸣吼声)——脱离控制的第二元神，反噬并抢占了本尊被封印的躯体，更以『解体化形』重组躯体强行解封，致该躯体损失约1/3修为；体内尚残存上古封印，正以魔火炼化。
  - **双头四臂魔魂**——主元神/本尊，即魔化的『南陇侯』(圣界夺舍者夺舍南陇侯躯体，世界状态须知第829章起)，现用临时魔化之躯，修为不及本来魔躯。当年它放出分魂去吞噬一名修士的元神法，分魂遂生异。
- 二者反目，按圣界规矩『分魂反吞噬主魂』，只能一存一灭。但它们**暂时联手**：先吞噬在场修士元婴大补，再单独寻僻静处决一从属，以免人类修士渔翁得利。

## 战况经过
1. 魏无涯隐于毒云、化出碧绿毒蛟苦撑，向众修士与**慕兰法士**发传音符求援，韩立等以拖延待援为策。
2. 双魔不给商量时间即动手。魔首古魔放出**黑紫色魔幕(方圆十余里)**笼罩全场；魔魂(双头四臂)则隐形遁入光幕。
3. 令狐老祖、白衣女子(持紫光宝镜)退入魏无涯毒雾庇护，协助对付魔首古魔；银发老者三人结梅花防守阵自保。
4. **韩立独战双头四臂魔魂**：
   - 发现魔幕能**直接吸收护罩法力**，遂以辟邪神雷金弧贯入青蓝双层护罩，电弧逼退魔光、止住法力流失。
   - 施明清灵眼(天眼术)仍难锁定隐形魔魂，遂放出**噬金虫**布二十余丈感应虫网。
   - 以中指弹出压缩成球的**辟邪神雷雷球**奇速偷袭，击伤魔魂怪舌(其舌不惧雷电却仍受创)。
   - 魔魂以『撼波魔功』黑色波纹震晕大片噬金虫；又坦言这些是**未成熟的噬金虫**，否则它无合适魔器时也只得暂避，并提及自身**本体为圣界魔蝶**。
   - 韩立催虫群、傀儡、天煞魔尸齐攻，欲争取布『大庚剑阵』之机；魔魂却结印爆发黑芒气浪，长啸震散攻势，现出**修罗战甲般、布满倒刺的乌黑硬壳**，气势冲霄。韩立目瞪口呆、惊骇落入下风。

## 归属说明(据世界状态须知)

- 魔首人身古魔为另一独立第二元神，与本尊(南陇侯/夺舍者)为反目敌对关系。
