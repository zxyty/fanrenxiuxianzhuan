# 第416-421章 土山·封灵柱

## 背景与缘起
- 散修金青请韩立出手帮忙破解一处古修士遗址的大阵；韩立因青竹蜂云剑培炼未久、天星城外乱星海暗流汹涌而迟疑，但念及金青交情与人情，加之对遗址收获有兴趣，两日后应允。
- 次日韩立与曲魂随金青出天星城而去。曲魂为韩立分身一事，附近修士多已心照不宣；此行对外仍以「师兄」身份掩饰其真正身份。

## 荒岛与土山
- 飞行约两月余抵达乱星海偏僻海域一座方圆千余里的大岛，岛上尽是灰黄土丘、无灵脉、草木不生，被凡人视为荒岛。
- 西行百余里有一座高约千丈、通体土黄的土山，近山时狂风飞沙(护体灵光可破)，山脚石屋处会合先到的胡月一行。

## 人物会合
- **胡月**：白衫年轻修士(实为善保养的年长者)，结丹期，善交际，金青旧识，与金青同发现两颗珍珠的同道。
- **石蝶仙子**：红月岛石真人之女，筑基期女修，姿色一般而神情倨傲，阵法理论造诣极高(略逊辛如音)、推算细腻，但破阵手段偏少、有纸上谈兵之嫌；先索取先挑一件之权。金青因受石真人旧恩而处处迁就。
- **简姓修士**：高瘦沉默，结丹中期，暗藏洪荒古宝三杆黑幡。

## 以阵破阵
- 阵法为风(土)双属性幻阵，石蝶先点出属性。韩立掏出十几道阵旗布成「北斗两仪阵」，凭辛如音所赠破阵心得「硬套」施法，破去外层障眼黄雾，露出方圆百余丈黄色光罩(内含数层、中心一数丈圆柱、七色光带游走)。
- 韩立自陈阵法领悟有限，全靠辛如音心得；众人误作谦虚，更生敬意。
- 韩立与石蝶分工合作(石蝶推算弱点、韩立设法破除)，历约三月破至最后一道禁制：该处被化为「三阳之地」，孕育罕见火系妖灵「翅恶」(寿短、喷妖火胜丹火、嗜食凡人魂魄与修士元神)。
- 破阵需水属性相克法器：韩立分发小尺、小锤、小叉给金青、曲魂、简姓修士，石蝶给胡月蓝色丝帕，并布下数十蓝色水属性阵旗阵盘。

## 岛主怪人之战
- 临施法时，一枯瘦白发、满脸红斑三角眼、结丹中期的怪人乘蓝虹而来，自称已住岛数百年的岛主，喝令众人滚出。
- 怪人持**洪荒古宝花篮**(化白气困住法宝)，胸前左右各咬一颗乌黑骷髅头(可飞出咬断其手指、喷碧渊鬼火/碧绿磷火)。先后困住胡月飞刀、吞毁简姓修士的洪荒古宝三杆黑幡，威势惊人。
- 韩立本欲藏拙，但不愿数月心血落空，且想试「噬金虫」威力——放出灵兽袋中数十金银甲虫(奇虫榜有名)，吞噬怪人乌黑法轮，遇磷火不伤、逼两鬼头自爆(中心数百只噬金虫被毁，余无恙，可见未成熟噬金虫超限亦会被毁)。
- 怪人收花篮、弃飞剑逃遁。韩立以七柄翠绿小剑合化绿色巨剑(绿煌剑)率曲魂(混元钵)穷追数十里，飞剑分化合一、携淡金色电弧贯穿怪人胸膛，将其化为灰烬。

## 战后所得与封灵柱
- 韩立夺得花篮古宝、储物袋，及一幅红光闪闪的画轴(展开后令其深思、合上不语)；无主黑骷髅头邪气未减但失灵性，单独收入玉匣。
- 韩立返回称「妖人已灭」，胡月、简姓修士对其暗生敬畏。
- 众人顺利破去最后禁制，水属性法器尽灭翅恶，露出中央石柱：盘旋铭刻八个古文符号与无数怪纹，镶嵌均匀分布的阴阳玉(黑白之光)。
- 修为最低的石蝶最先认出此为「**封灵柱**」——据其洞府阵法典籍，混合十几种珍贵炼器材料铸成，一根价值五六千灵石。

## 设定要点
- **洪荒古宝**：古修士所炼法宝，神通单一但威力极大，无法收入体内、只能随身携带——此即辨别其与普通法宝的唯一办法(往昔六连殿制「婴鲤兽」的「干天戈」亦属此类)。
- **翅恶**：火系妖灵，极罕见、寿仅数时辰，妖火胜丹火且嗜食魂魄元神。
- **噬金虫**：金银色甲虫，奇虫榜有名，可吞噬法宝/磷火，威力惊人但未成熟时超限会被毁。
