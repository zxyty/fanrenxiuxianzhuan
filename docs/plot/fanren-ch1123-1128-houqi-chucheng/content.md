# 第1125-1127章 后期初成 / 黄沙门 / 谣言

## 一、脱困虚天殿
- 虚天殿距上次开启不到三百年便诡异现世，半隐白光中。一道青濛濛光柱洞穿大海，光柱中浮现白色法阵，传送出一男一女及一只青色巨鼎（虚天鼎）。
- 巨殿一声巨响后虚空消失，光柱溃散，空中只剩一男一女与巨鼎。
- 女子（冰凤/十级玉凤，银衫女子）长袖一抖，上百口晶莹飞剑将逃走的火云魔道修士追上、乱剑分尸，连元神都未及遁出。
- 围观的甘霖等推断火云中即魔道元婴老怪「混老魔」本人——与甘霖同为元婴级，却被随手灭杀。

## 二、韩立后期初成的缘由
- 男子即韩立，在虚天殿借「灵眼之泉」苦修八十余年。先花一甲子修至元婴中期巅峰，又花二十年巩固达突破条件。
- 心存侥幸尝试突破后期瓶颈，竟水到渠成、几无阻碍地进入元婴后期；他自感诡异（中期当年极为艰难）。
- 可能助力（韩立自己也未理清）：所修大衍决、服用化婴丹；先前补天丹所化五色珠改善灵根资质；青元剑诀进阶后期本就较易；修为远比同阶深厚——或数种/全部共同作用。
- 修成后重新祭炼八灵尺与虚天鼎，轻易炼成通宝决第二层。
- 冰凤（早已是后期）见韩立竟轻易进阶后期，郁气发泄于火云魔修身上，随后离开韩立自寻返回大晋之法；韩立未把返回天南的上古传送阵之事透露给她。

## 三、苦门岛·黄沙门
- 甘霖（甘姓中年人，黄沙门大长老，元婴初期，双目灰白如瞎子）主动迎向韩立，恭谨相邀。
- 韩立化名「姓韩」（用本名行走，未冒充他人）入黄沙门做客，得知：前方苦门岛由黄沙门独占；北方数百万里为内十二星岛之一的癸星岛；此地离天星城近、离魁星岛远，途经天星城。
- 韩立用银色圆环、黄色古戈两件古宝（虚天殿中灭高阶妖兽所得）换取中阶灵石、材料及七八块高阶灵石。
- 高阶灵石之多源于百余年前外海孤岛发现的高阶灵石矿，星宫、逆星盟与外星海妖兽争夺后三方瓜分，灵石短缺稍缓。韩立人形傀儡耗灵石将尽，遂起意去该矿，并打算借星宫传送阵前往。

## 四、谣言与封山
- 青衫老者石宣认出韩立即近二百年前逆星盟「剿杀令」所追之人（令上记为结丹修士），翻出鬼面玉简幻影核对无误。
- 甘霖震惊：对方短期内由结丹进阶元婴后期。决定此事绝不外泄，不愿卷入逆星盟恩怨；命三名晚辈十年内不得离门，悄悄封闭山门。
- 甘霖亦猜出那巨鼎即虚天鼎、韩立即昔年得虚天鼎之人，但自知无力夺宝、泄露反招杀身之祸，遂作罢离去。

## 五、追杀令的背景与后续
- 当年极阴祖师等元婴老怪搜寻虚天鼎无果、大怒，借逆星盟剿杀令追杀韩立逼其现身；多年后此令未撤却已被正魔两道大半遗忘。
- 虚天殿再现世消息扩散：传闻中讹变为「混老魔被两名元婴修士合力击杀」，淡化了双后期同现的骇人程度；但仍有不少元婴修士将韩立与昔年身份联系，蠢蠢欲动。
- 韩立不知剿杀令及自己真容图流传，未变容。沿途换得二十余枚高阶灵石。

## 六、章末悬念
- 飞行月余将抵天星城圣山时，对面出现十几道遁光成队迎来：含四五名结丹、一名元婴初期高阶修士，且元婴修士气息给韩立「似曾相识」之感；韩立未隐匿，原地观望。

# 第1128-1130章 密语 / 元磁神光 / 拦截

## 场景一：天星城外重遇凌玉灵
- 韩立大咧咧悬浮于星宫修士队伍前方，施展**敛气之术**彻底掩盖修为，神念扫之只见一团云雾。
- 队伍中唯一元婴期者头戴白色斗篷、不辨男女、韩立神念也无法透视。韩立欲动用**明清灵目**强行透视时，对方摘下斗篷，正是**凌玉灵**——当年在天星城外被韩立救过的、貌若处子的星宫执事，如今已**凝结元婴大成**，部属敬称『淩长老』。
- 凌玉灵奉命去支援被**逆星盟**攻打的海岛；因防逆星盟潜入，星宫外海传送阵已大半关闭，仅留两座备用。他赠韩立一枚**令牌**可动用传送阵。
- 凌玉灵看出韩立绝非元婴初期（实已**元婴中期**之上，距二人分手不足二百年），大为震撼羡慕。

## 场景二：传音符与改头换面
- 凌玉灵离去后以淡银色**传音符（遁音符）**向天星双圣汇报。
- 韩立以人形傀儡截获此符，捏碎读取信息后面色阴沉：『原来我身具虚天鼎的事情，竟然整个乱星海都无人不知了。』遂施秘术**改头换面**，化为脸色紫黑的魁梧大汉，潜入天星城。

## 场景三：地下洞窟·天星双圣密议（元磁神光真相）
- 天星城圣山地下洞窟中，**淩天雷**（男，元婴后期，五百年进阶元婴、自负天赋不输历代星宫之主）与一名**蒙面女子**（女宫主，元婴后期）密谈。二人同修**元磁神光**。
- **元磁神光**：上古修士所创，大成可将五行之力化为己用、排斥并削弱五行之宝威力大半；但最后一步需修炼者**身具五行灵根**，而五行（混乱）灵根连筑基结丹都难通过，故历来无人修成，疑为上古修士故意布下的陷阱。
- 二人因此**功亏一篑**，无法进阶**化神期**，提前出关停修方免爆体而亡，如今仅剩**不足百年寿元**。淩天雷查得一种上古秘术可血祭纯属性法宝代替一行灵根，但二人皆天灵根，于事无补。
- 当年正因元磁神光排斥五行之宝、虚天残图对后期修士无用，二人无法进入虚天殿，才让外人（韩立）得到**虚天鼎**。曾派人调查得鼎者来历，疑为大晋某大宗门或外来修士。
- 培养计划：全力栽培**凌玉灵（玉灵）**凝结元婴，数十年后以灌顶之法助其进阶元婴中期以勉强执掌星宫。
- 仇敌：**六道、万三姑**（昔年败将），因**元磁山**限制未能灭杀，决意除去。
- 对韩立的处置：拉拢入星宫则暗种禁制、取走虚天鼎给玉灵进阶；不愿则灭杀。女子自信独力即可对付持虚天鼎的元婴后期修士。

## 场景四：星空殿传送阵·拦截偷渡
- 韩立白天在坊市补齐材料、售卖鸡肋法宝换灵石，入住小客栈（露结丹中期修为）。三更潜入圣山五十层**星空殿**。
- 撞见一群面遮灰气的偷渡修士：**田姓儒雅男子**及其妻结丹女修**月仙子**、一名病重的炼气期少女（与月仙子七分相似，似其女）、四名筑基弟子。欲去**银鲨岛**求**白鹭鱼妖**内丹救女。
- 星宫执事**张姓修士**与**黄姓老者**收灵石、发传送符，私放七人。七人刚入阵，天星双圣女宫主（头戴白色纱巾的蒙面白衫女子）现身，灵光击飞逃遁的田姓夫妇，二人喷血现形。执事跪呼『参见宫主』。
- 女宫主无视众人，直点入口处暗中旁观的韩立。韩立现身（青衫修士，双手倒背，似笑非笑），女宫主神念一扫震惊失声：『**元婴后期**！』——满厅结丹修士目瞪口呆。
- 结丹女修**月仙子**一见韩立面容即露极度震惊之色，似与韩立旧识，但无人留意。

## 悬念
- 天星双圣女宫主专程等候韩立，凝重确认『阁下真的姓韩？』；韩立反问微笑，对峙未决。
