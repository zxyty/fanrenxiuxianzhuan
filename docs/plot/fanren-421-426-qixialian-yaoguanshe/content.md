# 第422-424章 七霞莲 / 剧毒 / 妖蛇

## 背景
第四卷 风起海外。韩立一行人（散修金青、胡月、石蝶/石仙子、简姓修士，及随韩立的曲魂）在乱星海一处古修遗址寻宝，遇上一根「封灵柱」。

## 封灵柱（第422章）
- 封灵柱为古修士封印灵气、隔绝外泄的巨型罕见法器。两种用途：封印珍稀灵药灵草，或镇压厉害鬼灵妖魔。
- 众人权衡后决意一探。简姓修士透露自己功法有专克鬼魔的神通；石蝶自称有家父所赐辟魔法宝；韩立则倚仗七十二口天雷竹炼制的青竹蜂云剑。
- 柱身需以巨力旋转数圈方能放倒。一向沉默的曲魂主动施巨力符，双臂虬筋、隐隐红光罩体，转动三圈推倒石柱，露出深入地下的白石阶。

## 七霞莲与白骨（第422-423章）
- 阶尽是二十余丈方厅，夜明珠照明，左右各有半圆侧门。
- 水池旁有一具洁白骸骨，头颅被一枚尺许翠绿小箭钉死在地。
- 池中一朵青红黄三色「七霞莲」花苞（碗口大），上空凭空浮现七色彩虹，池水为粘稠白色乳液（疑似千年石乳），异香扑鼻。
- 胡月解说：七霞莲只见于口耳相传的传说，凡人服之起死回生、白骨生肉，修士服之修为狂升甚至升一阶，乃仙家之物，拍卖必出天价。
- 韩立看出众人因忌惮其实力（噬金虫之威）而起排外独吞之心，遂佯装无意，转而细查白骨。

## 毒池圈套（第423章）
- 胡月支使简姓修士采莲、石蝶查禁制。提醒七霞莲只能用手摘、不可用法器，否则枯萎。
- 简姓修士发现花根异常结实，改拉为掐；韩立喝止未果。花苞缩入池中，巨力将简姓修士扯入池中，乳白水液四溅。
- 石蝶被数滴溅中，满脸黄泡糜烂、容颜尽毁，急服火红丹药。简姓修士坠池后音讯全无，旋即浮出一具千疮百孔、白骨淋漓、眼珠尽化的尸体。
- 韩立点破：千年石乳本无毒，是有人在池中另行下毒；并指白骨右手乌黑、食指被咬断的破绽。随即以火弹术五颗火球一字射向重新浮出的七霞莲。

## 妖冠蛇真相（第424章）
- 「七霞莲」真身为一条三四尺长、头生肉角的乌黑妖冠蛇，其肉冠被人特意炼成七霞莲模样，七色彩虹实为其口吐妖气。
- 妖蛇奇毒、吐七色彩珠（妖珠）与彩雾，令石蝶紫色云帕、金青白色大印、胡月双飞刀皆黯淡受制。
- 曲魂以血灵钻射出血芒在蛇肉冠打出血孔，韩立以青竹蜂云剑一剑钉住蛇首——刻意不放辟邪神雷，以免被认出此剑为天雷竹所炼。妖珠被逼落，由石蝶收入储物袋。
- 石蝶因容颜被毁，手握短剑将妖蛇一分为二。
- 韩立判断此处确是古修洞府，但早被他人捷足先登、借此布置陷阱专害白骨主人；此行实为失败的寻宝。

## 悬念（第424章末）
- 死去多时的简姓修士尸体竟手脚乱动爬出水池、含糊呻吟，直直走向众人。胡月喝止后尸体猛然扑来，被胡月、金青法宝齐出击爆。
- 旋即胡月背后一凉，一只洁白骨手从其腹部伸出，五指紧捏一枚豆粒大小、蓝灿灿的圆丹——章节于此收尾。

## 备注

## 第四卷 风起海外 · 第425章 脂阳鸟

### 场景
乱星海地底一处大厅（有通往下层的地洞台阶、石柱林立）。韩立一行与金青、胡月等修士同陷此地，遭埋伏袭击。

### 关键事件
1. **胡月之死**：结丹期修士胡月被一具白骨傀儡跳起暗算，骨手自其腹部贯出、捏碎其蓝色圆丹。胡月炼有三把飞刀，第三把以秘法炼成、通灵护体之能远超普通法宝且新近炼到通灵，本应感应杀机自动护体——但因偷袭者是『死物白骨』而失效。胡月含恨自嘲而亡。
2. **困局与脂阳鸟**：墙壁中飞出大片阴寒鬼雾，将韩立、曲魂、石蝶困住。韩立及时展开刚到手的画轴，放出无数浑身冒火、形似燕子的『脂阳鸟』分身，结成火环驱散黑气。曲魂周身血光罩体、双手放紫色火焰助阵；石蝶手捧白濛圆珠，珠光克制鬼雾。
3. **金青与白骨**：另一侧白色骸骨傀儡紧追金青，金青裹银芒倒飞躲避并以法宝轰击。金青怒喝一声震得大厅嗡响，鬼啼暂止、白骨被击得七零八落、鬼雾缩回墙内——实为怪影故意收手。
4. **石蝶之死**：石蝶趁乱逃上台阶，刚走两步即被石壁中窜出的碧绿鬼爪（十指尖尖、通体碧绿）贯胸，黑气包裹后被吸成一具皮包骨的干尸抛出。
5. **怪影现身**：四周传来阴冷冷笑『本座需要一具合适的躯体，你们三人谁打算奉献？』。韩立放出一只丈许长的巨型脂阳鸟撞向石柱，柱中飞出黑气怪蟒迎战，展开鸟蟒之斗。随后石柱中走出一碧绿色人影：浑身绿光晶莹、缠黑雾带子、双目滴血般鲜红，仅以目光便给韩立、金青极大压力，疑为元婴期修为；其状似鬼魅又言语条理清晰，非神智已失的厉鬼。

### 设定要点
- **脂阳鸟**：诞生于精火，专食阴鬼厉魄，著名阴物克星，早已在修仙界灭绝。画轴中仅封印其一分精魂，只能释放分身，威力远逊真身。
- **驱灵术**：乱星海中已失传已久的秘术，可操纵白骨傀儡、鬼雾等阴物作战；怪影精于此术。
- 韩立判断白骨并无灵识、乃被人操纵的傀儡，但屡击不损，另有奥妙。

### 人物归属（据世界状态须知）
- 『曲魂』的言行实为夺舍占据曲魂躯体的【御灵宗修士】所为，以曲魂之名随侍韩立。

### 悬念
怪影身份、修为深浅，及其『索要躯体』的真正目的；白骨傀儡屡击不损之奥秘。

## 章节概要（第四卷 风起海外·第426章 诡诈）
虚天殿之行接近尾声。韩立、金青与曲魂三人被神秘『绿影/怪影』困住。绿影言语戏弄、深不可测，最终以诡诈手段逐一料理三人。

## 经过
1. **脂阳鸟试探失败**：韩立无征兆展开卷轴，放出无数火鸟与巨型『脂阳鸟』白色火团围攻绿影。绿影以双手黑气化出两只独角墨蟒吞食火鸟，又一拳击碎脂阳鸟火团，逼出其残魂所化乳白晶珠并一口吞下。随后韩立的画轴无故自燃化灰，火鸟尽消。韩立确认对方绝非纯粹鬼魅之流，专克阴鬼的宝物被举手废掉，毫无底细可摸。
2. **绿影放走二人之诈**：绿影只对韩立的躯体感兴趣，扬言放金青、曲魂离去。金青因今日目睹众多修士惨死，首次生出惜命、来日方长之念，向韩立抱拳告罪后匆匆奔向石阶。
3. **金青之死**：金青奔过白骨堆时，散落骨架骤然如弩箭强射，白芒犹若无物般穿过其护体银芒射入体内，金青身体瞬间破烂、当场栽倒身亡——绿影言而无信。
4. **身世线索**：韩立放出曲魂（护体血光）并肩而立，体内另有数柄青竹蜂云剑待发。绿影见血光惊问『极阴还是极炫门下』，并怒斥自称祖师的极阴祖师为其『逆徒』，暗示绿影乃极阴祖师之师傅——疑可解开黑煞教与极阴岛功法之秘。
5. **辟邪神雷御敌**：绿影趁韩立被言语震惊时以绿芒分身偷袭，韩立反射性放出全部七柄飞剑（共九柄青竹蜂云剑，此为其控剑极限），剑光放出淡金色『辟邪神雷』电弧轰灭绿色光团。绿影由此认出飞剑以『金雷竹』炼制。
6. **曲魂倒戈与夺舍**：绿影真正杀招在此——韩立侧旁空气异动，被熟悉的血芒（血灵钻）穿透胸膛血涌如注，又遭混元钵黄芒猛击（被飞剑弹回现形）。曲魂犹若未闻地射向绿影，韩立心念虽在却完全失去对曲魂躯体的控制。韩立咬牙掐诀喝『收』，曲魂天灵盖飞出一小团绿光倒射回韩立体内消失；绿影所化长虹同时没入曲魂躯体。曲魂缓缓睁眼，露出一对鲜红似血的眼珠——绿影夺舍成功。

## 关键转折
- 韩立失去得力傀儡曲魂的躯体，身受重伤，孤身面对夺舍后的强敌。
- 引出绿影与极阴祖师/极阴岛/黑煞教的师承渊源。

## 第四卷 风起海外 · 第四百二十七章 玄魂炼妖

### 局势承接
- 韩立从储物袋摸出一张蓝色符箓，白光闪动止住涌血，重伤暂时控制。此前他遭曲魂以**血灵钻**偷袭，险些被在心脏处穿孔，全凭神识与敏捷远胜常人才避过必死之局。
- 血灵钻阴损防不胜防，但那枚已是曲魂此前修炼出的**最后一枚**，韩立不必再担心其阴毒。

### 身外化身被夺
- 韩立按「**身外化身**」之法，已将分出的神识与曲魂肉身炼到人神合一（**煞丹分身**），却仍被对面老鬼不知不觉控制，连一丝警告神念都未发出；幸而见机极快收回神识，才未被吞噬。
- 对方能附身，印证「**绿影**」并非人类，而是近似鬼魅与元神的**妖灵**。

### 引魂钟失效
- 韩立取出搁置已久、含曲魂精血的**引魂钟**敲响，想创造可趁之机。
- 曲魂红光闪动却纹丝不动，冷笑道：既已成这具煞丹分身之主，那些小禁制再无用。并道煞丹与分身之术原是他传给「两个逆徒」之物，使用此术炼制的分身他可轻易控制。
- 韩立不愿替「极阴祖师」背黑锅，淡称分身法得自一块玉简；对方不信，扬言擒下后抽魂炼神。

### 玄魂炼妖凝骨化形
- 曲魂招来一堆白骨重组骨架并融入己身，体表涌出碧绿幽火、漆黑鬼雾与自身血炼神光三色光芒。
- 体内豆粒爆响，躯体四肢收缩：原本高大的身躯矮成常人，面孔变作**眉清目秀的少年**，眼珠血色大减转为淡红——瞬间彻底变成另一人。
- 少年自承所修为元婴期才有资格修炼的密功「**玄魂炼妖**」：练成后不再受夺舍限制，可以**玄魂之身随意附于他人身上**；如今玄魂凝骨之术已成，欲拿韩立血祭其『重出之日』。十指变利、长出数寸碧绿指甲。

### 韩立列阵迎敌
- 抛出灵兽袋，数千只**噬金虫**化作金银色云霞悬于头顶。
- 放出三四十只**巨猿傀儡**环绕四周。
- 九柄**青竹蜂云剑**凝成丈许翠绿巨剑，剑芒不定、淡金电弧跳跃（**金雷竹/天雷竹**所炼）。
- 少年见『天雷竹飞剑、驱虫术、傀儡』神色大变，判断韩立或非极阴极炫之弟子，提议『好好谈谈』。

### 交手
- 韩立拒谈，点指放出粗如巨蟒、雷鸣阵阵的绿虹巨剑。
- 少年以一爪挥出五道半月黑芒迎击，黑芒被绿虹上弹射的淡金电弧瞬间击溃、烟消云散，少年暗惊金雷竹威力远超预料，却仍胸有成竹冷眼看剑近身。（待续）
