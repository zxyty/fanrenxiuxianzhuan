# 第1474-1476章 惊空魔撕空·联手破冥河禁制

## 一、引出惊空魔（第1474章）
- 四大妖王（六足、木青、白发美妇、两名血袍人）以巨大血球（血食）散发奇香，引诱『惊空魔』现身。木青在血球前念念有词主持血祭。
- 血球上方虚空浮现淡银色云霞漩涡，空间被挤压如薄纸般皱裂；韩立以**明清灵目**（天眼术）看清漩涡深处情形。
- 空间碎裂，伸出一对毛茸茸银色巨手，撕开百余丈空间，化出一个更大的黑洞。
- **惊空魔真容**：一颗数十丈大的银色头颅，表面疙瘩寸毛不生，眉间一颗凸出翠绿独目，长嘴满布细牙，鼻处只有两个黑窟窿；**无身躯**，仅双耳处伸出一对银色手臂。
- 惊空魔『咕咕』如鸡鸣的怪叫令周围妖物乃至四大妖王受制摇晃，唯独**韩立安然无恙**。
- 六足解释：韩立以**金雷竹宝物（青竹蜂云剑）为本命法宝**，又未在地渊久留、未沾黑暗气息，故不惧惊空魔的尖叫。
- 惊空魔被血食吸引，一口吞下血球，浑身染红后没入黑洞消失。

## 二、潜入冥河之地（第1474-1475章）
- 妖王真正目的：利用惊空魔撕裂的空间裂缝进入**冥河之地**。此前已多次血祭，且上次正是惊空魔出现在地渊，才让他们发现冥河之地。
- 六足、木青率先入洞；韩立在**金猿（金灵）**监视下硬着头皮跟上。空间裂缝如巨型隧道直通极远处。
- 木青在最后一批血食中暗藏**万年醉灵药**，可让本无自我意识的惊空魔昏睡数月，保证通道畅通。
- 妖王所图为**冥河神乳**；冥河之地极险，六足因事先听闻其事方敢冒险。

## 三、冥河与破禁（第1475-1476章）
- 出口外为**冥河**：乳白浑浊、左右无尽，水面寒风呼啸却平静如镜、死水一般。
- **冥河怪鱼**：体细如针，群居，对靠近水面的生灵不知疲倦地飞起攻击；危险性甚至不如冥河之水本身。
- 破禁需联手分开此河：
  - **六足**：背生两臂，持三角乌黑令牌（赤红雷火）与梭形蓝色宝物（蓝虹），口喷黑色光柱。
  - **木青**：金色巨花化数丈金灿灿巨轮，发风雷之声斩入水面。
  - **血袍人**：紫血傀儡涨百倍持十丈紫红血斧；十几只金属傀儡合成数十丈铁傀儡持巨刃。
  - **白发美妇**：聚八道黑影（阴甲鬼王）阴气，催动残缺漆黑小钟，放黑色光波撑起黑色通道光幕。
- **黑丝魔气**：冥水中浮现晶莹黑丝缠裹妖王攻击使其黯淡颓势。
- **韩立出手**：以辟邪神雷（祭雷术）凝金色圆球与金符，召出乌云中的金色骄阳，喷出粗若水缸的金色光柱击水，化金色雷弧四散，将黑丝魔气灰飞烟灭，助妖王不断推进峡谷。
- 一刻钟后通道全部打通，**可维持数个时辰**。

## 四、韩立的八门引雷阵（自保设计）
- 韩立祭出八块青色小旗成雷属性法阵将自己簇拥中心，对外称『稳定祭雷术并稍增威能』的辅助法阵。
- **真实用途**：自知施展祭雷术可能被**辟邪神雷反噬**，闭关时独创此**八门引雷阵**——以雷符凝成、具吸纳放出雷电之力的阵旗，可在反噬时将辟邪神雷之力暂时引导到法阵上，避免自身损伤；增幅之说只是随口掩饰。其奇特全在阵旗雷符，妖王亦看不透。

## 五、驱众入河
- 通道维持期间，妖王驱使数万**傀儡、阴甲鬼兵、低阶妖物**先入河；傀儡与玄鬼分列两边，低阶妖物居中。
- **阴甲玄鬼**：身披发亮半身黑甲、持戈叉长兵，肌肤灰扑如干尸，头罩绿色鬼火骷髅；八名阴甲鬼王为**炼虚中后期**。
- 低阶妖物一入黑色峡谷通道，四周光幕即射出无数银丝（怪鱼）专攻中间低阶妖物，对两侧傀儡与玄鬼毫无兴趣。

## 备注
- 『韩道友』『韩立』本章言行均为真实人物韩立本人，无化名。
- 金猿自称『金灵』，为木青属下（口称木青为『主人』）。

## 背景
地渊妖物大队（数万低阶妖物 + 八千银甲玄鬼 + 上万傀儡）在四大妖王带领下进入灵界『冥河之地』。韩立与金猿（金灵）混迹其中，跟在木青（青木）身后，受妖王体内标记节制。

## 第1477章 冥灵之晶
- **黑色通道**：通道两侧黑色光幕激射针状怪鱼所化银丝，对无护体灵光的低阶妖物致命；继而浮出『阴鬼』——无数奇形生灵尸体所化、无神智的僵尸鬼物，口带奇毒（水箭/撕咬），普通法术打在其身上尽数没入无效。六足喝破：阴鬼无法离开冥河范围太远，冲出通道即安全。冲入通道的低阶妖物约损失五分之一。
- **冥河之灵**：一截通道上空降下巨大黑影，巨蟒般触手破幕卷食妖物。六足、木青、白发美妇、两名血袍人五名合体期存在依事先演习合力出手——四只巨大光手抓住触手硬拉，将白色无鼻无目、巨口巨大肉球（冥河之灵）拽入通道，六足以黑剑光劈开取出**冥灵之晶**（乳白色鸡蛋大小晶石，发光）。六足称有此物『以后过那一关时可轻松多了』。血袍人解释：冥河之水一地只盘踞一只冥河之灵，大动静必惊动它。
- 高阶妖物随后通过通道，进入一处巨大无边空间：下方黑沙土地、数寸黑黄小草，头顶冥河之水形成白濛濛『天空』。队伍按『阴甲玄鬼在前、傀儡断后』编队前进，飞出黑沙地后见到一望无尽的白骨平原，腐臭弥漫。

## 第1478章 遭遇激战（连环鬼战）
- **阴骨怪/骷髅大军**：白骨荒原远处涌来灰白阴气，地面白骨耸起为红目骷髅腾空扑来。两名血袍人放血虹化血色飓风、白发美妇背后浮现披发巨大鬼影喷绿气——绿气卷过骷髅则灰气溃散、骷髅解体坠落，未待木青六足出手即扫荡一空。木青点评：阴骨怪是冥河之地最好对付的存在。
- **血色蝙蝠/人面血蝠**（两日后，血色沼泽）：蝙蝠碎尸可复原，需多次灭杀或火属性法术；最大的十几只蝠身人面、口吐赤红血光，高阶妖物亦难正面承受。四大妖王齐出。韩立以金色电弧（辟邪神雷）触之即化血气灭杀，由此明白木青等为何对辟邪神雷艳羡——冥河鬼物多含魔气，辟邪神雷克制犀利。
- 其后又遇：蓝雾中黑色鬼影（以鬼啸惑乱低阶妖物，被一只黑色大钟龙吟震散）；黑湖三头腐肉巨人（与紫血傀儡相争）。一个月后地渊妖物已不足万余，多为中高阶，鬼兵傀儡折损不少。
- **韩立单战**：以金光斩牛首鬼物、灰色光霞挡白骨长箭、乾蓝冰焰（五色寒焰）冰封青面恶鬼；闪避独眼绿毛巨尸十余丈狼牙棒后，以黑色小山化千丈巨山一举压杀巨尸及方圆百丈鬼物。
- **高原中伏**：后半月十余处凶地反常无鬼物。走到木青称『无厉害鬼物』的高原时，三十余只炼虚级厉鬼率遮天蔽日鬼物大军放浓阴雾，将队伍团团围住、出其不意冲散，韩立等各自为战。阴雾使人稍一远离即迷失方向，韩立大半日厮杀后与众人失联，杀不胜杀。

## 第1479章 脱逃之机
- 韩立循战声最弱处突围，仗明清灵目（天眼术）在阴雾中避战穿行；冒险升高数百丈即遭密集攻击射落，赖元磁神光与灰霞护身、风雷双翅退回雾下。数个时辰仍困雾中。
- 闻黑色火焰爆发与女子娇叱（似元瑶声），赶去见数百细长大头血红骷髅（喷血箭、克制阴风黑火）围攻百余阴甲玄鬼；元瑶（持黑色小瓶）、妍丽（持黑色芭蕉扇）放黑色火球死守。韩立以七十二口飞剑化数百金色剑光斩碎血骷髅，残骸聚成血红巨鬼又被其以乾蓝冰焰冻成冰雕、辟邪神雷金弧弹碎，一招了结，救下二女。
- 二女自陈：与韩立同样被冲散；所控阴甲玄鬼乃当年参与炼制、仅余此一队。韩立指出至今只遇低中阶鬼物，炼虚级厉鬼应都在缠斗六足等妖王——**正是脱离妖王掌控的良机**。
- **决策**：趁乱逃离六足、木青（鬼婆）等妖王，寻精纯阴气凝结之地为韩立驱除体内标记；冥河之地广大，妖王再寻如大海捞针。三人达成一致。
- 元瑶、妍丽透露：二人修炼鬼道功法**阴阳轮回决**，本身为半鬼之躯，能在旁门左道上脱离此鬼雾，让韩立随其同行（先前因一路遭鬼物袭击未能脱雾）。

## 关键决定 / 行动项
- 三人合谋脱离妖王队伍，由元瑶、妍丽以鬼道之能带路出雾。
- 目标：寻精纯阴气之地驱除韩立体内的妖王标记。
