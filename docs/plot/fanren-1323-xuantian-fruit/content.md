# 第八卷 初入灵界 第一千三百二十三章 玄天果实

## 一、啼魂进化成黑茧
- 兽室中出现一枚蚕茧状诡异黑物，通体黑幽幽光芒，茧上浮现独角三目狰狞恶鬼图案，与啼魂兽背后图案一般无二——此茧正是**啼魂兽越阶进化**所化（当年曾有过一次同样形态的进化）。
- 回溯斗法那日：韩立被摄入**阴魂空间**，空间内浓浓阴气、无数凶魂在一名高大**鬼王**带领下扑来。
- 韩立放出啼魂兽，以其天生**啖魂**神通（鼻中喷霞光）将普通阴魂尽数吞入腹中。
- 鬼王将身体修炼得犹若实质，啼魂霞光一时无法撼动其身躯，颇为棘手。
- 韩立动用**风雷翅**出其不意遁到鬼王身后，一道**辟邪神雷**将其击伤；鬼王负伤后再无法以秘术凝固精魄，终被啼魂趁机一吸而入。
- 鬼王实力堪比化神修士，吞下后啼魂原本进化的瓶颈当场突破，化为黑茧。
- 韩立顾不得斩尽凶魂，用**破灭法目**强行洞穿破开阴魂空间，带茧逃出。
- 现将巨茧搬入兽室，如上次进化般静等一段时间即可。

## 二、药园与玄天仙藤之毁
- 药园中九曲灵参、龙鳞果树等灵木，即便在灵界也是罕见灵药。
- 唯独韩立精心培养的**玄天仙藤**已不存：当年在人界一开花结果、果实被收走后，整根仙藤通体枯萎化为灰烬；更不可思议的是事先切下的数截灵根也同时枯萎消失。
- 韩立判断他人手中其余几截仙藤灵根亦不复存在。
- 推论：一界天地法则独一无二，不会有两个一般无二的玄天仙藤、也不会有两个相同的天地法则；灵界化神、炼虚等大神通者也只能稍稍顺应法则、利用下界元气，真正触及/掌控法则估计要**大乘期将度劫之人**或**天地真灵**般存在方可——足见玄天之物之逆天。

## 三、研究玄天果实
- 韩立于洞府密室布好禁制，取出贴满七八张禁制符箓的洁白玉盒，撕符开盒。
- 果实形态：数寸（半尺）长，细长扁圆、表面圆滑淡黄，一端微钝、一端如刀切般平整，似一扁圆小木棍；遍布墨绿不知名花纹，似金非金似木非木，触之微有弹性、温凉。
- 据负责培育的人形傀儡告知：果实原为一颗泛白光的圆形果实，经常年绿液催育不知何故长成此般古怪模样。
- **逐项测试，皆无效**：
  - 青色灵光点入即四散，无法侵入——此物排斥一切灵力注入；
  - 金光五指用力一捏可将其捏变形、五指深陷如面团，松手即恢复原状，弹性惊人；
  - 青竹蜂云剑所化纤细金（剑）丝绕过，表面毫无痕迹；
  - 辟邪神雷金色电弧劈上即反弹爆裂；
  - 抛入鸡蛋大银色火球（**噬灵天火**，阴阳交泰下形成的至阳至阴之火），火球化作**噬灵火鸟**现身，却死死盯着此物、目露惶恐、急促清鸣、害怕之极；
  - 取出**噬金虫**放其上，号称无物不噬的灵虫竟慌忙远飞、丝毫不咬；
  - 烈焰、极寒、风刃等各属性攻击及破灭法目，皆无反应，如同滚刀肉。
- **唯一有效**：韩立将全身法力狂注双目，瞳孔暴射深蓝刺芒（**灵目/天眼术**），透视见果实内部一豆粒大乳白色光点轻轻闪动、微弱异常，余无其他。
- 此后十余日苦思无果，甚至在密室布几个玄奥小型法阵欲借阵力祭炼，仍丝毫无效。

## 四、收获与悬念
- 韩立感慨此等逆天宝物若无机缘，宝在手中亦枉然；但经久违的无力、焦躁等负面情绪磨炼，**心境反进了一层**。
- 明日即返回**天渊城**报道。
- 章末，韩立目光落在先前未在意的墨绿色花纹上，忽然联想到另一物——其表面花纹形状、颜色与玄天果实有惊人七八分相似，神情古怪而匪夷所思，留下悬念。

# 第1324-1325章 混沌万灵榜·啼魂化形·入主丙五十六队

## 一、绿液滴养玄天果实
- 韩立取出神秘小瓶(绿瓶)与玄天果实对比，发现两者表面墨绿色花纹相似，怀疑神秘小瓶亦由某种『玄天之物』炼成——既可解释小瓶材质至今未明，又能解释其不可思议神通。
- 韩立开瓶滴一滴碧绿绿液于果实(『棍子』状)上，绿液青光一闪没入，溶入果实中心一点白光，白光随即粗大一分。
- 结论：唯有神秘绿液对玄天果实有效，韩立决定日后一有机会便以绿液滴养，徐图壮大。

## 二、混沌万灵榜·玄天灵物现世
- 与绿液溶入白点同一时刻，距天渊城亿万里外的人族『天灵境』一座万里高空飞岛中心，一块高达百丈、写满金银符文的玉璧金光大放、清鸣冲霄。
- 玉璧上方有十余丈银色牌匾，金色古文书『混沌万灵榜』。十几名绿袍修士齐聚观看：第三行以下金文下挪一行，空出的第三行浮现陌生金文——新晋玄天灵物『玄天斩灵剑』，竟排名第三。
- 众修士震恐，白发绿袍老者喃喃恐为灵界又一次『真灵之劫』降世。讯息随惊虹传向人界三境七地；灵界蛮荒各异族禁地的同类玉璧亦以各族文字显现相同消息。
- 人妖等各族高层暗中骚动，密令外松内紧、加强戒备。罪魁祸首韩立却毫不知情。

## 三、啼魂进阶化形
- 韩立感应牵动，赶往兽室，见黑色巨茧顶端碎裂、内里空空，石室顶悬一团漆黑乌云。
- 韩立喷出已炼化的鸣魂珠召唤，乌云抗拒(如当年六翼霜蚣欲脱控)，韩立全力催动法决压服，乌云化为黑影落下——竟是一名黑色长袍的『另一个韩立』，面容身材服饰皆与韩立无二。
- 确认即啼魂(化形模仿主人，因长期亲手培育、对韩立更恭敬)。韩立取出第二元婴(当初顺手凝回，仅元婴期神通，斗法已用处不大，宜寄付傀儡管理洞府)钻入啼魂体内，灌输语言功法约一顿饭工夫，啼魂遂能应答；第二元婴飞出时元气损伤不小。
- 变化之术展示：撕袍变两丈乌黑巨猿(即化形前本体)→ 再膨胀变背后图案上的红毛三角恶鬼(六七丈高，背生三根丈许黑骨刺，黑气阴森；韩立疑此骨刺源自当日吞噬的化神级鬼王)→ 缩为乌黑小猴。小猴撕食空茧吞下，亲昵窜上韩立肩，被收入灵兽环。

## 四、神秘小瓶在灵界的效用大涨
- 韩立查看噬金虫群后再观神秘小瓶：在灵界，生成灵液由七日缩短为三日；夜晚成液异象大减——昔日夺目奇景演变为仅放淡淡微白光，数丈外细察方能发觉，旁人多误为低阶法器。
- 推因：灵界天有七月、灵气远浓于人界。好处：催熟灵药提速过半，更隐蔽安全，外出亦可悄然动用。
- 灵地被人惦记，韩立将噬金虫、豹麟兽等重要之物随身收入储物镯，离开洞府返天渊城。

## 五、入主天渊城丙五十六队
- 韩立遁回天渊城，入当日传送进来的青色巨塔，依赵无归所言先于塔中『天甲殿』领取一套青冥甲(其威力曾在翁姓汉子身上亲见)穿上。
- 寻至丙队五十六队所在(转七八层，殿门书『丙五十六』)，以青色玉佩破白濛禁制入内。厅中两名黑甲修士：其一正是当日带他去飞灵殿的卓姓碧眼大汉——卓冲，另一为枯瘦三十许岁的东郭峰。
- 韩立出示玉佩确认身份，正式出任五十六队队长，二人参拜。卓冲介绍小队成员与巡逻任务：另有马道长、许仙子未到，余者静室打坐。
- 一道金光(上面金剑传书)飞入，韩立虚空抓得插淡红玉简的金色小剑。随即马道长(面容儒雅道士)进门，得知韩立为新任队长上前见礼；韩立赞其已修炼至元婴后期大成，距化神仅一步。

## 关键设定/伏笔
- 玄天斩灵剑、玄天果实同属『玄天灵物』，与混沌万灵榜排名相关，或牵动灵界真灵之劫。
- 神秘小瓶或为玄天之物所炼，其神通与玄天果实可能同源。
- 啼魂化形取韩立之貌(非冒名行骗，乃灵兽化形随主)。

## 背景：分派浮黎沼泽

- 韩立解读红色玉简，得出本队下一阶段巡查区域为 **浮黎沼泽**，碧眼大汉、马道士、东郭峰等闻之色变。
- 浮黎沼泽是天渊城划定的 **第一类危险区**：地形可掩蔽神念扫描，异族探子常潜伏其中监视天渊城，每年都有同道陨落。
- 该地极大，通常由 **四支队伍** 分头巡视，绕行一圈约半月。
- **两年前**：第十二队队长 **轩广**（化神中期）在此被 **影族** 偷袭陨落，另两名同道一同阵亡。
- 命令由长老会直接下达，无法抗拒。韩立态度淡然，吩咐次日出发。

## 队伍构成

- 队长：**韩立**（化神期，修炼过大衍决，对外以‘韩前辈’身份统领）。
- 队员：**碧眼大汉**（卓兄）、**马道士**（马姓道士）、**东郭峰**、**赢姓少年**（元婴后期大成）、**许仙子**（许姓女修，二十余岁，飞升修士嫡系后裔，性情冷淡），以及两名道骨仙风老者、一对中年夫妇——共十一人。
- 韩立暗中以神念扫视，对赢姓少年多看一眼；对许仙子莫名生出熟悉之感。

## 进入沼泽

- 众人经传送阵传出天渊城，出现在荒凉丘陵。天渊城面向蛮荒世界的巨墙无城门，进出全靠传送阵配合禁制，杜绝异族可乘之机。
- 目的地浮黎沼泽在万里之外，是一片朦胧雾气笼罩的山峦林木。
- 沼泽特性：神念无法离体百丈，迷雾幻象重重。
  - **圭虫**：蜈蚣状群居怪虫，表面遍布利口，可喷毒雾腐蚀法宝、吞噬护体灵光，常数十条乃至三四百条群涌。
  - **银头兽**：潜伏密林的古兽，可分裂身体、忽视斩杀，但行动缓慢且地域性强，出其地盘即不追。
- 韩立曾以元磁神光定住超大圭虫群、再以灵火尽数烧灭，令部下由怀疑转为信服。
- 每名守卫配发 **异灵盘** 法器，可在一定距离感应异族，但对身具特殊神通的异族不一定灵验。
- 深入二十余日未见异族踪迹，气氛稍松。

## 伪装陷阱：水潭翠竹

- 众人于一处百余丈清水潭、潭边翠竹处歇脚。潭中有银色小鱼，看似无害。
- 韩立察觉异常：暗中弹出十根红丝没入碎石地，随即一剑斩断翠竹，断口喷墨绿腥臭液体，地面震动、翠竹化为绿色肉须如章鱼触手抽来。
- 真相一：碎石地实为一头 **古兽**（圆乎乎、无口鼻、背生十余丈肉须），瞒过众人神念与异灵盘。
- 真相二：清水潭为一头 **透明巨兽** 幻化，银色小鱼凝成两枚银眼，水面现满布冰刃獠牙的巨口，扑向韩立。

## 蛮兽人之战

- **透明巨兽**被韩立以元磁神光化无数光丝洞穿缠缚，再喷出一座黑色巨峰（元磁山）涨大百余丈镇压入地，使其无法动弹。
- **古兽身上现出绿色怪人**：淡绿肌肤、无毛发、双目无情，下半身嵌入兽体——马道士惊呼为 **蛮兽人**（罕见异族），众人忌惮异常。
- 蛮兽人操纵黄色飓风（可吞没雷火等属性攻击）与绿色肉须（反弹法宝），令碧眼大汉等手忙脚乱。
- 怪人钻入兽体 **合体**，化为数十丈高、紫黑鳞片、碧绿灯笼眼的巨人，一拳击碎众宝光霞。
- 许仙子以本命白色冰剑虚空斩下，反被巨人黄风吹偏、大手擒住，危急。
- 韩立出手：先以碗口粗黑光柱击退巨人，再放出银色火鸟（**噬灵天火 / 银焰**）；以红丝化火索缚住巨人双足。
- 银焰扑上巨人背部，连其黄色飓风一并点燃，山岳般巨人转瞬被焚为乌有，尸骸无存。
- 噬灵天火之威令碧眼大汉等部下目瞪口呆。

## 收尾

- 韩立收回银焰火鸟与红丝火索。
- 命部下用铭刻银蝌文的乳白葫芦（临时收押异族法器，可吸纳非真灵级异族）收起被元磁山镇压的另一头古兽，回城交 **异灵堂** 发落。
