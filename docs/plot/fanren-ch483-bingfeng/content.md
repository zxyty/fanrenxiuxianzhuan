# 第四卷 风起海外 第483章 冰封

## 场景
虚天殿高台秘宝出世前夕，正魔双方元婴期老怪在高台上空对峙争夺。韩立、玄骨、乌丑围绕一件化为银狼的幻形古宝争抢，而极阴祖师、儒衫老者、万天明等在上方因夺宝失败而对峙。

## 银狼古宝之争
- 一头双头怪兽两颅向中扭挤、打滚后化为**独角银毛巨狼**，自银色独角射出半尺大小的**银光弹**：脱角即发刺鸣，一闪即逝在黑蟒身上击出大洞，最后射在绿网上爆裂；绿网一触即化为灰白之气消散（银光弹属性不明）。
- **乌丑**急掐法决射出黑光，将即将溃散的黑蟒大洞弥合、身形重新凝固。
- **玄骨**喷出一把数寸长、短而无柄、刀芒伸缩不定的**碧绿色小刀**。韩立感应到此刀实为**金雷竹小箭**以法术遮面所化——正是当年**极阴**用以暗算玄骨之宝。玄骨竟敢当极阴之面使用此宝，令韩立惊疑：莫非银狼另有古怪，值得老魔冒此奇险。（韩立能识破，因其本命法宝亦为金雷竹所化）
- 银狼骤缩为尺许迷你小狼，自黑蟒半合的大洞遁出；玄骨小刀迎击，砰然对撞后银狼与小刀同时弹飞，银狼身硬如铁丝毫无伤。
- 银狼倒飞恰冲韩立而来。韩立抛却顾虑，以**花篮古宝**祭出相困，又自储物袋取出**五枚连成一串的铜环**，喷青气后铜环隐没；银狼刚止身，四肢头颅浮现铜环并一紧，如中箭飞鸟坠落。花篮化白气将其包裹飞回。
- 铜环为银狼克星，将其锁死不能动弹。韩立揉狼首后以青光收之，银狼化为虚影，显出一柄**玉如意**——幻形通灵的古宝。

## 各方反应
- 乌丑气急败坏（本以为凭玄阴魔功取此无主之宝手到擒来，却接连被玄骨抢先、被韩立轻易收去）。
- 玄骨懊恼困惑，盯着银狼身上的铜环多瞅两眼。
- **极阴祖师**暗皱眉，但面临大敌万天明无暇顾及；认为宝在韩立这小辈手上无妨，事后自会让其交出。极阴提防万天明的**天罗雷珠**（少有能直接伤害元婴的器物之一）。

## 乾蓝冰焰封台 · 虚天鼎将出
- 高台剧烈晃动，仿佛地面将塌。一团夺目**蓝色火焰（乾蓝冰焰）**自洞口露头，以洞口为中心绽放蓝色巨花蔓延全台，地面迅速结起蓝盈盈诡异冰霜。
- 韩立、玄骨条件反射飞遁升空；**乌丑**稍一迟疑，蓝冰沿双足全身冰封而上，玄阴魔气丝毫挡不住，双足牢冰封于石台。
- 眼看乌丑将成冰雕，一道纤细黑光自天而降射中其身，黑色火焰大起、蓝冰消融——**极阴祖师**以**天都尸火**随手相救，冷哼：「小心点，下次我可不一定有机会救你了。」乌丑死里逃生飞起。
- 流传不知多少年的**乱星海第一秘宝——虚天鼎**，终于当着正魔数名元婴期修士之面即将出世，连极阴这般阴沉老怪也全神贯注于洞口。
- 三只护宝灵兽**血玉蜘蛛**与一对**火蟒**已精疲力尽、身躯微颤，唯其红光能排斥寒光；洞口尽成蓝光世界，老怪们不敢飞低，韩立等早退至离洞口二三十丈外远观。（待续）

## 备注
- 据【世界状态须知】，第482章起标注「乌丑」言行实归极阴老祖（夺舍）。但本章中乌丑（在高台被冰封）与极阴祖师（在上空出手救乌丑）为同时出现、彼此互动的两个独立行为体，故按文本忠实分别记录，未强行合并。
