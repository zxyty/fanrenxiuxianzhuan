# 凡人修仙传 第2293-2298章｜青元之劫

（第十一卷 真仙降世 第两千三百一十二—二千三百一十七章）

## 一、炼制极元晶（前情收束）
- 韩立在飞灵族大乘**越隆**洞府帮其炼制**极元晶**：越隆喷精血催灵焰熔炼，韩立以石阵+越隆所授黑色丝网拦截『天地金力』，按需放下，控制炼制节奏。
- 黑云中金力积至七八百道时韩立首现凝重，遂投入越隆所给三颗**银珠**之一——银珠为专克天地金力之异宝，一爆将黑云中金力尽数同归于尽。
- 历十余日，金、木两种极元晶炼成（上百块淡金晶砖熔为金属性极元晶）。越隆豪爽各分韩立一块，挽留被婉拒。

## 二、青元子渡劫准备
- 地点：**地渊空间**某隐秘山谷。谷中立翠绿七层巨台，顶层插上百幡旗组成法阵、置银色蒲团；谷内上千座半跪雕像、满地五色玉砖。四周万里布数百禁制。
- **青元子**：地渊隐居大乘，精阵法，以秘术苦苦拖延劫期、以最佳状态迎劫。本质为『异族元神侵占长元族人肉身』修炼至今（见敌方自述）。
- **元瑶**：青元子义女，原半人半鬼之身被秘术转为常人并速进**合体初期**；身具『天阴之体』，将施罕见大神通助渡劫。
- **妍丽（妍儿）**：青元子徒弟，**炼虚后期**；非天阴之体，无法直接助劫，受命主持谷外『十八天煞阵』。
- **十八天煞阵**：十八座连环相扣法阵，须同破或逐座破解；第一座相当于十八座威能叠加，最难破——青元子镇定之凭。

## 三、大敌降临
- **黄元子**：长元族人，原姓被弃、改名图报仇——其嫡亲族兄肉身被青元子（异族元神）侵占，欲夺回族兄肉身。掌『**九幻如意门**』大神通（可松动界面之力、破阵），备三颗『**子母尸阴雷**』、玄天残片凝成的玄天残宝晶环。
- **不灭天尊**：身高两丈→可化小山巨人，头生弯角、背生骨刺；身具『**不灭之体**』，重伤血丝交织瞬愈；以变化之身硬撕虚空入地渊。
- **三全道人**：青年道士，面带墨绿蛇纹、瞳孔竖立如蛇瞳；阵法大师，催三面巨令牌幻化三四百杆小旗、放『化血炼蛇』三色小蛇。
- 三人受黄元子之邀联手，预定三日内扫平谷外法阵，待青元子渡劫元气大伤时联手诛之。

## 四、韩立入场与窥视
- 韩立携**血魄**（黑袍女子，本体被封印于他大陆）等乘**魔灵圣舟**，用**玄天斩灵剑**斩开地渊空间障壁进入；凭青元子当年所赠地渊地图辨位，知离青元子洞府不远。
- 感应到大天劫波动及三股不加掩饰的大乘气息，韩立以**灵目/天眼术**喷青色晶球+黑竖目放黑丝远程窥视，看清三大乘破阵之景。
- 不灭天尊察觉并一拳破其秘术。三人推断窥视者非妍丽（炼虚、已被重创）、非青元子帮手、非地渊其他大乘（半年前已被青元子引出），决意强破阵一探。
- 妍丽以青元子所赠传讯符（原韩立留给元瑶）联络上韩立，惊知第一座天煞阵已被破，符自爆中断。

## 五、谷外大战
- 韩立赶到，自陈与青元子有交情，劝三人退去未果，黄元子放话『谁坏我事灭谁』。
- **首回合**：不灭天尊留傀儡化身、本体隐匿偷袭，韩立反手金掌硬撼黑拳，将其震退十几步——肉身相碰不灭天尊反吃大亏，三人骇然。
- 三方联手：黄元子喷黄丝、三全道人三蟒首、不灭天尊化巨人踩落。韩立放灰色光幕挡黄丝、现**三头六臂梵圣金身**（梵圣真魔功+多重秘术，已近极致）：
  - 金身按住不灭天尊巨人、紫金拳碎其墨绿战甲与胸骨，将其砸入地面数丈；又以青色长剑（剑气剑山）斩出数道深痕——但不灭天尊以『不灭之体』瞬间愈合。
  - 三全道人以三巨令牌幻化幡旗困阵+黄元子赤红玄天残宝晶环偷袭。韩立化**百余丈金毛巨猿**，震碎大半禁制与困阵幡旗、崩开晶环（毫发无损）。
  - 三全道人放三色化血炼蛇，两条被弹飞、赤蛇咬中巨猿指尖后被捏爆；三全道人却冷笑称韩立已中血毒，片刻将精血凝固。
  - 巨猿反掌再压不灭天尊入地；不灭天尊变身生骨刺暴起再扑。韩立放出**噬金虫王**（紫金小人，喷晶光剑光没入黑风），黑风中一声惨叫、血雨洒落。
- （战局未结，承下章）
