# 第1825-1826章 万灵台

## 一、序幕：六翼道友与虬须大汉的逆灵之约
- **白袍青年（六翼道友）**：已化形开灵智，**炼虚后期巅峰**，欲借**逆灵**之法成就**真灵之身**。
- **虬须大汉**：身份神秘的异族大能，自称是唯一能助对方修成逆灵之人；因魔劫降临须留族中准备，无法亲随。其条件是白袍青年大成后助其渡即将到来的天劫——『合则两利』。
- 计划：白袍青年吸收**玄阴晶壁**中阴气后突破合体瓶颈，再孤身入**蛮荒世界**（风元大陆），专杀寒属性古兽、吞噬妖核、吸至阴之气，千年内冲击**合体后期**，进而谋真灵之身。
- 一股黄风卷过，二人消失。**两月后**场景转至韩立。

## 二、万灵台大典背景
- **三境**所有真灵世家每三千年聚于**万灵台**——四面环山的巨大山谷，中央方圆里许青色巨石圆台，边缘高大阵旗布百余丈光罩。
- 三四十堆人群（世家）环列，最前排大圈为**五大真灵世家**。
- **谷家**：上一界排名第五，五大世家之一，得居最前。韩立作为明显『助拳』性质的**临时客卿**出席，本应到场的太上长老**天黎仙子**（合体中期）缺席，引各家关注与试探。

## 三、韩立立威·小惩陇东
- 谷家一方：谷家之主**晓风仙子**、唯她马首是瞻的**萧长老**，身后排列谷家化神元婴级弟子。
- 十余道目光（含夹带神念者）肆无忌惮打量韩立。韩立先隐忍，后捏碎金色酒杯化粉，体表浮现一层**银色火焰**逼退恶意神念，再冷扫五名合体存在。
- 其中**陇东**（陇家少主，嘴角鲜红血痣）用极怨毒目光盯视——当年木族之行被韩立坏事，未得叶家天凤真血，反失大半真龙之血。韩立以**天眼术（瞳孔刺目蓝芒）**令其脑中剧痛惨叫。
- 陇家老祖反手一掌金光救下陇东，面无表情质问韩立；韩立毫不给面子回『眼神不敬，略给小惩』。陇家老祖竟不追究，称孙儿心高气傲该惩，遂闭目。

## 四、陇家老祖的盘算
- 老祖传音晖姓修士：当年九仙山与其动手的神秘妖猿，是为一名被误带到陇家、却是『韩道友记名弟子』的炼气期小辈而来——疑韩立与**圣岛**有关。
- 结论：魔劫之际不宜竖敌；韩立能如此快进阶合体，必有莫大神通或奇宝。但大典排名比试照常出手，正好试探其真正实力，若有大神通『那件事』或可算他一个。比试由晖姓修士出面。

## 五、各世家暗谋
- **叶家**：老祖（五色羽衣的羽衣少女）戏谑孙女**叶颖**（白衣少女，与羽衣少女有五六分相似），笑言韩立不肯入叶家、不肯娶颖儿是眼光差。
- **丰家**：华服男子与青铜面具怪人（均合体初期）商议，天黎缺席便对谷家、对韩立动手消耗其法力，苦战交给劳家。
- **劳家**：老祖密令——遇他家放水，轮到谷家则不惜元气竭尽全力，视此为劳家大兴良机。

## 六、大典开启
- 韩立把玩一颗拇指大、忽隐忽现仿佛无实体的**金色葫芦**，爱不释手。
- 两个多时辰后，万灵台中心浮现乳白法阵，狂涨至亩许，符文嗡鸣；四周山壁射出数以百计粗若水缸的五色光柱；峡谷上空乌云密布、紫金色（粗若巨蟒）**天劫之雷**电弧浮现，与光柱组成巨大电网牢笼。
- 陇家老祖（在场修为最高）起身称『万灵大人就要出现』，受众家推举主持祭奠仪式，命各家奉上祭品。
