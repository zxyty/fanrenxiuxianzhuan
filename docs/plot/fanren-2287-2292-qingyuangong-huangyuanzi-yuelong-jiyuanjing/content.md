# 凡人修仙传 第2306-2308章 剧情梳理（第十一卷 真仙降世）

## 一、青元宫立、韩立远游（第2306章 黄元子 前段）

- **青元宫成立**：韩立将自己居住的银色宫殿命名为『青元宫』，坐落于元合岛（无涯岛、风元大陆一带）。门下弟子已数千，多为器灵子等人旧收门人，及三年间海大少等去人妖两族新招者。韩立计划每隔十来年扩充门人，志在数百年成普通大宗、数千年后岛上化神炼虚层出不穷，潜力者乃至合体可期。
- **托付与远游**：韩立当众宣布离岛远游——岛上闲杂事交『月天』负责，青元宫内事务交『银月师姑』处理；海大少、蓝药等弟子躬身领命。
- **银月留守**：银月已跌落炼虚境界，但神通仍在且有合体期灵躯化身，自称遇一般大乘有自保之力，请求同行。韩立以其灵躯化身刚转移真元、须巩固十余年为由坚拒，命其留守青元宫。
- **启程**：韩立携朱果儿（文中亦写作白果儿）、花石老祖（话石老祖）登『墨灵圣舟』（漆黑巨舟）腾空远去，自言顺利则百余年可返。

## 二、黄元子谋诛『青元子』（第2306-2307章）

- **碧原小筑**：翠绿草原上，三全道人以三颗巨蟒头颅（青黄红三色霞光）一口吞尽万头牛头马身兽群。黄元子（枯瘦黄袍男子）破其禁制现身。
- **九幻如意门**：黄元子凭一座迷你牌楼状法宝轻易入内——此为『九幻如意门』，九幻老怪的本命法宝。黄元子以三颗『长元丹』（长元族圣药，每颗可推迟天劫三千余年，三颗可为九幻老怪多争万年准备）说服九幻老怪相借。
- **冥狱真经为饵**：长元丹打动不了三全道人，黄元子改以『半部冥狱真经的线索』相诱，三蟒首大惊变色，显见此物对其极重要。
- **结盟目标——青元子（即韩立）**：黄元子坦言此行求三全道人助其对付大敌『青元子』，并已另请『不灭洞』的『不灭天尊』。黄元子与青元子有不共戴天之仇（渊源讳莫如深，二者名号『黄元子/青元子』似有关联），称青元子是大乘中较厉害者，一对一自问非其对手；但青元子马上要经历一场大天劫，正是千载难逢的灭杀良机。
- **三全道人考虑**：三全道人化形为满脸蛇纹、瞳孔倒竖的青年道士，称同阶难杀（打不过可逃），需三日考虑。黄元子限三日内答复，否则改请『血怨峡』的仇怨道友（修炼吸髓秘术，对付同阶有奇效）。黄元子另有金碧辉煌宫殿、宫装女子称其『主人』。

## 三、韩立入蛮荒、飞灵族遇越隆（第2307章 后段-2308章）

- **同行血魄**：离无涯海两年多后，韩立乘墨灵圣舟入蛮荒世界，途经飞灵族区域。同行的白衣冷女、眉间鲜红符文者为『血魄』——冰魄仙子的血魄化身；两年前韩立先去许家将其接出。
- **赫连商盟与跨大陆传送**：血魄告知，赫连商盟为灵界少数横跨三大陆的隐秘势力，实力不逊角蚩等超级大族，拥有跨大陆传送法阵，每隔多年拍卖传送名额，今年正逢拍卖；外人参加需有人介绍且具身份，或合体以上修为可无条件参加。韩立欲借此传送进『雷鸣大陆』，并需在该陆停留寻一样东西。
- **越隆现身**：墨灵圣舟惊动飞灵强者，无人敢拦。第五日，飞灵族大乘『越隆』以千余丈九头怪鸟（九头狰狞、五彩翎羽）现身，灵压不逊巨舟。得知是『在魔界截杀螟虫之母的人族韩立』后大喜，化形为披斑斓羽衣、拄黄木拐杖的两丈丑陋老者，热情相邀。
- **救命渊源**：越隆好友为长身族『袁飞』（飞灵族近邻），袁飞曾应邀去魔界对付螟虫之母、被困始魔之地，蒙韩立救命并见其斩杀螟母神通，归来再三提及。袁飞此时因损元气正闭关。
- **越隆洞府**：韩立随越隆横跨数百万里至一盆地，中心为千余丈高、百余层的巨大石塔（越隆七大洞府之一）。

## 四、极元晶与重元塑灵（第2308章）

- **款待**：越隆以『琥元酒』（七七四十九种灵草酿制，恢复法力兼清神醒目）相待。
- **极元晶**：越隆出示三块巴掌大晶砖（深黄/浅蓝/赤红，对应土火水属性），称为自炼的『极元晶』，纯净胜极品灵石。韩立以晶丝、青丝（剑光）勘验，得出：所含灵气约同体积极品灵石四五倍，含『先天灵纹』、炼法器或可形成特殊神通；剑痕被淡蓝光晕迅速自行修复——
- **重元塑灵**：此材料具『重元塑灵』奇效，所炼宝物除非本体彻底飞灰湮灭，否则只要时间足够皆可自行修补如初，远胜需主人真元长期培育的本命宝物，价值极大且坚韧异常（能硬接韩立剑光一击不断）。
- **越隆所求**：越隆历数千年苦工只炼出水火土三色极元晶，欲集齐五行炼成『五行至宝』以抵挡下次大天劫；但金、木两种天地之力威能为前三者数倍，已非他能抵控（试两次即不敢再试，人多亦无用，须一人独控）。故请韩立（最合适人选）助其引控天地之力炼出金木极元晶，事成以三块极元晶为酬。越隆就『为何不请旧交』『是否有大危险』『韩立能否胜任』逐一解释（金木天地之力远强、做好准备可将危险降至最低、需修为神通更高者方能掌控）。

## 待续 / 悬念
- 三全道人是否在三日内应黄元子之约；『青元子=韩立』的杀局正在成形，韩立尚不知情。
- 韩立是否答应越隆炼极元晶；金木天地之力引控之险。
- 韩立赴雷鸣大陆所寻『一样东西』为何、赫连商盟拍卖跨大陆传送名额。

## 背景：第十一卷 真仙降世 第2309-2312章

地点：灵界·风元大陆。飞灵族大乘老祖**越隆**居于一座巨塔。

## 一、炼制极元晶的交易
- 越隆此前已花极大代价请族中占卜师耗费寿元，准确预测星辰变化、引下天地之力，炼成数块极元晶。
- 金、木两属性的星辰变化极难出现，越隆**必须在所剩一年内**炼成金木极元晶，否则下次出现不知多少年后。
- 越隆原打算冒险请一名声不佳的老怪物相助，恰遇韩立（信其曾赴魔界斩螟虫之母、救灵界众修，诚信可信），遂改求韩立。
- **韩立条件**：三块极元晶当作预付定金现交，新炼成的另两种属性极元晶各要一块。越隆毫不还价应允，并称引来天地之力的灵地就在附近。
- 越隆请韩立在塔中住一晚，次日同往灵地。

## 二、打听五光族奕姓老者
- 韩立向越隆打听一名出身**五光族**、身具五色神光神通的飞灵族道友。
- 越隆大惊：五光族近百万年仅出一人，此人刚入大乘便离族远行、音信全无，族中以为已陨落。
- 韩立称多年前在蛮荒某险地偶遇此人一次，知其**姓奕**、见过其**五色孔雀变身**，未深谈。
- 韩立看出越隆笑容僵硬、与该五光族大乘关系不简单，遂随口岔开话题，**对奕姓老者真正下场只字未露**（按世界状态：奕姓老者已于第2269章死亡）。

## 三、再见金悦
- 飞灵族侍女禀报天鹏族金长老求见，越隆怒斥不见；韩立认出乃**天鹏族大大长老金悦**（昔日有数面之缘、有过交易），请越隆放她入内。
- 金悦初未认出韩立——当年韩立为族中『圣子』、其心目中应是炼虚期且早该陨落于地渊，如今竟与越隆并驾齐驱。
- **金悦诉冤**：半年前赤融族无视族规，侵占天鹏族十几条灵脉、数座城池，掠走该区域族人；金悦亲往交涉反被赤融族数名长老联手击伤，养伤半年方来申辩。
- 越隆称三月前赤融族大长老亦来哭诉，反指天鹏族一年前先侵其领地。越隆以全力应对下一次大劫、不理会鸡毛蒜皮为由打断，并以恐怖气息镇压金悦。
- **韩立出手**：以波动化解压在金悦身上的无形巨力，看在情面为天鹏族求情；越隆笑允，称会抽时间调解两族之事。
- 金悦终认出韩立，惊骇施大礼（『韩老祖』）。韩立传音：『此次相帮后我与贵族恩怨了清，以后不必再来找我』，随即掐诀施**雷阵术**，十几道银弧化小型雷阵，将金悦**虚空传送**出巨塔（落于远处一普通建筑上空）。越隆赞『好一招虚空传送』。

## 四、灵地·金罡风魔阵·引天地之力
- 次日，一青一黑两道遁光出塔，大半日后落于小型山脉两山间的洼地。洼地寸草不生、遍布土坑，被前两次天劫轰击而荒凉，中心立一根百丈高、锈蚀残破的青铜巨柱——名**金风柱**（越隆特意打造的引金属性之力辅助器物，上次失败受损未收回）。
- 越隆挥手放出上百根崭新青铜巨柱，掐诀布成法阵：**金风柱共一百零八根，组成金罡风魔阵**，可大幅削弱所引天地之力大半威能。
- 越隆又布下七八种玄妙禁制，并喷出五件宝物：白色细颈玉瓶、紫金葫芦、青色罗盘、淡银色鼎炉、赤红羽扇。
- 越隆另抛给韩立三颗淡银色圆珠与一面黑纱丝网（抵挡天地之力之宝，即昨日所诺两件宝物）。
- **星辰变化**：青色罗盘幻化漆黑星空，两个时辰后数颗星辰光芒大放，越隆开始引发天地之力。
- 越隆操作流程：紫金葫芦喷出沉重金粉 → 赤红羽扇自燃成炙热飓风卷起金粉升空 → 喷数团漆黑精血化黑色魔焰助风火，化作直冲九霄的黑色火柱 → 高空金粉化斗大金色符文、铺成超级符阵 → 天空雷鸣，万千白色光球（流星）坠入金色符阵被吞噬 → 符阵酝酿更恐怖气息。
- 银色鼎炉胀至百余丈，喷出各色散发金灵气的材料悬浮待炼。
- **金光坠落**：金色符阵喷下水缸般粗的金光（内有锋利晶丝）。越隆催动一百零八根青铜柱化为青铜甲士傀儡，举法器爆出狂风（带金戈煞气）罩住洼地、削弱金光近半，金光射入巨鼎；巨鼎钟鸣巨颤。
- 金光由单道→两道→三道→四道渐次密集坠落，越隆渐感吃力（黑色霞光闪动）。
- **韩立出手**：三道金光时，散发深不可测灵压；以**辟邪神雷**弹出粗大金弧、瞬移至金光前一分为三击之削弱；四道金光时，倒转淡黄色葫芦倒出上千晶莹沙粒，于巨鼎上方虚空化出磨盘大晶石组成的巨大石阵，金光击中石阵中心被斩、削小数倍方勉强穿过。
- 越隆以细颈玉瓶爆出灰白色灵焰、化三十余丈灰白火蟒于鼎下燃烧，再投黑色精血使白焰转淡金，加速熔炼鼎中材料。

## 关键关系与归属
- 越隆与韩立为交易/合作关系（炼极元晶）。
- 天鹏族、五光族、赤融族均为风元大陆飞灵族下属分族；赤融族与天鹏族因灵脉领地相互侵占而敌对。
- 韩立隐瞒奕姓老者死讯，仅作旧事询问。
