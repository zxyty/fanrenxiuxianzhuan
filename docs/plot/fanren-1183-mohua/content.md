# 第七卷 纵横人界 第1183章 魔化

## 背景
韩立星夜赶至万丈魔渊，欲擒回此行的‘心腹大患’——失控出走的第二元婴。第二元婴借一具天煞魔尸躯体行魔气灌体之事。

## 半魔化的真相
- 万丈魔渊魔气虽不及昆吾山镇压之魔气，但魔化程度远超第二元婴预料。
- 第二元婴因顾及韩立寻来，在灌体最后阶段心急加快魔气灌注。
- 它虽有大衍决在身，但忙于修炼玄阴**(玄阴功法)，只练到大衍决第二层，神念虽强于同阶却难以长时间抵挡魔气。
- 结果魔尸之体承受住魔化，元婴却在将完成灌体时未抵挡住魔气侵入，迷失心智。
- 此刻魔尸成了只知杀戮、依本能活动的半魔之物，无灵智(腰挂储物袋却赤手空拳应敌)。

## 战斗经过
1. 黑影(魔尸)鲜红双目满是暴虐，魔臂抓入韩立银盾光幕半尺；韩立以数尺长金剑(青竹蜂云剑)反手斩落半截毛臂。
2. 韩立辨明对方仍是天煞魔尸大概样子，但黑袍褪去露尸毛、要害生漆黑甲片、脸罩黑气五官狰狞、伤口不流血，确认其神智已失。
3. 韩立取出三焰扇(三色羽扇)欲强行化去魔躯。魔尸聚魔气瞬间再生手臂——‘不灭之体’神通。
4. 三焰扇放出三色火鸟，魔尸吐漆黑光柱硬接，竟以元婴中期修为僵持，令韩立意外。
5. 三色光晕压过黑团；韩立背生青白二色风雷翅(二次炼化)，瞬移至魔尸背后虚空一拳青光巨拳将其击入三色光晕。
6. 魔尸黑气被化去、躯体消融瓦解，被三色符文淹没。

## 第二元婴脱逃
- 三色光晕被血光击穿豁口，一道乌光射出，化作浑身黑绿、手持漆黑小幡的第二元婴，目光清澈、神智彻底恢复。
- 韩立风雷翅消失欲追；第二元婴将当年韩立亲手修复的阴罗幡抛下自爆，迸出无数黑色光丝结成方圆数里天罗黑网，自身破空高飞。
- 韩立现形，金光初斩黑网不破；以金光中辟邪神雷(金色电弧)再斩，撕开大洞，化青虹追击。

## 魔渊出口异变(并线)
- 万丈魔渊海面出口处，数十名修士持阵旗、阵盘分队守在白色封印旁，灵光已被激发。
- 封印突传巨响，一口数丈长漆黑巨剑洞穿封印，划开十余丈长口子，一团乌光从中激射而出。
