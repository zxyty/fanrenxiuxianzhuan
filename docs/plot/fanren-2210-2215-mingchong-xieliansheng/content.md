# 凡人修仙传 第十一卷 真仙降世 第2210-2215章

章节范围：第2210章《螟虫》、第2211章《黑葫城》、第2212章《威震黑葫》、第2213章《邪莲圣祖》、第2214章《白光界》、第2215章《真相》。

## 一、再入魔界，初遇螟虫（2210）
- 韩立、银月、蟹道人经传送大洞再次进入魔界（文中魔界与『圣界』混用，指同一界面）。韩立此时已是大乘期，外加蟹道人相助，自言面对魔界三大始祖之一亦无惧意。
- 落点为一处沼泽，附近石制建筑（数百间房屋、数座防护法阵）凌乱空无人迹。三人决定先寻最近魔族城池问明所在。
- 西行半日，遇数百名高低魔族、魔兽拼命逃窜，身后是由无数狰狞怪虫组成的灰色虫海。蟹道人判断此虫『魔气不纯，非魔界本土魔虫』。
- 韩立断定怪虫与『螟虫之母』有关——即当年制造魔界大劫、逼得银月祖父（傲啸）与莫前辈（莫简离）入魔界之虫母，并推测可能只是从封印偷跑出的虫母后裔。
- 韩立放出三座迷你极山，化为万丈巨峰迎击：黑峰喷灰色霞光令怪虫凭空消失（并能瞬移突袭），青峰射无形剑气，五色光焰峰以光晕压成肉酱。顷刻荡尽大半虫群，并擒获一头化神后期威压的双首巨虫。

## 二、黑葫城问讯（2211-2212）
- 逃亡魔族中乱云山采晶客（一名魔族女子）献上区域地图，告知最近城池为北面『黑葫城』。
- 韩立对双首螟虫施搜魂之术，发现螟虫无真正灵智、仅有本能意识，搜魂无果；遂爆碎虫首、赤焰焚尸。银月与韩立据此判断：纯靠本能的螟虫纵然数量再多，也难真正威胁大乘存在。
- 一日后抵黑葫城（三四百里大），城正被一望无际的灰色虫海四面围攻，黑色光幕将熄、上百高塔黑色电弧（防御手段）耗尽停摆。
- 韩立决意保城：祭出三座极山，催动三头六臂『梵圣法相』发龙吟啸声（无形音波令大批螟虫自爆/坠落），抛洒银焰火海；蟹道人化银色电弧扫荡巨型螟虫。二人之力硬生生扫空虫海，城中数千万魔族欢呼，奉二人为『圣祖』。
- 城主葫岩（皂袍枯瘦老者，合体中期，魔尊，任城主千余年）赶来拜谢，单独向韩立交代情报。

## 三、始印之地与圣祖失踪之谜（2213）
- 葫岩供述：螟虫乃普通魔虫感染螟虫之母从封印泄露的气息所变异，近百年现世，越灭越多，屡致屠城，圣界动荡。
- 三位始祖（元魇、六极等）与众圣祖前往『始印之地』应对螟虫之母，初期借异界强者之助封印持续加固；但约二十多年前始印之地忽与外界失联，四周聚起数之不尽的螟虫群，出现实力不下合体的高阶螟虫，外人无法闯入。
- 留在外界、最近黑葫城的圣祖为万花山脉『邪莲圣祖』。葫岩献万花山脉地图（邪莲居中心『朝天峰』）。

## 四、邪莲圣祖与白光界（2213-2214）
- 一月后三人抵万花山脉（景色秀美似人界），朝天峰被障眼法伪装成被冰雪/黑雾笼罩的巨峰。韩立以第三竖目（天眼）射晶色光柱破阵，现出真容：五色霞光笼罩的秀丽山峰，顶部翠绿宫殿。
- 邪莲圣祖现身：绿色宫装女子，面容酷似『宝花』，但神情阴冷。她乃宝花同胞妹妹。她认出韩立即当年被元魇六极通缉、离开圣界时尚为合体、如今已进阶大乘的『韩姓修士』，并认出蟹道人为魔源海蟹道人；对银月（合体初期）不屑一顾。
- 殿前另有白光界两名大乘：光头壮汉金差、披发壮汉石定（黑肤、银灰战甲、赤红灵纹、银瞳无瞳孔），比韩立早半月到。邪莲称外界圣祖一直暗中联络，各界来援强者必先找上他们。

## 五、真相揭示（2215）
- 始印之地仅传出元魇以秘术发出的一条模糊信息：螟虫之母提前冲出上古封印大半禁制，且能反操控部分封印力量，将所有前去镇压者（含各界来援道友）尽困其中；但虫母力量消耗后再度沉睡，仍受始印之地禁制压制，无法离开该区域；现世螟虫即其沉睡中泄露气息所致。
- 此为始印之地最后一条讯息，此后彻底隔绝。
- 最先接到消息者为宝花（被认为是唯一有能力解救者）。宝花据元魇消息推断：封印力量有轮回期，数十年后的今年是封印力量最衰弱、同时始印之地禁制对虫母压制最强之时——故重新镇压或营救皆事半功倍，但二者不可兼顾（须集中全力只做其一）。此结论源自宝花亲自潜入始印之地探查。
- 宝花的探查惊动了螟虫之母：包围始印之地的虫海激增十倍以上，并出现连圣祖也退避三尺的可怕螟虫。
- 该可怕螟虫为『血色螟虫』：体长尺许、浑身血红，仅始印之地四周虫海存在，数量不超千余；气息仅炼虚等阶，却具两种神通——『瞬移』（避无可避、无视防御贴身）与『自爆』（任何功法宝物皆无法防护，只能肉身硬抗；自爆威能会污秽精魂，连累各地分身/化身重创乃至当场陨落）。两位试图闯虫海的道友因此身负重伤，圣祖再不敢轻试。
- 邪莲与宝花似不和。一月后众圣祖将聚会：一为聚集金差、韩立等各界来援强者增胜算，二为等宝花炼制好两件专门克制血色螟虫自爆的特制宝物。

## 关键人物状态（本片段）
- 韩立：大乘期；持三座极山、施梵圣法相、用天眼破障眼法。
- 银月：合体初期。
- 蟹道人：大乘，魔源海，化银弧雷电。
- 葫岩：合体中期，黑葫城城主（魔尊）。
- 邪莲圣祖／宝花：圣祖级（宝花为圣界始祖）。
- 金差、石定：白光界大乘。
