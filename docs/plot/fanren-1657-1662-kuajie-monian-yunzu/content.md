# 凡人修仙传 第1661-1663章｜跨界魔念 · 云族

第九卷 灵界百族。地点：灵界·云城·灵山一带。

## 第1661章 跨界魔念——修补天外魔甲险酿大祸

- **修补法阵**：韩立在晶族女子纤纤的商铺私人空间中，取出贴有禁制符箓的玉匣，放出一件半尺大的紫色天外魔甲——造型狰狞、肩膝带尖刺、黑纹遍布、魔气冲天，胸口却有一惹眼大洞、边缘碎裂。纤纤以铜鼎（黑青色火焰）收纳魔甲，催动法阵与四角深黄石台上的四杆黑色魔幡（由小变大至五六丈）。约定：修补中韩立只需听吩咐及时向四杆魔幡注入法力。
- **突发跨界魔念**：修补历时三日，前半顺利、后半频出小故障（法阵停工、铜鼎裂缝）。魔甲完成最后工序飞出铜鼎时，一道阴冷强大神念强行破开空间障壁降临，直扑魔甲，欲借此地精纯魔气凝聚出狰狞魔体。
- **辟邪神雷灭魔**：韩立曾见过此类跨界神念，先以一道辟邪神雷击散其小半；魔念不退反进扎入魔气欲成魔体（一缕分念凝聚的未成魔体竟能连受十余道辟邪神雷），韩立遂倾尽体内所剩辟邪神雷，成百上千金色电弧齐出，将魔念连同魔体彻底化为乌有。
- **魔甲归属**：经此一役，二人皆知魔甲大有来历。纤纤头顶尺许大的青色麒麟虚影叹此甲非寻常天外魔头之物，建议找借口留下另以他宝相换。纤纤却忌惮韩立——其神通几不下于圣族初阶，而自己几件救命宝物已在魔金山脉用尽、毫无把握逃命；且魔器留下也无法祭炼驱使、反失这位难得交好的大援，遂作罢，让韩立将魔甲顺利收入贴满十几道符箓的青玉匣带走。韩立对纤纤好感又增。
- **纤纤的暗线**：纤纤与青色麒麟另谋一件下落只有几个圣阶老怪知晓的东西，为闯入某『上古遗址』所必需；麒麟点出纤纤当务之急是尽快提升修为神通，否则机缘虽多也难掌握。纤纤打算从那几个老怪的门人子侄处买或借。

## 第1662章 云族——许老怪登门

- **不速之客**：韩立返回灵山山腰洞府，遇红袍六十余岁的『许老怪』（徐老怪）——即此前以朱红葫芦吸走他失控真麟本源所化天地元力之人，腰悬该葫芦——及一名十四五岁清秀少年。
- **洞府背景**：洞府系万古族千机子长老吩咐安排；韩立自称数年前偶对万古族另一道友有小恩，那道友恰与千机子有关，故得照拂。许老怪称万古族在天云十三族中排名不低，千机子随口一句便可保韩立在云城无忧。
- **云族揭示**：许老怪自陈是天云十三族中的**云族**人——擅变化隐匿之道，刻意收敛气息时高其数阶者亦难辨；人数稀少却稳居十三族前三，与晶族同为人数最少之族，但个体实力普遍不弱。许老怪为族中异类，修炼至阳火属性功法，幻化之道稍逊。
- **本命云兽**：云族真正威慑种族的天赋神通是介于灵兽与化身之间的『本命云兽』——修出方算真正云族人，相当于多出一个与己一般无二的身外化身，危机时可与云兽合一化为『云兽巨灵』，修为暴涨并施展原本无法施展的大神通。许老怪点明此行与身后少年（其本命云兽）大有关系。

## 第1663章 换取——许老怪求聚天地元气

- **后辈许伦翔**：少年许伦翔（翔儿）是许老怪唯一堪培育的嫡系后裔，仅十几年即修至人族筑基巅峰、再进一步即结丹，资质在整个云族中亦称过人，不足百年应可进阶上族境界。缺陷在于本命云兽卡在最后一步始终无法凝练。
- **所求之物**：许老怪无意中发现，上次『天象』收集到的天地元气有助许伦翔凝练本命云兽，普通天地元气无用；惜旧存大半已用尽，故求韩立再聚一次同样的天地元气。（韩立心知那天象实为真麟本源晶粒溃散所化，难怪与寻常天地元气不同，却没想到对云族本命云兽有用。）
- **韩立婉拒与交换**：韩立以该天象并非秘术功法可凭空释放、而是动用一件『消耗性宝物』所致、所剩寥寥且另有大用、绝不愿为初识异族牺牲（暗指不愿再为啼魂兽吸取珍稀真麟本源）为由婉拒；但松口可考虑交换。许老怪取出七八个玉盒珍藏，其中两件险使其陨落。
- **炎熔蝶灵卵**：韩立先抓取一火红玉盒，内为鸡蛋大、布红斑、散红光的蛋白色兽卵——许老怪当年于地熔深处历时一年灭杀一头火属性神通不下于己的『蛮荒炎蝶（炎熔蝶）』所得虫卵，孵化精育可成上佳灵兽。韩立因已有灵兽未取。
- **结果**：其余玉盒含珍稀材料、一两件成品法宝、乃至一瓶能增进炼虚期修为的丹药，皆价值不菲（炎熔蝶卵卖给火属性修士或拍卖可达千万灵石以上），但韩立自有合体修士不敢妄想的玄天之宝两件，故自始至终笑而不语、无一看中。许老怪神色渐难看，本章在僵局中收束。

## 关键设定提炼
- 跨界魔念可强行破空降临、借魔气凝魔体，修为或不逊真灵存在；辟邪神雷为其克星。
- 云族＝天云十三族前三、擅变化隐匿、核心天赋为本命云兽。
- 韩立真麟本源所化天地元气对云族本命云兽凝练有奇效。

## 一、炎金之精交易（第1664章）

- 韩立与许老怪议价。许老怪为求后辈前途，先后取出多件珍宝，最后掏出一只奇寒灵玉盒，内装拳头大小、琉璃般赤红金属——**炎金之精**。
- 韩立以普通法器短剑试探，剑尖一触即被无声溶解为青烟，确认其为**炼制火属性法宝的最顶阶材料**（普通法宝掺入一点即可凭空增加火属性神通）。
- 成交：韩立以一粒**真麟本源**晶粒（自**啼魂**体内取出）作主要对价，另补**五百万灵石**，半买半换。
- 兑现方式：韩立放出啼魂取真麟本源，以灵火炼化引动天象搅动天地元气，许老怪在山顶祭出朱红葫芦化**火蛟**吸尽元气；全程隐秘，几未惊动他族修士。韩立以铜镜秘术远程旁观。
- **关键收获**：炎金之精 + 此前所得**金髓晶虫** + **血杏** = 凝练**梵圣真魔法相**实体之躯的主材料齐全，只待配齐辅料即可开始凝练。
- 随后韩立赴云城数家大材料店，豪掷天文灵石购入三十余种材料（赖云城为天云超级大城方能配齐），返回闭关。

## 二、云城圣族议事（第1664-1666章）

半月后云城突发大事：角蚩族奸细盗走**刚布置好的禁制大阵图解**。城门戒备倍增，巡查近倍，外来可疑者被神秘人盘查，身份有问题者诡异消失。

隐秘大厅中召开圣族高层会议：
- **主持**：翁姓青年（大乘期，曾现身四族拍卖会，浑身阴森之气）。
- **列席大乘期**：沙老夫人（沙姓老妪，金发满皱、眼皮难睁；来历神秘，无人知其出身何族、修何大神通，乃十三族现存最长寿者，连翁姓青年都以半个晚辈相待）。
- **诸圣族长老**：万古族千机子、晶族彩流罂、石茧族段天刃、水魅族白发人（白道友，自称**幻化/勘察之术（**术）在十三族首屈一指）、阴妖族苍影（灰影）、黑须老者宁道友、墨绿肌肤者、黑炭大汉等。

议事要点：
- 翁姓青年判断：奸细仅炼虚期修为，却躲过水魅族勘察，说明其**早已在城中潜伏多年、另有身份**；盗走的禁制图解关乎云城大阵，绝不可让其出城。
- 彩流罂献策**『外松内紧、引蛇出洞』**：一月内人手尽出大搜，无果则佯装放松、实则四门暗增人手并专查欲离城可疑者；另有一半计策以**传音**密告翁姓青年（青年由意外转惊喜）。沙老夫人懒于谋划、自称只负责对强敌时帮衬，中途离席。最终采纳此策，交彩流罂、白发人全权处理。

## 三、广寒界将启（第1665-1666章）

- 万古族置于**通灵大殿**的**广寒仪**两日前有反应，按常例广寒界**多则三四年、短则数月**即将开启——较以往为早。
- 广寒界开启是雷鸣大陆各族实力大增之良机，但适逢角蚩族犯境。十三族此次仅得两块**广寒令**（云城留存），故只需布两座开启传送法阵，由万古族千机子三人负责，法阵方圆十里设禁区，戒备增三倍。
- 进入名单由各族自定，须连续后期顶阶（炼虚巅峰）后辈。
- **广寒令激发**：一枚人选已定；另一枚**已被韩立提前激发**——千机子说明韩立来历（非天云、非本大陆之人），几族答应事后借**大陆传送阵**一用（灵石自备），并约定暗中监视他。

## 四、战况与三年后（第1666章）

- 角蚩族增兵、新增圣阶五六人，已破三城，天云处下风；翁姓青年冷然抽调五名圣界、上千中高阶增援前线。
- 会议持续近半个时辰散。
- **三年后**：韩立密室三人影盘坐——
  - **金色鳞甲、如赤金者**：韩立本人，睁目爆发两团刺目蓝芒（灵目神通），居中淡金色人影。
  - **黑气化出的三头六臂、似魔似神金像**（身高两丈、金光灿灿，两面孔清晰、一面无口鼻，瞳孔幽黑深邃）：即**梵圣真魔法相**。
  - **紫光翠绿、面容与韩立一般无二、双目清澈者**：**人形傀儡**（化形成韩立面容，由韩立/第二元婴操控，据世界状态须知起于第1183章）。
- 来客：晶族**彩流罂**携一名头戴白色斗篷、面容被晶光遮挡的**『新收小徒』**到访。韩立以千年以上稀有茶树所结灵茶款待，并赠乳白色茶盒。

## 备注
- 人形傀儡之『韩立面容』言行据世界状态须知归属于人形傀儡（化形伪装），非韩立本人。
