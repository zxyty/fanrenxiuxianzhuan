# 第1045章 吞婴（第六卷 通天灵宝）

## 局势
- **白袍儒生**（叶家大长老）以黑血刃逼出**大头怪人**，斥其为叛逆：七叔与古魔圣祖勾结，欲将**八灵尺**交予妖魔。
- **黑血刃**：叶家传承之宝，亦是本族处置叛逆的法器，由白袍儒生掌控。
- **八灵尺**：佛宗灵宝，叶家必得之物；古魔欲破其禁制夺取。
- 大头怪人冷笑不辩，露出凶煞之气，与先前儒雅形象大相径庭。

## 珑梦 vs 元刹
- **黑袍女子**即古魔**元刹圣祖**，正借一具**化神末期的银狼妖体**行事（珑梦称『我那具银狼妖体』）。
- **珑梦**在宫殿中为**无躯体的残魂**，自称『本妃』，与白袍儒生**联手一次**（强调只是联手、非与叶家合作），欲拖延、阻其破八灵尺禁制。
- 元刹以白色大手抓碎珑梦弹出的五道晶莹利芒，展现强横神通。

## 吞婴
- 元刹反手一弹**黑风旗**（乌黑巨旗），喷出乌黑光柱瞬移至**方脸修士**身前炸开，化出黑色飓风、无数黑色风刃。
- 方脸修士（元婴中期）以**弥天镯（骨环）**所化黄光光罩护身，风刃斩上不爆而留豁口，光罩与骨环寸寸崩裂。
- 方脸修士人剑合一化青虹强突，被乱刃分尸而亡（**本章死亡**）。
- 碎尸射出三个元婴（方脸修士元婴 + 另两名叶家元婴），被风刃困住、定身、裹成三颗黑色光球。
- 元刹以大神通挪移光球至跟前，双目血色一闪喷血霞，将三元婴缩成指头大小吸入口中、吞进腹内——**吞婴**。
- 白袍儒生欲以黑血刃救人，已迟。

## 珑梦的点破
- 珑梦：吞噬元婴只能令老魔法力暂时提升，却需短时间强行以魔气炼化元婴，反令魔气大耗。
- 珑梦判断元刹准备动用银狼妖体神通，不惜大耗魔气。
- 元刹坦认：原想只凭黑风旗破八灵尺禁制，但嫌拖延太久、迟则生变，故拼着魔气大损，凭化神末期狼妖之体尽快灭杀对手。

## 双首银狼现身
- 元刹掐诀，通体银光大放如银色骄阳，狼嚎动九霄，强大灵压降临，空间嗡鸣、大地裂缝。
- 银光暴涨数十倍，现出**巨山般的双首银狼**：两只狼首一黑一银，银首双目紧闭低垂，黑首仰首龇牙怒目；前爪利爪如巨刃。
- 白袍儒生与大头怪人皆倒吸凉气。

## 备注
- 珑梦另据世界状态须知自第1033章起夺舍占据花天奇躯体；本章其以宫殿内残魂之声现身，言行归珑梦。

## 背景
众人被困于元刹圣祖『黑风旗』所化的封闭空间内。韩立一方（韩立、徐姓青年、圭灵等四人）围住一名偷偷藏匿的红光人影——四散真人（郑某），神色不善。银翅夜叉与狮禽兽远远旁观、不愿插手。

## 第1046章 血刃再现
- 四散真人伪装成五十余岁和蔼老者，自称散修、连声叫苦推脱。林银屏指其遁术神妙、潜藏其旁，幸被灵犀孔雀五色灵光扫到才察觉。
- 韩立揭穿：进入镇魔塔前是四散真人以宝物暗算自己；其身上隐透凶煞血气，一路吞噬了不少元婴修士（同行修士皆失踪）。
- 韩立喷出青竹蜂云剑（寸许金色小剑）斩杀，四散真人以蓝色飞剑、青色铜盾抵挡均被秒斩；徐姓青年因金剑掺『庚精』而暗自警惕。
- 四散真人被圭灵、徐姓青年封住退路，被逼放出血光——化十几道血芒，凝聚成半尺血雾短刃。韩立点破：『你这血刃是仿制魔龙刃炼制的吧？』『魔龙刃』一出口众人（含银翅夜叉）皆动容。
- 血刃狂斩仍无法伤及韩立金色飞剑分毫。韩立遂改提议：不杀，反而联手——利用四散真人血刃的大威力，合力强攻空间一点破壁脱困（韩立有灵缈园击破空间的经验）。
- 徐姓青年、银翅夜叉皆同意；约定离开后各行其事、不秋后算账（四散真人提及若有太一门人则需顾虑）。四散真人爽快答应，却暗中与在场某人传音，手背悄然冒出数寸白毛（一闪即收）。

## 第1045章 破壁突变
- 韩立暗以神念安抚灵兽袋中暴躁的啼魂兽，心生异样警觉。
- 众人各布辅助法阵；韩立与徐姓青年共主『风炎龙烈阵』（威力约抵元婴后期一击）。徐姓青年以乳白光球标定障壁攻击点。
- 发动：徐姓青年激十余道赤红火柱，韩立以青色阵盘凝十余条青蛟，风火交融化作风火之蛟。韩立再抛法盘催涨，风火蛟暴涨数倍直撞障壁。
- 韩立数十口青竹蜂云剑化数百剑光、凝成十余丈金色巨剑，剑身浮现金色电弧（辟邪神雷）。
- 风火蛟连撞 + 四散真人血刃巨刃斩击，障壁现裂缝。韩立辟邪神雷化金色雷电斩下，效果数倍于众人合击，疑黑风旗为魔道灵宝被辟邪神雷所克。
- 韩立人形傀儡（先）与无声黑色飞刀破开最后薄障壁，孔洞现出熟悉景色。

## 反噬（突变）
- 人形傀儡率先冲出，众人争相遁出。韩立未立即瞬移，圭灵动作一顿。徐姓青年等抢先遁出后即遭洞外血光伏击，传来四散真人得意阴笑。
- 韩立以风雷翅化银弧遁出，避开席卷血气，停于百余丈外。
- 洞外现数亩血海：四散真人现真身——身穿其服饰、浑身尸气、双目绿火、口露獠牙的白毛熊脸高大怪物，操血刃将徐姓青年压入血海（其仅剩半身、缺一手一腿，靠青色圆珠与银钩法宝苦撑）。
- 灵犀孔雀被狮禽兽所化紫雾困住；林银屏被银翅夜叉的煞魂丝（更煞魂丝/灰丝）围攻，靠锦帕银蚕银丝勉强自保，落败只是迟早。
- 圭灵瞪着白毛怪物惊呼『是它！』。

## 关键判断
- 四散真人自始即是潜藏的吞元婴凶徒，联手破壁实为借众人之力脱困并伺机反噬；银翅夜叉、狮禽兽倒向四散真人一方，原韩立联盟瓦解。

## 背景

地点为昆吾山虚天殿外殿空间。韩立一行破开黑风旗禁制的障壁而出，发现空间已变诡异：四处空荡无人，宫殿上空原悬浮的**八灵尺**与禁制他们的**黑风旗**皆不翼而飞，地面满是数丈长的爪痕沟槽与深坑，宫殿被毁去一小半，似经历惨烈激战。韩立神识因宫殿残骸内仍存隔绝灵觉禁制而无所获，但也因避开了压力极大的**元刹圣祖**而暗松一口气。唯一传送阵已被隔绝，需另寻脱身之策。

## 圭灵与白毛炼尸的渊源

圭灵『姑且』认得那只**炼尸**：它变成炼尸前也是一只灵兽，当年与圭灵等有过交情，其主人亦是从灵界下到人界的人类修士。关键设定——**当年从灵界下界的修士修为不能太高、所带灵兽亦受限，且随行灵兽是在被抹去原有记忆后才获准下来**；故圭灵除知道自己来自灵界外，对灵界毫无记忆。韩立由此动念想从银月（身份不简单，或对灵界有记忆）处探问灵界之事，但暂压心底。

## 万年尸熊的真身

这白毛炼尸有元婴后期修为、修炼神妙的隐匿功法，连韩立都看不出其炼尸身份，韩立判定其即**万妖谷副谷主——万年尸熊**，不知多少万年前已修炼成灵，在大晋妖族中大名鼎鼎。它化身为人类散修『四散真人』出现于此（究竟是杀害真散修后幻化、还是其行走人界的固定化身，存疑）。其隐匿功法令修为大受限制（否则当初血刃偷袭阴罗宗黑衫长老时不会让其元婴逃掉）。

它先前在障壁中密谋联合银翅夜叉，待破障后趁众人警惕最低时偷袭先出谷的人类修士（不论对象是突兀人还是韩立一伙）；此谋被韩立袋中**啼魂兽**察觉，韩立遂偷听到大概。韩立故意迟出一步，致徐姓青年与林银屏一伙倒霉。

## 救人与逼退

天澜圣殿的**林银屏（天澜圣女）**与**徐姓青年（徐仙师，元婴后期）**陷于尸熊的血刃血海、银翅夜叉的煞魂丝、狮禽兽的紫雾围攻；唯徐姓青年的上古灵禽**灵犀孔雀**以五色灵光克制狮禽兽金波而稍好。林银屏许诺以圣女身份化解恩怨求援，被韩立冷然拒绝；徐姓青年则厉声许诺相赠噬金虫成熟体秘诀，林银屏以『不传之秘』反对，被徐以『族运重于秘术』压下。

韩立本意是待徐毙命、血刃吞噬元婴血肉精魂的间隙趁机夺取血刃（此邪宝须握于己手方稳妥）。当尸熊催血刃化巨大鬼脸扑向徐姓青年时，韩立一道**金弧（辟邪神雷）**击散鬼脸小半；徐姓青年元婴破残躯而出欲遁，被尸熊黑气大手罩住。此时**人形傀儡**（银光中人影）以翠芒带金芒一击洞穿大手、瞬移至元婴旁救下。灵犀孔雀射出长翎炸开五色灵光、破紫雾脱困，与徐姓青年元婴合一（元婴遁入灵禽体内，灵禽明显大了一圈，头颅浮现徐姓青年乳白光脸）。

与此同时**圭灵**现身，传音几句令银翅夜叉长叹收回煞魂丝放出林银屏；随即圭灵以银色巨斧（浮现独角巨虎幻影）斩断林银屏锦帕、以黄气罩之，将昏迷的林银屏生擒带回韩立身旁。尸熊全程未拦阻。

韩立携人形傀儡逼近血海，以强硬口气宣称要暂带走二人。**万年尸熊**问明韩立是天南修士后，警告：此次救天澜二人可『视若不见』就此放手，仅此一次；下次再坏万妖谷好事，即使有这两人相助也绝无法活着回天南。随后尸熊将血海尽数收回血刃，率银翅夜叉、狮禽兽径直没入宫殿，似另有目的。

## 噬金虫催熟秘诀

韩立向灵犀孔雀上的徐姓青年面孔追问。要点：
- 噬金虫上古凶名赫赫，以『无物不噬、几乎不灭』著称；天澜圣殿的噬金虫得自上古遗留虫卵。
- 按普通方法需数万年才能自然进化到成熟体，远超修士／宗门寿元与坚持。
- 天澜圣殿由精通驱虫术的长老耗**上千年**摸索出催熟法：**用灌注之法、借天地之力，强行往灵虫体内灌注五行灵气催熟**，可将进化时间缩短至原来的**三分之一**；若不惜珍稀材料、着重培养其中几只还可更短。瓶颈是虫卵稀少、辅助材料珍稀，无法大规模催熟。
- 徐姓青年坦言：韩立的噬金虫群虽进化到最后一步但未开始成熟体进化，纵得秘诀也需数千年苦养，故不怕泄露。秘诀写入白色玉简交予韩立，韩立神识一阅便能领悟。
- 韩立自承：他能一次次催生进化大量噬金虫，全靠**神秘小瓶**与**霓裳草**的神奇效用。

## 圣兽之印与和解条件

交易完毕后，韩立作势要灭杀徐姓青年（风雷翅银白双翅闪现、金色雷衣、人形傀儡持小弓翠芒对准灵禽）。徐姓青年元婴飞出求和，提出：
- 双方仇怨主因是韩立以元婴中期身份在天澜草原击杀击伤诸多高阶仙师、并收走**圣兽分身**（犯圣殿大忌）才被追杀；如今韩立可驱使两名后期修士、本事不下大修士，已有与天澜圣殿平起平坐之资格。
- 和解担保：由圣女林银屏施秘术召唤**天澜圣兽真灵**为韩立种下**圣兽之印**、聘其为天澜圣殿**客卿长老**，条件是韩立归还圣兽分身（好让圣殿对外交待）。
- 圣兽之印性质：种印者皆为圣殿最核心之人，互相不能起杀戮之心，否则事后印记发作，染上古怪诅咒、神识长年刺痛、生不如死。

韩立眉头紧锁、神色阴晴不定，似有动心；徐姓青年元婴识趣住口静待其决定，章节止于此。

## 实力与势力注记
- 韩立本章为**元婴中期**修士；自评有人形傀儡（近乎分身）与圭灵配合，可同时周旋三四名元婴后期。
- **万妖谷**为大晋可与**太一门**、**天魔宗**并驾齐驱的一大势力。
- 天澜方仍有三大仙师未至；韩立判断三大仙师不可能全离草原（否则突兀人将遭慕兰人灭族之祸），故不惧其威胁。
