# 第1443-1444章 危机 / 地渊深层

## 一、韩立三人穿越地渊一、二层

- **同行者**：韩立携天鹏族两位圣子**雷兰**（娇美俏脸，初阶灵将）、**白璧**（初阶灵将），进行飞灵族三百年一次的地渊试炼。
- **沿途战斗**：以**银弧**（疑为辟邪神雷）瞬杀十几只乌黑巨大螳螂；以无数**金丝**（噬金虫）绞碎石妖；灰雾山谷中轻易灭杀带翅怪飞蛇。三人足下飞车一度现身后又踪迹全无（毁于妖物或被韩立收起）。
- **地渊之风**：穿过朝下巨大裂口时，喷出酷寒黑色劲风，将石壁化为冰壁。雷兰、白璧修为损耗严重、脸色煞白，**韩立却毫无亏损**。雷兰由此意识到初阶灵将与高阶灵将修为相差悬殊，并感叹多亏韩立出手才度过难关。

## 二、行进决策

- 韩立决定**放弃在二层寻找冥焰果**，直奔三层。
- **入口形势**：二层到三层仅三个入口，两易一险，沿途各分支（如**赤融族**等大族）都会埋伏，目标包括最弱的几支分支。韩立分析：赤融族不可能将全部人手布在三层入口（地渊试炼对其同样重要），至多留一半甚至更少，故闯关并非不可能。
- **选定路线**：选离当前位置最近的**万藤道**（许多古藤连接二、三层的入口，除妖虫外基本无险），顺利则半月可达。

## 三、韩立警觉

- 行进中韩立背后突生莫名寒意，停下回望，灵目神通（瞳孔蓝芒）细查却一无所获，疑为灵觉感应出错，但心生阴影，此后行动谨慎，频用灵目提前避开低中阶妖物。

## 四、地渊深层的阴谋（关键剧情）

### 血袍人（地血老怪）
- 居地渊深层小山山腹，以铜镜观察韩立等人，惊觉自己寄于**泥傀儡**上的一丝分神竟被韩立感应到，断定此人神识强大。
- 因『鬼婆子』正搜集神魂强大的生灵，遂起意将韩立精魂当作礼物，换取一条**万年血鱼**。

### 数十年前定下的『大计』（四方分工）
- **木青**：搜集血食祭祀；
- **六足**：收集阴灵之力；
- **血袍人（地血老怪）**：炼制大量傀儡；
- **老妇人**：培养计划中最重要的**八千名阴甲玄鬼**。
- 统率玄鬼的**八名阴甲鬼王**需神念强大的极品精魂炼制，成功率极低，老妇人仅炼成三名——故对韩立精魂极为合适。

### 交易
- 血袍人开价：仅给消息＝一条万年血鱼；出手擒下并送上门＝两条。老妇人压价，最终**以精魂换一条万年血鱼**，约定**十日内送达**。
- 对飞灵族试炼圣子的处置：大事处于关键期，不能惊动飞灵族；只要圣子们不影响大计、不进入四层以下，就任其在前三层行动。
- 血袍人放出**三只血木傀儡**（灵帅初阶），令其杀韩立三人、拘回精魂。

### 老妇人真容
- 地渊六层绿光宫殿、黑色深潭旁，一名**白发无血美妇**（即与血袍人对话的老妇人/鬼婆子）冷视潭中数名乌黑怪甲模糊人影（阴甲玄鬼）。两名黑扇美貌女子（一娇小圆脸、一妙曼冰冷）以**乌黑小瓶放出黑焰火蛟**、以**黑色芭蕉扇**扇出黑焰，动用**冥火**淬炼潭中人影。
- 另有：一层神秘山脉中心祭坛供奉头颅大小、布满血丝的灰白巨眼球，喷灰丝吸纳灰气，坛下立一乌黑斗篷高大神秘人。

## 五、章末突变
- 韩立三人于二层墨绿山脉翻山后，对面天空一道**翠芒**激射而来，其后三团**黄光**以不逊遁速紧追。遁光出现诡异，三人来不及躲避，韩立眉头一皱、带二人徐徐停下遁光。（悬念）

## 备注
- 此处『韩立』正以飞灵族灵将身份行走于地渊，按世界状态须知归属真实人物韩立。
