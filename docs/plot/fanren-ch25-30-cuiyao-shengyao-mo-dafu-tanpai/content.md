# 凡人修仙传 第25-30章｜绿液催药 · 墨大夫摊牌

## 一、绿液催药规律（第25-27章）
- **第25章 插柳成**：韩立发现前夜滴过绿液的黄龙草、苦莲花、忘忧果一夜之间从一两年药性变成十几年的成熟稀有药材。他买灰毛兔栓在房门口、以催生草药做的培筋壮骨药喂兔试毒；同时夜里取瓶试验是否再生绿液——连等数夜无动静，险些以为瓶子是一次性消耗品。
- 顿悟关键：瓶子异象只在**晴天、能见星月**时发生；此后半月连绵阴雨，谜底迟迟无法解开。
- **第26章 催药生**：天放晴当晚再现四年前的光点光团异象，确认瓶子可反复使用；再等七天产出一滴绿液。试验得出规律：
  - 稀释后洒大量草药 → 仅得一两年药性的普通药材；
  - 不稀释直接滴一株人参 → 次日得**百年人参**；
  - 同株反复滴加可**叠加年份**：在三乌草上反复滴加，叶色由绿→黄→黄黑→乌黑，终成**千年三乌草**。
  - 绿液离瓶后任何容器（瓷/玉/葫芦/银）都存不过一刻钟，超时自行消失，**无法大量储存**。
- 韩立警觉「怀璧其罪」，决心守口如瓶，趁墨大夫不在、神手谷只他一人时安全使用瓶子。

## 二、配制圣药与突破第四层（第27-28章）
- **第27章 配灵药**：韩立按墨大夫平日所授、却因缺药材无法配制的极品配方，催生药材并炼药，屡有失败（每次损失大量极品药材）。最终炼成：
  - **黄龙丹**、**金髓丸**：增功力、脱胎换骨，对练功帮助最大；
  - **清灵散**：世间少有解毒圣药，可解千百种剧毒；
  - **养精丹**：内外伤奇效灵药。
- 韩立对墨大夫的提防：察觉师徒间有隔膜，墨大夫常以古怪目光看他，疑其暗藏图谋，故决意瓶子之事绝不告知任何人，墨大夫回山后即收起不再用。
- **第28章 墨回谷**：韩立服黄龙丹+金髓丸当晚冲破瓶颈练成**口诀第四层**，五感质变（视/听/嗅大增，数十丈内动静皆觉），身轻、精神大进，并意外发现能**随意操控真气强弱**、可瞒过墨大夫把脉。墨大夫携神秘人回谷，气色灰败如大限将至；韩立把脉时谎称进度无变化并伪装第三层真气。

## 三、冲突与身份揭底（第29-30章）
- **第29章 冲突起**：墨大夫把脉后不满，进屋后突现狠厉之色、脸罩张牙舞爪的黑气，瞬间点住韩立胸前穴道、搜出其铁筒。自称「墨居仁」，赞韩立临危不乱「不枉我下了那么大本钱」，挑明「我也没真把你当徒弟」。
- **第30章 枭雄末路**：墨大夫全盘托出——
  - 真名**墨居仁**，外号「**鬼手**」，早年称雄**越国岚州**武林；
  - 实年仅**37岁**，外表却如六七十岁，因中年遭亲信暗算重伤、武艺大减，抛家隐匿求医；
  - 后得一本**奇书**，按法恢复功力却被**邪气入侵**，急速衰老、每日透支生命（活一天耗十天精力），靠自配秘药续命；
  - 从书上研出破解之道：需一个练成**长春功（即韩立的「无名口诀」）第四层**之人为其运功推拿、以长春气刺激，方可找回精元；
  - 长春功苛刻：须年少从头修炼且必具「**灵根**」体质，此前找过数百童子皆失败；
  - 流浪时救下同样被暗算的**七玄门王门主**，受邀成供奉，将韩立等招入谷试练长春功，唯韩立有反应；
  - 此次下山被仇家认出，苦战击毙对方却耗尽精力，自估只能再活一年，故等不及而提前摊牌制人。
- 神秘人：头戴黑斗篷、身罩宽大绿袍、不露肌肤、异常高大魁梧（高韩立两头），紧随墨大夫寸步不离。

## 关键转折
1. 「无名口诀」真实身份揭晓 = **长春功**。
2. 墨大夫真实身份、年龄、图谋全面曝光，师徒情分彻底撕破，韩立陷入被制的险境。
3. 韩立掌握绿液催药全部规律，具备源源不断获取珍稀药材的能力。
