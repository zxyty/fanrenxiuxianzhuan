# 第1003章 群魔隐现

## 三妖密谋（昆吾山·巨树之下）

封印已破，三个被封印的妖物在巨树下密议：

- **银翅夜叉**：人类修士昔日饲养的灵禽修炼成灵；擅土遁；欲夺本命牌并离山，因须吸取**山外阴月精华**方能进化为**金身月尸**，不愿终生困守。
- **圭姓丑妇**：本是一名人类修士的**肉身**修炼成灵，实为**十级玄岩龟**所化（有玄岩龟妖丹），神通更胜银翅夜叉一分；起初只想钻入灵脉深处再睡千年躲祸，被人囚禁怕了。
- **狮禽兽**：同为那名修士饲养的灵兽。

三者皆当年被人类修士一同封印在**困灵阵**中。

### 关键往事与筹码
- 当年古修在三妖身上打下**暴血咒**，欲使其神智全无、嗜血成狂；银翅夜叉曾暗分一粒**清虚丹**给圭姓丑妇保其神智，圭因此化形至今，并欠其一个人情。
- 银翅夜叉以「若本命牌被人类夺去，必会供出圭、被派来追杀她」相要挟，又重提清虚丹之恩，终说动圭姓丑妇再出手相助。
- 约定：一得本命牌即刻走人，圭从此隐修不出。

### 行动计划
- **昆吾殿**设有专克三妖的禁制，平常无法靠近；计划尾随替他们开道的人类修士潜入殿中，再一齐出手夺取本命牌。
- 提及尚有**第四个被传来的家伙**下落不明，银翅夜叉遍寻未得困灵阵，疑其因无清虚丹保神智、相隔多年早已不存。
- 三妖商议毕，一起飞向石阶方向。

## 潜伏的第四方

三妖离去后，附近一颗看似普通的大树骤然缩小、现出两颗**晶绿碧色眼珠**——竟是一不知名妖物所化，潜伏近在咫尺却未被三妖察觉，冷冷望向其离去方向后钻入地中消失。

## 韩立一行的处境

- 韩立等人待狮禽兽离去后遁出石殿，**一出殿外即遭禁空**纷纷跌落，确认已至昆吾山要害之地。
- 白瑶怡发现数里外石阶没入一大片**紫色巨竹林**：一人抱的紫竹密布，原白玉广场尽被掩盖，林中紫气弥漫、灵压惊人。
- 花天奇判断此阵胜过**冰焰两极阵**，应是那群人布置的**最后一道禁制**，破之即可追上前人。
- 乾老魔驱动**五魔子**化霞光率先卷向竹林；花天奇带毒圣门数人随后；紫竹林嗡鸣，**七道紫濛光柱**冲天，紫气化波浪抵住五魔子灰色寒气，相互撕扯。毒圣门四老、大汉等也各施神通宝物攻阵。

### 韩立惊觉古魔
- 攻阵前，韩立体内**数十口飞剑同时无故一颤**，由此惊觉**古魔**也在此山中，疑其混入前方修士里，心神彻底大乱。
- 此时韩立已拥有**三焰扇**与**元婴后期傀儡**，并非惧怕古魔，而是觉此魔出现时机过于诡异（联想天南万丈魔渊旧事），心生不祥。
- 韩立强自镇定，默喷数口飞剑心不在焉加入攻击。

## 山外异动

封印外那条因幻阵自爆被重新封死的**地下通道**外，又来了**数批不速之客**。

## 备注
- 韩立体内飞剑即青竹蜂云剑（金雷竹飞剑）。
- 圭姓丑妇为玄岩龟所化，行文中以「丑妇/妇人/圭道友」相称，本质为妖。

## 一、封印裂缝入口的三方对峙（第1004章）

小湖北边二十余里地下深处，数百修士聚于封印裂缝入口。幻阵自爆后裂缝一目了然，但入口银光如银浪席卷，禁制狂暴，进入极险。散修远避，三伙高阶势力僵持：

- **化仙宗**：全为翠绿衣衫的南疆年轻女修。为首者为**木夫人**（元婴中期，姿容平常）与一名**秀丽师妹**（元婴初期）。化仙宗名头胜过毒圣门，弟子多女修、长老皆女子，靠防不胜防的诅咒阴毒之术，可远隔万里施法杀人于无形；木夫人曾一日同时咒杀三名异地同阶修士。
- **阴罗宗 + 天澜**：三十余人，隐分黑衣、白袍两拨。五名元婴修士站前，含**葛天豪**（阴罗宗）、**天澜圣女林银屏**，以及最惹眼的**徐姓大仙师**（天澜草原、元婴后期大修士、清秀年轻男子）。
- **太一门**：人数最少，三名青袍道士，为首**玄青子**（白发红面、仙骨道风的元婴后期大修士）。太一门号称正道第一门。

玄青子与木夫人旧识、有难明恩怨，故身为大晋数一数二大宗长老仍客气相劝。木夫人反讥封印身处化仙宗势力范围，化仙宗不惧危险。葛天豪则坚称阴罗宗大长老已下去，必要入内。

**玄青子的警告与底牌**：他得密报，封印是被人处心积虑解开，下面绝非好事；裂缝禁制狂暴，他自己也不敢贸然闯入，要等**两日后天魔宗七妙真人**赶到——借七妙的**灵龟飞车**通过裂缝。此事原牵涉正魔两道，由玄青子与七妙分头负责。

**各方强行进入**：
- 化仙宗亮出新得的**日月梭**（三灵梭之一），木夫人与师妹各祭金、银一梭，空中合二为一化为丈许金银巨梭，二人遁入梭中飞入裂缝。
- 徐姓大仙师放出**灵犀孔雀**（五色灵光天生隔绝天地灵气），为葛天豪一行开路，众元婴修士护身尾随入裂缝。
- 玄青子不阻拦，言明利害后率两名结丹道士在入口盘膝守候。

## 二、封印内部：巨物、宫殿与白玉石人（第1005章）

**昆吾山顶某封闭空间**：一具大得难以置信的小山般巨物悬于半空，遍贴极高阶禁制符箓、缠满乌黑渗血锁链，周围上千巴掌大铜镜放黄濛光柱组成怪阵，禁制层层无尽。巨物近乎死物，仅偶尔起伏（疑为后文古魔）。

**相邻空间**：灵气充沛、奇花异草如仙境，中央一座数百丈华丽宫殿建于巨型法阵中心；宫殿四周八十一座高约十丈的小祭坛，每坛供奉一尊数丈高、身披金甲、双手合持金色巨刃、面朝宫殿的白玉石人，肃然如活人，万古寂静诡异。

## 三、韩立独破紫薇七星阵

韩立周身金光，逼近紫薇七星阵的七道紫色光柱。应对：
- 紫雾所化碗口粗巨蟒近身即被金光斩断；头顶银白电弧被布满金弧的手掌引开。
- **巨峰**（拳头大、黄斑、三寸毒刺）：碧绿虫血腐蚀性极强，曾损韩立飞剑灵性，且抗火球冰锥，遂以**辟邪神雷**粘之立毙（灭三四十只）。
- **血蝠**（通体血红、身体坚硬）：掺入庚精后犀利无比的**青竹蜂云剑**也需顿一下才劈开，令韩立暗惊。
- 光柱附近设幻术禁制（紫竹困人），韩立以**天眼术**（瞳孔蓝芒）瞬间破幻，七八口金色飞剑合为金色巨剑斩毁光柱根部。

韩立未急于破其余光柱，取灵石盘坐回复法力，思量古魔出现后的对策——此山大修士妖魔众多，他纵有**三焰扇**和**元婴后期傀儡**仍有性命之忧。

**晶碑**：韩立取出当初在昆吾山石碑中（凭**土甲龙**天生识宝神通）找到的缩小蓝色晶碑。此物注灵力前穷重、注后轻飘；其上刻有比所谓“上古”更早的不识古文，确为货真价实的古物异宝。

## 四、破阵后的变局

其余光柱陆续被毁，最后一根自行黯灭，神妙大阵被众人联手破除。乾老魔暴怒：**毒圣门四老**认得此阵、悄然穿阵溜走；元婴大汉怒骂撺掇众人同入的**郑卫**也不见踪影——皆欲独占夺宝。白瑶怡指出脚下广场一端有十几条石阶通往不同方向。富姓老者推断那几人觉得只要不与人同路便可独力应付。乾老魔迅速冷静，扬言独行、谁阻取宝即杀谁；随后五道白影查看所有玉牌石碑后，齐往标有**昆吾殿**的石阶飞射而去。

## 在场人物速记
- 太一门：玄青子（+2 结丹道士）
- 化仙宗：木夫人、秀丽师妹
- 阴罗宗：葛天豪
- 天澜：林银屏（天澜圣女）、徐姓大仙师
- 山内夺宝者：韩立、乾老魔、白瑶怡、富姓老者、元婴期大汉（毒圣门四老、郑卫已溜走）
- 未到场被提及：天魔宗七妙真人（携灵龟飞车，两日后到）

## 背景
昆吾山秘境，众修士联手破去入口大阵后，联盟瓦解。此地通向多处去处，多人同往一地易起争抢，故众人决定分头行动。

## 分头探宝（第1006章 铸灵堂）
- 九幽宗长老『富姓老者』自言无自保之力，与白瑶怡同走一条石阶；『大汉』一行两两分组飞向另外两条石阶。
- 韩立神通明显高出富姓老者与白瑶怡一截，遂独自行动。石碑所列去处中，『昆吾殿』『镇魔塔』『灵宝阁』等重地必有人争，他避开，选了疑与炼器有关、十有八九无人的『铸灵堂』。
- 韩立离开后，银翅夜叉、丑妇、狮禽兽三妖以土遁术现身广场。它们意在趁人类修士分散时抢夺『本命牌』，并由隐匿术最高的银翅夜叉遮掩身形跟踪。
- 铸灵堂外有丈许厚光幕笼罩整片庄园。飞剑、巨剑术皆难一击破开（屏障类禁制不能瞬毁便会复原），噬金虫吞噬又太费时。韩立遂取出金银红三色的羽扇『三焰扇』，以三色火焰融穿光幕、化青虹潜入，代价是耗去近三分之一法力。

## 化灵殿夺鼎（第1007章 巨鼎）
- 铸灵堂内楼阁、石屋、仓库皆被上古修士撤离时清空，一无所获。最后韩立来到通体赤红、炙热逼人的『化灵殿』。
- 殿内十几根盘踞赤蛟的火柱喷吐红霞，煅烧一只六七丈高的古朴巨鼎，鼎中雷鸣不断，似在锻炼某物。
- 韩立判断附近有控制型法阵长年激发，遂飞剑击毁角落阵盘，火柱与红霞俱止。这才发现整座大殿建于一座**极品火池**之上，火霞源源不绝。
- 以灵目透视巨鼎，韩立发现此鼎经上万年地火淬炼，吸收的火力精纯可怕，已由普通法器**自行进阶成火属性异宝**，故而通灵。推测当年古修炼制某物到关键时无法停下，被迫撤山，遂启动法阵令其自炼。
- 韩立布下蓝色阵旗与水属性护罩，放出已进化生翅的**六翼霜蚣**盘旋鼎上；再以**紫罗极火**凝成紫焰大手抓向鼎盖。鼎面火焰凝成赤红火鸟相抗，被霜蚣寒气与紫手联手击碎，鼎盖被拍飞数丈。
- 一道赤芒自鼎中暴射欲逃，撞融蓝色光幕。韩立十指弹出十余道**青元剑气**（青色剑气）将其击得摇摇欲坠，再以紫罗极火大手擒下。失去内容物的巨鼎被霜蚣寒气冻成巨冰。

## 太阴真火与银月苏醒（第1008章）
- 韩立惊呼疑为『太阳精火』。沉睡数年、在韩立步入大殿时刚苏醒的**银月**以神识纠正：此乃与太阳精火齐名的**太阴真火**，人界三大真灵之火之一。
- 银月此次苏醒恢复了小部分久远记忆，觉前事恍如隔世，情绪略有伤感；但修养仅够勉强苏醒，暂无法为韩立出手。三大真灵之火等知识为其见火时脑中自然浮现。
- 此太阴真火为最低阶、刚诞生不久，故被几道剑气击伤、可被紫罗极火困住；若是太阳精火则最低阶也能拼本源破紫罗极火逃走。培育至高阶后威力将在紫罗极火之上。银月建议日后以太阴真火反吞紫罗极火（保其灵昧），并提议设法以此火相克相生之性捕捉太阳精火；韩立忆及大衍神君曾言太阳精火非本界修士所能捕捉，此事暂搁。
- 韩立仿当年囚禁乾蓝冰焰之法，以纤细电弧将紫焰连同阴火缠成拳头大金球，收入赤红玉盒。
- 击碎巨冰后，韩立从鼎中取出一团透明温润如暖玉、略有弹性之物（原材料经万年锻炼已不可辨，疑与太阴真火诞生相关），收入储物袋。
- 韩立以红色法决将巨鼎收为半尺**小鼎**（灵性十足，可释放无数『至阳火鸦』御敌，是地火之精所化），收为日后对敌手段——因三焰扇虽更强但太耗法力。
- 临行又以飞剑齐根斩断十几根吸饱火气的青铜**火龙柱**尽数收走（珍稀材料/灵器）。此后搜遍铸灵堂，仅另得几件普通炼器玉简。

## 要点
- 韩立此行已先得『培婴丹』，安然脱身为根本目的，故避开昆吾殿等争夺激烈的重地。
- 韩立现为元婴中期，禁空禁制下仍能短时强行漂浮。
