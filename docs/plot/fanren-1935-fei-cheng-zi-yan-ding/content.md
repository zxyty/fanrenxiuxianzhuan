# 第十卷 魔界之战 第1935章 废城

## 一、逼退血光圣祖三化身
- 韩立以**金毛巨猿**之姿（三头六臂法相）与血光圣祖化身、灵躯协同作战。远处一具与灵躯缠斗的化身得传讯后血光暴涨逼退灵躯，三化身聚作一团遁光向天边逃去。
- 为首少年（血光圣祖化身）阴森放言：此次因『趁手宝物未带在身边』才奈何不了韩立，下次见面必将其『抽魂炼魄』报仇，说罢血光在天边消失，退走异常干脆、头也不回，似笃定韩立不会追杀。
- 韩立收起法相、缩身褪去金毛恢复人形，并未露喜色，反生凝重：『只是几具区区的化身，竟如此棘手。若本体到来不知强横到何种地步……大乘期的厉害果然名不虚传，比传闻更可怕三分。』
- 灵躯化淡绿影没入韩立体内；十三只**紫纹噬金虫**被绿霞收入袖中。

## 二、追回并封印紫言鼎
- 韩立掐诀念咒，眉宇间浮现拇指大**血文**，确认『紫言鼎并未跑太远』。施法后另一方天边一道黑色遁光携紫呼呼圆鼎射来——正是先前自行飞遁的**紫言鼎**。
- 鼎现紫色小鼎原形，对接近韩立迟疑，但血文产生的冥冥联系又令其不愿离开。韩立放飞血文、弹射五色法决没入鼎中，血文亦没入，小鼎泛血色、龙吟欢鸣，化数寸迷你落入其手心。
- 韩立以双目蓝芒（灵目）查看：鼎色淡紫、铭有不知名附文、笼一层若有若无黑气。遂以金银符文贴封使其灵光呆滞，再以金色电网包裹缩成金色小球，装入白玉盒并贴十几张符箓嵌成微型符阵，将紫言鼎**暂时封印**。

## 三、紫言鼎来历与战局复盘
- 紫言鼎乃**玄天残宝**，是韩立按车骑恭所授秘术，从血光圣祖化身手中夺取并最终抓到手中；但鼎内藏有老魔一丝神念，未驱除重新祭炼前不敢直接使用、贴身收藏。
- 韩立喜得玄天残宝，却对刚祭炼完成、原备对付魔族大军的玄天残宝**万剑图**一战报废大感肉痛。
- 复盘：此次能逼退三名大乘化身，主因对方未做准备匆匆找上门、两件依仗的玄天残宝出意外，外加其影响**彩光塔**的手法。若三魔从统领魔族大军的另一具化身处借来宝物、又对手法有了防范，下次未必能再退敌。
- 韩立旋即宽心：对方若去魔族大本营往返起码三四个月，足够自己偷偷跑回**天渊城**；日后只要不离天渊城便无碍。

## 四、下一步打算
- 韩立决意先去离此不远的**黄泉地火**一趟：一为履行当初对车骑恭所发誓言、免遭心魔反噬，二为对**混沌二气**大为动心；趁血光圣祖化身受损短期内难卷土重来正是良机。
- 但车骑恭同为魔族圣祖，韩立绝不全信，必暗自防范后再行事。**倚天城**在合体存在败退后想已难保，无须再回。随即化青虹腾空而去。

## 五、魔族一方动向
- 数万里外，巨魔（血光圣祖本相/化身）收起法身，放出血色小舟载另两具化身赶往倚天城。数日后见倚天城城墙倒塌近半、上空遍布黑魔气。
- 一队骑魔兽的**万象魔骑**甲士恭迎，为首少年冷令：『马上开启离圣族大军最近之处的传送阵，立刻离开此地。』骑士躬身称『遵命，圣祖大人』。

## 六、废城异象（章末钩子）
- 半月后韩立化青虹于万丈高空疾行，遁光奇淡、波动尽敛，非大神通神念或灵目难以察觉。
- 飞过一座人族**废城**（魔族降临前亦不过小型城市，今遍布荒草废墟、无一人影、被人族放弃逾百年）。韩立神念下扫后骤『咦』一声，停遁现形，目光闪动、满脸意外之色。

## 备注
- 文中『灵躯』『金毛巨猿』均为韩立本人之化身/法相形态，行为归属韩立。

## 第1936章 遇旧 / 第1937章 虚皇鼎

### 一、韩立偶遇许芊羽（第1936章）
- 韩立循异动现身一座被乳白雾气（禁制）笼罩的住宅上空，五名服饰各异、皆**化神期**的邪修正围屋窃语。
- 韩立一句『给我滚』，并放出**合体期**庞大气息，五邪修魂飞魄散、落荒而逃。
- 屋中走出三名许家女子，为首者为**许芊羽**（许世家族），另有圆脸侄女**小青**等二女。许芊羽行大礼致谢。
- 韩立当年在许家与此女一别后再未见面，魔劫期间重逢出人意料；据称二人当年曾在**天渊城**共事，交情不浅。

### 二、魔劫情势与许家机密
- 许家在魔劫到来前已迁入由**圣皇**镇守的**圣皇城（圣城）**。
- **血灵**（许家『血灵大人』）外出办一件要紧事，带部分族人离城；办完返程途中遭遇魔族大军，厮杀战败后各自逃命。许芊羽与两侄女被小队魔族追杀，逃至许家多年前荒废的落脚处，借残存禁制对抗，又遇邪修。
- 许芊羽离开前亲见血灵杀出包围，虽有高阶魔族尾追，但自保无虞。
- 三女所办之事**事关冰魄先祖能否回归**，是许家极少数人才知的机密，故许芊羽未向韩立透露，亦未挽留他相助，以免节外生枝。
- **本次魔劫性质**：不同以往破城劫掠，魔族大有长久居住灵界之意——驻扎被攻破的凡人城市、奴役人族凡人；目标不止人族，妖族及木族、夜叉族等异族恐皆在入侵之列。人族据点多被攻破、宗门世家全灭，符箓传讯亦被割裂。
- 韩立宽慰：有**圣岛**与**莫简离**在，魔族阴谋难逞。
- 韩立告辞，忠告三女早回圣皇城，化青虹离去。

### 三、血灵取虚皇鼎（第1937章）
- 万里之外地下洞窟，一身血色战甲的娇媚女子拔出血光冰刃，已击杀三名高阶魔族（尸覆深蓝色冰霜）。此女即**冰魄仙子血魂所化的血灵化身**。
- 她将追兵引至此地实为有目的：祭出**虚天鼎**（韩立当年依仗之宝，现于血灵手中），念诀弹指，以青色法决冲开石壁后古铜大门上的银色禁制符箓。
- 门内大厅数亩空荡，仅一石桌置一小鼎，与虚天鼎七八分相似。血灵摄取，喜形于色。
- **虚皇鼎**：血灵出发前刚仿制出的第三只宝鼎，功效『远胜』虚天鼎、虚灵鼎，是『再进入那地的钥匙之一』。
- 取鼎后血灵以血色雷火轰塌整座大厅，化血虹离去，方向正是许芊羽三女所在小城。计划**魔劫一退即动身前往雷鸣大陆**。

### 四、一月后·韩立屠魔族精锐（第1937章末）
- 一月后，韩立高悬空中，俯视下方上百名紫甲魔纹**魔族**（最低元婴以上，为首两名**化神中期**精锐）围攻数十名人族修士。
- 人族同属一宗/世家，男女老幼皆有，修为参差：最低数名筑基少年，最高一名化神后期大成白须老者，多数为结丹期；摆玄妙法阵仍处下风。
- 白须老者催动大印放出白色蛟龙拼命，魔族放出鹿首蝠身魔禽虚影相抗，僵持。
- 韩立一声冰寒冷哼震慑人魔，爆出青濛光球，袖中抖出上百青芒化数百剑光——正是**青竹蜂云剑**之术：剑光虚空隐没后于魔族近空鬼魅闪出，将上百魔族连同元婴瞬间绞为碎片，血雾弥漫。

### 关键设定/线索
- 虚天鼎系列：虚天鼎 → 虚灵鼎 → 虚皇鼎（虚皇鼎为血灵新仿制、功效最强）。
- 雷鸣大陆为血灵后续目标地，需『钥匙』方可进入。

## 背景
第十卷《魔界之战》。韩立一路上已五六次撞见魔族围剿人族散修，在确认无魔尊在场时果断出手扫荡。

## 第1938章 黑雨上人
- 韩立以漫天青竹蜂云剑剑光斩杀围攻天马门的魔族，救下白须老者**简飞天**一行，嘱其向西避祸（西边暂无魔族），随即化青虹遁走。
- 简飞天推断救命者为合体期以上「姓韩」前辈，附近仅二人：新投圣皇麾下的**九泪真人韩清**，或隐居天渊城附近的**韩立**。
- 韩立抵达阴气极盛的**黄泉地火**——七座山峰联结的小型山脉，欲借此地炼制法器。
- 山顶遇两名合体中期修士：紫袈裟白胖僧人**天蝉大师**（圣皇两大手下之一，与韩立有一面之缘），及枯瘦丑陋的**黑雨上人**（本名雨啸天，现居圣岛）。天蝉惊讶韩立进阶合体不过数百年便达中期。
- 韩立透露现以客卿身份落脚天渊城，除天渊城外其余大型据点几乎尽数沦陷。

## 第1939章 黄泉地火
- 天蝉、黑雨上人坦言此行奉**圣岛**之命，为擒取黄泉地火中孕育五万年、已生灵智的**黄粱石灵**。
- 黄粱石灵生于布置大型幻阵的极品材料**黄粱梦石**，天赋幻术瞬息而发，须生擒不可伤命；曾有人族前辈被其幻术困半年。
- 圣岛大能占卜得知近期正是石灵度劫之日，是擒捕唯一良机。其体内**妖晶**为人族对付魔劫的重要之物，系**莫简离（莫大人）**经圣岛亲自交代。
- 韩立闻莫前辈之名而动容，慨然应允相助，负责万一时缠住石灵、不让其逃回地火。
- 黄泉地火奇寒源自九幽黄泉，韩立投白色法盘下探，瞬间被冰封碎裂；传闻或为直达冥界的入口之一，但奇寒大异普通冰寒，无法被生灵炼化。

## 第1940章 陷阱
- 高空黑云汇聚、银弧孕育，石灵天劫将至，巨坑黄雾翻滚。
- 二人合力布下森严禁制大阵；天蝉放出三只**暴炎鸟**（仅带一丝火凤血脉，形似火凤），黑雨上人放出碧蓝**水晶蝎子**，皆隐匿气息。
- 二人以晶球（奇寒）、青色羽扇（飓风）制造暴风雪，清除先前留下的气息。
- 黑雨上人祭出**银霄雷网**（混沌万灵榜排名不低的灵宝），化银雾笼罩山头，封死石灵从空中逃逸之路，且不受雷劫影响。
- 韩立自持神念之力不逊大乘，承诺拦阻石灵逃走。三人各自隐匿，静候石灵度劫现身。

## 关键设定
- **黄泉地火**：阴气极盛，奇寒来自九幽黄泉，修士最多下潜千丈，大乘度劫者亦仅万丈，再下有陨落之危。
- **圣岛**：每隔千年邀请新进阶合体修士上岛，部分留岛修炼，此事对合体以下修士保密。
