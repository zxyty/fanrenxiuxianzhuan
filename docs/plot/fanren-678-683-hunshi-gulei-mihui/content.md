# 第678-683章 魂石、上古傀儡术与南陇侯密会

## 背景
地点为天南**阗天城**百年一度的交易会前后。韩立身为**落云宗**三大长老之一，将修为掩饰为**结丹中期**，时戴面具，与同门**吕洛**、**古剑门**长老**火龙童子（蓝某）**同行。火龙童子幼年误食异草，容貌身形永葆童子模样，为人豪爽，是古剑门中唯一对落云宗友善的长老（暗示古剑门其余长老对落云宗不善）。

## 一、魂石交易会（第678章）
- 傀儡师**天晶真人**在**晶龙阁**举办交换会，以上古修士亲手制作的高阶**傀儡兽**换取一种上古之物「**魂石**」——一颗拇指大碧绿晶石，散发阴寒莫名波动，似阴魂之力却精纯得多。
- 韩立认出魂石即自己从**阴冥之地**收集的「**阴冥兽晶**」（土人称兽晶，上古称魂石），联想到虚天殿诸多傀儡皆暗含魂石，遂留心天晶真人。
- 韩立拿出鸡蛋大的巨型魂石，以一颗换走天晶真人剩余全部六只傀儡兽；交易中以传音暗示，令天晶真人惊疑，约定夜间再会。

## 二、制符与残符（第678-680章）
- 韩立逛坊市欲购稀缺制符材料并换一杆顶阶**制符笔**，为日后炼制威力非凡的「**降灵符**」（骨片所载密符，远超普通五行符箓）做准备。设定补充：天南专职结丹制符师寥寥几人，元婴期制符师尚无；高级符箓有时比顶阶法器还贵。
- 韩立撞见火云峰青年**孙火**因一张火云符被本地杂货铺掌柜讹诈三百灵石。韩立放出元婴灵压震慑，九国盟执法使**武斐**赶到，掌柜自曝符为普通货色认罚。
- 韩立见孙火所押符箓中有半张**残符**，带其至僻静处，取出另半张对合完全吻合。孙火乃**孙二狗第七代玄孙**；孙二狗当年发誓孙家世代奉韩立为主，但韩立当年不在天南，故孙家从未真正为仆。韩立给孙火两个选择（两不相干受惠 / 继续奉主、被种禁制、受提携指点，结丹有望），令其交易会后回宗再答复。

## 三、夜会天晶真人，换得上古傀儡术（第681-682章）
- 韩立先回住处，叮嘱随行的**慕沛灵**交易会期间自行行事，赠其一对**灵心佩**（碧蓝玉佩，注灵生护体灵罩并令韩立千里内感应）。
- 夜访晶龙阁，韩立先后赠出数颗巨型魂石以问两事：
  1. **庚精消息**：天晶真人透露拍卖行交易会期间或将拍卖一块庚精，原主似想以物易物尚未定，剑修门派恐倾家荡产抢拍。
  2. **上古傀儡术**：韩立点破天晶真人在上古遗址得到上古傀儡炼制之法、靠魂石融入半成品傀儡。天晶真人最终将一枚记载上古傀儡术的玉简直接相送，换走一堆魂石。
- 玉简所载最高级傀儡全力一击可媲美**元婴中期**，但主料为**万年铁木**等逆天材料；另有相当于结丹修士的低阶傀儡之法，材料同样稀绝。韩立离场佯装吃亏，实则窃喜：万年铁木他可用绿液培育，其余材料多可从**虚天殿**傀儡残骸中寻得，有望拼凑出几只元婴级古傀儡，重拾傀儡术杀手锏。

## 四、庚精拍卖落空（第682章）
- 半月后交易会正式召开。庚精（核桃大、淡金色矿石）最终拍出**76万灵石**；韩立超50万即放弃。
- 此前夜寻庚精原主（九国盟元婴修士，住西门外小山），对方欲换七级兽卵或「**化一神泥**」，韩立皆无而归；且该庚精仅够掺入七八口飞剑仍不够。**大庚剑阵**短期无法施展。韩立趁交易会搜集了部分上古傀儡材料。

## 五、南陇侯密会与旧敌重逢（第682-683章）
- 交易会第四日，韩立依**南陇侯**所赠玉简至阗天城一角石屋，南陇侯（玉冠紫袍长髯、元婴中期）引其入地下大厅密会。
- 厅中七八名修士，含正道、魔道、散修，皆**神识异常强大**（修炼特定功法 / 天生神识过人 / 有强化神识之宝）。
- 站立男子认出韩立失声——竟是当年追杀韩立的**鬼灵门少主王蝉（王婵）**，同戴银色面具，震惊于韩立已结元婴。其**二伯**即**鬼灵门门主王天古**出面，称当年是小辈无知，请韩立既往不咎；韩立表面应承，暗藏旧仇。王天古借称赞韩立「近二百年由筑基修至元婴」将其推为众矢之的。
- 王蝉身旁结丹后期美貌少妇为**燕如嫣**：**燕家**（越国第一家族）潜出越国后加入鬼灵门，将天灵根之女燕如嫣嫁与王蝉。王蝉二人精通秘术，可联手合二神识。
- 另有元婴诸修，含一名与南陇侯同为元婴中期、目光冰寒刺骨的白衫无须老者，及一名与王天古有纠葛的冷面尤姓修士。

## 六、密会真正目的
南陇侯揭示召集众人的目的：**随他同往幕兰草原（慕兰草原）**。话出，满座包括韩立皆脸色大变。

## 关键设定/伏笔
- 魂石=阴冥兽晶，上古傀儡术核心材料；虚天殿傀儡残骸暗含魂石。
- 韩立的傀儡术（千竹教数百三级傀儡曾几乎倾家荡产）将借上古傀儡术再度升级。
- 庚精用于强化本命法宝青竹蜂云剑、布大庚剑阵，暂时受阻。
- 幕兰草原之行成为后续主线。
