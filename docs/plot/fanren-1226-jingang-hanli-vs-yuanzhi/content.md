# 第1226章 金刚

第七卷 纵横人界。地点：天机阁悬空的天机殿。在场：韩立、雷音宗高僧元智（银袍僧人）、天机阁阁主粲苦、少年模样的焰竹僧人，及众多敬畏旁观的修士。性质为切磋而非生死相搏。

## 斗法经过
- **冰剑压境**：韩立催动青竹蜂云剑，上百道剑光聚为金剑，再借此前祭炼时掺入的极多万年玄玉之寒，化为数十丈擎天冰剑，凌空斩下，气势如冰山。
- **金刚现身**：元智以白莲遁术闪避，结手印放出五六丈高、金光濛濛、面目狰狞、半身的佛门『金刚虚影』（罗汉金身），与僧人动作同步，双拳侧击冰剑使其碎裂飞旋，又以双臂硬夹冰剑。韩立判断此秘术与明王决相辅相成，须修至第四层方能撑住，且与『六极真魔功』形似而本质不同（金刚乃自身法力所化，非借古魔真魔气）。
- **化剑为丝**：冰剑受损后被韩立神念瞬间修复如初；继而冰剑化为道道金丝从虚影大手中逸散，元智惊呼『化剑为丝』，认定韩立为剑修。青竹蜂云剑还放出粗大金弧化金色电蟒攻击。
- **三焰扇与紫罗极火**：韩立先弹出十道红丝（化为红色火链捆束金刚虚影），再以三焰扇放出三色火鸟，并喷出白濛濛圆珠催动紫罗极火的紫色火浪火海。
- **银瓶克火鸟**：元智祭出银光灿灿的佛门顶阶银瓶，七色佛光将三色火鸟卷入瓶中，似为火鸟克星；正欲再收紫罗极火时，韩立喝『爆』，引爆银瓶。
- **胜负**：银瓶炸裂，三色光晕暴涨，连同困住金刚虚影的火链一并吞灭金刚虚影与白莲。元智金蝉脱壳逃出，但金刚虚影与其本命精血心神相连，被灭时他喷出一小口精血、淡金肌肤黯淡，损伤元气。韩立再以紫色火蟒（口含白珠）相逼，元智苦笑认输：『住手！……老衲甘拜下风』，并惋惜顶阶银瓶被毁。

## 结果与后续
- 韩立收回紫罗极火白珠与火链（火丝原形），抱拳道『承让』落殿。在场众人（含粲苦、焰竹僧人）大感意外。
- 元智服火红药丸，称静养数月即可；评价韩立三焰扇连其罗汉金身都挡不下，应是某件灵宝的仿制品，韩立承认。元智态度转亲善。
- 阁主粲苦满脸笑容、毫无沮丧，主动履约：取令牌交一结丹修士，命其前往地面阁楼取『戒子空间』为韩立炼制秘术。
- 粲苦取出上古灵酒『津龙吟』（以人界已灭绝奇果酿制）款待。元智自述当年正是被粲苦以数十坛上古灵酒请为天机阁客卿长老。一行人重入大殿。

## 设定要点
- 明王决：佛门功法，修至第四层方能支撑金刚虚影硬接大威力一击。
- 金刚虚影 / 罗汉金身：元智的秘术，与本命精血心神相连，类本命法宝，可随意涨缩、刚猛抗寒。
- 青竹蜂云剑：此次祭炼掺入万年玄玉之多到极限，每口飞剑蕴含极寒，可化冰剑、化剑为丝、放金弧电蟒，并能自我修复。
- 银瓶：元智佛门顶阶宝物，能以七色佛光收摄克制三色火鸟，毁于此战。

## 章节梗概（第1228章 半途截杀）

### 一、得芥子空间秘术
- 韩立才离开**天机殿**不久，于高空青光遁行中把玩一块**淡红色玉简**——记载**炼制芥子空间秘术**之物。
- 韩立感慨芥子空间名声虽大，炼制步骤并不复杂，所需材料多为普通之物，故炼制不难。
- **童子(啼魂)**懒洋洋反驳：此术炼出的顶多是「可定位传送的空间裂缝」，真正的**芥子空间**是可随身携带、随时随地进入的**至宝**；据其所知，即便灵界，真正戒子空间也一只手数得过来，拥有者几乎都是灵界最顶阶存在。
- 韩立认为以人界修士之力无法炼化此等宝物，自己的**黑风旗**、童子所提**乾坤带**也只是稍具空间类神通之宝，不能称芥子空间。

### 二、定计灭阴罗宗、夺鬼罗幡
- 拿到空间炼制秘术后，韩立下一步要去**阴罗宗**夺**鬼罗幡（鬼罗枪/幡）**。
- 韩立不愿直接闯山：阴罗宗虽现仅余一名大修士，但为**魔门十大宗之一**、上古传承宗门，禁制与杀手锏难测。
- 计策：**擒贼先擒王**——先解决阴罗宗宗主，余下元婴长老便好对付。据坊市打探，近年多有大魔道宗门窥视阴罗宗十大宗地位，阴罗宗宗主近来常出门，可寻机在**邡莽山**外一举灭杀。
- 顾虑：动手须速决，不能拖延过久，以免被**化神修士**察觉插手，酿成大麻烦。

### 三、半年后的屠杀（封州·苀郡交界）
- 大半年后，封州和苀郡交界处某不知名山脉上空，五六名黑袍修士驱使上千只**铜甲炼尸**，围攻数十名各色修士。
- 尸气滔天，一顿饭工夫，除几名结丹修士外余者皆被群尸吞噬。
- 一道金石长啸直冲九霄，黑袍修士面色大变；一团数亩大小、遁速极快的**碧绿魔云**呼啸而来。
- 阴罗宗**房老魔**现身，认出来者为**血骨门**的**毕道友**，化灰光(转黑光、遁速倍增)追击；魔云仓皇后逃，两人前后遁出百余里消失天际。
- 失去外援的被困修士面露绝望，终被黑袍修士尽数灭杀；黑袍修士以乌黑皮袋收回铜甲尸，却滞留窃窃私语，目光屡望远空。

## 第1225章 韩立堵截阴罗宗宗主

### 背景
- **血骨门清剿**：阴罗宗宗主（房姓修士）回到属下黑袍修士聚集处，确认潜入本宗的**血骨门**七十三名修士已被尽数灭杀，无一逃脱。
- 血骨门长老**毕火**被宗主毁掉肉身，只放其元婴回去示警；宗主称若非本宗人手不足，本想直接杀上血骨门。

### 遣散弟子
- 宗主称此地暂无事，命众黑袍弟子先行回宗，自己另有事要独自处理，可能晚归。弟子化遁光离去。

### 喝破行踪
- 弟子散尽后，宗主喝破暗中观看许久的「道友」，自称**房某**。
- 韩立（青衫年轻男子，两手倒背）现身，点出房道友身为**阴罗宗宗主**，是先打发弟子、避免连累门下，才喝破其行踪。
- 韩立为准确堵截此人，曾在阴罗宗总坛所在的**邡莽山脉**附近监视半年多，跟踪至此。

### 对峙
- 宗主一眼认出韩立，黑气散去露出二十七八岁年轻面孔、脸色苍白、双目碧光妖异。
- 宗主确认韩立**已进阶元婴后期**；韩立明言此来二人只有一个能生离。
- 宗主提及传言**乾老魔**与**小极宫的寒骊上人**皆陨落于韩立之手，本就不信，更有**杀妻之仇**及数位长老之仇要报，遂狂笑迎战。
- 宗主身形一转浮现大股黑气将其淹没，黑气中鬼哭声起，数个高大身影诡异浮现。
