# 第2031-2032章 圣砖与血鸦城主 / 真灵遗骨

第十卷 魔界之战。地点：魔界·血鸦城拍卖会。

## 一、圣砖拍卖
- 主持人**越连天**（老年魔人）应炼虚后期魔族**津角**之问，说明**圣砖**来历：物主不愿招惹骚动；圣砖残缺，所记两套功法均缺关键口诀，且非**泣灵**圣祖当年主修神通，故由其主持拍卖。
- 津角认为纵功法不全，参悟泣灵神通一二亦远胜同阶，价值在「暗血五色铃」之上。物主要求不得事先透露功法名称。
- 竞价火爆：四千万→五千万→……血鸦城主以八千万出价时，奇寒气息席卷全场，众魔认出**血鸦城主（炳千刃，别名炳前辈）**——中阶魔尊、本城之主，银目鹰钩鼻、灰袍、散发血气。
- 众魔慑于魔尊威势皆噤声，唯一道飘忽不定的声音连出『九千万』『九千九百万』。血鸦城主跟到九千五百万后脸色阴沉，最终称『不夺人所爱』转身离场。
- 拍下者现身：竟是容貌普通、淡蓝长袍、炼虚初期的中年魔人『**羊老二**』。据邻座青鳞魔族介绍，羊老二是本城小有名气的炼器师，居住百余年、为人低调，竟敢与城主争抢圣物且身家可疑，举动诡异。羊老二拍下后未离场，似要继续参加压轴拍卖，令众魔起疑。

## 二、上古真灵遗骨（压轴）
- 拍品为十几丈长、通体青翠的巨兽残骨，浩然蛮荒气息逼人。越连天揭示其为**上古真灵遗骨**，从空间裂缝流入本界。
- 因岁月太久，骨内已无真元灵髓，无法分辨真灵种类，气息万不足一，价值大跌，仅相当罕见炼器材料（坚硬，炼宝威能不下普通灵宝）。越连天以青色香炉燃半尺香烛限时供众魔上台查验，并贴银符封印气息。
- 韩立从蛮荒气息中嗅到一丝似药香的香甜气息；神识中**曲儿**（九曲灵参幻化）惊喜传音，称此物对其大有用处，恳请务必拍下。韩立放神念扫骨后轻咦。
- 竞价冷清，底价五千万，仅六七人小幅出价。一名贵妇打扮的魔族女子出至七千万领先。
- 韩立加入：以兑换魔石（市价三分之二、不退还）为由到后殿兑换，回座后连喊八千万、九千万、一亿。
- 美妇自报为**枯寂城海家**之人（有枯寂圣祖坐镇的圣界炼器世家），称此骨对海家大有用途、愿事后重谢，出到九千一百万，但终未再争。
- 韩立以**一亿魔石**拍得真灵遗骨，收入储物镯，拍卖会结束。
- 街上行走时，曲儿兴奋表示：有此骨，韩立的**芝仙灵躯**日后拥有大乘神通大有希望。

## 关键点
- 韩立全程斗篷遮面、伪装炼虚后期魔族身份。
- 圣砖归羊老二（神秘人物，伏笔）；真灵遗骨归韩立（关乎芝仙灵躯成就大乘）。
