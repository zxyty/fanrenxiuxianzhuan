# 第485章 元婴现

## 场景
地点：虚天殿（虚天殿之行进行中）。补天丹现世，正魔元婴期老怪混战夺宝。

## 补天丹现世与争夺
- 五色光团漂浮于洞口上方，中心有一颗寸许大小的丹丸（**补天丹**）缓缓转动，每转一圈光团便收缩闪烁，仿佛有生命。
- **虚天鼎并未取出**，只此一颗补天丹现世，故正魔人人觊觎。
- 众人陷入『缠住对手 + 另想取宝』的僵局：极阴想放天都妖尸去取丹，却被黑瘦老者（玄骨）的绿丝预判缠住，无法脱身。

## 魔道联盟瓦解
- 极阴（极阴祖师）传音呵斥乌丑去收补天丹；乌丑化黑气前往。
- **玄骨**突然出手，双手一晃放出两道绿色怪蛇直袭乌丑背后；乌丑机警察觉，惊怒回身招架，与玄骨的绿色鬼气斗在一处。
- 极阴怒斥蛮胡子（『蛮胡子！你这子侄在干什么？』）。蛮胡子哈哈狂笑，一边以巨大金手狂击万天明，一边宣称：同属魔道也绝不会把补天丹让出——等于公开宣布**魔道联手解体，补天丹各凭能力来取**。

## 韩立死里逃生
- 韩立此前被**星宫长老穿心一剑**（白虹一击）竟未死，大出意外。
- 救命之物：
  - 贴身所穿的**皇鳞甲**（向蛮胡子暂借）——外衣焚毁过半，内里银鳞闪闪，中剑处仅略凹陷。
  - 真正消减白虹大半威力的是**红黄两色玉如意**——挥动抵挡时，如意中探出一颗**银色狼首**（即银月），低消了大半威力。韩立有意细究玉如意真实功效，暂收入储物袋。
- 韩立以白光自愈虎口剧痛伤处。

## 韩立欲逃被喝止
- 血玉蜘蛛已惨死，护身/利用价值尽失；韩立不愿把性命寄于老魔善念，凭超常记忆欲循原路冒险逃回第一层。
- 刚飞出十余丈，**玄骨**平静冷漠地喝问：『韩立，你要去哪里？』——声不大却让极阴、蛮胡子等尽数听到，冰寒目光齐扫韩立。
- 韩立自知破光罩的耽搁足够极阴灭他数次，只得苦笑退回原处。玄骨对付乌丑游刃有余却不去取丹、专盯韩立，意图不明，令韩立疑惑。

## 万天明元婴出窍（点题）
- 万天明屡攻补天丹皆被蛮胡子金手击退，恼羞威胁动用**天罗真雷**（极阴与青易皆忌惮此物）。蛮胡子不信其真敢用，出言相激。
- 万天明终被逼急，一拍天灵盖，一片紫光透顶而出，一个二寸大、白嫩、紫光缠绕、相貌与本人一般无二的婴儿浮现头顶三尺之上，双手各抓一颗紫色圆珠（晶莹温润）——即**元婴出窍**。其肉身本体双目紧闭、人事不知。
- 蛮胡子神色大变：『元婴出窍！万天明你真想魂飞神灭？』婴儿（万天明元婴）以细弱含糊之声答：若得不到补天丹则大限同样将至，早死迟死无别，顶多当被蛮胡子帮忙兵解——说几句已显吃力。

## 世界状态归属说明
- 据【世界状态须知】，本章『乌丑』的言行实归**极阴老祖**（自第403章夺舍占据乌丑躯体），故乌丑被玄骨绿蛇所阻、受极阴传音驱使等，归属于极阴老祖。
