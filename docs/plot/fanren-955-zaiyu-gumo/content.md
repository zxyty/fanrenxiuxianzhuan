# 第六卷 通天灵宝 第955章 再遇古魔

## 一、交换会拍卖平山印
- 主持白霞人影介绍重宝「平山印」——一件**通天灵宝仿制品**。韩立与心识中的大衍神君判断：连叶家这样的修仙世家都能炼出仿制品，其余渊源长久的大宗大派存有一件仿制品也大有可能；但炼制材料多已在此界绝迹，皆为多年积攒偶得。
- 大衍神君酸溜溜提醒：韩立手中尚有一件**正牌虚天鼎**，若能彻底掌握，此界无人能奈何他。韩立苦笑：没有**化神期**修为，掌握虚天鼎只能空想；但若真有化神神通，纵无虚天鼎亦可纵横人界。
- **竞拍方式**：叶家备好材料清单，凡能提供清单材料价值最多者得宝，且最高材料价值不能过低；最后再以**三百万灵石**为底价拍卖一次。三百万已可比一家中型宗门大半灵石储量，在场修士骚动却无人反对。

## 二、韩立的盘算与离场
- 韩立联想：自己的两枚**魔髓钻**若拿去拍卖也值不少灵石。魔道至宝**魔龙刃**上古仅有人炼成一柄，持刃魔修曾斩杀化神期修士，威力似不逊通天灵宝，且是地道法宝仍有潜力；但一两块魔髓钻无法炼出此刃，若仅当灵丹吸收精粹魔气则价值大降。
- 乌冠老者照玉简念诵材料清单，多为各大宗门必备储存之物（非五行玉、庚精这类世间难寻者），韩立判断叶家意在让各大宗门大出血。
- 韩立有自知之明：外来散修无法与大晋本地宗门争此影响大派牌势的重宝，遂起身离场，举动惹众人惊疑注目。

## 三、与富姓老者的密约
- 临行，**富姓老者**传音相询。韩立答已成众矢之的须早走，承诺数年后炼完一件宝物再详谈其事。
- 富姓老者点破韩立身怀魔髓钻至宝却不漏口风，难怪**阴罗宗**数人紧追；约定**四年后**到**南疆郡潮云府**相见，并会约其他几位道友于**双蝎山**会面，称此事虽要借韩立之力但对韩立大有好处。
- 韩立应「三年/到时赴约」后，步入禁制离去，化青虹向东激射，一口气飞出千里。其后亦有数名修士悄然离场，但多数仍留下争夺平山印。

## 四、再遇古魔分魂
- 遁光掠过一处山峰时，韩立体内飞剑发出龙吟剑鸣示警。韩立放出数十口金色飞剑护顶，喝令暗处者现身。
- 现身者为**古魔分魂**：相貌与韩立近似、一身青袍、面目平凡，瞳孔闪黑紫两色异芒。它当年在**坠魔谷**主魂被韩立灭杀，主魂之亡使其避免被吞噬、却也永远无法恢复魔界全盛修为。
- 古魔已夺走韩立**两口金雷竹飞剑**（金光小剑被层层黑丝缠缚，拼命欲飞回韩立而不得），并惊讶韩立竟有如此多**金雷竹**（破邪之物）。
- 古魔逼问获取金雷竹的方法，许诺可放韩立一马，否则将抽元神搜魂。此事涉及**神秘小瓶**秘密，韩立坚拒，反讥古魔在天南曾被重创未必痊愈。
- 古魔杀机大起，黑紫光芒大涨；韩立飞剑化金幕护身，另一手翻出装**噬金虫**的灵兽袋，战端将起。

## 五、生死备注
- **大衍神君**于本章（第955章）死亡。

## 第九百五十六章 魔焰滔滔

### 古魔屠头陀
- 韩立正欲试探古魔分魂伤势是否尽复，远处一道挟风雷之声的惊虹疾射而至，现出一名披发满脸横肉的头陀(后文知为大名鼎鼎的**恶火头陀**，元婴中期)，一手托赤红圆钵(**血焰钵**)，一手抓碧绿禅杖(**蟠龙杖**)。
- 头陀见韩立(元婴中期)与古魔(元婴初期)面容相像，误以为二人一伙，欲谋夺交易会宝物；又有三道遁光(三名元婴初期修士)接近。
- 古魔不耐，化黑芒突袭。头陀以血焰钵喷血红火焰结火幕、蟠龙杖化绿玉蛟龙相迎，再以青色法决催火成青红色火焰旋风护身。
- 古魔瞬移现身侧后，乌紫魔爪一击破其金霞光盾，洞穿丹田、将头陀身躯撕成两片；红色元婴驭金色飞剑瞬移逃二十余丈，仍被一道紫线(紫舌)洞穿头颅卷回火中。
- 三名元婴初期修士大恐，以更快遁速逃散。

### 韩立脱身
- 古魔现出半魔半人之态：手臂双腿乌紫巨大化，口吐数尺紫舌挑着头陀元婴。**大衍神君**(残识)提醒此处离地下交易会不远，恐引来更多修士，不可久缠。
- 韩立收青竹蜂云剑(金色飞剑)入袖，喷精血化血雾，掐诀背生**风雷翅**(银白翅膀)，化血影遁走；古魔瞬移截杀，被脑中一声冰寒冷哼所阻(疑大衍神君出手)，韩立趁隙化血影连遁逃出其神识极限。
- 古魔惊疑：此术似**铁翅魔**秘术**化影遁**，但又不太一样。旋见六七道遁光来犯，古魔自语数年前所吞元婴已炼化干净，欲再吞一批以求数年后愈伤，遂化为**双头四臂巨大魔物**。

## 第九百五十七章 天符门

### 背景与地理
- **华云州**：大晋东部沿海大州，海外修士常出没，坊市兴隆。**开江镇**离海二百余里，镇外数里有**白竹山**(二三百丈高、灵脉低劣)。
- **天符门**：白竹山中的修仙最底层小派，掌门长老加弟子仅百余人。数万年前由**天符真人**以自创三大密符威震大晋，曾差点跻身十大正门、门人数万雄踞一州；然主修功法平平，“成也符箓败也符箓”，今仅靠符箓秘术延续。非太一门、阴罗宗那般上古宗门。

### 坊市危机
- 两名炼气期守山弟子(李师兄等)议论：**三元坊市**(天符门与**金霞山**、**明阳谷**共理)被**灵风门**下最后通牒——三月内合并或关闭；合并后每年仅给上千灵石(不足原收入三分之一)。背后有中等宗门**煞阳宗**推动，非小宗门可抗。掌门当日请来另两家掌门商议对策。

### 韩立来访
- 距逃脱古魔四个月后，韩立伪装成相貌普通、气势非凡的青袍青年(实为元婴修为)落于白竹山门前，称带来一件“与贵门大有渊源”之物，要见掌门。神识扫山，仅感应到一名结丹初期气息，印证此乃末流小派。
- 此行缘由：恶蛟出没之地正在华云州，且韩立已学得**降灵符**炼制之法。
- 掌门**岳真**(筑基期、黄色锦袍老者)惶恐出迎，请入白竹山大殿(殿中另两名他派中年修士告退)。随后**温师叔**(白袍老者，门内唯一结丹初期长老)以白光射入殿中，众修士纷纷见礼。

## 关键人物状态(本章)
- 韩立：元婴中期，伪装青袍青年；拥风雷翅、青竹蜂云剑，施血影/化影遁，习降灵符之法。
- 古魔分魂：元婴初期(实力远超表象)，半魔半人→双头四臂巨魔，吞元婴愈伤。
- 恶火头陀：元婴中期，本章被古魔撕杀、元婴被吞，身死。
- 岳真：天符门掌门，筑基期。温师叔：天符门长老，结丹初期。

## 背景
第六卷 通天灵宝 第958-960章「李师侄 / 化神传闻 / 密符和八级恶蛟」。地点：大晋皇朝·华云州·白竹山·天符门。

## 主线事件
1. **归还降灵符**：韩立以元婴修士身份造访没落的小宗门天符门，奉其失踪掌门「云师兄（云师伯）」之托，归还刻有降灵符炼制法的骨片。云师兄数十年前赴海外途中误入鬼雾、困于阴冥之地（该地无法动用法力），与韩立相遇后托其转交。韩立坦言自己也曾遭遇阴冥之地、费力才逃脱（隐去阴冥兽晶等秘事）。唯一长老温姓老者、岳真等极为敬畏。
2. **借阅制符典籍**：岳真主动邀韩立观阅门中符箓典籍，韩立心动留住数日，在藏经阁二、三层翻看并暗中复制所有制符典籍。
3. **三大密符设定**（韩立读典所得）：
   - 降灵符：因材料问题，自创立起几无人能炼。
   - 化灵符：将特制符箓纳入丹田日夜培炼，类法宝；修至一定程度可起魔道「化劫」类神通、卸去功法反噬，结丹后以丹火即可培炼，为历代结丹修士必修。
   - 六丁天甲符：借符力凝聚天地灵气成六层护罩，灵气不绝则护罩生生不息，是少数不需化神修为即可驱使天地灵气的法术；玉简于天符真人坐化后离奇失踪，历代寻而无果。
4. **识破向之礼**：藏经阁打理者「李师侄」被韩立一眼认出，实为二百余年前黄枫谷血色试炼中幸存的旧识**向之礼**（当年以炼气期十层走出禁地）。其表面炼气期一层为幻术，容貌与二百多年前几乎无异。向之礼始终以「姓李非姓向」否认。韩立与大衍神君均无法以神识看穿其真实修为——身体如空荡荡一片。
5. **化神传闻**（大衍神君之论）：此异状要么源于特殊秘术/至宝，要么对方已进阶**化神期**——化神可将全身灵力散入体内经脉、连元婴短时化为无形。大衍神君称这一界确有数名化神期修士滞留，虽有飞升灵界之能却未必真飞升，且进阶化神后便不能常在人前露面，否则恐遭奇祸；故古今化神修士皆似约定般从修仙界消失。韩立遂决定佯装不知、暂按原计划再留几日，并布下淡青色防护禁制。

## 后续计划
- 韩立血影遁所致精元亏损已基本恢复，准备斩杀恶蛟后即开始为大衍神君炼制傀儡，关键是先取得**赤火蛟鳞片**；乌凤翎之事待傀儡炼成后再办。
- 大衍神君自称大限将近，盼于辞世前得见此傀儡。

## 恶蛟与屠蛟大会
- 临江府/海岛一带有恶蛟群（约七级），近来出现一只**八级化形蓝蛟**，伏击中重伤过一名元婴修士；南海门数位长老联手才击退群蛟，未能尽杀。
- 八级蛟龙身为天地灵兽，实力恐胜元婴中期，且在海中如鱼得水。南海门已邀众多高阶修士齐聚**琼英岛**召开屠蛟大会。
- 群中确有一只红蛟（疑赤火蛟）随往海边，正是韩立所求之物。

## 客卿长老之邀
金霞山常道友、明阳谷谢道友连夜赶至确认韩立身份；天符门、金霞山、明阳谷三家小派欲聘韩立为三派共同的**客卿长老**。客卿长老为大晋中小宗门特有的挂名靠山职位，不受门规约束、不听号令，仅于宗门有难时可名正言顺出手相助（无强制性），待遇优厚。韩立未明确应允。
