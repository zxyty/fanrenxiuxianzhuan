# 第九卷 灵界百族 第1542章 乌甲兽 / 第1543章 敌现

## 背景：火云岛戒严
- 妇人与少女（白珠儿）离去后，火云岛戒备森严数倍。火阳族蛇人不再离岛打猎采料，十余万族人收缩入土城。
- 土城被赤黄绿三层光幕笼罩，数千精锐蛇人巡视；祭祀齐聚大殿打坐养气，大殿另有五色光霞防护，源头为广场中央圆锥状法器。

## 韩立出关入殿
- 韩立以话语声穿透禁制求入：『火道友，在下已炼化丹药出关』。妇人大喜开禁放入。
- 韩立此前用**噬灵火鸟**吸尽**烈阳神丹**中火灵力后吞服炼化；精血、神念亏损恢复大半，法力亦将复原。
- 韩立自承『只恢复七八分』，但已**进阶炼虚期**，多种神通大进，即便碰合体级存在亦有自保之力，故不惧火阳族之敌。
- 白珠儿因送丹时险被**啼魂**吸走神念而对韩立心存惊惧。

## 敌踪出现
- 韩立到来半日，四面地动，乌罗族来犯。妇人以红色令牌催动赤红法阵与丈许赤色晶球（功用近似韩立的**万珑珠**），由八名白袍蛇人催动法决，映出城内外景象。
- 城墙外钻出高七八丈、长十余丈、披黑色鳞甲的巨牛怪物——**乌甲兽**，背负巨框载满人影，无声逼近。蛇人惊呼疑为早该被娲氏灭族的**乌罗族**。

## 第1543章 敌现
- 乌甲兽上人影：上半身淡紫肤色、长发披肩、黑甲持乌黑兵刃的眉清目秀女子；跃下后显露**下半身为半截蜈蚣之身**，数十肢足，狰狞裂口尖牙吐蛇芯。
- 妇人以精神秘术（一声冷哼）鼓舞低阶祭祀战意，被修炼**大衍决**的韩立轻易看破。妇人下令分四队赴四城门督战。
- 妇人称娲氏与乌罗族为生死大敌，乌罗族擅隐匿潜入，惯先杀对方最高修为者，料其高层会主动寻来。

## 殿内交手
- 韩立指出旁有隐匿道友，单手虚抓木柱逼出一名银瞳乌罗人；其错认韩立为雷鸣大陆以灵目神通闻名的『白目』族或『千幻』族。
- 韩立又以一口**青色小剑**斩向阴影，逼出第二名隐匿者（黑丝/黑影）。青丝紧追黑丝绕殿疾驰，速度提速一倍有余；黑丝化七八道分身遁走，重聚现出**淡金眼珠**乌罗人。
- 妇人辨出『金银双瞳』为**乌罗王族**特征，断定其余两族大祭司亦为其所杀。两名乌罗人身罩诡异灵光、有异宝护身，连韩立神念亦难看穿修为。
- 白珠儿以黄色大弓放骨箭抢先发难，漫天黄箭罩向二女；妇人红色短棍化赤红火蛟、四名白袍蛇人挥银色长刃齐攻。乌罗二女各发乌芒、化两团黑色光晕旋转护身应战。

## 关键人物修为
- 韩立：炼虚期初。
- 四名白袍蛇人：结丹初期，为族中仅次于妇人者。
