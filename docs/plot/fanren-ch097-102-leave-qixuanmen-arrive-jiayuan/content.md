# 凡人修仙传 第97-102章 剧情梳理

## 第97章 背后交易
- 七玄门高层接厉飞雨通报、得知韩立将于次日午时来落日峰后，内部争执：一派主张趁会面除掉韩立（实为图谋其修炼法门），一派主张温和待之。最终由一名‘幸存下来的灰衣人师叔’一句‘害死他不怕其长辈找上门’点醒众人，统一为温和对待。
- 韩立深谙‘鸟尽弓藏’，回谷后闭门不见、故意拖延，并未按时赴约。为安全计，他提前半日独自密会王绝楚（王门主）。
- 韩立以天眼术试探，王绝楚露出马脚，证实高层确曾密谋下手。王绝楚以‘已拿下几个败类’搪塞。
- 韩立表明将远走他乡、永不再回彩霞山，提出与王绝楚做‘一笔对双方都有好处的交易’。
- 午时正式会场，韩立未现，王绝楚最后入场宣布韩立已离开彩霞山、不知所踪，麻烦消失。

## 第98章 回村
- 韩立离开后，王绝楚把厉飞雨收为关门弟子、扶正其外刃堂堂主之职，宠信有加；数年后还力保犯帮规的韩立三叔。
- 王绝楚日后屡遭重伤，皆靠玉瓶中药丸奇迹生还，秘而不宣；临终才留下药名‘养精丹’，时瓶中仅剩三颗，日后为后人招来血雨腥风。（此为闪前交代）
- 厉飞雨晨起发现韩立留下的数瓶延寿秘药与纸条（署名处画一张笑脸，祝其与张袖儿早日成亲），登峰远望、黯然送别。
- 韩立与曲魂租整辆四轮篷车东行，肩头停着黄羽‘云翅鸟’；墨大夫所遗法器、信件书籍因关系重大贴身携带。目的地为韩立出生的山边小村。

## 第99章 离开
- 韩立入村正逢办婚庆——竟是自家小妹（韩家四丫头）出阁，许给城里秀才。
- 他躲于树后，认出胖三叔、黑大个大哥韩铁及大嫂，最后目光落在两鬓灰白、腰背躬下的双亲身上，神色复杂。
- 将至亲面容刻入心底后，韩立不相认，转身大踏步出村。自学长春功、知晓修仙者存在起，他已决意走与常人迥异之路，无论祸福不悔。

## 第二卷 初踏修仙路

## 第100章 嘉元城
- 岚州为越国十三州中富足程度第二、首屈一指的产粮大区；其中部嘉元城为岚州第一大城，乡鲁大运河穿城而过，水运枢纽、商贸要道。
- 码头苦力头目孙二狗（四平帮）与黑熊（铁拳会，绰号黑爷）两帮素来不和，每日以手下对骂为例。两帮实为同盟，正联合对抗较大的毒龙帮，故暂忍。
- 韩立与曲魂赶路三月、初抵墨大夫故乡嘉元城，乘扁舟登上这处又破又偏的小码头。

## 第101章 钱财惹祸
- 韩立自故里东南行，途中曾以野狼为晚餐，穿两州抵岚州，惊讶于其四通八达的水道。
- 依码头不成文规矩雇脚夫；按孙、黑两帮轮流接活的约定本轮该黑熊一方。脚夫接过曲魂包裹（内藏数千两白银及杂物）才知份量惊人，连唤两人方抬起。
- 孙二狗、黑熊均看出包裹藏巨财，于垃圾堆后密谈，最终‘四六分账’、击掌为誓暂结盟约。黑熊已暗令两名手下把韩立引向黑水巷设伏。

## 第102章 格杀
- 两脚夫七拐八绕将韩立、曲魂诱入黑水巷，巷内窜出十余持械大汉，身后又现孙二狗、黑熊率七八人，前后合围。
- 韩立认出谋财害命之局，冷令曲魂‘杀了他们，不用留手’。
- 曲魂暴起：一拳碎人头颅、挥臂震飞兵器、扫腿踢飞两人，连毙数人，凶威骇人。
- 孙二狗加赏‘杀巨汉每人二十两’驱众；黑熊眼力老到，看出曲魂之强连帮主亲来亦未必能胜，遂放弃银子之念，暗暗向韩立靠近，意图劫持这位‘世家公子’为人质逃命。

## 关键转折
- 韩立彻底斩断与七玄门、与家乡凡人世界的牵连，正式踏上修仙之路（第二卷开篇）。
- 嘉元城为墨大夫故乡，是韩立此行目的地，新的码头帮派冲突引出初入岚州的第一场杀戮。
