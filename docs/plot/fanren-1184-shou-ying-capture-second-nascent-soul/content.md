# 第1184章 收婴

第七卷 纵横人界。承接韩立追擒脱离躯体的第二元婴一线。

## 一、封印外的伏击
- 奎焕与妇人率三十六名落云宗弟子，将阵旗阵盘抛向魔渊封印上空，五色光霞凭空浮现下卷，将逃出的乌光(第二元婴)裹困其中。
- 韩立反常地一改单独行动作风，事先在封印出口召集众多低阶修士、布下威力不小的禁制法阵封锁出口，专等第二元婴自投罗网。

## 二、第二元婴的来历与逃法
- 它本被三焰扇火焰笼罩，下意识将灌体魔气尽数激发保命；魔气被火焰化为乌有、魔躯也被炼化大半，却误打误撞解除了元婴的魔化，瞬间恢复灵智。
- 它拥有韩立先前的记忆，生死一线下果断自爆魔器赤血剑破开三色火焰，又怕韩立以风雷翅追来，再自爆修复好的罗幡，借两宝自爆从韩立眼皮底下逃出。
- 计划：遁出魔渊后立刻施展元婴瞬移之术，逃往七灵岛中某岛预设的隐秘藏身之所(动用特殊宝物+隐匿气息高明法阵，欲瞒过韩立神念扫描)。

## 三、被困与挣扎
- 被困五色光霞中，第二元婴挥手放出漆黑剑气狂斩，光霞虽被斩开却又瞬间恢复。
- 它双手搓出两道黑色光柱化为两口丈许巨剑同向斩下，劈开数尺孔洞；因孔洞过大、修士法力供应不及，孔洞未即时合上，第二元婴趁机瞬移逃出光霞。

## 四、韩立收婴
- 第二元婴现身光霞之外，欲连续瞬移逃离，韩立以风雷翅随风追上现身；元婴不照面即瞬移走。
- 韩立施展**天眼术**：眉间裂开血痕，黑气冒出凝聚化作一只乌黑眼珠，射出黑芒击中二十余丈外虚空，爆出隐遁的第二元婴。
- 韩立喷出**虚天鼎**(青色小鼎)抓在手中，背生青白色羽翅(风雷翅)随风隐匿；从虚空喷出一蓬**青丝**(青竹蜂云剑系)守株待兔，将元婴五花大绑，乌黑剑气破之无损。
- 韩立单手托鼎现身，弹指开鼎盖，青色霞光冲起，青丝灵光大放使元婴灵力转动不灵，将其卷入小鼎；鼎盖落下重新密封。身处虚天鼎中，元婴后期大修士也绝无法逃出。

## 五、善后
- 奎焕、妇人率筑基弟子上前恭贺。韩立赏两瓶丹药给奎焕分予众人，嘱咐魔渊须增派人手严加看管，以防妖人混入。
- 韩立婉拒回灵鳖岛休息，称另有要事，化青虹离去。红袍修士艳羡丹药，奎焕不愿多谈，众修士重新加固表面灵光黯淡的封印。

## 六、回子母峰
- 十余日后韩立回到云梦山，不回落云宗而直回**子母峰**母峰洞府。
- 洞府前有记名弟子**柳玉**(白袍貌美女修)恭候；她与慕师姐、宋师姐三人担心、轮流守候，二人此时在小师妹洞府中小聚。
- 韩立提及新收弟子**田琴儿(琴儿)**：身怀**龙吟之体**，大道恐注定无望，收她另有原因，让柳玉不必试探。
- 韩立命柳玉召集三人前来，称正有话要对她们几人说。柳玉化白光奔田琴儿洞府而去。
