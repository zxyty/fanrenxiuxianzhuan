# 凡人修仙传 第1498-1500章｜虚天殿混战

第九卷 灵界百族（第一千四百九十八章 出手 / 第一千四百九十九章 混战 / 第一千五百章 意外迭起）

## 背景
灵界虚天殿内，韩立潜伏一旁，护着神色木然、被抽取血丝的元瑶与妍丽二女，等待唯一一次现形救人的机会——若失败，此殿便是其陨落之所，千余年苦修付诸东流。

## 妖王围攻黑色禁制
- **木青 / 白发美妇 / 血袍人**三方联手攻打高空中两只**冥雷兽**操控的**黑色光阵**（黑风裹黑色晶沙，狂猛如刀）。
- 木青催动翠绿小树放出无数树叶翠影；白发美妇巨大化（高七八丈、古战甲、八骷髅头怪锤，八阴甲鬼王化为骷髅喷绿焰）。
- 冥雷兽以银弧加持光阵，并开十丈黑洞将血袍人祭出的七只金属傀儡尽数吞没消失。
- 血袍人（地血道友）由**紫血傀儡**承载，怒祭血球。

## 金甲傀儡持五龙铡突袭
- **金甲傀儡**借另一鬼物诡异神通逼近，却因木青预布的隐形警戒灵丝而暴露。
- 五龙铡（银色铡刀）犀利绝伦，斩碎青丝、切开青色纱巾。木青被寒光淹没『陨落』——实为**桃李代僵秘术**诈死潜伏（木妖之体，尸体不流血）。
- 金甲傀儡转攻紫血傀儡。他急于出手，因『六足』已潜入池下、欲切断灵脉，若灵脉被断，纵杀外敌回族亦获大罪。
- 血袍人以万年秽气血球反被五龙铡洞穿，又以血色飞剑（喷精血再凝）缠住铡刀；五龙铡以一化五，分袭血袍人与白发美妇。白发美妇祭百余阴甲鬼兵，交由琉璃马面鬼物抵挡。

## 韩立出手救人
- 趁众妖王分心，韩立现身二女身旁，双手按肩，无数金弧（**辟邪神雷**）涌入，逼出血丝并化为乌有，卷二女化青虹奔出口。
- 白发美妇惊怒拍击、灰色鬼爪自空抓下，韩立机敏避开；身后碧绿翠芒（诈死复出的木青）袭来。
- 韩立瞬间加持三层防护：金色鳞片（梵圣真魔功）、黑色怨甲、金色雷袍；袖出十几颗雷珠化青色雷云，将大意的木青困在原地（雷云不致命但难脱身）。

## 意外迭起：血傀儡与金甲士
- 韩立将欲冲天遁走，血雾中探出血腥大手抓破雷袍、黑甲两层，却被金色鳞片滑开无法抓破；韩立诡异身法化虚影脱身。
- 现身者为**血傀儡**，厉害程度不下合体期，远胜韩立当初所杀两只；紫血傀儡脑后双目自主盯向韩立，显已不受血袍人控制。
- 韩立化数条各抱二女的虚影四散，本体被血傀儡锁定，以**元磁山**（元磁之力，爪芒不能抓）硬挡，并祭**雷蛟剪**（天鹏族，化两条青色雷蛟）缠斗，趁机带二女化晶丝遁入虚空。
- 血傀儡击飞雷蛟欲追，地下钻出两名紫金甲士（一持金枪化金蟒、一持双刀放金色风刃）拦截，为韩立再次奔至出口、化青虹飞射争取时机。

## 悬念
封印绿池的金色光阵剧烈波动、地面震动，众妖王最忌惮的**六足**即将冲破封印而出。

## 背景
地点为灵界（灵界百族卷）。韩立先前从白发美妇（鬼婆）、血袍人、紫血傀儡及金甲傀儡等围困中救走元瑶、妍丽二女，体内被妖王种下封印印记，随时可能被催动，故一路狂遁并设法除患。

## 第1501章 困敌脱身
- 血傀儡追击韩立：其血色光柱洞穿韩立两名金色甲士（约有韩立七八分实力与部分神通），甲士虽自愈、施展金色电蛇与五色寒焰，仍被血傀儡放出的血色光球所化血焰溶解为血水消灭。
- 韩立早已遁出百里之外。血傀儡循蓝色通道追出，却接连遇上『第三只冥雷兽』（实为啼魂兽化形诱敌），被迫据守。
- 韩立事先布下的**九宫天乾符**于入口虚空触发，化为巨大银色光阵；冥雷兽（啼魂）悬停阵上，下方**噬灵火鸟（银色火鸟）**喷出十几颗青色雷珠逼退血傀儡，将其卷入光阵所化白雾禁制世界，短时间无法冲出。
- 雷珠倒射回被火鸟吸回；啼魂化回半尺黑色小猴遁走，火鸟化银焰消散。百里外韩立收回火鸟与啼魂（入灵兽环），知晓全程；随后展开风雷翅化晶丝破空远遁。
- 地下宫殿：木青被困雷云；白发美妇、血袍人、紫血傀儡苦撑五龙铡攻击，黑色光阵将落；池中封印破裂，一道高大人影飞出。

## 第1502章 转轮聚阴阵
- 黑色光幕区域地面塌陷成超级大坑，数团灵光追逐飞出、互相争斗，一团脱离单独离去。
- 韩立内视体内印记，封印暂凝固未松；他清楚妖王只是暂时无暇，催动印记后自己压制不久。
- 飞出百余万里，于黑气弥漫、阴气凝重的丘陵带落脚，用金色小剑（青竹蜂云剑/金雷竹飞剑）开洞、布旗造黑雾，作临时藏身处。
- 救治二女：因辟邪神雷洗荡加二女半鬼之体、肉身不强而昏迷；韩立以十几根银针、青色符箓、掐诀引导，并以一分惊神刺神通唤醒二女，确认体内异物清除。
- 决策：此地阴气有六七成把握驱印记，时间紧迫不另寻地。妍丽提议布**转轮聚阴阵**短时间聚集惊人阴气，三人合力驱除印记。妍丽以灰旗阴丝测定邻近丘陵为阴气最浓的阵眼，二女着手布置可笼罩整座丘陵的巨阵。

## 第1503章 施法除患
- 巨阵布置两个多时辰完成。正欲施法时，一名妖王（六足等之一）催动印记，韩立灵压忽高忽低，仍称尚能镇压，决意先驱除被催动的印记。
- 妍丽提醒：阴气入体秘术本用于鬼修/鬼物，常人之躯接阴气灌注会剧痛甚至当场昏死，一旦昏迷无人引导则施法失败。韩立自信肉身强横（修多种炼体秘术、灵药淬炼），不以为意。
- 阵中阵小型法阵：二女各喷乌黑圆珠嵌入凹槽激发巨阵，百余丈高空形成吸纳阴气的黑色漩涡，中心徐徐凝成漆黑墨球（含强大灵压）。
- 鬼物受精纯阴气吸引蜂拥而来；韩立放出啼魂兽与进阶后的**豹麟兽**（化十余丈巨猿、幻出残影），迅速驱逐/吞食低阶鬼物。
- 二女抛黑幡引漩涡黑气，以指尖弹出无数黑丝阴气灌入韩立手背、体内；韩立仅身上青光微闪，承受远超二女当初辟邪神雷之苦，令二女钦佩骇然。韩立称已感印记松动，催二女继续灌注，二女遂以黑光裹身喷出墨色光柱灌注，韩立闷哼晃身、青光闪烁。

## 关键设定与道具
- **九宫天乾符**：韩立预设的困敌符阵，可远程触发，化禁制世界困住合体巅峰级敌人。
- **转轮聚阴阵**：元瑶、妍丽所布，短时聚集海量精纯阴气，用以驱除体内印记。
- **辟邪神雷**：韩立用以洗荡清除二女体内血丝控制。
- **噬灵火鸟/银色火鸟**：可喷收青色雷珠，与啼魂兽配合困敌。
- **印记**：妖王（六足等）种于韩立体内的封印，可远程催动，是本段除患主线。
