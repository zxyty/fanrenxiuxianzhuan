# 凡人修仙传 第2234-2239章 虫母之战

第十一卷《真仙降世》。海底深渊封印之地，韩立、宝花、蟹道人三人探查螟虫之母封印，引发恶战。

## 一、水面之战：两具化身（2234-2236）

- 韩立（幻化成妖魔模样）、宝花、蟹道人面对螟虫之母的两具化身：
  - **神念怪虫**：透明如液、遍体眼珠、长触须，施『煞雷』（灰白色电弧，蕴一丝毁灭法则，连玄天之宝都可硬破），神念秘术强大。
  - **肉身怪虫**：青色尖甲覆体，一对鬼爪般巨大前肢，纯以肉身/巨爪之力压人。
- 分工：**蟹道人**（伪仙傫傀儡）就地化金色巨蟹，浑身雷光、放雷球缠斗神念怪虫——傀儡不受神念秘术影响。**韩立**以玄天斩灵剑斩肉身怪虫。**宝花**催土皇钉所化擎天巨柱对抗肉身怪虫巨爪，并出大成元婴（粉红小人）出窍。
- 宝花放出玄天灵域困住神念怪虫，韩立十二道绿线一斩将其灭杀。两具怪虫被杀后均化白光消散、尸体如泡沫爆裂。
- **关键疑团**：怪虫并非真正肉身，韩立判断其形体像『投影之力』凝聚，怀疑有其他界面之人（甚至仙界真仙）插手螟虫之母之事。两怪虫之间、与元神女童之间隐隐有敌意，非同路。

## 二、深渊之变：骸骨觉醒（2234-2237）

- **古朴老者**与**黑袍妇人**（两名下界大乘）潜入近万丈深渊，发现螟虫之母本体——一副万丈白玉巨虫骸骨，散发可怖灰白气息，似早已坐化许久。
- 黑袍妇人以**天机枪**（漆黑短枪）劈骸骨头颅，毫无痕迹；继施**黄泉秽焰**（采数界八十一种污秽之气炼成，可抹去轮回资格，消耗性绝技），古朴老者祭**万兽牌**放万兽虚影——皆无法近骸骨数丈。
- 骸骨觉醒站起，恢复实力。螟虫之母元神女童（先前被三人缠住）赶来，与骸骨用未知语言争吵；女童施**时间法则**（黑光指尖）逼骸骨吐出漆黑晶块，骸骨却反手将晶块化巨口吞掉女童，吞噬其元神。
- 古朴老者、黑袍妇人欲逃，骸骨以时间法则令二人不知不觉倒退至其身下，一口吞噬。二人陨落，残破玉牌、短枪弃于骸骨旁。

## 三、本体复苏：真极之躯（2237-2239）

- 骸骨眼眶燃起黑色魔焰，化出银纱女子（女童长成十七八岁之貌，即螟虫之母元神主体）。她收回两具怪虫化身之力，重铸血肉：白骨化为**一大两小、三颗头颅的巨型怪虫**——中间人面（银纱女子之容），另两头狰狞遍布眼珠、覆五色鳞片，身躯水晶般透明，背生四对乳白蝉翼。外形与水面上被毁的旧躯壳截然不同。
- 三人返回，识出此即螟虫之母本体，已完好复苏。

## 四、灵域之争与恶战（2237-2239）

- 螟虫之母放灰丝弥天大网；蟹道人化巨蟹雷网托住，宝花放玄天花域护身，韩立出梵圣金身法相（三头六臂、紫金鳞片）持玄天斩灵剑斩十二绿线。
- 虫母弹黑色符文，以**时间法则逆反之力**将绿线反弹回韩立等人——非普通反弹。韩立连番动用玄天之宝，法力大耗、脸色发白。
- 虫母施展『**真正的玄天灵域**』：深渊翻转，化漆黑乌云+白骨骷髅海，骷髅头喷苍白鬼火，再合体为百丈白骨骨架、白骨甲士（持白骨巨叉、气息不下大乘）。宝花的玄天花域被斥为『伪灵域』，只得缩小至亩许护身、香汗淋漓难支。
- 韩立化三头六臂金猿，放金光球、洞玄金光漩涡吸碎骷髅（虫母讥其为模仿『轩元神光/元磁神光』的假货）；宝花插**土皇钉**入花树枝爆发生气，挺过白骨甲士最后一波骨叉齐投。虫母灵域威能耗尽褪去，但宝花玄天花域亦彻底被毁。

## 五、真极之躯无敌与夺宝（2239）

- 螟虫之母自述：早在十几万年前即可破印而出，甘愿被困是借上古封印玄妙之力打磨玉骨、重铸『**真极之躯**』；今真躯已成，纵全身无一丝法力，肉身也不惧任何宝物攻击。
- 宝花暗藏第二枚土皇钉偷袭（趁天地法则碰撞之机祭出），细针没入虫母前肢、化尺许巨钉，黄符将山岳之躯锁缩至数丈，真元大半凝固难调。
- 但虫母凭真极之躯一击震飞第二枚土皇钉、举手夺回，并自身躯划出无数裂缝吞掉黄色符文与法则之力挣脱，从前肢拔出古钉——『土皇钉本为一对，合二为一才发挥全威』之谋彻底破产。宝花面如纸灰，再无战胜信心。

## 旁注

- **封印之灵**因力量流逝极虚弱，宝花已无法令其现身，除非让上古封印彻底失效。
- 神念怪虫与肉身怪虫的神通若集于一身，三人将只能逃命；韩立、宝花约定遇无法应对之险不硬拼。
- 开篇阴森声音曾以螟虫之母肉身之力与『**螟二**』相较（螟二仅被提及）。
