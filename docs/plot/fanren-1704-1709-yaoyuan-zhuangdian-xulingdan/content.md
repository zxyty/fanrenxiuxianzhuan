# 凡人修仙传 第1704-1709章｜药园破禁·主殿虚灵丹之争（第九卷 灵界百族）

## 背景
韩立与石茧族修士石昆、晶族女修柳水儿三人合伙，进入灵界一处疑为远古仙人居所的秘府寻宝。出界后他们仍要回云城面对彩流罂与段天刃，且附近有角蚩族游荡，故三人维持着脆弱的平衡合作。

## 一、药园破禁（第1704章 掷山）
- 韩立独自先抵药园，遭五色光霞禁制困身、五根紫金柱放出紫色电蛇围攻，一度性命堪危。
- **梵圣金身登场**：韩立催出三头六臂法相，并以新炼成的『金身法相（梵圣金身）』使其通体如赤金神像；金身六臂可徒手抓住、捏爆紫色电弧并反震。
- **山岳巨猿变身**：韩立施惊蛰决中的『山岳巨猿幻化术』，化身十余丈高蓝眼金毛巨猿，肉身强横程度骤增四五成。
- **掷山破禁**：巨猿单凭蛮力，把约十余万斤的元磁极山当石块连续投掷五次，逐根砸毁五根紫金柱（柱坚但远逊元磁极山），五色光幕失去支撑后被银色火鸟吞噬崩溃。韩立惊觉此变身比其他真灵化身更适合自己（直接强化肉身）。

## 二、采集灵药（第1705-1707章 红罗仙酒·蚀毒草／束手无策／银色莲蓬）
药园虽小，灵药珍稀：
- **红罗果**：叶淡黄、结红辣椒状红光果实，是酿制传说仙酒『红罗仙酒』的主材料。红罗果千年开花、万年结果，培育极难；此园上古遗留，年限药性惊人。韩立不嗜酒，视为鸡肋但价值连城。
- **蚀毒草**：丑陋黑紫浆果灵草，含剧毒（合体误食可毙命），是炼制十几种顶阶灵药（含直催合体修为灵丹）的必需材料，百万年前已在灵界灭绝、无法移植培育。此田七八十株。
- 另有一种带金丝清香小草，韩立不识。
- **曲儿采药**：韩立唤出芝仙元神所化的白衣女童『曲儿』（昔以白兔随他多年），以十指碧丝成排拔取、净根入玉匣，瞬息采空成片灵药。
- **单株灵药自灭之谜**：十几个独立药圃中的灵树灵花多为仙界独有、韩立不识。其根部被药园主人暗设禁制，一离土即枯萎/自爆/化液/电灭。韩立先用元磁神光（借元磁极山催至极限）破去灵树下方金灵气丝网，仍无法整株移走；遂改以青竹蜂云剑剑丝斩取果实/种子部分，再用乾蓝冰焰冰封、符箓禁锢带回慢慢研究。
- **银色莲蓬**：泉眼上漂浮的银莲连剑丝都反震而灭。韩立发现银藕根须不入淤泥、蜷成银雾团，遂布水属性法阵、催动水灵气制造漩涡，连泉水带银莲银藕一并吸入蓝色玉瓶，成功完整收取（唯一完整到手之灵物）。
- 韩立把角落杂项灵药也尽数采空，大有所获。

## 三、联手破主殿（第1708章 平衡与破门）
- 出药园时，主殿方向传来灵压——石昆、柳水儿被广场幻术折磨后，动用超出自身掌控的秘术/宝物硬破幻术，但已耗去近半法力，且火锤、宝镜先后被殿门紫电毁掉。
- 韩立赶到，提出『此处禁制需三人联手』。石昆、柳水儿因忌惮韩立深不可测的神通（自付联手亦非其敌），又有彩流罂、段天刃在云城静候为后盾，遂请韩立主持破禁，并各自亮出底牌（柳水儿一套破禁法器、石昆一枚家师所赐破禁符箓）。
- **破门**：韩立以天外魔甲护身、催梵圣金身（柳水儿失声认出『金身法相』，惊其稀有——法决与凝练材料皆罕）。金身抓挡门上紫电，元磁极山被电网裹住；石昆喷银符、爆白色光莲，柳水儿射三柄金戈钉网吸电，削弱禁制。最终元磁极山二度暴涨猛击，主殿大门寸寸碎裂。

## 四、主殿现宝·虚灵丹之争（第1709章 屏风·金鼎）
- 主殿近千丈广，列上百根紫金柱，四壁悬近千件长戈巨斧、堆放五色盔甲（皆非凡品），但正中尽头只有：
  - **青光屏风**（七八丈高）：以天眼术（灵目神通）看清，绘有金银两色符文的古朴巨门，符文为真仙界的银蝌文与金篆文。韩立放出一缕神念探查，神念被屏风吸入、与本体失联；石昆同样吃亏更甚，柳水儿亦察觉其诡异。
  - **金色古鼎**：表面螺旋云雾纹令人头晕，完全隔绝神念；两侧各有方形凹槽，疑专为镶嵌两物、取出鼎中宝物之用。
- **冲突起**：石昆、柳水儿一见金鼎凹槽便露狂喜，韩立追问下，柳水儿言此鼎应是盛放『虚灵丹』及他们家师所需之物的法器，二人身上各带一物（钥匙）可开鼎。
- 石昆奉段天刃之命、柳水儿奉彩流罂之命，皆『虚灵丹势在必得』，互不相让：柳水儿以晶族镇族之宝『晶月液』诱石昆退让被拒，气氛凝重。韩立从中劝阻——『未见丹药就剑拔弩张太早』，建议先入殿确认宝物再议归属，二人冷静接受。章末石昆翻手取出开鼎之物，韩立凝目看个真切。

## 关键设定/进展
- 韩立此卷已掌握：梵圣金身（金身法相）、惊蛰决·山岳巨猿幻化术、元磁神光、天外魔甲、元磁极山、青竹蜂云剑、乾蓝冰焰、天眼术。
- 元磁极山乃天地奇物＋灰白异石＋真仙界炼制秘术二次炼成，坚韧近乎无敌、可借肉身蛮力为投掷武器。
- 引出势力/种族线索：石昆出身石茧族（其家师为段天刃），柳水儿出身晶族（家师为彩流罂），晶族镇族之宝为晶月液；附近有角蚩族。
