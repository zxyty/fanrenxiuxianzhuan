## 章节定位
第六卷 通天灵宝 第一千一十三章 万斤压身。昆吾殿夺宝场面。

> 注：据【世界状态须知】，自第1009章起『韩立（韩长老/叶家客卿）』身份为【古魔分魂】化形冒充。但本章片段中所有以『韩立/韩长老』之名行动者，结合上下文（弹金弧逼妖、用三焰扇、修明王决服天尸珠、施辟邪神雷化电衣）均为本尊韩立的标志性手段与法宝，故本片段行为归于本尊【韩立】；古魔分魂冒名一事以 note 标注存疑。

## 一、韩立破幕抢宝、金弧逼出三妖
- 韩立靠近光幕，银弧雷击袭来；他以金色电蛟自爆化出金色巨网护身，银弧尽数化为无形。
- 乾老魔与方脸中年修士同时惊呼『韩立，是你！』『韩长老！』。韩立回首见方脸中年人一脸吃惊却不认识对方，心生奇怪。
- 韩立以三焰扇击出三色火球破开光幕大孔，闪身脱出光幕之外。
- 见徐姓青年与林银屏现身平台、脸色难看，韩立冷笑，弹指射出一道金弧。金弧没入光幕银弧、自另一端弹射而出，击中平台边一处不起眼石墩。
- 一声闷哼后，逼出隐形的三妖：**银翅夜叉、狮禽兽、腰粗如水桶的丑妇**。徐姓青年一眼认出狮禽兽，众人戒备。
- 葛天豪等人则看到光幕中的『五子同心魔』，惊喜呼『大长老！』。

## 二、三妖处境
- 银翅夜叉认出韩立是差点要其小命的仇人，破口大骂，并纳闷对方为何三番两次看破其『随风掩形之术』，令其被动。
- 三妖并非真怕人类修士，而是想借这些修士之手破除『后面一道关键禁制』；如今行迹暴露，进退两难。
- 徐姓青年衡量：银翅夜叉与丑妇皆已化形，妖气惊人不下于狮禽兽，不敢轻举妄动。
- 光幕中乾老魔与叶家长老虽欲住手，但两头青铜狮（傀儡）追扑不停，二人只能边应付边互相警惕拉开距离。各方僵持。

## 三、韩立闯山门遇『万斤压身』禁制
- 韩立不理乱局，直奔后方抢先取宝，落在碧玉所制山门前。山门奇特：仅两排不知名焦黄巨树（高二三十丈，却给人茂盛之感），四周空荡。透过大门沿白石路千余丈可见古朴巨大宫殿——**昆吾殿**。
- 韩立以神识反复扫视巨树与整座山门，均确认是普通树木、无隐藏禁制，遂迈步进门直奔昆吾殿。
- 踏上白石路十余丈后，骤觉寒气直冒、双肩重压加身，双膝一软几乎跪地。他大喝放出青濛濛护体灵光，仅单膝着地，仍浑身颤抖、骨骼嘎嘣爆响如要被压碎。
- 护体灵光仅抵消一小部分重压，浑身如万斤巨力加身。**幸赖修炼过明王决并服下天尸珠，身体坚韧远胜普通修士，方未被当场压碎趴伏**。
- 此前明明搜查无任何法阵，却莫名陷入厉害禁制。韩立连手指都无法动弹，只能以咒语催动神通：以辟邪神雷化出一层金色电衣强行起身，但直起半身后重力骤增，再次被压半跪。韩立脸色大变。

## 四、殿外：乾老魔 vs 叶家方脸长老
- 山下众人因光幕遮挡与山门地处高处，只见韩立没入群殴，未见其被困实情，心悬重宝者皆按捺不住。
- 乾老魔暴怒，将禁制中五道白影合五为一化为身高数丈的**白色巨影**（五子同心魔），手臂化白蟒一口吞住一只青铜狮，盘身脱离化为巨大光球将青铜傀儡彻底困住。
- 叶家方脸修士点动身上**巨大骨环『弥天镯』**，幻化无数黄濛濛环影困住另一只青铜狮，使其翻身栽倒。
- 乾老魔评弥天镯：传为上古灵宝『映月环』的仿制之宝（一直未被证实），即便是通天灵宝仿制品也差不了多少。
- 叶某称若无此宝早已命丧乾兄之手；并讥讽乾老魔若只顾纠缠，不怕宝物被韩立先取走。
- 乾老魔追问：方才叶兄称此人为『叶长老』，难道他是叶家修士？叶某眼珠微转打哈哈称『看错了人，并不认识此人』。
- 乾老魔指出：纵此人天南外姓人，叶家不可能真信他；遂提议——既一时奈何不得，不如**合力先破昆吾殿禁制，到了昆吾殿再各凭手段取宝**。叶某出乎意料地一口答应『各凭手段！正有此意！』，反令老魔踌躇。

## 关键设定/伏笔
- 昆吾殿山门白石路设有【隐形重压禁制】，神识无法探知，踏入即触发万斤压身。
- 弥天镯 ←传← 上古灵宝『映月环』之仿制品。
- 韩立护身底牌：明王决、天尸珠（身体坚韧）、金色电蛟/金色巨网/金色电衣（辟邪神雷系）、三焰扇、青濛濛护体灵光。
