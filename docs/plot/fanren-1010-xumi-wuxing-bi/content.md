# 第一千一十章 须弥五行壁

## 真魔塔入口被封
- 毒圣门长老**花天奇**当机立断，以翠绿霞光卷起本门四人，化绿光抢先射入真魔塔下一层入口。
- **大头怪人**暴怒，以黄色剑光击散围困的阴云欲追，但花天奇已遁入出口消失。
- 大头怪人与**古魔**（本章自称『韩兄』，实为冒充韩立的**古魔分魂**）联手击散阴气后，入口处凭空生出一堵晶光闪闪的浑厚晶壁，将通道堵得严丝合缝。
- 大头怪人剑光劈在晶壁上，被五色霞光化为乌有；他与叶家老者（**叶月圣**）惊呼此乃**须弥五行壁**——南疆大宗**须弥宗**的镇宗之宝，因须弥宗与毒圣门颇有渊源被借来使用。
- 古魔（韩兄）冷笑劝慰：下层囚禁之物更棘手，毒圣门先下去是自讨苦吃，待先收拾鬼王再破壁，正好坐收渔利。

## 毒圣门的盘算
- 晶壁另一面，毒圣门长老埋怨此举得罪叶家。花天奇反驳：叶月圣早年假死逃过十大宗门灭杀，既被撞见绝不会轻易放人；况且连本派祖师爷都念念不忘之宝，多半是传闻中的**通天灵宝**。
- 花天奇透露已随身携带本宗之宝**毒龙珠**，称拿宝后大不了拼着静养十年服珠，可与元婴后期修士周旋以脱身。
- 花天奇惋惜**元师弟**（元姓大汉，已于第979章身亡）不在，否则联手可斗元婴后期；同门猜测元师兄或已应口讯赶来。

## 真魔塔第八层：传送阵与蛮荒仙境
- 下行十余丈即见一方石室，内有一黑一白两座小型**短距离传送阵**（精通上古阵法的**苗师弟**判定射程不足百里、双向）。
- 花天奇择白色传送阵而行（黑阵令众人心惊肉跳）。一阵天旋地转后众人现身于第八层：蓝天白云、灵花奇草、灵气扑面的仙境。
- 正前方有一座金碧辉煌的小型宫殿坐落于超大法阵之上，四周众多祭坛各立一尊手持金色巨刃的巨大石人，默对宫殿而立，透出蛮荒诡异气息，令花天奇等瞠目结舌。

## 韩立白玉广场遇围
- 同一时间，真正的**韩立**回到白玉广场，因后悔过早走出**铸灵堂**，被一群自石阶两侧潜行包抄的修士围成半圆、堵死下山之路（此地神识大受限制）。
- 围困者：**天澜圣女林银屏**（韩立曾取其玉佩祭炼成感应法器以防备，数年后法器功效已失）、面熟的白袍青年、**葛天豪**、两名黑袍老者。
- 白袍青年自报为**突兀族**四大仙师之一的**徐姓**修士（元婴后期，对韩立威胁最大），点名要与韩立切磋。
