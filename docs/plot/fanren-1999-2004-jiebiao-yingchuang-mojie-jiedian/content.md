# 第2013-2015章 天卜 / 灭口 / 硬闯（第十卷 魔界之战）

## 一、行程与节点选择
- 韩立随**陇家老祖**、**千秋圣女**等一行（含**羽衣少女**、**林姓披发男子**、陇家**晖姓黑袍长老**等合体期高手）渡海前往进入魔界的节点。
- 该节点位于**玄武境**偏僻角落，可直通魔界荒凉的**血石山脉**（魔族驻扎极少）。羽衣少女曾惋惜未选天渊城地域节点，陇家老祖称天渊城附近节点不可用，否则不是羊入虎口就是落点离目标太远。
- 沿途遇数队魔族巡逻卫士（与天渊城魔族不同：身材更高、魔甲铭刻蓝色魔纹），修为最高仅化神期，众人以合体修为掩遁光大模大样掠过，未被发觉。

## 二、隐匿与混入手段
- **千秋圣女**祭出**烟云幡**，喷出白色奇花化成亩许白雾护罩，再以五色符文令众人化为虚无；该雾隐匿玄妙，连韩立放出强大神念都扫之空空。
- **韩立**以瞳孔蓝芒催动**明清灵目神通（天眼术）**，才看到化为护罩的薄薄白雾，判断除精通特殊秘术的魔族尊者外，一般高阶魔族无法察觉。
- 陇家老祖分发**伪魔珠**（黑黝黝圆珠），吞服敛息后可激发纯正魔气蒙混过关，但对禁制抵抗不足。

## 三、宫装女子的『天卜』（第2014章 灭口）
- 后方百余里高空，一名**白色宫装女子**（清冷、把玩紫色铜钱=**天命铜钱**）携黑甲魔族大汉**黑鄂**现身，欲再开坛**天卜**一次（上次反噬未消，但不愿错过法力恢复良机）。
- 黑鄂请求赐下**震元锏**代为击杀这群合体修士夺灵药，被拒：黑鄂修为发挥不出震元锏几分威能，且天命铜钱示警表明对方有抗衡手段。
- 布坛：黑鄂放出**八只半尺高大头木人**（替身傀儡，幻化喜怒哀乐鬼脸）；以数百石板聚成七八丈碧绿石台（蛮荒气息）；抛蓝晶球掀巨浪化碧蓝水幕罩坛。
- 祭品：黑鄂以黑色皮袋倾下千斤血水（击杀合体期古兽所得精血）注入紫盆；宫装女子登粉红→淡银花树，以白玉骨刀割腕滴**淡银色血液**入花树，再踩爆花树化漫天银色星辰，配金色罗盘弹指演星图卜算。
- 反噬强烈致最终祭坛突然爆裂、八只替身傀儡尽毁，短期内无法再行卜算之术；但仍得含糊、似自相矛盾的卦象，唯一确定：**这群人与她法力恢复大有关系**。

## 四、灭口与暗追
- 一队数十人暴魔族卫士骑带翅鲨形魔兽路过，触爆隐形水幕，未受伤但发现宫装女子与黑鄂（神念探不出二人修为，黑鄂魔气纯正）。为首化神期**暴魔族石烁**躬身探问。
- 宫装女子冷然下令『灭掉』，黑鄂放黑气妖虫尽数绞杀逃散卫士、提头颅回。
- 宫装女子已在**人族女子（羽衣少女）**身上暗下印记，明暗之势令对方无法摆脱追踪，遂尾随暗追。

## 五、硬闯节点（第2015章）
- 数时辰后众人抵一座数百里珊瑚岛上空，见**魔族要塞**占岛近半：百丈灰墙、层层禁制、数层黑色光幕、双头怪鸟巡空、上空数百亩黑云翻滚银弧闪烁——疑有魔族尊者坐镇。
- 陇家老祖解释此地原仅小要塞，魔族突增大量人手才扩建至此，出乎预料。
- 议定：改换节点已来不及（此处最适合且落点最准），伪魔珠抗禁制不足，遂改为**直接隐匿身形向节点飞去，一旦被识破即强行硬闯**，凭众人合力可斩一两个出面阻拦的魔族尊者；入魔界后即龙入大海难追。林家披发男子、晖姓长老附议，羽衣少女虽顾忌走漏消息仍从众。
- 千秋圣女极力压低烟云幡威能避触禁制，接连擦过七八波巡逻魔族未被察觉；至距要塞十余里时触动一层巧妙法阵禁制，无形利刃四射。
- 千秋圣女令云雾凝成厚光幕挡黑芒，众人合力以宝物雷火击碎禁制，化十余道惊虹直扑要塞——魔族大军被惊动，尖鸣四起、禁制灵光遍布、魔族甲士骑魔兽气势汹汹腾空迎战。

## 备注

## 章节
第十卷 魔界之战 第2016-2018章《宝煌圣花 / 天泣和鹤颜 / 魔源海》。

## 一、强闯节点通道
- 韩立与陇家老祖、千秋圣女等一行十人(一半人族、一半灵族，皆合体期以上，含四名合体后期)到达节点黑色巨大光幕前受阻。
- 披发林家男子以青色巨爪击打光幕无效；灵族老儒**藏形**(经千秋圣女介绍、精通阵法禁制)以七八道符箓轻易破开光幕，开出孔洞，众人穿过后孔洞弥合。
- 巡逻魔族围来，十名合体存在合力化七色长虹，焚灭魔族卫士，连破七层禁制。
- 要塞乌云中魔族放出擎天雷网，**陇家老祖**祭出陇家镇族之宝『**净雷瓶**』，化白气光丝将漫天电弧尽数收入瓶中。乌云中魔族惊呼『净雷瓶！你是人族陇家的人！』。
- 通道入口处魔尊**客云**(化名/本体后述)率数百魔族阻路，但见对方合体后期众多，惊骇之下让出通道。韩立等径直没入灰色节点通道。

## 二、宝煌圣花/宝花圣祖现身
- 客云让路后，迟来的**蜂兄**(带翅男子)赶到追问异族下落，二人忧心圣祖责罚。
- 虚空女子声音命二人不得上报。高空显现十余丈晶莹**宝煌圣花**花树，粉红奇花化火焰飘落，瞬间烧灭包括拼命逃遁的蜂兄在内的所有守卫魔族，只剩客云一人。
- **宝花圣祖**(白色宫装女子)与黑甲大汉**黑鳄**自花树现身。客云本体现形：头生漆黑弯角、面容四方的中年大汉，激动跪拜。
- 宝花自述：当年在『那两人』联手下保住性命，但被打落一个境界，强行破空带部分手下(暗部)逃至灵界。
- 宝花以晶丝探查客云神识，揪出敌方暗设的阴毒禁制(逼出耳鼻中灰色光点擒回)，化解后以虚假禁制替代，除非『那二人』亲查否则不会被发觉。

## 三、宝花的任务与图谋
- 宝花问明确有一队异族(半人族半灵族、皆合体以上)进入节点通道，遂命客云设法掩盖此事、尽量拖延不让圣界知晓(客云称监视他的另一人已被宝花灭杀，可隐瞒年许)。
- 宝花携黑鳄进入通道，返回魔界一端；令黑鳄清扫围困的中低阶魔族。
- 宝花透露此行因一次卜算卦象(残缺、仅参悟一半)而冒险返回圣界。明言敌手为**天泣**、**鹤颜**两位身具始祖之位者，二人联手必能杀她。
- 宝花指示黑鳄：在『前边那些人』(韩立一行)到达目的地前，不得打搅、不得让其发觉，甚至要暗中相助。许诺黑鳄日后进阶大乘瓶颈时出手相助。黑鳄表忠。

## 四、出通道与魔源海三路线之议
- 韩立、陇家老祖一行出通道，落点正位于魔界**血石山脉**(魔气资源极缺、占地广大)，飞行半月解决一群魔禽后到一片草原(魔气较凝厚)。
- 目标：寻**洗灵池**与**净灵莲**，据上届灵王遗言坐落于魔界最东**魔源海**的神秘岛屿，具体线索须到当地方能用。
- 陇家老祖提出三条路线：
  1. 经**冲月大平原**横穿魔族繁华地域、借魔族巨城传送阵，顺利则1-2年；但巨城有魔尊乃至圣祖化身坐镇与重重禁制，即便用『**伪魔珠**』也仅约两成蒙混机会。(羽衣少女/叶道友、千秋圣女反对)
  2. 沿**第五魔河**入海走海路，穿多片海域至魔源海，约5-6年；魔海海兽无数、不乏魔尊乃至圣祖级化形海兽。
  3. 沿草原边缘西行入魔界荒地，穿**幻啸沙漠**(魔界十大禁地)：任何修为入内法力被压制九成、遁术宝物皆不可用，只能步行；主要威胁为幻啸魔风(神念无损)，但徒步穿越需五六十年。(披发林家男子以耗时太长否决)
- 千秋圣女透露另有取巧之法可将穿幻啸沙漠时间缩短近半至三十来年，但仍嫌过久。
- 众人(韩立、千秋圣女偏稳妥)初步倾向风险居中的第二条海路，韩立则希望第三条能再缩短时间。议程未定。

- 天泣、鹤颜为圣界始祖级强者，是宝花圣祖的死敌。
