# 第2300章 族誓

第十一卷《真仙降世》。本章双线并行。

## 一、神秘青年下界（疑为上界/真仙修士）
- 一人影破阵显形：身穿黑色长袍、两手倒背的青年，皮肤白皙异常，黑白交错衬出病态瘦弱之感。
- 他射出银色符文打通一道直通地表的巨型峡谷，飞出地下，停在离地千丈高空。
- 周围是遍地黑色泥石、仅有寸许野草、死寂无虫鸣的荒凉死地（下等界面）。
- 掐诀凝聚出一头**狮首麟身**、上半身血焰滚滚、下半身银光灿灿、足缠艳丽霞光的百余丈凶兽；但旋即脸色一白、闷哼，巨兽无法成形，只化为虚影。
- 自语：法力被压制到此种程度，『这次下界果然不是什么好事，怪不得观中会派我走一趟』；但也因此『那些老家伙也无法干预』。
- 图谋：手中一件宝物『还差点火候才能炼制大成』，欲借下界生灵血气（虽不纯但数量多）重新**血祭**弥补；并提到暂不急着去找『那名叛徒』。
- 取出金灿圆筒，内插遍布银纹、缭绕五色霞光的**翠绿竹签**；抛起化为金光球，浮现巨大八卦虚影，射出翠芒指向『生灵气息最多』的方向，遂化金虹追去，狮首麟身凶兽虚影随行。

## 二、韩立炼化山海珠 + 空鱼族族誓
- **山海珠本来面目**：原淡蓝色晶珠经空鱼族长放精血抹去内中历代先祖所留痕迹与掩饰之力后，缩小至蚕豆般大小，表面光滑、幻出五色、缭绕一缕乳白之气。
- 韩立承诺炼化此珠后立刻带空鱼族离开小修罗界。
- 进入空鱼族长平日修炼的三角建筑密室：除常见摆设外，有一排黑色木架（竹简玉盒瓶罐）与数尺高赤红丹炉——乃族长孙女炼丹之物（此界灵药单一，只能炼少数几种）；韩立称若其孙女确有炼丹天赋，回去后可加以培养。
- 韩立闭门布禁制，于蒲团坐定，先以黄色玉简（族长所献，记载空鱼一族**专为山海珠所创的祭炼之术**）领悟秘术。
- 施法：定珠于半空，数团精气罩上，掐诀显**三头六臂梵圣法相**，眉心黑光一现睁开**第三颗法目**，射出密密麻麻神念晶丝。**两日两夜**后成功炼化山海珠，并感到此界排斥之力渐起、不能久留。
- **族誓（族誓本章题眼）**：广场上数百空鱼族人肃立。族长率先拜下，代表全族『五百一二口』下血誓——自今日起全族世世代代永不悔改地侍奉主人韩立，违誓则全族遭天谴、血脉之力断绝、世间再无空鱼一族。族长以短刃插入胸口数寸，身后族人亦以利刃插身发誓，天空隐现亩许大血色符文一闪而逝。
- 韩立现身众人上空，受誓并承诺：收下后绝不歧视，自当作族人看待。

## 章节
第十一卷 真仙降世 第两千三百零一章《再见灵王》、第两千三百零二章《元魂灯》。

## 一、收纳空鱼族入山海珠
韩立喷出山海珠，霞光所过将广场上空鱼人尽数挪移入珠内「洞天」——一处中心有一大三小四岛、大半为蔚蓝水面的神秘空间。韩立命空鱼族长率族人暂在岛上修屋居住，待其回族寻得安置之处再放出。空鱼族跪伏称「主人」。

## 二、返回灵界
韩立在排斥之力萦绕、银弧霹雳中触发预设传送，从小修罗界经空间光阵返回灵界一座荒凉石山顶。放出神念勘验，确认未偏离原设坐标，此行成功。三个时辰后莫简离寻至，二人同往伏灵山找灵王换取雷霄符，以防符数不足被血燃等抢先换走。

## 三、伏灵山交易（核心事件）
- 灵族圣灵「灵引」（光头男子）于禁制处迎接引路，入殿见血燃、黑鳞已先到。
- 灵王（白袍老者）出关，验看晶核：成年修罗蛛晶核蕴含**时间之力**，足以提炼**光阴之力/光阴之丝**。
- 黑鳞先交修罗蛛晶核换得贴符玉盒中的**三清雷霄符**（半尺玉牌，电弧缭绕、金文隐现，雷电之力惊人）。莫简离、血燃随后各以三颗晶核换符。
- 韩立按兵不动，反问灵王尚余几枚符。灵王坦言三清雷霄符为**灵族镇族之宝、仙家密符**，换给血燃三人后仅余两枚。
- 韩立拿出一只玉盒（内装三颗晶核）与一口尺许晶剑。该晶剑以修罗蛛晶核（三四枚）炼成，原为奕姓老者催动时间之力的五口晶剑之一，四口在激战中被韩立毁去，仅余坠地这口被韩立悄悄收起。韩立以此共换得最后两枚雷霄符，四方皆大欢喜。

## 四、归途清点所得（墨灵圣舟密室）
韩立携莫简离归队，会合银月、朱果儿、花石老祖等，乘漆黑墨灵圣舟返人族。密室中清点此行全部所得：一盏古灯、两块三清雷霄符玉牌、一颗五色山海珠、一个乳白小瓶。
- **试符**：韩立指点雷霄符，电弧反击致手掌麻木焦黑，提真元方复原。叹「不愧仙家密符」，望能助抵天劫。
- **元魂灯**：参悟一日一夜，得知古灯名「元魂灯」，是盛放、滋养并壮大分魂的奇宝。寻常法器寄放分魂久则衰散，而元魂灯能令分魂渐壮，配特殊秘术祭炼可成「元魂化身」，且因灯在手而无反噬，甚至能育出强于本体的化身。此灯可召唤出似慕兰圣禽的孔雀化身（因其乃一丝精魂所化），与五光族孔雀之身的奕姓老者关系耐人寻味。韩立因第二元婴已失，遂种入一缕精魂以再育第二化身。
- **玄武真血**：装于小瓶，吞下后以惊蛰决炼化，可推进惊蛰变化神通至玄武之身。惊蛰决载十二种真血/变化，韩立已修成真龙、天风、雷鹏、青鸾、五色孔雀、山岳巨猿等半数；每多成一种，余变身威力以不可思议程度递增。韩立当即仰首吞下玄武真血。

## 关键设定点
- 修罗蛛晶核 → 时间/光阴之力来源，可炼制飞剑、提炼光阴之丝。
- 三清雷霄符：灵族镇族之宝、抗天劫的仙家密符，存量稀少。
- 元魂灯：养魂育化身奇宝，疑与五光族孔雀（奕姓老者）同源。
- 惊蛰决：十二真血变化体系，韩立已成半数。

# 第2303-2305章 收徒蓝药 · 敖啸之陨 · 元合岛立基

## 一、玄武变身与山海珠洞天（第2303章 法则心得）

- 韩立于密室中炼化**玄武真血**完毕，结印化身一只五六丈高的黝黑巨龟（**玄武之身**）：甲壳裂纹间晶文流转、自发结成光幕，综合战力虽不及巨猿等变身，但**防御冠绝诸上古真灵**。
- 韩立以神念遁入**山海珠**洞天，发现空鱼一族已被族长集中至一座小岛，岛上遍布三角形建筑。
- 韩立允许空鱼族自由活动其余小岛（最大岛屿另有他用），并准其整理药田。
- 韩立考校族长孙女**蓝药**的丹道悟性，对其天赋大为赞赏，遂**收蓝药为徒**：赠丹药与修炼玉简，命其先修至**炼虚**境界，届时再亲授丹道，许其终成丹道宗师。
- 临行前空鱼族长献上**空间法则参悟心得**（石片、玉简、卷轴等十余件，原仅历代族长可阅），因身为族上能感应到韩立体内的空间之力。韩立收下后离开洞天。
- 数月后墨灵巨舟载众返回人族，旋即得知一个惊人消息。

## 二、敖啸之陨（第2304章）

- 在天渊城传送大殿，众人得知**银月（玲珑仙子）**之祖父**敖啸**因闭关出错，七日后即须提前度劫，连一日都拖延不得。
- 韩立此次灵族之行取得**两张雷霄符**，原已表示愿赠敖啸一张以增度劫把握，银月为此欣喜。莫简离推算即便全程传送也多半来不及送达。
- 众人急赶圣岛，九日后抵达，得知敖啸在**离圣岛十万里的红云谷**度劫，岛上长老尽往护法。
- 途中银月怀中**敖啸画像玉牌**爆裂、化为粉末——敖啸已**陨落于天劫之下**，仅留一枚灵核及两件残宝（半截淡白小剑、缺半黄色小钟）。
- 圣岛长老依敖啸遗嘱将灵核交予银月。银月取灵核后闭关静养。
- 半年后，韩立亲传弟子**海大少（月天）**勘定新洞府选址：**无涯海**中一岛，灵气精纯、大半海域天生被两仪磁力笼罩，可改造为护府禁制。银月随后出关，与韩立同往新地。

## 三、元合岛立基与忘情决之解（第2305章）

- 韩立在无涯海一座万里巨岛（**元合岛 / 元和岛**）以最高巨峰为中心营建殿宇、遍布阵旗，将黑白圆石放入巨型法阵，催动**两仪万引大阵**。
- 此阵参考上古法阵布成，借岛下数条大型磁光地脉催动**两仪磁光**御敌于岛外，全力可扩展十倍范围；韩立更发现整个无涯海磁脉皆聚于此群岛之下，诸岛可布简化阵相互呼应，使无涯海固若金汤。
- **忘情决（无情决）之解**：韩立查典籍发现银月所修忘情决虽难彻底废弃，但可将其真元**单独分离**——炼一具特殊化身，把忘情决真元尽数转移过去，连同负面情绪一并脱体；约七八成把握，需修为神念远超银月者（韩立）辅助。代价是银月修为掉回**炼虚**，但日后重修合体再无瓶颈。
- 韩立将自己的**芝仙灵躯**（纯净无灵性、合体后期大成、始终无法突破大乘的空壳）改造化形为银月容貌相赠，作分离忘情决的化身。银月收下，定百日后祭炼完毕再行分离。
- 韩立急于解决银月忘情决，是为**数年后的异大陆之行**铺路：拟只带朱果儿（探查小灵界通道需其神念）与花石老祖（精水属性神通）同行，不愿再让银月涉险。心头一倩影（倩影真人）若此行顺利或可再见。
- **三年后**元合岛建成：灵气盎然、灵禽异兽出没、傀儡甲士巡守；钟鸣声起，数千中高阶修士齐聚巨峰银色宫殿前参拜出关的韩立。
