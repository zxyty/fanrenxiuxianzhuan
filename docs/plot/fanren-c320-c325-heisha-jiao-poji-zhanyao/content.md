# 凡人修仙传 第320-325章 剧情梳理（破茧·隐斗·斩妖·黑煞教主）

## 一、攻打血侍光茧（第320章 破茧）
韩立随刘靖、宋蒙、王师兄、雪虹、陈巧倩师兄等黄枫谷修士围攻黑煞教，四名「血侍」结成坚固的血色光茧准备妖化。趁其未破茧，众人全力强攻：
- **刘靖**（筑基中期）祭出金色符箓，催动金系群攻法术「**千刃术**」，化漫天金光为千百寸许金色小剑下刺。
- **宋蒙**与**雪虹**各以「**符宝**」放出灰色长枪、蓝色光华（符宝乃身家深厚或得师门宠爱者方有之物）。
- **韩立**只放出四头**兽傀儡**（旁人以为得自李化元的「傀儡术」）喷射光柱，省力以恢复法力，刻意保存实力以待黑煞教主。

集中火力先破最左的「冰妖」光茧，宋蒙灰枪洞穿，露出半妖化的白色青年怪物（头生白角、白尾白鳞、利爪）。

## 二、隐形苦斗（第321-322章 隐斗 / 刘靖与真宝）
冰妖半妖化即破茧，隐形偷袭，**掏心杀死正催动符宝、收起防御法器的雪虹**。随后又欲杀王师兄，被韩立施展「**罗烟步**」以血色试炼所得**银剑**及时救下并缠斗。
- 冰妖身体水晶透明、急速移动造成「隐身」错觉，并将地面冻成晶冰。韩立凭丰富肉搏经验压制之。
- **刘靖背景**：出身修仙大家族，幼年曾被邪修掳走折磨，后拜入李化元门下；嫉恶如仇之名带来声望，渐迷恋受人尊崇之感。为保名声，他决定动用保命之物。
- 刘靖取出火红玉盒，内为画有火鸟、散发炎热的奇异符箓——王师兄惊呼此乃「**真宝**」。

### 真宝设定（lore）
- 普通符宝至多含本体法宝十分之一威能；**真宝可容纳约三分之一威能**。
- 真宝由法宝主人以**血脉之法**炼制，只有特定血脉后人能驱使，他人注灵即自毁。
- 缺陷：**一次性消耗**，威能一次全放；且本体法宝被炼真宝后**威能永久损失、无法弥补**，故真宝极稀少。

## 三、斩妖（第323-324章 斩妖 / 黑煞教主）
- 韩立趁缠斗将**透明丝线**缠上无名指，假意被冰妖击退后猛然回拉，使冰妖倒射回身前，以银剑无数银芒切割——冰妖透明冰身虽硬却失触觉，整个化为冰渣而亡（仅利爪与银剑可硬接不损）。韩立挑得其储物袋。
- 暗写：假山密室中一白净中年人睁开血红双目，自语「冰妖已死也好」，身前横卧一具仅剩皮的黄枫谷修士骸骨。
- 刘靖激发**火鸟真宝**成功，化红色小鸟，再涨为丈许火焰巨鸟，一啄烧杀妖化的**青纹**，又喷火柱炼化另外两名破茧血侍。众人盛赞，皆以为只剩闭关的黑煞教主不足为虑。
- 韩立察觉王师兄盯着地面、神色古怪却讳言「无发现」，料其有重要私藏。

## 四、黑煞教主现身与同门陨落（第324-325章 黑煞教主 / 诡异）
- 钟卫娘、陈巧倩擒回身着金黄、惊慌的「**越皇**」。陈巧倩闻雪虹「兵解」脸色大变。
- **蓝袍人**（约四十、白净无须、慈祥模样）凭空现身，一手从王师兄胸腔抽出，杀之，并捡走王师兄手中拇指大的**青色珠子**——自承即创立黑煞教的教主。
- 刘靖正气凛然欲领众围杀，话音未落即被黑煞教主一手洞穿前胸捏碎丹元而死；同时另一臂洞穿陈巧倩一位师兄。**未及交战已折两名筑基中期修士，领队刘靖陨落**，钟卫娘失神。

## 五、越皇真面目与五行血凝丹
- 「越皇」骤显**筑基后期**修为，自言缺几个筑基修士血祭；其能将修为掩饰到黄枫谷众人毫不察觉（韩立联想到当初小王爷、王总管）。
- 蓝袍人从冰妖冰渣取出**蓝色珠**，越皇从两名被炼化血侍处拍出**一金一黄两珠**。韩立认出这些即小王爷所言、与**结丹**有关的「**五行血凝丹**」，此处共四颗（蓝/金/黄/青），尚缺一颗。
- 韩立令众人升空。越皇竟一手插入蓝袍人胸口，蓝袍人毫不挣扎、主动将护体血光经手臂输向越皇（邪功），蓝袍人干瘪、越皇返老还童至三十许岁；二人相貌不同却给人「神情如一人」的诡异感。
- 韩立掷出一两百张火系符箓化作铺天火浪，越皇凭血光无伤；继而暗藏一颗「**天雷子**」引发白光巨爆，重创越皇（法力大损）却未杀死。越皇满身血污走出，怨毒盯住韩立，封住众人法器。

## 关键转折
- 黑煞教此役血侍尽灭，但真正的威胁是伪装成越皇的筑基后期修士 + 黑煞教主，连斩刘靖等多名黄枫谷同门。
- 「五行血凝丹」与结丹相关、被敌方夺取，正是韩立此行的隐秘目标所在（仍缺一颗）。
