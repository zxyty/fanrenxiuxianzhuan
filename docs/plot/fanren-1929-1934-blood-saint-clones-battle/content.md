# 第1929-1934章 引念 / 血海 / 激战 / 困敌 / 逆转 / 退敌

## 背景
血光圣祖（圣界圣祖）为重夺**镇魔塔**以行其大计，冒奇险将**三缕分神**同时隔界降临**灵界**。三化身法力远逊本体全盛、趁手宝物也难带至灵界，但自恃三人联手并持**彩光塔（七色小塔）**与**紫言鼎**两件**玄天残宝**，足以横扫灵界大乘以下。降临后的主要目标——韩立——却极为棘手。

## 韩立的战力配置
韩立此时修为**合体中期**，以化身手段应敌：
- **金毛巨猿（山岳巨猿）**：本体变身，逆天神力，可化数百丈，三头六臂，金色拳影、六口金刃/环枪锏锥等兵器、金色雷火。
- **梵圣金身**：由**梵圣真魔功**修出的真魔法相直接凝练成的三头六臂金身，由无数珍稀材料凝练、近金刚不坏，六臂金刃、竖目金光、吞噬吸力。
- **灵躯**：绿肤、依仗**芝仙肉身**不死特性，墨绿战甲、青丝、银色短尺、绿色丝网，自保绰绰有余。
- **十三只紫纹噬金虫**：吐紫丝，躯体坚不可摧，风刃难伤。
- **万剑图**（新祭炼完毕）、**九宫天乾符**（化宫殿楼台困敌）、青黑两座**极山**、辟邪神雷所化金色电网、大量符箓临时法阵与剑阵。

## 战斗经过
1. **引念/血海**：韩立先布下剑阵与五六座临时法阵、放出噬金虫、并以血色符文晶球等手段做手脚，化金毛巨猿迎敌。为首少年以背后**巨角魔影**（触手）、**血海**（魔界奇毒之水，可化万物为乌有、生**带翅血蚕**与血丝）猛攻；韩立剑阵化青色墙壁、五爪青龙、青莲挡下。韩立暗令灵躯自爆一件低阶空间宝物扰乱，使**紫言鼎失控**长鸣飞遁，一具化身被迫追宝离去。
2. **激战**：另一化身的魔影将巨枪化为**灰色巨蟒**缠斗梵圣金身；金身以吸力、竖目金光、金拳与之皮糙肉厚地肉搏。手托彩光塔的化身以**七色飓风**（现金篆文）欲强夺韩立两座极山，被金刚拳影、极山银符与剑气合力震碎。
3. **困敌/逆转**：彩光塔化身张塔布下不断扩张、蕴含法则之力的七色光晕；韩立以新悟音波神通**金刚吼**轰溃光晕，再以金色电网（**辟邪神雷**）、灵躯绿丝、符阵将彩光塔困成绿色巨球，复抛入**万剑图**摄走、再收入九宫天乾符宫殿。彩光塔虽一度凭空间神通破出万剑图，却被图中无穷飞剑化成的**剑海**反将其困死原地，灵性大损、短期内无法再动用空间神通。
4. **退敌**：为首少年纳入八颗青色鬼头狂涨成巨魔，喷血鬼、血色巨棍硬撼巨猿。韩立施**念剑诀**（伤神识、自身遭反噬）一剑斩落巨魔正中头颅，又灭其魔相所化灰色巨蟒、致其本体反噬。彩光塔化身脱困而出却气息衰弱、宝塔黯淡。韩立终亮出金身手中的**玄天残宝（金色断刃）**，引动天地法则之力，巨魔骇然失色，与彩光塔化身一同破空遁走。

## 结果与影响
- 此役韩立成功退敌：血光圣祖三化身一个紫言鼎失控、一个彩光塔灵性受损、一具化身（灰色巨蟒/魔相）遭重创，夺塔大计落空。
- 韩立自身亦付出代价：兵器残损、金身受创、念剑诀反噬激射血丝；万剑图等重宝有损耗之虞。
- 点明设定：金刚吼为合体中期山岳巨猿变身新领悟；念剑诀与仙界功法相关、需变身山岳巨猿并合身梵圣金身方敢施展；彩光塔、紫言鼎、金色断刃均为玄天残宝。
