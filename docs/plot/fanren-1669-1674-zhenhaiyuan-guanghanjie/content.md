# 凡人修仙传 第1673-1675章｜广寒界海域历险

第九卷《灵界百族》。韩立、柳水儿（斗篷女子）、石昆三名修士结伴，行于广寒界海域，意在寻找禁制遗址；因海上无法定位，须先抵陆地。

## 第1673章 各现神通——斩章鱼海兽
- 小岛下方潜伏一头巨型海兽：浑身乌黑的巨大章鱼，头颅一面隐现凶恶男子模样，身侧除巨大触手外另生两只粗大手臂，各持一件赤红色长枪状兵刃。
- 触手遍布黑绿花纹与车轮般肉盘，被斩断后伤口喷绿气可瞬间幻化复原，具『不灭之体』神通；肉盘可喷黑色光霞硬抗刃芒。
- **石昆**：单足跺地激射、挥臂发十余丈黄色刃芒斩断触手；拳头可气球般膨胀五六倍、闪现黄色符文爆裂出深黄光晕波纹，将触手连皮带血绞碎溃散；肌肤可转黑成『石人』。肉身强横，独力压制海兽分身。
- **柳水儿**：自称修水属性功法，化蓝色惊虹潜入海面，短时间内斩杀海兽本体（黑色漩涡浮出巨兽尸体）。
- **韩立**：以五指探出五道五色寒焰注入乳白光幕，使光幕化淡五色，将百余丈来袭巨浪凝结成晶莹寒冰，绕岛形成淡蓝色冰墙护岛；事后喷出银色火球化为尺许银色火鸟，钻入冰墙使之消融回落为海水。柳水儿惊讶其兼具冰、炎两极神通。
- 韩立评石昆为传闻中的『斗法狂』。

## 第1674章 切磋——元磁神光合击秘术
- 三人欲操练『元磁神光合击秘术』，需由威能最大者主持，故先各展元磁神光。
  - **柳水儿**：天生元磁之体，操纵随心，背后凝灰色光轮（直径三四丈），可放灰光、催动巨大符文吞没石昆数百枚灰色光球，新近参悟出更多变化；威能最强（自承仅用六七成）。
  - **石昆**：后天元磁之身，当年吃尽苦头修成，可借外力变强；身化灰色骄阳、双臂虚影连击化数百灰色光球。与柳水儿首击平分秋色，次招败北、爽快认输。
  - **韩立**：借一件元磁之宝施展元磁神光，自称威能不值一提、不参与比试。**实情：其宝为整整一座元磁神山**（二人不知）。
- 结论：以**柳水儿为主**、韩立与石昆为辅。三人布巨大幻阵掩饰岛上狼籍，操练十余日（再过三四日，合计约二十日内）即操纵如意，遂撤阵启程，认准方向寻陆地定位。

## 第1675章 镇海猿——天吼神通
- 飞行四日四夜，沿途岛屿皆被气息强大的海兽占据，三人联手亦难轻取，幸早听韩立之言未贸然停留。第五日晨望见陆地。
- 柳水儿提醒：陆地危险更甚海中，登陆需收敛气息以免惊动上古凶兽。
- 远处海面白雾翻腾，浮出一只比巨型章鱼大数倍、毛茸茸、头生独角的巨猿状海兽。
  - 其尖鸣令三人耳膜刺痛、精血翻滚、手脚酥软；继而化雷鸣狂吼，震散海雾、形成漩涡、震死方圆百里海鱼。
  - 吼声喷出肉眼可见透明波浪，海面凹陷十余丈、激起数百丈海浪向四面扩张。
  - 三人被『天吼』同时禁制肉身与体内法力，定身无法移动；柳水儿玉容苍白颤抖，石昆青筋暴跳热汗滚滚，唯韩立稍轻松。
- 透明音波逼近不足里许时，**韩立脸上金光一闪、冷哼一声**（精神秘术）解除三人禁制，三人遁出三四十里脱险，身后七八里处音波威能丧尽溃散。
- 事后辨认：石昆疑为『真灵山岳巨猿』；柳水儿否定（若真灵第一吼便陨落），判定为上古凶兽『**镇海猿**』——传说可撕裂蛟龙、生吞圣阶，其『天吼神通』上古即鼎鼎大名。三人能逃，因此兽攻击非针对他们，乃城门失火殃及池鱼。
- 韩立自称当年凑巧修炼过一种精神秘术，可稍抗其吼。柳水儿、石昆均承下韩立救命之恩，约日后回报；韩立淡然，称广寒界凶兽众多，日后或反需二人相助。
- **结尾**：远处镇海猿对岸海岸飞来一只紫色大鸟，体积不逊巨猿，清鸣如天籁、丝毫不受巨吼影响，直扑镇海猿而去。

## 关键设定点
- 元磁神光分三种来路：先天元磁之体（柳水儿）、后天元磁之身（石昆）、借元磁之宝（韩立，实为元磁神山）。
- 镇海猿『天吼』可双重禁制（肉身＋法力），威能笼罩百里。
- 韩立隐藏实力：对外轻描淡写元磁之宝与精神秘术，未暴露元磁神山及真实底牌。

## 背景
第九卷 灵界百族。韩立与柳水儿（斗篷女子）、石昆（深黄肌肤大汉）三人结伴进入**广寒界**——一处保留大量灵界早已灭绝的上古凶兽/异兽的界面，目标是寻找某处『禁制遗址』并破除其禁制。

## 八凶海脱身（第1676章）
- 三人误闯**八凶海**（地图上标注为**九凶海**），正撞上**镇海猿**与一头不知名巨鸟两头上古凶兽相斗。柳水儿急令撤离，三人化遁光遁出数百里，身后千余丈巨浪淹没原地。
- 安全抵陆后，韩立升空护法，柳水儿、石昆各祭白色圆盘（化作巨大明月）与黄色卷轴（展开化黄色光幕），三人成三角阵施法。

## 广寒图定位
- **广寒图**：柳水儿之家师与段前辈搜集历代进入广寒界前人所留残图，汇总铭印炼制而成，记载广寒界十之七八地域。施法后一颗鸡蛋大白光球悬停于地图某方块，显示三人身处**万禽岭**（万禽领）边缘，所经海域即八凶海。
- **万禽岭**：居住数以万计各种灵禽，含三只上古凶禽，威名不下镇海猿；与八凶海紧邻。三人一致决定绕开。

## 择路决策
- 万禽岭两侧：一为**蓝阳沙漠**（又作旭日沙漠），诡异日光对未进阶圣族者半日暴晒即化烟灰，比万禽岭更可怕；一为**暗兽森林**，栖息灵界已灭绝的暗兽一族，夜晚实力暴增数倍。
- 石昆起初反对入暗兽森林。柳水儿以时间紧迫（赶往禁制遗址至少三四月，绕万禽岭半月、再绕暗兽森林近两月，另需一两月突破瓶颈）及『空手而归会遭段前辈严惩』说服石昆。
- **韩立立场**：自称受两位前辈所请、已收不小好处协助二人；只要二人意见一致、去处非必死之地便无所谓。最终三人定下穿越暗兽森林。

## 界中诸族暗涌（第1677章插叙）
三人离去后，广寒界各处异族活动并行展现：
- **两名异族**（矮胖套乌黑战甲 / 高瘦如干尸）：在八凶海被怪雾困十余日、又遇两凶兽，自爆数件宝物脱身；用从拍卖行高价购自**晶族**的定位盘在此地仍无法定位。晶族每逢广寒界开启便靠炼制定位宝物大捞横财，传闻能炼更玄妙之物。此二人此后彻底销声匿迹。
- 三名**天云**异族被十几只鹿首狮身怪兽追逐。
- 冰川风雪中数名遍生白毛怪人低空飞行不受冰寒影响。
- 两名绿袍枯黄男子持红色小幡，以不灭之炎烧死一株生有恶鬼面孔的擎天巨树，从灰烬取得翠绿晶体，欣喜若狂。
- **角蚩族**：一名额生三金角青年率十几名生角男女，于沙漠传达机密目标——族中长老不惜大损元气、用玄天之宝分身炼制五件逆天宝物，并动用潜伏在天云的数个棋子引开天云高层注意；入广寒界十几支队伍中仅五支持宝队伍负责正事，余者作掩护。
- 一对鱼尾人身异族男女以相似吼声驯服了一头额生九金眼珠的怪鲸。

## 暗兽初现（第1678章）
- 暗兽森林：一日三分之二为黑夜，白昼奇短；暗兽数以千计，连万禽岭、八凶海的上古凶兽都不侵入此地。仅少数夜晚能增幅变身的异兽可与暗兽一搏。
- **青夜蟒**：七八丈长青鳞巨蟒，夜晚可狂化变身（蛇皮碎裂换更大蛇躯、银白符文鳞、生独角、眼变赤红），实力近成年暗兽，但变身后数年实力大减并损寿元；其内丹对暗兽有莫大滋补，二者遇即不死不休。变身刹那七寸处有淡红弱点鳞。
- **暗兽**：体型如放大三四倍黑狼、豹尾、可隐于夜色、能分出两只影子般分身。一只刚成年的暗兽与一条狂化青夜蟒在森林边缘遭遇厮杀。
- **三人观战**：韩立、柳水儿、石昆耗时大半月赶到暗兽森林边缘，藏身一株巨树内两丈树洞，围一面宝镜（背银光、镜面青光）远观缩影战况。近半个时辰后，暗兽化数百黑色蝙蝠将遍体鳞伤的青夜蟒磨死、剖腹吞兽丹离去。柳水儿一掌法决散去镜中画面，问二人观感。
