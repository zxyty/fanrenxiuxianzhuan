# 第四卷 风起海外·第440-442章「鬼王（2-4）」

## 场景
虚天殿「鬼冤之地」（第一关），黑雾翻滚、鬼音交织。韩立、紫灵仙子与黑袍魔修临时同处，面对一群强大妖鬼。

## 经过
1. **勾魂鬼音**：紫灵仙子误把鬼音当普通鬼嚎，几乎全身血液逆流、无法自主狂舞。韩立一声「呔」震退鬼音，救其脱困，并点明「以你修为只要守住心神，不会被趁虚而入，是大意了」。
2. **战场全貌**：黑袍人以火红魔火锤（喷蓝色炎火）与黑色鬼影缠斗，鬼影祭绿珠放玄阴寒气大占上风；啼魂兽（吸魂啖鬼、喷黄色霞光吸纳鬼火）被两只绿毛**鬼夜叉**缠住。紫灵仙子识出鬼夜叉为罕见可在白日活动、肉身遨游天地的僵尸异类。
3. **韩立冷眼旁观**：黑袍人因先前傲慢拒邀，被韩立故意按兵不动、险些被黑鬼影一抓而亡；千钧一发时韩立才以四五道青色剑气逼退鬼影。黑袍人靠一面古宝**铜镜**（射出巨鸟白芒）救命，但铜镜随即无故碎裂报废。
4. **群鬼围攻**：又涌出七八只头生角、十指尖利的**炜吾**悍鬼（实力不逊筑基后期）。韩立部署：黑袍人、紫灵对付炜吾，自己迎战两鬼影，并放出噬金虫助阵。
5. **斗绿珠器灵**：韩立以青竹蜂云剑化青蛟破玄阴寒气，令鬼影意外。鬼影点珠，绿珠化为实体凝形的黑色妖虎——竟是**器灵**。鬼影再啸，妖虎生出第二颗虎头成「双头怪虎」，妖气狂涨，连喷黑色光球，气势汹汹。
6. **噬金虫灭灰影**：擅长化形匿踪、化无形之体偷袭韩立金丹的**灰色鬼影**，被悬浮的噬金虫铺天盖地缠住啃噬——甲虫能无视无形鬼体，灰影变雾、变鳞甲均无法摆脱，恐惧求援。韩立再放两袋虫加入吞噬。
7. **附灵术**：黑色鬼影舍弃灰影，化黑雾扑向双头妖虎，与之融合——一虎头现出毛茸茸人脸罩黑气。韩立惊认此为**附灵术**。

## 关键设定（韩立的内心解说）
- **器灵**：法宝一生仅有一次将妖兽鬼怪元神精魄封入的机会，封成则法宝化形为器灵生前模样、得其神通，威力大增；失败则永不再得。元神越强成功率越低，故多数法宝无器灵。
- **附灵术 vs 附身术**：附身术只是远程借体操纵、只能发挥几层实力；附灵术只能由修士对智慧不高的妖兽施展，修为叠加数倍、签生死之约（一方魂飞神灭另一方俱灭），并会形成拥有独立神智记忆的全新个体；施展越多新个体越占主导，最终不可逆。代价是寿元惊人消耗、独立后不久即泯灭；传言施术者死后魂坠无尽深渊、不入六道轮回。
- 疑点：附灵术本应只有正常修士可施，妖鬼竟能修炼且向自己的器灵施展，令韩立警惕大起。

## 人物与伏笔
- **韩立**：金丹修为，散修，自叹见识不足（黄枫谷时修为低看不到隐秘典籍、乱星海所购典籍杂乱），打算战后到**妙音门**系统翻阅（自己名义上为该门长老）。
- **回忆**：血色禁地中，**南宫婉**取**墨蛟**元神，估计即为封印器灵之用；韩立一直想用蛟龙做器灵而未得。

## 结果
战局未决，黑色鬼影与妖虎器灵融合（附灵术）形成半人半妖怪物，韩立陷入更险境，章末「心中警惕心大起」。

## 概要（第443-445章 第四卷 风起海外）
地点：虚天殿内的鬼雾区。韩立与紫灵仙子、黑袍人（元瑶）暂时同行，陷入与鬼怪的苦战。

## 鬼王之战（443-444章）
- 韩立的**噬金虫**（奇虫榜有名）正啃噬一只灰色鬼影，将其吞噬大半。此虫不仅吞灵气，亦食阴鬼厉魄。
- 对面**虎怪（人面虎首）**借附灵、变形、合体之能，发出范围性怪笑，威力惊人，连噬金虫都被震落、半死不活。韩立全靠**大衍决**凝神定魂之效硬撑，并将虫收回灵兽袋。
- 虎妖吞回灰鬼影后再生出第二、第三头颅，化为**三首虎妖（合体虎妖）**，体形暴涨如两三层阁楼，徒手抓住韩立的**青竹蜂云剑**且不惧剑光，喷黑阴气消磨剑上灵光。韩立判定其实力比结丹后期只强不弱，疑近元婴期，即所谓「鬼王」。
- 韩立以百余只**巨猿傀儡**掩护，趁虎妖徒手握剑的良机，引爆飞剑上积攒的「**辟邪神雷**」——数十道淡金色电弧结成金色光网，一声「疾」字爆裂，白光过后虎妖化为飞灰。
- 虎妖将主魂藏入一颗**三色圆珠（灰黑绿）**法宝中诈死，欲诱韩立夺舍。韩立一剑斩珠，再弹出三颗鸡蛋大小**小火球（火弹术）**将逃窜的三道黑气尽数烧灭。
- 代价：四柄飞剑的「辟邪神雷」耗尽需长期修养，另四柄飞剑沉睡。

## 黑袍人真容（444-445章）
- 韩立由黑袍人沙哑假音识破其为女修；紫灵仙子亦早看出。黑袍女子脱帽，露出美艳惊人的容貌（肌肤赛雪、乌黑披肩长发、额戴金灿发环）。
- 韩立认出她为**元瑶**：百余年前在**魁星岛「天都街」坊市**外，与韩立及**曲魂**（韩立称之为「曲师叔」，实为其分身，紫灵知情）有一面之缘，当时元瑶与一位『妍道友』同行。
- 元瑶起初被叫破本名时惊慌否认、自称『姓阮』并露敌意；确认认错人后放松，并因韩立如今神通不小、自身法宝元气大损而不愿得罪他。
- 元瑶当年为炼气期，如今已结丹初期，百余年间修为飞速，韩立暗忖其资质远超常人或另有奇遇。韩立则自陈当年已是筑基期，靠『**三转重元功**』方得结丹（对元瑶『从炼气修到结丹』之说心中摇头）。

## 花园集结（445章末）
- 三人离开鬼雾，来到一处花园（虚天殿内）：二三十名修士多带伤而面露兴奋，**极阴祖师、万天明**等元婴期修士分处玉亭闭目养神，**星宫**两位白衣长老居中盘坐如石雕。
- 一名面目清秀、身披绿袍的**年轻男修**毫发无伤地走出鬼雾，神色从容；瞥见极阴祖师时暗闪怨毒。他未见韩立身影而微蹙眉。
- 其脑中突然响起一个懒洋洋的浑厚男声，讥问『你的那位帮手不在吗（指韩立），莫非被厉鬼吞噬了』；年轻修士急以神识喝止，警惕在场元婴期修士的强大神识感应到体内之声。

## 关键判断
- 「辟邪神雷」是韩立此行的杀手锏，凭金雷竹专克妖魔邪法的『辟邪』特性，四剑齐发足灭鬼王级恶鬼。
- 黑袍人因目睹虎妖合体之恐怖与被秒杀之轻易，对韩立由戒备转为震惊、欲拉拢为靠山。
