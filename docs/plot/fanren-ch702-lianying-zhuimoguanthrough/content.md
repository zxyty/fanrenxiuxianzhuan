# 第五卷 名震一方 第七百零二章 炼婴

## 场景
坠魔谷外通道附近、玉矶阁废墟。鬼灵门云姓老者一行围杀刚逃走的南陇侯，并处理被韩立重创的同伴。

## 各方与图谋
- **拉拢**：云姓老者透露南陇侯与天极门有关（但南陇侯并未真加入天极门），邀老妇人、黑脸汉子加入鬼灵门，称成为鬼灵门长老便不惧天极门；二人皆拒绝。
- **目标——坠魔谷**：此地隐秘洞府藏有**苍坤上人**当年进出坠魔谷的**路线图**，按图进入安全大增。云姓老者更看重南陇侯脑中关于坠魔谷的隐秘消息，欲先擒下、留其元婴用**搜魂术**套取情报，而非当场灭杀。
- **路线图去向**：王天古担心路线图恰在**韩立**带走的**玉盒**中。云姓老者对策：若真如此便捅破坠魔谷之事，引正魔两道乃至**天道盟**向独享宝物的**落云宗**追问，鬼灵门再浑水摸鱼；王天古补充可先尝试以代价换回。

## 韩立的诡异神通
- 据王天古向王蝉（蝉儿）打听，韩立原只是**黄枫谷**普通弟子、修为确为**筑基期**，如今神通暴涨、争斗经验老辣狡诈，连尤道友都着了道，连云姓老者也自认单打独斗未必能胜。
- 韩立曾以**黑色山峰**攻击，并在脱身时驱使**太妙神禁**阻挡众人。
- 韩立的黑气秘术斩断王蝉双腿：燕如嫣与王蝉联手施血灵护体血雾仍无法阻挡，断口无血却有黑气聚而不散，多种续肢法术失效、王蝉因此昏迷。

## 王天古取样研究
王天古以黑芒手指抓出断口黑气，原欲真火烧毁，转念用**小瓶**将黑气吸入收好，打算日后研究以了解韩立功法。又提到韩立临走放出的黑红色光片，竟似自己所修的**魔血斩**。燕如嫣见状面露复杂之色，旋即恢复如常。

## 尤姓修士之死（炼婴）
尤姓修士被韩立的**蓝冰**秘术冻成冰像：碎玉触之即化蓝冰碎裂，真火触之被吸纳。云姓老者判断其肉身已废，令其元婴出窍另寻躯体夺舍。尤姓修士元婴捧玉佩、喷元阳婴火欲破冰而出，红白光焰一碰冰壁即被引燃化为蓝焰，连元婴一并吞入，元婴惨叫打滚后被炼成白光消散——尤姓修士**从此形神俱灭**。

## 后续打算
众人决定先回去（韩立秘术已无法感应到对方），待南陇侯所施秘术失效后再追。

## 备注
- 苍坤上人据【世界状态须知】已于第685章死亡，本章仅为其遗留路线图之提及。
- 韩立表面身份/修为（黄枫谷弟子、筑基期）系敌方所知，实际已远超此。

## 背景：慕兰入侵与九国盟战事
- 慕兰族普通族人无法修炼灵术，但慕兰法士进攻时会组织青壮凡人结成临时部落，趁天南修士无暇旁顾时抢占灵石材料原矿，以人命换取珍稀资源。
- 九国盟（天南一方）对此一度派修士重夺矿洞产地、顺手灭掉慕兰凡人，但慕兰人不惜伤亡、反复组织并设伏重创修士，九国盟损失后干脆睁一只眼闭一只眼。
- 韩立返回天南期间，九国盟被慕兰法士全力偷袭，初战不利、暂时后退，真正大战尚未开始。丰原国是最靠近慕兰的三国之一。

## 第703章 盒中之宝
- 韩立先以雷遁术一口气遁出百里，换上血色披风化为血光续飞千里。
- 见一支数万人、多为年轻男女、往天南而去的慕兰临时部落队伍（仅一名筑基期、三名炼气期法士），遂隐身遁入一辆装杂物的破皮蓬马车，盘膝坐定，意图藏身随队安然穿越慕兰草原（约两三日）。
- 以强大神识，韩立短时间内已学会慕兰语；并以淡青光罩封闭马车隔音。
- **银月取宝经过**：银月（白色小狐／器灵）受命趁众修士争夺时遁入取宝。第一层古宝因遁术无法掩其灵气未敢动；第二层众人争夺的玉盒原应六只，银月只取走三只（怕取多引疑）。加上韩立自身一只，共得四只。韩立点明『苍坤上人一共留了六个玉盒，四个落在我手中』。
- 韩立当着众元婴修士暗中放银月遁入取宝，以释放灵光掩饰得手，自信大不了施『血影遁』脱身。银月谦称玉盒本身能掩饰灵气，方能瞒过老怪物神识。
- **开盒所得（四盒）**：
  1. 淡蓝色玉简——记载苍坤上人功法神通，主修功法『望月决』韩立无法修炼，但其中数种秘术与修炼体会可借鉴，用处不小。
  2. 乌黑无光小指环——材料古怪，经神识扫描判定只是一件普通法器，用处不明；韩立认为苍坤既肯收盒中，必有他用。

## 第704章 路逢旧人
- 续开盒：
  3. 绿色小瓷瓶——开盖辛辣异常之气扑鼻，韩立脸色大变如见毒蝎，急塞盖谨慎收入储物袋。
  4. 拳头大紫濛濛软团——经法决打回原形，现为层叠薄如轻绢的紫色晶网，银月认出是蛮荒时期名气不小的古宝『紫铖兜』（又作紫云兜）。
- **紫铖兜设定**：古宝中顶阶存在，但远不及通天灵宝；单论防御神通妙用无穷，施展可遮蔽百丈乃至千丈，最顶尖者可一下罩住百里之内生灵不受伤害；亦擅困敌，可自行释放『玉阳真火』灭杀强敌。银月在被炼化成器灵前似曾有此宝，故知之甚详。
- 韩立推断紫铖兜十有八九出自『坠魔谷』；惋惜四盒中并无坠魔谷信息，料想相关信息在南陇侯所持的另两只玉盒中。他也清楚得坠魔谷秘密利弊难说，恐成众矢之的。
- 韩立携银月随队两天两夜，期间遭两拨法士检查皆被其略施法术蒙混；出慕兰草原后立即神不知鬼不觉脱队，另走荒原直奔天南，凭强大神识远在百里外即可避开法士，顺利进入丰原国。
- 进入丰原国数日，一路不见九国盟修士、只见游荡巡查的法士气息，判断九国盟已初战后退。
- **路逢旧人**：又行三四日，韩立飞过无名小山时感应到剧烈灵气与冲天煞气——一处斗法。其中一道为元婴期、其余四五人为结丹期合力对抗，且有一两股气息似曾相识。
  - 现场：五名服饰不一的结丹男女修士围攻一名黄袍光头法士（满脸横肉、元婴初期），法士未放法宝、仅凭功法与修为以圈圈黄雾逼得五人节节后退，且对其中一名绝色女子频频留情、欲活捉。
  - 韩立目光落在一名肥胖老者身上：周身银白色雷弧、驱使不停放电的巨剑，威力居五人之首。韩立认出『是他？这世间的事，还真是巧合啊！』，神色复杂。
  - 黄袍法士不耐，欲收绝色女子为炉鼎（自称功法缺上佳炉鼎），祭出黄色羽扇（灵光大放、隐画图案），喷黄濛濛精气、念咒一扇，卷起数十丈深黄飓风将五名结丹修士尽数卷入。

## 关键物品归属
- 苍坤上人六只玉盒：四只归韩立（含望月决玉简、黑色法器指环、绿色辛辣瓷瓶、紫铖兜），两只在南陇侯手中（疑含坠魔谷信息）。

## 章节
第五卷 名震一方 第705章《拘灵土龙》、第706章《雷万鹤的震惊》、第707章《南宫之讯》。

## 背景
丰原国大半已落入慕兰法士之手。黄枫谷雷万鹤一行五人奉九国盟（天道盟系）之命外出执行任务，归途灭口数名低阶法士时被一名元婴初期法士发现追杀，被困于偏僻之地的飓风禁制中。

## 主要事件
### 救援与斗法（705）
- **覃姓光头法士**（黄袍、慕兰法士，元婴初期）以羽扇法宝放飓风困住五修，并凭空召黄色巨手欲活捉绝色女修聂盈。
- 隐身旁观的**韩立**远放一道**青元剑气**击飞羽扇，飓风、巨手尽散，随后现身。法士看不破其存在，被韩立指出『才进阶元婴不久』。
- 法士以两颗黄灿弹丸催出两只长十几丈、土石所化的**拘灵土龙**（弹丸镶于龙首眉间为核，赋予活物气息）护法扑击。
- 韩立口喷**乾蓝冰焰**（中途一分为二），将土龙冻成冰雕，再以剑气击碎，从容收走两颗弹丸。
- 韩立放出精心培育的**噬金虫**（首次真正驱使对敌）。法士竟一口叫出『噬金虫』真名，惊惧之下化黄虹遁逃；韩立因元婴出窍后雷遁术稍逊、空旷处无十足把握，放弃以风雷翅追杀。

### 旧识相认（706）
- 获救五人拜谢：黄枫谷雷万鹤、巨剑门石齐云、掩月宗唐明骅、天阙堡钱环，以及女修聂盈。
- 雷万鹤即当年与韩立换丹方的『雷师伯』；聂盈即当年维护雷灵根慕容兄弟的『聂师姐』。二人时隔近二百年、加之韩立容颜未改，初未认出。
- 韩立点破身份后，聂盈惊呼其为『李师叔的弟子韩立』。雷万鹤因晚辈师侄骤成元婴前辈，惊骇万分，改口尊称『韩前辈』。
- 韩立取出新得的**御风车**（白色精致四方飞车，可涨至十余丈，遁速极快），载众人脱离险地。

### 故人消息（706-707）
- 韩立得知师父**李化元**百余年前与慕兰法士争斗中陨落；其夫人未结金丹后坐化；同门师兄弟（于坤、宋蒙、钟卫娘等）无人结金丹，多已坐化。韩立黯然。
- 韩立自言现为**天道盟落云宗长老**，无意回黄枫谷（二人猜其当年作诱饵被放弃，故心冷）。
- **萧翠儿**（萧师妹）数十年前已入结丹期、已嫁人，对韩立引其入门之恩一直感念。黄枫谷令狐老祖大限将近。
- 韩立向掩月宗唐明骅打听**南宫婉**：南宫婉百余年前已凝元婴，现为掩月宗长老。三个月后将与**化意门魏离辰长老**（三百年即结元婴的天南奇才）举行**双修大典**。韩立闻之骤寒，旋即强自平静，表示要去参加。
- 韩立追问『南宫屏』，唐明骅称掩月宗并无此结丹女修、南宫婉一向孤身修炼。韩立由此悟出：当年的南宫屏正是易容改貌的南宫婉本人——故当日并未对他下狠手，临别神情古怪。

### 分别与评价（707）
- 韩立于近丰原国边界处放下五人，独自驭车离去。
- 天阙堡钱环指出：聂盈修炼二百余年方至结丹初期，而韩立同样二百余岁即结元婴，意味其极可能有望冲击**化神期**——天南上一次有记载的化神期修士还在数万年前，曾短期横扫整合正魔中立各派。

## 越国/天南格局
越国六派如今几近一体（同属天道盟体系），互不愿生事端。
