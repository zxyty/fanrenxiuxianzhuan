# 第817-819章 斩火蟾兽 · 古修遗骸

## 背景
地点：坠魔谷。韩立与两名元婴期修士南陇侯、鲁卫英（鲁姓老者）于禁制水幕中围困上古火蟾兽，谷外另有诸多机缘。此行韩立入谷目的之一即获取火蟾妖丹。

## 一、斩杀火蟾兽（第817-818章）
- **火蟾遁逃与断肢**：火蟾兽中圈套后化烈焰欲从弥合孔洞遁出，韩立情急放出一道**阴魔斩**（黑芒），斩断其一只前肢，使其惨叫坠落。
- **火属性灵盾难破**：火蟾身上火属性光盾「不灭不休、源源不绝」，南陇侯碧绿指环幻影、蓝色长戈所化冰蟒、韩立六只蓝色巨龟傀儡的光柱皆收效甚微。
- **巨剑成形**：韩立放出十余口金灿飞剑（**青竹蜂云剑**，掺庚金炼制）凝成丈许巨剑，再口吐纤细**蓝色寒焰**包裹剑身，并激发剑中**辟邪神雷**（金色电弧）。
- **破火弹与蟾舌**：火蟾喷出拳头大小火弹（精元真火所化、非实体妖丹）阻剑，火弹爆裂化红色光蛇缠剑；韩立以电网震开光蛇。巨剑瞬移连破十余面光盾，火蟾以坚韧乌黑长舌格挡仍被一斩两截，头颅终被斩落。
- **不灭之体**：断头后躯体与头颅放出赤红光霞欲重新接合。韩立识破「不灭之体」（生命力极强而非真不灭），制止南陇侯贸然撤水幕，以冰焰将头颅、躯体冻成冰雕再击碎，逼出拇指大小赤红**妖丹**收入手中。

## 二、实力对峙
- 南陇侯、鲁卫英身为元婴修士，斩兽时明显未尽全力，几乎全靠韩立神通灭兽。
- 韩立名声近来如雷贯耳，二人自忖联手亦只与韩立五五之分，故虽对古兽妖丹动心，仍掐灭抢夺念头。韩立则暗中提高数倍警惕。

## 三、古修遗骸分宝（第818-819章）
三人进入巨峰下山洞（内有熔岩之湖），石台上有青袍包裹的上古古修遗骸。
- **青蚕袍**：南陇侯神识被青濛光华反弹，揭出此为「青蚕袍」——上古灵蚕吐丝织成，可挡神识探查；强行取下后露出乌黑储物袋。
- **储物袋所藏**：
  - 大量赤红**灵料**（炙热铁块，古修炼制火属性古宝的特殊材料，需珍稀材料结合天地之力炼成）。
  - 白玉盒（内含古玉简）、紫色小镜、黄色小剑、墨绿色飞针、两个黑色药瓶。
- **玉简——七焰扇**：玉盒带禁制，南陇侯以金光手强开，内为玉简，记载古宝**七焰扇**炼制法（需八十一种火属性灵料及众多辅料，部分本界已绝迹）。南陇侯、鲁卫英判为鸡肋。但**银月**（韩立器灵）密语提醒：七焰扇乃**通天灵宝**之一（排名靠后）；**大衍神君**亦兴奋欲研究炼制法，与银月因旧怨斗嘴，被韩立喝止。
- **两枚古灵丹**：两药瓶各倒出一枚翠绿色古灵丹，珍稀难辨。

## 四、分宝结果
- 南陇侯、鲁卫英以「寻丹妙药为入谷主要目的」为由，各取一枚绿色古灵丹。
- 韩立佯装势在必得争丹，迫二人让步后，反而放弃古灵丹与多挑古宝，仅取：**紫色古镜** + **玉盒（七焰扇炼制法）** + 全部**灵料**（韩立自称对炼器颇感兴趣）。
- 二人虽疑惑韩立不要古宝，但为古灵丹懒得深究，一口答应。

## 五、伏笔/支线
- 南陇侯先前经**慕兰草原**一行，古宝几乎尽毁于**鬼灵门**一干人围攻，故对古宝甚为不舍。
- 韩立暗藏借银月先入洞寻宝之念，因二人警惕及遗骸古怪而作罢。

## 章节概要（第六卷 通天灵宝 第820章 血咒之门）

坠魔谷内谷、熔岩湖石台旁，韩立与南陇侯、鲁卫英（鲁姓老者）三人分配所得宝物。

### 分宝与换灵石
- 剩两件古宝、三人难分，韩立以『不缺古宝』为由，主动让二人取走两件古宝，只索取等价灵石。古宝在天南修仙界有价无市，二人欣然搜刮身上灵石凑足数量交付。
- 韩立随手将南陇侯丢弃的『青蚕袍』吸回收起。此袍稀有但用途不大，二人未加阻拦。
- 三人又摘走熔岩湖旁灵草——火属性灵草『金阳芝』（炼制火属性灵丹的顶阶材料），各分数株。

### 韩立佯装出谷
- 韩立称已有收获、不愿在险地多留，欲按原路返回外谷，并邀二人同走，被婉拒。
- 涉及出谷通道：通道中有『北极元光』，需『两仪环』方可通过。南陇侯透露苍坤上人在内谷寻到一处传送阵，可直接传回外谷，故二人不必随韩立原路返回。
- 韩立告辞，临行若有若无地瞅了那具晶莹骨骸一眼，化青虹离去。

### 二人的真正图谋——血咒之门
- 韩立走后，南陇侯放出驯养灵禽『千里鹂』搜查方圆十里，确认韩立已离去；鲁卫英仍布下隔音罩谨慎防备。
- 据苍坤上人遗书：这些透明古修骨骸正是破解『血咒之门』的钥匙。血咒之门上古多用于存放重宝，禁制厉害；古修遗骸变透明即施展过血咒的标志。
- 当年苍坤上人在血咒之门附近偶然发现这具坐化的古修遗骸，二人借此可破门取里面重宝（远胜眼前所得）。二人随即出洞奔向血咒之门。

### 韩立的盘算与地图
- 韩立其实早已飞返出谷之路，已在百里之外。器灵『银月』提醒二人有所隐瞒、骨骸有异。
- 韩立明言：自己更关心『灵烛果』，已落后鬼灵门许久，无论南陇侯二人有何图谋都不愿插手。
- 韩立揭破玄机：撕开青蚕袍一角夹缝，取出一小块非布非绢之物，上以碳笔勾勒粗细线条——竟是一张『坠魔谷地图』。他凝望地图上某个显眼粗大标记，双目微眯。

### 关键人物认知
- 南陇侯、鲁卫英始终看不透韩立底细：修为不高却高深莫测，二人对其最后看骨骸的举动心存疑虑。

## 章节概要
第六卷 通天灵宝 第八百二十一章。两条线并行：韩立筹划仿制七焰扇，南陇侯一行揭开血咒之门。

## 韩立线：地图与七焰扇之约
- 经银月（未出声而由韩立自行看出）提示，韩立看出一张地图轮廓：他处粗糙，唯标记处详尽，连附近地势都标得清楚。标记处距韩立现处甚远，途中险情未知。
- 韩立判断标记处必含重大秘密，但在搞定**灵烛果**之前不愿另起事端，遂收图，待灵烛果到手后再定是否探查。
- 韩立取出新得玉盒塞入背后竹筒，向**大衍神君**提出：以前辈天纵之材，或可为七焰扇寻到已灭绝/难寻材料的替代之物，仿制出通灵之宝。
- 大衍神君识破其激将之法，但因自己对通天灵宝早有好奇、且仅余数十年寿元不愿留憾，遂接受。
- **交易**：韩立承诺，若大衍神君真能做到，便在力所能及范围内答应其**任意一个要求**作为报酬。双方说定。
- 大衍神君随即研究玉盒中炼制之法，声音全无。韩立加速向前遁去，约小半日后再次接近**紫纹蝎**巢穴所在山头。
- 旁白：韩立手中已有通天灵宝**虚天鼎**，但离打开使用尚不知何年；此等逆天宝物自然越多越好。

## 南陇侯线：血咒之门
- 同时，**南陇侯**与**鲁卫英**（鲁姓老者）行于熔岩地带中一处阴潮地下通道。
- 鲁卫英称对血咒之门之事如今有七八分相信；点醒南陇侯自己存有戒备，以防对方过河拆桥。
- 关键设定：
  - 血咒之门为南陇侯**先人代代口传的秘事**。
  - 当年南陇侯与**姓云贼子**同得**苍坤上人**遗书，遗书中亦未提及此门，否则**鬼灵门**绝不会放过此地。
  - 开启血咒之门**需两名元婴修士之力**，故南陇侯拉上鲁卫英，并一直瞒着「姓韩小子」韩立。
- 二人进入约三十余丈宽的**钟乳洞窟**，中心有十余丈大的**碧绿水潭**。
- 南陇侯祭出一根蓝色**分水旗**没入潭中，掐诀催动，潭水旋成巨大漩涡并高涨数丈分开，露出通道。二人下潜，深达二三百丈方至潭底。
- 潭底十几丈见方、圆形，铺青石地板，分水旗插于其上散发蓝濛灵光，四周碧绿水幕高耸直通潭口。
- 南陇侯祭出当初在内谷入口用过的**白色玉佩**，连打各色法决喂入；玉佩朝一面水幕发出凤鸣清音，喷出大片白霞，撕裂水幕。
- **血咒之门**显现：高约三丈的拱形石门，闪烁血色红光，表面雕一巨大独角鬼头，几占大半面积，狰狞栩栩。鲁卫英见之心惊肉跳、大感不安。

## 疑点
- 鲁卫英质疑苍坤上人当初如何得知潭底数百丈处此地；南陇侯推测其神识过人、分神探得，但自己也心生惊疑。

# 第六卷 通天灵宝 第822章 灵缈园

本章在坠魔谷内谷展开三条并行的寻宝线索，核心目标皆为传说中的「灵缈园」。

## 一、南陇侯、鲁卫英——血咒之门
- 二人来到一扇不大的石门前，门上血气魔性极浓。
- 南陇侯解释：此门为上古修士以全身精血所封印，异于普通禁制属正常。
- 鲁卫英主张谨慎；二人遂取出布阵器具，在水潭底部设下数层禁制作为退路，准备解禁血咒之门。

## 二、鬼灵门门主、魏无涯、王天古一行——直奔灵缈园
- 一行人攀登一座黝黑山石巨山，山后即目标之地。随行另有四名结丹期鬼灵门弟子。
- **禁制**：内谷上百里设禁空禁制，离地数丈亦不能飞行；入谷以来已折损三名弟子。
- **灵缈园真相**：鬼灵门门主断言灵缈园确有其地，位于灵界与人界交界处；整个坠魔谷即上古修士为看守灵缈园所建。本宗有七八成把握确认其存在（证据未明言）。
- **各方诉求**：若事成，鬼灵门可成魔道第一大宗；魏无涯欲借此突破化神期。魏无涯原本半信半疑，认为多半是上古修士以讹传讹，但因王天古几乎尽出门中一半精锐而决意掺一脚——借其名声镇住入谷修士，分得宝物。
- 鬼灵门门主仅元婴初期，面对元婴后期的魏无涯却毫无不安，颇具枭雄本色。
- 魏无涯追问钟长老下落，门主坦言钟长老另赴谷中一处密地寻另一宗宝物（珍稀但不及灵缈园）。

## 三、钟长老——另一处密地
- 钟姓老者（元婴中期）漂浮于一片乱石堆中心，神识可罩数十里，却难寻特定标记，只能靠六七名鬼灵门弟子大海捞针般搜索。

## 四、天晶真人、东门图——破阵入内谷后分手
- 天晶真人（老道）与御灵宗大长老东门图（绿袍老者）携众冲出五色霞光峡谷，形象狼狈。
- 天晶真人有两具恶鬼傀儡，威力近初期顶阶修士；东门图麾下五名绿袍修士各修一种五行功法，联手可与元婴后期抗衡一时。
- 二人互相戒备：天晶真人故意拉开距离、傀儡护身，婉拒东门图的联手提议，独自离去。
- 东门图表面笑送，背后阴沉，叹未能谋得那两具傀儡；随即放出一只银光巨大蝙蝠（银色巨蝠），喂下一颗乌黑药丸，令其追踪某方向，自己带五名御灵宗修士追随而去。

## 关键设定
- **灵缈园**：灵界与人界交界处的空间，坠魔谷为守护它而建。
- **血咒之门**：上古修士以精血封印的石门，魔性浓重。
