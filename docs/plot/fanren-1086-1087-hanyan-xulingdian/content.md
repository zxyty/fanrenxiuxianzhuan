# 凡人修仙传 第1086-1087章 寒焰修士 / 虚灵殿

## 背景：韩立滞留小极宫
- 接下数日小极宫戒备增加，白瑶怡消失踪影，但无想象中骚动。
- 韩立此界除遇化神期全力追杀或身陷绝地外已无性命之忧，故对漩涡镇定不走。
- 留下三因：①已答应助寒骊上人冲击化神期；②寒骊所授冲破瓶颈秘术只含他需施法部分，须亲身参与一回才能窥全貌(韩立为此郁闷)；③暗查太阳精火，藏经阁遍寻无线索亦不愿离开。
- 此后一个月韩立闭门打坐，小极宫风平浪静，却有暴风骤雨将至的凝重。

## 第1086章 寒焰相试
- 来访者白梦馨：面容酷似白瑶怡，自称其堂姐、小极宫内宫长老，元婴中期巅峰，修炼极寒之焰。
- 韩立曾听寒骊言宫内除他另有两人修炼极寒之焰，白梦馨即其一。
- 白梦馨晶白面孔显露功法之寒，提出以所修‘风离冰焰’与韩立紫罗极火切磋。
  - 韩立布数道阵旗结淡红光幕禁制四周。
  - 白色冰焰若淡青烟，吹气后涨大化白色小蛇扑咬；韩立喷紫色火焰化紫色小鸟迎击。
  - 紫白相撞起雷鸣，紫焰大涨吞噬白蛇，白蛇荡然无存，紫鸟光芒更胜，韩立从容收回。
- 白梦馨动容，承认此为白瑶怡不放心而出手相试，引韩立去大长老处——另两名寒焰同道已到。

## 三大外援齐聚万年玄冰大厅
- 厅中乾蓝冰焰巨鼎依旧。座中四人：
  - 寒骊上人(元婴后期，小极宫大长老) + 青衫中年人(小极宫修炼另一寒焰者，相貌普通但目光阴沉)。
  - 龙夫人：须发皆白老妪，‘柳翠派’大长老，元婴中期，功法特殊故声音永葆年轻；脾气古怪冷淡。
  - 摩鸠大师(魔鸠)：灰袍长眉僧人，元婴中期巅峰，慈眉善目；与韩立同样非冰寒属性功法却拥极寒之焰。
- 寒骊称韩立为其所寻‘第五名’修炼极寒之焰同道，苦研数百年的冲破化神瓶颈之法今日可试。
- 龙夫人欲见韩立独创新寒焰(新寒焰出现已是数万年前之事)；韩立谦称寒焰未经精练，打哈哈避免结怨。
- 寒骊坦言成功几率仅三成，但寿元将近无法再拖，只能冒险；并称已安排后手，纵自身陨落亦不致满宗被灭。
- 摩鸠提醒北冥岛附近已有高级妖兽聚集，此时行险风险大；寒骊自信回应。三人皆同意施法。

## 第1087章 虚灵殿
- 寒骊念咒，乾蓝冰焰巨鼎缩成拳头大收入袖，鼎下地面现黑洞与晶莹冰梯。
- 众人沿玄冰通道(壁嵌鸡蛋大月光石如水晶宫)下行百余丈，转入青玉砌成的古朴地下殿堂。
- 殿堂尽头为乳白色巨大石门(高二三十丈，布满符文法阵与古怪花纹，各色灵光)。
- 门前已立十六七名元婴修士(多为元婴初期)，为首是白瑶怡与灰发青斑老者(叶长老)。
- 寒骊揭示：秘境中三座同名‘虚灵殿’各有奥妙，玄玉洞在其一之内，真正入口为各自地下密道而非殿门；借玄玉洞极寒之气可多一成突破把握。‘虚灵殿’之名令摩鸠、龙夫人皆惊。

### 叶长老反对与寒骊压制
- 叶长老(监察长老，元婴中期)反对：①历代开虚灵殿须本宫濒临大难无力退敌时；②让宫外修士进玄玉洞从无先例。态度桀骜。
- 寒骊称群妖压境、早开合情合理，且非三殿同开，身为大长老有此权。
- 叶长老执掌‘监察灵宝’(韩立疑为通天灵宝或灵宝仿制品，故元婴中期能抗元婴后期，自成一系)；寒骊提监察灵宝时眼角微跳，略有忌惮。
- 寒骊以争取留守长老三分之二同意压过反对；叶长老回首见数名同门躲闪，方知金师弟等已被调往外岛，乃中计。叶长老色变退去上面巡视，寒骊不留难。

### 开启封印
- 青衫人主持，众长老各取式样相同玉佩(与寒骊给韩立又收回的那件相同)祭于半空，咒语催动化光球聚顶，幻化出五色女子光影——身材修长、面目模糊、明眸如星辰。
- 全体小极宫修士(含寒骊)躬身恭敬，韩立断定此为创派上古女修。韩立暗思此女与乱星海虚天殿渊源——虚灵殿与虚天殿相似、乾蓝冰焰出现，绝非巧合。
- 光影飞向石门，‘噗嗤’爆裂，符文灵光尽消，石门变朴实无华；顶端虚空现众多失灵玉佩坠落，众修各自摄回。
- 寒骊双袖拂出蓝霞击门，石门如纸无声分开。

### 寒封暴涌
- 石门方开，寒骊忽身形一晃退远，急喝‘诸位师弟小心’。
- 门内传惊天巨响、如无数猛兽齐吼，白茫茫寒封挟雷鸣自内暴涌，所过寒冰蔓延、地面殿柱晶莹闪烁。
- 众修大惊：修为弱者倒射退避，高者放光罩或祭护身法宝；寒风席卷整座殿堂。(悬念止)

## 关键悬念/伏笔
- 太阳精火仍无线索。
- 小极宫创派女修光影 ↔ 乱星海虚天殿 ↔ 乾蓝冰焰，韩立笃定三者有渊源。
- 叶长老所执‘监察灵宝’性质未明(疑通天灵宝/仿制品)。
