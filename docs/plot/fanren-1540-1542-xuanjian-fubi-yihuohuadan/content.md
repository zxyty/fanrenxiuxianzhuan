# 凡人修仙传 第九卷 灵界百族 第1540-1542章｜玄剑附臂・以火化丹

## 一、火阳族的危局
- 灵界乱海一带有**同源三族**(皆为**娲氏一族**分支)，**火阳族**为三族中实力最强、岛上防守最严密者。
- 另两族已先后遭**乌罗族**灭杀，其男性族人被掠走；当年乌罗族与娲氏一族结为死敌正因此事。
- 从出手痕迹看疑有**乌罗王族**混入。娲氏一族数名天祭司曾以逆天神通探查，本以为乌罗王族血脉已断尽，如今乌罗人再现，恐有乌罗王当年逃脱探测、如今修炼大成。
- **乌罗王族(乌罗皇族)天生克制娲氏一族**：除娲氏皇族外，其余族人在其面前神通发挥不出一成；其修炼速度之快不逊雷鸣大陆几大族上层存在。
- 火阳族大祭司妇人向交好族群求援多半无望，决定与族中长老商议、将戒备增加一倍，并寄望于韩立。

## 二、烈阳神丹的抉择
- **妇人(火阳族大祭司)**已进阶**上族**之列，曾服**炼仙丹**，**寿元亏损**。
- 她判断韩立虽重伤，仍是远高于己的存在(『一出手就能轻易灭杀我』)，决意取出**烈阳神丹**(可恢复近半还多寿元)相赠，恳请其助火阳族度过乌罗族之劫。
- 其女**白珠儿(珠儿)**反对轻率赠丹，提议同往试探韩立虚实；妇人最终同意，叮嘱『不要过分、千万别得罪此人』。三日后取丹同去。

## 三、韩立的布置(玄剑附臂)
- 韩立法力略复，但精血与神念损耗严重，纵有众多灵丹也需**三四年**才能恢复巅峰；他敢留在异族自有自保把握。
- 召出**啼魂**与**豹麟兽**(后者已相当于**化神级**)。命**啼魂化形成与自己服饰相貌一般无二的『韩立』**坐于身旁打坐作替身，豹麟兽化残影隐匿屋中。又布下阵旗，木屋外现一层淡白光禁制。
- 查看手臂：劈出一剑后消失的**玄天果实**竟化作一条淡黄色印痕附于臂上(**玄剑附臂**)。聚灵力可使印痕稍清晰，但以现有法力难有更多动作。
- 此宝一剑威力极大却**伤敌先伤己**(劈出恐先丢自己大半小命)，韩立忐忑，决定法力恢复后先将其逼出手臂。

## 四、珠儿试探与以火化丹
- 两日后妇人携赤红玉盒(内装烈阳神丹)与珠儿至。屋中并坐两名『韩立』：一含笑睁眼(本尊)、一闭目无表情(啼魂化身)，气息一弱一强但服饰相貌全同，妇人神念竟均测不出修为，疑为身外化身之术，心生忌惮。
- **珠儿(结丹期、蛇人少女)**借袖中一片翠绿异常叶子催动秘术神念试探：
  - 探**啼魂化身**：神念如落死物，毫无波动。
  - 探**韩立本尊**：神念如入巨大漩涡，不由自主陷入，珠儿魂飞天外急收神念方脱身，被本尊以似笑非笑之色看破，窘迫不已。
- 妇人未察异状，介绍丹药药性与服用忌讳后将玉盒交予『韩立』告辞。事后珠儿确认韩立确有大神通，妇人遂安心，认为赠丹不算浪费。
- 啼魂化身在客退后变回乌黑小猴(自天渊城灵智初开、消化典籍后化形已难辨真假)，将玉盒奉上。
- 烈阳神丹为拇指大赤芒圆珠，手指靠近即冒赤焰，但韩立肉身强横视若无物。因丹火属性霸道、服用易被药性反噬(宜以大量寒属性灵丹中和)，韩立改以**噬灵天火**(喷出银色火球，化为拳头大银色火鸟)将整颗烈阳神丹包裹，**以火化丹**炼取其极火之力。

## 五、要点
- 烈阳神丹乃火阳族第一代大祭司当年于附近海域斩杀一头海兽所得。
- 妇人自言身为大祭司，耗寿元救族无憾，历代大祭司少有寿终正寝者。
