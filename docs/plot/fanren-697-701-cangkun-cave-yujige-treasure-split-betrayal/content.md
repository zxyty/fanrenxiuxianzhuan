## 背景
地点为苍坤上人（文中亦作『穹坤上人』）遗留洞府，与坠魔谷秘密相关。一行元婴/结丹老怪为夺宝同行，苍坤遗言称洞府仅设两层禁制。

## 第696章 破禁与卷轴
- 韩立以十几杆阵旗布阵、配咒语与法决将法阵『嵌入』四面晶墙（耗时半个时辰），再以碧绿色阵盘统合四壁霞光，喊『破』字震碎晶墙，露出青石壁与一扇石门。此即破开第二层『太妙神禁』。
- 南陇侯、白衫老者（云姓老者）承诺：二人各先挑一件后，韩立可优先挑选一件宝物。
- 石门后更大厅中立白玉『玉矶阁』（两层、十余丈、银字匾），阁前陈旧供桌供一银白卷轴，展开为苍坤上人背负长剑仰天的儒生绣像图。云姓老者以红色法决试探无异常，南陇侯建议暂收，云姓老者纳入储物袋。

## 第697章 瓜分
- 玉矶阁一层三排木架：第一排古宝/法宝16件、第二排稀有材料、第三排几瓶丹药。
- 窗口有『阴凝草』（炼阴寒丹药最佳药引，可增药性三分，唯百年期最佳；此株年久已无用）。韩立查看时燕如嫣主动攀谈欲化解其与夫君之怨，被韩立以『魔门与天道盟敌视』讥讽回绝。
- 南陇侯清点：古宝6、法宝10，材料丹药分八份。规则：要古宝则放弃下面法宝挑选权。白衫老者与尤姓修士弃古宝选两件法宝；韩立等各取一件古宝——韩立得一件竹筒状古宝（威力未知）及一份材料丹药。蒲团被白衫老者弃于地。
- 上二层：檀香扑面，神龛供三头六臂独角妖神金像，与韩立疑似梵圣真片的铜片妖兽完全一致；另有书桌、蓝濛濛寒玉床，床头三只无法以神识透视的玉盒。

## 第698章 算计
- 白衫老者抢抓玉盒被王天古以黑芒阻拦。王天古借『东西不够平分』为由，要求云、南二人不得先验玉盒。邰夫人（老妇人）出言相帮（建议改挑寒玉床）。
- 阁中分成两团：南陇侯+云姓老者 对 王天古+尤姓修士+邰夫人+黑脸汉子（王蝉、燕如嫣茫然不知联手内情）。王天古一方坦承出发前已小聚商定应对之策。韩立表态中立。

## 第699章 祸水东移
- 白衫老者（云姓老者）突然反水，以银轮重伤南陇侯——切开护体金光、击凹胸膛，幸南陇侯贴身穿『青犀甲』保命。云姓老者随即归入王天古一方。
- 王天古派尤姓修士缠住韩立，派王蝉、燕如嫣协助看住韩立，主力围杀南陇侯。南陇侯施秘术暂控伤势、面颊殷红如血，众人忌其困兽搏命不敢先动手。
- 南陇侯以金霞卷回三玉盒，主动抛一只给韩立行『祸水东移』，分散对方注意以求脱身。韩立识破不上当，收下玉盒。

## 第700章 先手 / 第701章 万尺一线
- 韩立先发制人：抛出黑色小山隔挡尤姓修士（尤以玉如意召白虎光柱轰击）；王蝉放血腥护体血雾、燕如嫣放香甜迷神血雾，二人融为紫色血雾。
- 韩立以风雷翅银弧瞬移贴近尤姓修士护罩，张口吹出乾蓝冰焰穿透光罩直击其脖颈，将其冻成冰雕（借此前对穆姓法士经验先封头颅）；淡金电网（风雷翅）吸去蓝色葫芦射出的蓝电，韩立夺走蓝色葫芦。
- 韩立以黑色小山压向王天古众人逼其闪避，给南陇侯出手机会；又以阴魔斩斩入紫雾重创王蝉（未杀），用雷遁离开。逃至一层大门时银月（化雪白小狐）自暗处飞入韩立袖中——原来破禁时韩立故意以耀眼灵光扰乱众人灵觉、强开禁制缺口将银月送入阁中盗宝，这也是他拒绝与南陇侯联手、先行脱险的底气。
- 玉矶阁被南陇侯一击崩爆（升起丈许金色骄阳秘术），南陇侯以金碗古宝暂困王天古、云姓老者等人后脱身，于通道中拦住韩立请其拖延（仅一杆开山旗，需时间解外层禁制）。韩立以阵盘重新封起蓝色晶壁（威力不及原十分之一），并放噬金虫施『虫甲术』护身。石壁开后二人先后遁出。
- 南陇侯施苍坤上人独创遁术『万尺一线』（耗元气精血、化遁光为纤丝、气息全敛、瞬间远遁）逃走，临走诅咒众人；韩立以雷遁化黑点远去。
- 收尾：王天古问遁术，云姓老者解释二者遁术来历；云姓老者透露已对南陇侯下手脚，只要其跑不出三百里仍可追踪，且南陇侯重伤强提法力又用万尺一线，元气大损、恐难再保元婴中期。邰夫人提及王天古曾先赠『黑玉莲』并许诺共享坠魔谷秘密，众人才肯冒险联手。云姓老者自陈早年出身鬼灵门、向不管事，因南陇侯（穹坤上人后人）私交正道天极门并欲借势入坠魔谷，故不得不灭之。

## 关键转折
- 联手内情揭穿：王天古一方早已串联，韩立从出发起即被定为南陇侯陪葬品。
- 云姓老者身份暴露：非散修，实为鬼灵门长老。
- 韩立『先手』成功：暗送银月盗宝 + 乾蓝冰焰秒杀 + 中立脱身，全身而退并截留一只玉盒。
