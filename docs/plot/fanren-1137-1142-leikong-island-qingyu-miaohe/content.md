# 凡人修仙传 第1137-1142章 剧情梳理（第七卷 纵横人界）

## 一、第1137章 青羽
- 韩立目睹一场远超人界修士的大战：一只大得出奇的巨禽攻击一团**活的『鬼雾』**。事后他放神念探海底，发现异常。
- 派**人形傀儡**潜入海底四五千丈深处（异常平坦干净），追上一团灵性十足的青色光团——内有一根丈许长、忽明忽暗、布满神秘符文的**青色翎羽（青羽）**。
- 青羽能**瞬移遁空**，傀儡数次扑空。韩立以**辟邪神雷**所化金色电网克制其瞬移，再喷精血催动符箓封住灵性，将其收服并缩小至尺许。轻晃可散发**极纯净的风灵力**，韩立大喜，以玉盒收入储物袋。
- 韩立推断：巨禽应是灵界/他界厉害妖物，鬼雾则是半灵半魔之物，疑与魔物**罗睺**、当年困他的**阴冥之地**有关（能撕裂空间、游离界面）。此等存在非他能左右，只当奇遇。
- 两月后赴**碧灵岛**途中，遇三名修士被六级妖兽追杀。

## 二、第1138章 雷空岛
- 被追三人：**辰京**（戴斗笠、结丹初期，自称雷空岛/青阳门）、一白发老者与一妙龄女子（皆筑基后期）。妖兽为**婴鲤兽**（六级，白雾缭绕、婴儿啼哭、四臂持刀剑鼓旗四宝、射蓝色雷球），是韩立当年在乱星海所见第一只高阶妖兽，此只更大已开灵智。
- 婴鲤兽神念扫出韩立修为后惊恐逃窜，韩立**一口尺许飞剑化十余丈金光**瞬间将其斩成七八截、元神俱灭，摄取妖丹。
- 三人惊为元婴高人。辰京拜见，自称**青阳门门下、师从三阳上人**。韩立想起当年元瑶杀青阳门少主被追杀旧事，但与辰京无过节，遂赏妖丹换其带路打听碧灵岛/高阶灵石。
- 辰京引韩立至**雷空岛**（方圆数千里，灵脉灵药最佳，逆星盟与天星宫忌惮下由多方联手占据）。岛中央**石城**新建，灰白石砌，住三四万修士；城中八根五六十丈石塔为**守护大阵**（阵法大师所设，每塔筑基期镇守）。
- 辰京提及岛上元婴修士中有**碧云门妙鹤真人**——昔年毁体后经碧云门以**夺舍**救回修为，此次来意在替碧云门拿下本岛。

## 三、第1139章 神念附灵
- 韩立令辰京三人离去并警告勿泄行踪，随即隐形。辰京回隐蔽洞府开禁制商议：女弟子提议上报门中调查韩立，辰京以师祖（三阳上人）闭关冲击后期、不愿招惹元婴高人为由拒绝。
- 韩立早将一缕分神**寄附在噬金虫**上潜随三人，确认其识趣后收回神念。说明其**元婴后期大成**后神识强大，可将一缕神念暂时寄附通灵活物（祭炼过的噬金虫等可直接神念操控攻敌）。
- 韩立变容成**瘦高老者、压制修为扮结丹后期**逛坊市。识出**六连殿**（已加入逆星盟），又撞见妙鹤真人与一**黄衫/黄袍老者**（皆元婴初期）从六连殿走出——妙鹤面容年轻二十余岁，乃夺舍重生后施转颜秘术、尚未将体貌年龄一并转回所致。

## 四、第1140章 守株待兔
- 韩立连逛五家商铺，用法宝、古宝、珍稀材料换得**百余颗高阶灵石**。出城时被数名觊觎灵石的结丹修士跟踪，他以隐匿法术诡异消失甩脱（本不想打草惊蛇惊动岛上元婴，否则尽灭之）。
- 韩立在石城外山坳布下隐匿法阵，放出**上万噬金虫**，施展借**大衍决**自悟的**『神识化千秘术』**（坠魔谷曾用过一次）：将大半神识化为上千细小神念依附噬金虫，化作千眼千耳搜索全岛，密切监视妙鹤等人动向。

## 五、第1141章 截杀
- 夜里数名斗篷修士（妙鹤、黄袍老者及结丹弟子）悄然离石城赴碧灵岛。途中黄袍老者以『白哀针』击中一只噬金虫却无法击毙、虫尸消失，警觉布防。
- 韩立现身索要灵石，自报姓韩、专为妙鹤而来。黄袍老者自称**离龙岛黄昆**。妙鹤终认出韩立——正是**当年得到虚天鼎、被逆星盟剿杀令追杀的结丹小辈**，竟已凝结元婴（二人误判为中期）。二人传音被韩立偷听，决意联手又暗备逃路。
- 韩立背生**风雷翅**雷遁逼近，逼出二人防御。

## 六、第1142章 极阴之讯
- 韩立以**紫罗极火（紫焰/紫冰）**轻破黄昆白丝、妙鹤碧绿玉锤与火盾，一手扼住黄昆颈项、紫冰封体擒住，再轰碎冰雕逼出其黄色元婴，以辟邪神雷金网+符箓禁锢。
- 四名结丹弟子四散逃走，被韩立四道金光（飞剑/噬金虫）尽数斩杀。妙鹤真人抢先化绿虹逃向石城，韩立不追，放暗藏的**人形傀儡**鬼魅尾随——傀儡一拳洞穿其红盾护体灵光、贯胸取命，再以银网罩下擒回，妙鹤已半死不活。
- **搜魂**结果：妙鹤实为暗投逆星盟、欲在其支持下控制雷空岛；黄昆为逆星盟收买的另一元婴散修主事人。二人突接命令收集所有高阶灵石速送碧灵岛、勿惊动他人。
- 关键情报：当年虚天殿中差点要韩立性命的仇家**极阴祖师（极阴老祖）**，竟是**逆星盟驻碧灵岛、守护矿脉的长老之一**。碧灵岛共有逆星盟元婴四人（一中期、三初期）。
- 韩立视极阴老祖为乱星海必灭仇敌，自信能神不知鬼不觉潜入碧灵岛击杀，遂携两袋（各百余颗）高阶灵石直奔碧灵岛。三日后抵达碧灵岛附近海域。

- 韩立本章全程以**伪装/化名瘦高干瘦老者、压制为结丹后期**行事，无冒用具体已名人物身份（非 impersonates）。
- 本片段中妙鹤真人被韩立人形傀儡击杀并搜魂、黄昆元婴被擒封印——据正文记其结局于本卷此处（与某些世界状态须知所记死亡章存在出入，此处依正文）。
