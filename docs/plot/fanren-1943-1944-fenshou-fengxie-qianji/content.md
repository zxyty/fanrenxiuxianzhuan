# 第1943-1944章 分手 / 风邪与牵机大法

## 擒黄粱石灵（第1943章）
- 韩立两手一搓扬出**辟邪神雷**，金色电弧化巨型电网罩住黄粱石灵；石灵以黄色妖火欲挣脱。
- **黑雨上人**传音阻止韩立近身，催动**碧蓝飞蝎**分化十余只，尾钩蓝芒没入石灵身躯，克制其黄色妖火使之黯淡溃散。
- 石灵暴怒鼓身欲**自爆**，韩立瞬移百余丈外避开；黑雨上人早有预防，抛出**银霄雷网**（纤细银网、满布银色符文），勒紧使石灵膨胀身躯迅速缩小、神通尽废，最终包成拳头大小银色丝团收入袖中。
- 黑雨上人言只借用石灵本体一用，并不取其性命，要带回**圣岛**。
- **天蝉**（僧人）原被石灵幻术所困，石灵被擒后脱困，谦称多亏韩立相助。

## 祭雷术之议
- 黑雨上人辨出韩立所用为**辟邪神雷**，赞其辟魔属性是对付低阶魔族的大杀器。
- 关键设定：辟邪神雷需以真正**祭雷术**催动，威能足以威胁合体级魔尊；但圣岛只保有**下半部法决**。
- 祭雷术上半部教人控制神雷威能、祭出神雷；下半部才是操控驱使之法，缺一不可。
- 韩立暗忖：当年在**地渊四大妖王**处所得祭雷术原来只是半部，难怪不堪实用——应是四大妖王故意藏私。
- 黑雨上人建议韩立去**蛮荒世界**异族中寻找全本祭雷术。韩立心动，决意走一趟圣岛与蛮荒。

## 分手
- 天蝉告辞回**圣皇城**助圣皇守城；黑雨上人携石灵返圣岛；韩立称要借地火、滞留两日，三人就此分手，两道遁光朝不同方向远去。

## 韩立布置与放出第二元婴
- 韩立重回巨坑边（坑底黄雾弥漫看不清），布下数十道阵旗化作白茫茫雾气笼罩巨坑，又放出十三团紫光、并放出一道绿影没入地下。
- 随后盘膝坐下，拍开天灵盖，放出一只**面容酷似韩立的元婴（第二元婴）**；元婴解封一只金丝圆球露出乳白木盒，并取出十几颗五色眼珠状灵物。

## 神念化身入镇魔锁遇风邪（第1944章）
- 韩立一缕神念所凝化身出现于草原低空，未见**车骑恭**，却被高空骄阳中的陌生魔头引出。
- 魔头自报无名（后称**风邪**），黑色战甲、络腮胡子大汉，称车老怪有事缠身，自己可代为交易混沌之气，且只要三分之一（车骑恭要一半）。
- 韩立以已对车前辈发下**心魔之咒**为由婉拒；风邪称韩立尚未得混沌之气使用之法、不算违誓，并愿先告知镇魔锁混沌之气提取之法、事后另给好处。

## 车骑恭脱困对峙
- 草原幻境骤变风沙之地，**车骑恭**（车老怪/车老魔）脱困现身，怒斥风邪暗中出手，凝黑云巨手、三头六臂魔相来攻。
- 风邪以战甲乌光卸去大半威能，亮明已修成**牵机大法**，并以'否则便宜了血光这贼子'相挟；车骑恭一听**血光**之名面露狰狞，暂停手听其陈说提取新法。
- 风邪索要独得混沌之气三分之一为条件；车骑恭要其先证实所言再议。韩立在旁冷眼旁观未表态。

## 黑雨上人遇伏（章末）
- 一日后，急返圣岛的黑雨上人飞经两座万丈巨峰之间，双峰毫无征兆同时巨响倒压，漫天山石雨点般落下——遭伏击。

## 备注
- 本片段中韩立以本名行事（黑雨上人、天蝉皆称其'韩立/韩道友/韩兄'），未用化名。
- '车骑恭/车老怪/车老魔'为镇魔锁内被封印的魔尊，与早先已死的'车老妖'非同一人，勿混淆。
- '血光'指血光圣祖（据世界状态须知，其自第1909章起以'阳刹'之名/身行事）。
