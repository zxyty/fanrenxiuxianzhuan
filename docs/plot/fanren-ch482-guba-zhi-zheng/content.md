# 凡人修仙传 第482章·古宝之争

## 背景
第四卷《风起海外》，虚天殿/虚天鼎之行。虚天鼎临近洞口，鼎中所藏古宝在接近洞口时竟各自独立飞遁而出，落宝全凭本事。极阴祖师原计划驱除正道后再将虚天鼎连同鼎中宝物一齐取出、按约分赃，此变故使瓜分约定彻底作废——宝物入谁手便归谁，不会再吐出。

## 双首火狼古宝之争（祭坛处）
- **乌丑（实为极阴老祖夺舍占据其躯体，自第403章起；本章抢宝言行归极阴老祖）** 催动法决，以黑色巨蟒收缩束缚火狼；**玄骨** 以绿网（绿丝、绿光）共同强逼。
- 双方加力，火狼身上光罩被挤压变小后反而更耀眼坚固，火狼安然瞪视二人，陷入僵持。
- 乌丑虽恼怒玄骨抢宝，但忌惮玄骨有 **蛮胡子** 做靠山（连极阴祖师都大为忌惮），不敢以势压人。玄骨表面只是结丹初期，真实底细更深。
- 极阴祖师与蛮胡子各自看好与己有关之人收宝：极阴因赠予乌丑多件宝物、且其玄阴功为魔功名列前茅，认为乌丑更有把握；知玄骨真实底细的蛮胡子则对玄骨信心十足。

## 韩立的隐忧与花篮古宝
- 祭坛附近唯一闲着的是 **韩立**。他原本不假思索取出花篮古宝（其神通正适合收取法宝），见玄骨抢宝后才惊觉：花篮是他击杀看守玄骨的丑汉时所得，当面拿出岂非暴露玄骨是他放走的。
- 韩立冷汗直冒、苦思应对极阴之策，却发现极阴的目光只扫了花篮一眼便毫无异常情绪。由此确认：花篮并非极阴所赐丑汉之物，极阴对丑汉拥有此宝一无所知。
- 真相：丑汉当日是私自外出取得花篮古宝的消息才离岛，得手后却让韩立一行将困玄骨的大阵破除大半，丑汉惊怒暴跳，最终死于韩立的青竹蜂云剑下，花篮成韩立战利品。
- 韩立暗想：若真被极阴发现，定要把玄骨一同拉下水。

## 洞口上方的抢宝（黄、白二光）
- 洞口又传沉闷雷鸣，六名元婴期修士运足功法、祭出防御宝物抵御奇寒，齐聚洞口正上方，正魔互相警惕。
- 蓝焰中“嗖嗖”飞射出黄、白两道光华，六人几乎同时出手抢离己较近者。
- **黄光之争**：万天明与极阴、儒衫老者同时盯上黄光。万天明幻化紫色火龙咬向黄光，同时喷出两颗紫色晶莹圆珠——**天罗真雷**。极阴失声惊呼“天罗真雷”，与儒衫老者皆不敢硬接、急忙闪避。略一耽搁，紫色火龙便一口吞下黄光缩回。万天明搓手现出古宝原形：一块四方古佩，符文流光、灵气荡然。那两枚紫珠飞出数丈后轻微爆裂消失。
- 极阴与儒衫老者脸色铁青，斥万天明用假雷欺诈、手段下作。万天明冷笑反讥：天罗真雷需自损法力才能修炼，非性命之危不舍得用；二人连真假都分不出还称下作；说罢两手一合，紫光闪过玉佩消失、手上又多出一颗一模一样的紫珠，逼得极阴与老者拿不定主意。
- **白光之争**：蛮胡子凭托天魔功超强防御，硬接黑瘦老者与天悟子一击，抢得白光所化、白光闪闪的铜钱状古币。
- 另：青易居士见状面露可惜，仍缠住天悟子不让其分身遁走，使祭坛附近只剩韩立闲着。

## 结局意外
- 儒衫老者轻“咦”一声，极阴顺其目光望去：祭坛附近，韩立正手持一件黄红两色玉如意端详，如意两端各雕一只栩栩如生的狼首——正是双首火狼所化的古宝。乌丑与玄骨一脸无奈、悻悻望着韩立。
- 此宝未落入乌丑、玄骨之手，反落到一开始毫无出手意思的韩立手上，极阴大感意外、目露疑色而沉默不语。
