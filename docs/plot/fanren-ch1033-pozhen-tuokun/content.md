# 第1033章 破阵与脱困

## 化仙宗的图谋（木夫人 / 秀丽女子）
- 众修士齐攻宫殿四周大阵；秀丽女子担心宫殿即「向老」所提古魔圣祖封印之地。
- 木夫人断定此处并非魔物封印地（否则化龙玺早有反应）。她真正目的：**控制整座昆吾山禁制的枢纽阵眼，就在封魔塔第九层内**；欲重新加固封印，除化龙玺外还须到第九层施法。
- 四周无入口，唯一可疑处即此宫殿——推断**进入第九层的入口在宫殿内某处**，八灵尺与此法阵或为封印第九层入口之用。
- **八灵尺**是化仙宗先人遗宝、属通天灵宝；秀丽女子手中的**四象尺**乃仿制八灵尺所炼，动手抢宝时发动或有奇效。木夫人称「向老」随时会赶到，宝物终将易手归己。
- 二女传送进来的石室中有一黑色传送阵（已失效），疑通往古魔封印之地；宗内典籍未载古魔被封具体之事。

## 古魔圣祖分魂脱困
- 另一处幽暗空间内，古魔圣祖正冲破封印束缚。
- **双首四臂的古魔**四手掐诀，催使**五件魔器**——墨镜、玉莲、圆环、令箭、黑色小瓶——向上空**银狼（双首巨狼）**喷黑色光柱灌注魔气，将其身上符箓与黑链推离。古魔身下黑色传送法阵被灰光阵旗笼罩、暂停运转以防修士闯入。
- 四面黄色铜镜组成囚笼；银狼黑色狼首魔化，独角涨大、黑电闪动、双目紫色魔焰可令符箓自燃成灰。
- 巨狼再度巨大化撑破黄光囚笼，数百铜镜爆裂消散。
- 唯**大周天星辰链**随之变粗，紧束巨狼，链表燃起乳白色火焰，无视护体魔光直接点燃银白毛皮。古魔露畏惧之色，低呼「**纯阳白骨火**！」——上古修士将此妖火融入链中，颇为棘手。
- 银狼甩头放出黑色雷球、化无数黑电击向黑链，白焰黑电交织；又喷黑濛光霞扫荡远处禁制。古魔面露喜色：圣祖分魂神通未失，不久即可脱困。

## 裂缝入口的对峙（玄青子）
- 昆吾山封印裂缝入口，聚集修士由上千锐减至二三百，但多为结丹以上、含十余名元婴期。
- **太一门元婴后期长老玄青子**盘坐阻拦；因太一门声名与其修为，众人未敢强闯。
- **罗姓老者**（元婴中期）不耐催问；玄青子答裂缝暂不可通过，再等数日便会恢复。罗质疑阴罗宗、化仙宗已有人进入——玄青子释为木夫人、乾兄等持**日月梭**与**灵犀孔雀**开路、有三大灵梭之一方可冒险。
- 碧磷涯**羽道友**（毒功名震大晋、罩于绿雾）讥讽再等宝物将被「乾老魔」们搜刮一空。
- **玄青子真实处境**：得密报开启封印者疑为叶家修士；旋即收到太一宗内一极特殊存在（连他也不敢违抗）的神秘命令，须与**天魔宗七妙真人**联手入封印一趟，入前尽量阻止他人进入。后又收到载有封印信息的玉简，大惊，悔放乾老魔等人进入，遂越发不敢放人——恐放出封印之人则人界无人能敌，且内有通天灵宝不容他人染指。

## 关键判断
- 命令两名后期修士联手方可进入，足见封印之物绝不简单。

# 第1034章 禁魔环、冰魂谷

第六卷 通天灵宝。本章双线推进：南疆裂缝口正魔两宗联手封锁，以及太一门禁地冰魂谷有神秘来客。

## 一、裂缝口：正魔两宗联手封锁

- **玄青子**（太一门，道号玄青子）正欲安抚欲闯入的修士时，**七妙真人**（天魔宗执法长老，皂袍鹰钩鼻中年文士）乘乌光赶到。
- 七妙真人是大晋赫赫有名的**魔道巨枭**，对敌手段残酷狠辣、凶名在外。元婴中期的**罗姓老者**认出他后暗吸凉气，不愿触怒，遂打哈哈退回人群；绿雾中修士亦默然退走。
- 七八名元婴期天魔宗魔修赶到，布下禁制大阵、以白雾掩住裂缝入口。正魔第一大宗联手姿态强硬，散修与小宗门敢怒不敢言。

### 阵中密谈（玄青子 ↔ 七妙真人）
- 二人皆**奉门内之命前来**，不得不来。玄青子坦言七妙真人凶名才能镇住同道；但他隐瞒了**化仙宗二女**之事。
- **叶家声东击西**：明面用一件通天灵宝仿制品吸引正魔两宗注意，暗地里取真正的通天灵宝。两宗早年在叶家埋下内应才得知，内应只来得及破坏部分法阵趁乱逃出。
- 封印内是**古魔圣祖的分身/化身**，被镇压多年已元气大伤。门内命令：**不到万不得已不要伤其性命，最好重新封印**——当年古修也未灭杀此魔，疑有特殊缘由。
- **乾老魔**及数名散修此前已进入封印，随后阴罗宗与天澜草原修士（含一位**天澜草原大仙师**）也进去了。玄青子当时不知封印重大，未坚决拦阻。七妙真人对无大晋顾虑的天澜草原修士颇不放心；并称天魔宗与天澜圣殿只是利益交换，戒心很重。

### 禁魔环与天阿神剑
- **禁魔环**：古魔界入侵时灵界修士带来、专门禁困高阶古魔的上古法器。七妙真人由宗内那位连同命令一起送来**五只**，称五只加身即便古魔圣祖分身也只能束手就擒。
- **天阿神剑**：玄青子随身携带的太一门法宝，最适合对付妖魔。
- 七妙真人取出**灵龟飞车（灵车）**，化为数丈龟壳状灵车，二人乘其化乌虹飞入裂缝。

## 二、太一门禁地：冰魂谷

- **真桓山**：大晋四大灵脉之一，广十万里，正道第一宗**太一门**位于此。太一门势力之大，全大晋仅天魔宗、万妖谷可勉强并驾齐驱。
- **冰魂谷**：真桓山脉一处不起眼小山谷，灵气匮乏、阴风冰寒、长年冰雪覆盖，是太一门罕有人知的禁地。数百年无人敢入，仅谷口一道幻阵遮蔽，外无禁制亦无人驻守。
- 七妙真人与玄青子进入封印裂缝的同时，一名**满面病容的灰袍老者**鬼魅般现身谷口，视幻阵如无物，化绿光射入谷中。
- 老者在谷尽头一块巨大**冰壁**前现身，自称『**呼某**』，冷喝：『**白老鬼**，明知呼某到此，何必还躲在棺材中不出来见客！』话声如雷在冰谷反复轰鸣，震得冰壁颤抖。

- 玄青子刻意隐瞒化仙宗二女之事，可能另有图谋。

## 一、冰魂谷密谈（化神修士线）
- **来访**：魔陀山的**呼姓灰袍老者**不请自来，到**白姓男子**所在的**冰魂谷**，因『昆吾山的事情』而来。
- **缘由**：**向老鬼**进入昆吾山现世的封印后疑似出事。**风老怪**飞剑传书告知——风老怪与向老鬼一向交好，互持对方**元神珠**；向老鬼进入后不久元神珠转暗，若非重伤即被困禁制。风老怪自身另有大麻烦无法赶到，故托呼、白二人代往，事后赠**两枚血气丹**为酬。
  - 向老鬼老奸巨猾，入封印前已先以飞剑传书给风老怪留言留后路。
  - 向老鬼被称为『几个老不死中神通最大的一位』。（按世界状态须知，向老鬼已于第1033章死亡；本章二人尚不知情，仅为讨论，不计新行动。）
- **白某拒绝**：他三年前**生服一枚火精枣**，正借其霸道火灵力强行冲击功法瓶颈，不愿因小失大。火精枣若不炼成丹药而生服，火灵力足以点燃体内法力，极其凶险。白某宁可搏一线突破，也不愿如向、风二人为省精元而扮小丑混迹低阶修士。决意将彻底炼化火精枣前封闭冰魂谷、不见任何人。
- **逆灵通道之争**：向、风等人苦寻**逆灵通道**——传说上界修士能压制法力降临人界，人界修士或可借此通道以秘术暂提法力进入**灵界**，从而无需进阶**化神后期**即可飞升。白某不以为然：天地元气大变后无数化神同道遍寻大半人界、大晋皆未果，疑此通道根本不存在，或藏于荒海之上无从寻起；他更信老实进阶化神后期自行飞升。
- **大背景**：上古魔界作祟致人界天地元气稀薄，众修资质虽佳却全停滞于化神初期；自最后一批古修飞升后，飞升者寥寥，近万年竟无一人飞升灵界。
- **结局**：白某下逐客令、声息全无。呼姓老者迟疑后为两枚血气丹（可续百余年寿元，弥补此行精元流失）决意独自冒险前往昆吾山，化绿色惊虹离谷。

## 二、镇魔塔破阵（韩立线）
- **地点**：昆吾山**镇魔塔八层**宫殿前。
- **韩立**两手掐诀，十余口飞剑（青竹蜂云剑）凝聚化为数丈金光巨剑劈下，击碎一具白玉石人——这已是他彻底击毁的**第四具石傀儡**。
- **石傀儡机制**：受大阵庇护近乎不死，但被击毁次数达一定程度后便无法恢复；需众多元婴中后期修士不停攻击方可耗尽，仅两人则纵然法力耗尽也难尽毁。
- 石傀儡手中**金刃**材料不明，犀利且带灭魔神通，魔功修士护体魔气一触即穿，不敢近身。欲直接入阵者会被祭坛石傀儡集中无数刀气以玄妙轨迹围攻，加之大阵兼具数种禁制功效难速破，故众修皆只闷头击毁附近石傀儡，待清空后再破阵事半功倍。
- 祭坛上可活动石傀儡仅剩几只，破阵在即。
- **变故**：韩立脑中**银月**毫无征兆地一声娇呼、连连低呼痛苦——称『宫殿中的东西好像醒了』，感觉非常奇特、很熟悉很重要，随即头痛剧烈无法多言。重要剧情伏笔。

## 关键悬念
- 镇魔塔宫殿中苏醒之物为何令银月感到熟悉重要？与镇压之魔物、逆灵通道是否相关？
- 独自前往昆吾山的呼姓化神老者将卷入塔中诸修与魔物之局。

## 背景
地点为昆吾山虚天殿（『此塔』）内，众修争夺通天灵宝**八灵尺**。在场势力含韩立一行、叶家（白袍儒生＝叶家大长老、大头怪人＝七叔）、徐姓青年与天澜圣女林银屏、化仙宗木夫人等，以及银翅夜叉、狮禽兽二妖。

## 珑梦现身（夺舍花天奇）
- 宫殿中冲出惊人灵压，一劈头散发人影现形，重创叶家辈分最高的**大头怪人（七叔）**，使其断一臂、狼狈逃脱。
- 此人着毒圣门蓝袍、形似毒圣门大长老**花天奇**，却发年轻女声、瞳孔变翠绿——众人认出其被附身。
- 据世界状态须知：自第1033章起，**珑梦夺舍占据花天奇躯体**，本章『花天奇』的言行实归珑梦。

## 银月身世揭露
- 韩立腰间灵兽袋射出雪白小狐，化为白袍美妇，即**银月**（借四瞳妖狐之体幻化的人形）。
- 珑梦称银月本名**雪玲**，二人原是同一人『**玲珑**』。玲珑为应付第五次千年雷劫，被逼修炼银月狼族无上秘术，半途出错，精魂硬生生一分为二，化作银月、珑梦两个独立元神。
- 当年元刹圣祖分身将玲珑元神逼离肉身；过去银月（雪玲）元神强、珑梦受压沉睡，如今银月衰弱，珑梦反占上风、扬言举手可灭之。
- 银月以『**噬血魂印**』反制：『你杀不了我，我若形神俱灭，你也好不到哪里去。』
- 玲珑身份：银月天狼族第一美女、**天奎妖王（天奎神狼）**之宠妃；珑梦讥讽韩立竟将其收为灵兽。
- 韩立以青色光手救下被珑梦无形巨力击飞的银月，并以**元罡盾**硬挡珑梦一击（盾面凹陷数寸）。

## 元刹圣祖与古魔登场
- 化仙宗**木夫人**取出**化龙玺**（绿色玉玺，浮现五爪真龙幻影），指认珑梦实为古魔圣祖化身欲惑众取走镇魔之八灵尺；化龙玺真龙之影暴怒。珑梦一掌击碎传送阵、又与化龙玺真龙同归于尽，逼木夫人与秀丽女子遁入日月梭。
- 八灵尺被激，引出**九真伏魔阵**：地裂升出九座大祭坛、各竖一柄十余丈金光巨刃；封印空间塌陷，一**双首四臂古魔**（自坠魔谷逃出、体内感应到被收走的两口青竹蜂云剑飞剑）脱困而出。
- 一身穿黑裙、绝色而满身魔气的**黑袍女子**现身挡下金光，其面容与当年玲珑仙子一般无二。银月确认：此即**元刹圣祖**——古魔圣祖一缕精魂所化、当年古魔入侵人界三大统领之一，所用肉身正是银月与珑梦当年合用的原身（被附身后曾化为器灵搁置虚天鼎中）。
- 银月坦言：已施展灵魂吞噬、久占妖狐之体，修为大降、功法忘却大半，难夺回原身；除非返灵界、且与珑梦联手，方有一线希望。

- 珑梦坐镇八灵尺下，宣称任何人取宝须经其同意，唯独不许大头怪人取（因其身染元刹圣祖魔气、曾持灭魂符图谋珑梦魂匣）。

- 白袍儒生留虚影、真身隐匿；徐姓青年与林银屏起撤退之意、暗中寻路；银翅夜叉、狮禽兽（被认出乃当年空玄丹士遗躯与灵禽通灵进阶）被毁传送阵后被迫滞留。

## 关键设定
- **八灵尺**：镇压古魔的通天灵宝，含八只灵兽幻影（含**八麒鹿**，可吞噬光柱），被激发可引动九真伏魔阵。
- **九真伏魔阵**：人界辟邪退魔第一等厉害法阵，上古大战曾灭杀众多上古妖魔。
- **化龙玺**：化仙宗木夫人之宝，真龙幻影克制魔类。
- **黑风旗**：通天灵宝，原镇压元刹圣祖，需修『通宝决』驱使。
