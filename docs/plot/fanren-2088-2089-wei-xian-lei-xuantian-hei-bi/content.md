# 第2088-2089章 伪仙傫·玄天黑匕

## 背景
地点为圣界禁地（一处带池塘的秘境）。黑袍青年实为一名魔族圣祖，先前一击已镇压陇家老祖等数名合体修士于地。本章承接其对韩立与灵族青年『止水』的出手。

## 止水真相：伪仙傫
- 黑袍青年弹出两颗豆粒大小漆黑丹丸，分射韩立与止水；丹丸接触即爆为滚滚魔焰（紫罗极火系魔焰）。韩立以青元剑光劈丹，仍被魔焰卷入，黑色火柱冲天，外界一时不知其生死。
- 止水以银色灵纹、银濛音波震碎魔焰，又化银盾、灰白触手抵挡，但被黑袍青年弹出的黑幽魔针洞穿银盾与头颅。
- 伤口涌出的是黏稠灰白阴气而非鲜血。黑袍青年由此判定『止水』是一具**伪仙傫**（傀儡），且被后天炼制过、气息远逊真正伪仙傫；其驱动之物不是**仙元晶**，而是**天外魔头的魔魂之力**。

## 千秋圣女之死
- 千秋圣女催动体内神秘器物、周身涌出金灿灵纹（与止水同款），原是将自身炼成**半傫之身**以操纵伪仙傫。
- 黑袍青年识破后冷笑：区区合体催动伪仙傫，仙纹一旦反噬足以致命。其以玄天之宝乌黑匕首隔空一划，千秋圣女身躯连同元神丝毫无法抵抗地一分两半，残尸被魔焰焚为飞灰。
- 其本体露出，为一面数寸大扁圆**通灵古镜**（周边镶紫色金属、背刻花纹、镜面灰濛模糊）。黑袍青年觉眼熟，欲取作祭炼魔阵的镇阵之物，遂将『止水』与古镜一并收至身侧。

## 续斩与众修保命
- 黑袍青年又以匕首连斩**老儒**、**晖长老**，二人同样一劈两半、化为乌有。
- **陇家老祖**（真龙血脉）：天灵盖飞出金灿五爪真龙击魔手，化四五丈半龙之体砸地脱困；放出七八件宝物，含仿制玄天之宝的玉玺与拳大金环苦撑。
- **羽衣少女**（天凤血脉）：喷精血激活五色羽衣化全身战甲与面具，背生光焰之翅，撕开十丈白濛空间孔洞遁走；被黑线追入虚空险些斩杀，靠**替劫**舍弃一条手臂方逃生（一臂尽失）。
- **白戚**：始终一动不动，被魔手镇压，去向意图不明。

## 玄天黑匕与韩立现身
- 黑袍青年单手抓起乌黑匕首竖起，匕首喷黑光柱破空开一漆黑大洞，引下上百斗大黑色符文没入匕首，半尺匕首暴涨为丈许**黑色巨剑**。
- 一声『斩』，月牙乌光却转向劈向淹没韩立的黑色魔焰——黑袍青年点破『区区遮掩气息之法瞒不过我』，并断定闯入者中以韩立神通最大，欲先除之。乌光化成千万黑丝织成收缩黑网。
- 韩立喝『封』，紫光化斗大『封』字爆开，凝滞空气、荡开黑网一线破绽；随即一根青白光丝激射，金弧（辟邪神雷）缭绕全身、背生一对轰鸣**青白雷翅（风雷翅）**，安然现身于陇家老祖与羽衣少女之间，双目凝重望向黑袍青年。

## 要点
- 伪仙傫为高阶傀儡，正品以仙元晶驱动；此具为魔魂之力驱动的劣化品。
- 半傫之身：修士将自身炼化以操纵伪仙傫的手段，有仙纹反噬致死之险。
- 玄天之宝（一界至宝）威能恐怖，可一招连斩近半数合体修士。
- 真龙血脉、天凤血脉为真灵中名气最大的血脉，黑袍青年觊觎其真血。
