# 第1177-1182之3｜第1181章 重临七灵

## 救回三女与真相
- 韩立驱使两条金色电蛟扫尽黑气后收回体内，三名结丹女修（慕沛灵、柳玉、宋姓女子）对其驱电化形神通骇然。
- 柳玉禀报：抓她们三人的是韩立**走失的第二元婴**，对方**夺舍过一具躯体**，约**元婴中期**修为。
- 韩立据其外貌描述，断定第二元婴所夺舍的正是**天煞魔尸**之躯。
- 韩立此番能轻易寻回三女，全靠**大衍宝经中的血月觅灵术**。

## 韩立现状揭示
- 三女以神念扫过韩立修为后大惊：韩立已是**大修士**。
- 韩立自陈**半年前已举行大修士典礼，现为宗内大长老**（境界元婴后期），对付中期第二元婴不成问题。
- 命三女即刻返回宗内，自己继续搜索内谷。

## 搜索坠魔谷
- 韩立在洞窟守候六七日，第二元婴（至木灵婴）踪影全无。
- 放出数万**噬金虫**扫荡内谷，未寻得第二元婴，却觅得不少上古修士古宝（虽不及三焰扇等仿制灵宝，但在天南已属珍稀）。
- 发现黑袍人活动踪迹：多处上古洞府被卷宝一空，判断第二元婴已携宝离开内谷。

## 推演与追踪
- 第二元婴夺舍魔尸后以**玄阴**魔功为主修；欲速进，最佳辅助为**玄阴真经**记载的**魔气灌体之法**（强行灌注可短时突破瓶颈，但有身躯魔化、神识迷失之险）。
- 因其本就魔尸之躯又懂**大衍决**，抗魔化能力强，用此法风险较小。
- 韩立闭目静坐一日一夜推演，得出结论：第二元婴前往**万丈魔渊**（七灵岛）灌体。
- 魔气灌体需年许完成初步，韩立化青虹以极限遁速直奔七灵岛。

## 双方时间差
- 黑袍人（第二元婴）虽察觉行迹或被发觉，但正施法关键期不能中止；它一出坠魔谷便潜行隐匿、未见任何修士，**至今不知韩立已进阶元婴后期**。
- 它按中期修士遁速估算韩立需两月方至，自计一个多月即可灌体大功告成，遂咬牙再滞留至多一个半月。
- 韩立进阶后期遁速快近半，一个月便赶到无边海，再行两日抵七灵岛。

## 重临七灵岛
- 七灵岛七座岛屿中的**灵鳖岛**：当年因镇压魔气，魏无涯等人赠予韩立，韩立转让给落云宗管理，现驻有落云宗弟子。
- 韩立落于灵鳖岛小山，放神识扫过数百里，找到两处驻扎修士所在，传音后静坐。
- 两队十几道遁光来见，恭称「大长老」「韩师祖」；为首两名结丹修士（小眼粗眉黄衫人、三十余岁妇人），余者皆筑基。
- 韩立认出黄衫修士为旧识，见其已结丹成功，展颜轻笑。（第一更）
