# 第2050-2051章：小灵天之秘与覆天居士

## 一、审讯朱果儿（第2050章 小灵天）
- 韩立看破女奴**朱果儿**真实修为非结丹期，而是正处于『**轮回功**』的『轮回之期』，本体已达**元婴境界**。她因身处轮回阶段才遭暗算被俘。
- 朱果儿欲自爆元婴抗拒搜魂，被韩立一拂袖以灰蒙蒙光霞封住四肢与法力，自爆不能。
- 韩立以**天眼术**（瞳孔蓝芒）令其神智恍惚、化为傀儡般状态逐一问答，最后以眉间晶丝施**搜魂之术**直接获取记忆资料。以合体修为搜元婴元神，刻意小心未损其元神。

### 问出的关键信息
- **轮回功**为其母**凌飞仙**所传；凌飞仙本人修炼的是**清气决**（道家普通法决），仅元婴境界，面容与朱果儿有五分相似。
- 凌飞仙对朱果儿轮回功的指点『不太平常』：朱果儿凝结元婴后，一些修炼疑问母亲常需隔几天半月才能解答，朱果儿怀疑母亲背后另有真正精通轮回功的『师傅』，但母亲屡次否认。
- 朱果儿来自『**小灵天**』，被追杀误落空间裂缝才来魔界为奴。

### 小灵天设定（韩立整理记忆所得）
- 是某界面遗失的小部分空间，长期演化成近似独立的微型界面，类似**广寒界**那般的碎裂界面。
- 大小与广寒界相仿，但灵气程度远不如，甚至比**灵界**还稍逊一筹。
- 内有上亿人族，并有七八个异族与人族混居。
- 不少居民是从人界飞升灵界失败、误入此空间者。
- 在虚空中按一定轨迹缓缓流动，能否找到进入门户几乎全凭运气。

### 对南宫婉的推断
- 据此推断**南宫婉**极有可能滞留小灵天——这解释了韩立在灵界久寻不得其消息的原因。
- 轮回功极罕见，灵界人族无门派拥有；人界**邀月宗**当初机缘巧合得此传承。（暗示传授朱果儿轮回功者可能与南宫婉有关，但韩立未能确定。）
- 计划设法再找从小灵天回到灵界之人核实信息。

## 二、收服与种禁制（第2051章 覆天居士 开端）
- 朱果儿醒后质问被搜魂，韩立坦言并称未损其元神。
- 韩立在其体内**种下禁制**，命其在**幻夜城**期间充当婢女、称韩立为『主人』，许诺若满意则离城时还其自由（朱果儿不抱希望）。

## 三、势力来访
### 赵家
- 第四日，幻夜城地主**赵家之主**遣长老送来银灿灿拜帖（中心金『赵』字，铭黑色魔纹），帖中虚影为眼角带黑鳞、瞳孔金芒的儒雅中年男子。
- 赵家从万奴塔**黄掌柜**处得知韩立这位魔尊到来，欲交好，并表示不会因一名女奴而不满。
- 邀韩立一个月后作贵宾观赵家**祭祖**之礼。韩立以银色火焰焚帖，吩咐回信答应赴约。

### 白家（覆天居士）
- 次日，**白芸馨**（白姓貌美少妇，韩立旧识）上门赔罪——韩立成魔尊之事传到白家。韩立此前将气息压制在**炼虚**等阶，今放出**合体期**气息，白芸馨大惊行礼。
- 白芸馨奉白家**覆天老祖**之命，请韩立过府一叙。
- 白家以八头独角马拉的巨型兽车（顶竖紫色幡旗、墨色白字）相迎，两侧十六名『**虎煞卫**』——骑双头巨狼、披黑甲背插双剑，皆**化神期**修为，每名曾在荒地独自生活百年以上、斩魔兽无数。
- 数个时辰后抵白家堡垒：浑然一体、墙壁光滑如玉、淡青发绿非金非玉的未知材料砌成，层层禁制；百余丈高巨门两侧列数十魔兽雕像，八名黄袍魔族赤手守门。
- 韩立于古朴厅堂候茶。一人步入，自称『**覆天居士**』，称韩立亲来是白家幸事。韩立一见此人，嘴角抽搐、目瞪口呆（暗示覆天居士身份/容貌出人意料）。

## 关键悬念
- 南宫婉是否在小灵天；传授朱果儿轮回功的神秘『师傅』身份。
- 覆天居士的真实身份（韩立目瞪口呆的原因，本章未揭晓）。
