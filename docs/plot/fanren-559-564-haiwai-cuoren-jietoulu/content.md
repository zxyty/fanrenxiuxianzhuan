# 凡人修仙传 第559-561章｜风起海外

> 世界状态须知：本段「姓厉修士／厉师兄」实为【韩立】化名（借已故好友厉飞雨之名，起于第162章）。本段韩立另以幻术幻化陌生容貌、自称『韩某』混入队伍。

## 第559章 诡异错认

- **韩立逃脱**：风希（裂风兽）追至海岛大厅，欲击毁传送阵擒韩立。韩立一道法决混一口精血、大耗真元瞬间催动古传送阵，弹出十余道剑气暂阻青光，乳白光芒将其卷走。风希无传送符、不知通向何处，不敢尾随。妙鹤骑白鹤赶到亦只见黯淡白光。
- **妙鹤之败**：风希暴怒迁怒妙鹤，化青虹猛扑。一月后海域传闻：元婴期妙鹤真人被九级妖兽打得肉身崩溃，仅元婴逃回洞府。
- **蛟龙之乱**：数月后深渊蛟龙一族派大批高阶成员出海，追杀人类结丹期以上修士，持续数年，伤亡堪比兽潮，另两名元婴老怪被数只蛟龙围攻重伤。蛟龙每次动手前必问『有无姓厉的结丹期修士』，答不出即攻击。修士谈蛟色变十余年，却始终找不到这位『姓厉』罪魁——因其实为韩立，已再次幻化容貌。
- **回到内星海**：韩立持大挪移令在白光中现身一陌生山洞，此处与范夫人描述的皇明岛传送阵一致，洞中另设两座传送阵。遇黑衣垂首之人——相貌如活骷髅头、两眼幽蓝、结丹中期。骷髅头看出韩立结丹后期，口气转客气，称『五人一齐行动、皆是接命令的执法使』。
- **暗中布置**：韩立趁机以青色剑气在脚下传送阵划出难辨裂缝使其失效，防妙鹤／裂风兽以传送符追来。随后又一法阵传出凶恶大汉（恶汉），再一阵两人同至——须发洁白灰袍老者（袖绣青色火焰标志）与精瘦干练汉子。
- **五人修为**：韩立结丹后期、骷髅头结丹中期、恶汉与老者结丹初期、精炼汉子（柯宇）筑基后期。

## 第560章 元瑶与青阳门

- **柯宇传令**：精炼汉子自称柯宇，奉令传达任务、带路，出示黑色令牌。骷髅头懒得查验身份（『能传送到此即已放行』），柯宇遂不再检验。
- **任务揭晓**：灰袍老者为**青阳门护法丁姓老者**。任务是助丁前辈捕拿青阳门一位要犯——结丹期、身怀数件青阳门至宝、逃缉多年。骷髅头惊问『可是三阳上人的青阳门』。
- **要犯元瑶**：祖师三阳亲向六道盟主提协助请求。要犯名**元瑶**，原是青阳门少主欲收的侍妾，仗姿色与狐媚之术伙同另一女暗害少主、卷走宝物珍稀丹药，借此结成金丹。三阳仅少主一位传人，恨之入骨。
- **韩立心思**：韩立认出正是虚天殿分别后的元瑶，恍然明白其当年为何短时间结金丹、行事神秘——原是洗劫少主又被青阳门追缉。元瑶就在皇明岛以北荒岛，韩立暗下决心灭此队伍、救下元瑶，事后借机讨要一样重要东西。
- **另处异状**：另一岛石屋内一披头散发修士发现自己脚下传送阵莫名失灵（即韩立所划裂之阵的连带影响）。
- **赴岛监视**：五人飞向北面荒岛，岛上灵气稀薄。丁姓老者发黄芒联络本门监视弟子（筑基中期中年男子，袖绣较小青焰标记），确认妖女未离岛，指认其洞府在某座石山中。

## 第561章 借头颅

- **破拘灵阵**：丁姓老者以黄芒破去石山表面禁制，景色骤变为灵气逼人的苍翠绿山——此乃青阳门密阵『**拘灵阵**』，天下少有可掩灵气之阵，元瑶修为不足只能掩部分灵气。韩立动了把布阵法弄到手以护洞府之念。
- **元瑶现身**：山腰现绿光石门，元瑶（黑衣黑裙、肌肤如雪绝色佳人）自门内飞出，媚术对在场修士无效，对丁姓老者直接无视。她暗察修为：丁与恶汉仅结丹初期，但骷髅头结丹中期、青袍人（韩立）结丹后期，四人联手则凶多吉少。
- **斗法**：元瑶以白色阵盘放绿黄浓雾幻阵退入雾中，老者以白色飞剑化巨大光轮驱散浓雾，恶汉以开山钺猛劈石门。骷髅头忽惊呼『妖女从后山跑了』，化黑光去追，恶汉随后追去。
- **借头颅**：韩立按兵不动，向半信半疑的丁姓老者诡称『确从后山逃了、已被两位道友拦住』，随即开口『就借头颅一用』，喷绿芒绕丁头颅一圈，头颅自行滚落，以青光卷回手中、一滴血不沾。又随手两道剑芒透心击毙吓得欲逃的柯宇与青阳门带路弟子，收走丁的储物袋。韩立暗叹『既在传送阵见到了我，只能算你们运气不好』。
- **后山战局**：骷髅头、恶汉与元瑶缠斗——元瑶黑云裹身、放出上百绿色魔火对轰，边斗边退欲逃，二人经验老到拖住她。韩立赶到，恶汉催促出手，并不满丁道友拖拉。韩立面无表情手一扬，将丁姓老者头颅甩向恶汉。

## 死亡与归属备注

- 本段『姓厉修士／韩立化名』言行均归【韩立】（化名借厉飞雨之名）。
- 死亡：丁姓老者、柯宇、青阳门带路弟子（本段被韩立所杀）；青阳门少主（多年前被元瑶暗害）。
- 骷髅头、恶汉于第561章末仍在与元瑶缠斗。

## 第四卷 风起海外 第562章 赠物

### 战斗反转
- 韩立（此前以换形术伪装混入）口气生硬，并掷出**丁姓老者**的首级，骷髅头与恶汉惊骇之下凑到一处、收回开山钺与怪刃，警惕戒备。
- 远处黑雾中的**元瑶**疑虑丛生，未趁机逃走，红色怪锤盘旋身前静观其变。
- 韩立深意地看了元瑶一眼，两手一挥放出二十多口青色小剑（**青竹蜂云剑**）激射而出。普通结丹修士倾家荡产才能炼一件本命法宝，二人见状知其绝非寻常结丹后期，转身分头逃命。

### 剑光分影术追杀
- 韩立掐诀一点，每口飞剑幻化三道剑光，24道青芒化为**96道剑光**，又吐「分」字分为两股，如青色蜂群分头破空追击。
- 他点出，除非有血色披风、风雷翅这等逆天宝物，否则二人绝无逃脱可能。片刻后剑群追上逃出千余丈的二人，骷髅头与恶汉回身祭宝抵挡，终被乱剑分尸。
- 韩立撤去剑光分影术，令飞剑卷回储物袋而归。

### 相认
- 元瑶率先轻笑唤「韩道友」，黑云散去现出绝色佳人。韩立散去**换形决**恢复真容。
- 元瑶称乱星海再无第二人拥有如此多飞剑，故已认出他；又因韩立修为远胜、若有恶意她绝无生机，遂大方上前道谢攀交情。

### 洞府叙谈
- 元瑶洞府不大却雅净，通道两侧种满奇花异草。她以花茶待客，开门见山提及**虚天鼎**与**补天丹**：传闻星海第一秘宝虚天鼎落于姓韩结丹修士之手，疑其短时间晋入结丹后期是服了补天丹。
- 韩立坦言虚天鼎确在己手，但修为见识浅薄一直无法开启，于他暂如鸡肋；修为精进另有机缘，与补天丹无关。元瑶表面不再多问，神情却不太相信。

### 求购与获赠
- 韩立说明来意：身怀虚天鼎处境凶险，原有的灵液已消耗殆尽，欲高价收购元瑶手中剩余的**万年灵液**以备救命。
- 元瑶沉默片刻，取出一个青色小瓶推给韩立，称瓶中万年灵液一滴未用，无须灵石交换，直接**无偿相赠**，美目清澈。

### 关键设定点
- 韩立现修为：结丹后期；元瑶：结丹初期。
- 万年灵液在外海是韩立的救命之物，此番收齐另半瓶以增保命之资。

## 背景
第四卷《风起海外》第563-564章。韩立（化名行走外海，本章以本名/『韩兄』『韩道友』相称）与结丹修士元瑶于虚天殿共患难、屡受其救后，于洞府中交易、互赠并揭开元瑶身世秘辛。

## 互赠与交易
- **元瑶赠万年灵液**：以报韩立数次救命之恩，并请韩立此后直呼其名，不再以『道友』相称。
- **韩立回赠丹药**：取出两瓶以六级妖丹为主料炼制的精进法力丹药，可令元瑶由结丹初期顶峰顺利突入结丹中期，节省十余年苦修。元瑶愧领。
- **拘灵阵**：韩立见青阳门拘灵阵可隐匿洞府灵气，向元瑶求取布置法。元瑶赠一块乌黑玉简——青阳门阵法典籍，内含拘灵阵及数种威力不小的法阵。

## 元瑶秘辛（往事追述）
- 元瑶与妍丽师姐出身同一修仙小派，资质、功法皆平庸，自知筑基乃至结丹无望，遂相约离派寻修为高深的未婚男修结双修之缘，屡屡碰壁并招来觊觎其姿色之徒，后二人分道。
- 数年后重逢，妍丽已沦为青阳门少主炉鼎。少主又以纳妾为名诱元瑶，称若元瑶应允便放过妍丽；元瑶见少主身份显赫、修为高深、皮囊俊美，遂应允随其回门。
- 归途中妍丽揭穿少主真面目：所谓侍妾实为骗回门中肆意采补的炉鼎，入青阳门后女修身不由己。少主更在途中逼元瑶同房。
- 二人冒死设套，趁其手下不在时暗算杀死青阳门少主。计划虽成，妍丽却被少主临死反噬，肉身受损，只能元神出窍暂栖阴魂法器；普通阴魂法器中魂魄日衰，元瑶用尽方法仅能减缓，妍丽元神灵性大失，已无法主动夺舍。

## 还魂术计划
- **养魂木**：元瑶从青阳门少主处所得玉简记载虚天殿藏有养魂木，故冒险入殿（凶险远超传闻，全赖韩立相救）。妍丽元神经养魂木多年滋养，魂力已复大半，可用『还魂术』重得肉身。
- **还魂术**：令魂魄进入新死肉身重新还阳的逆天秘术，需结丹以上修为、于玄阴之地施展，施法前后动静大、忌人打扰。代价极大——结丹期施术者多跌一境，以元瑶修为十有八九当场碎丹跌回筑基（碎丹后再结丹几无可能）。
- **紧迫性**：妍丽元神已间断迷失，至多一两年彻底失性，届时还魂亦成痴呆；养魂木无法阻挡。元瑶已寻得合适躯体、布好法阵，本拟一两月后施术。她不惜碎丹也要救活妍丽。
- 元瑶因被青阳门通缉，除韩立外不识其他高阶修士，遂请韩立护法数日，任开条件。

## 韩立的决定与鸣魂珠之秘
- 韩立估量：以自身修为，除非遇元婴中期修士否则无碍，普通偷窥者可轻易剿灭，遂应允护法，且不另提条件，只求元瑶解释鸣魂珠蹊跷（韩立心存顾虑尚未炼化此宝）。
- **鸣魂珠/啼魂兽秘辛**：啼魂兽原为青阳门少主之物，仅完成一半祭炼，故控制它的鸣魂珠也有瑕疵，会令炼化者定时头痛难耐，且随祭炼深入愈难压制（元瑶当年为闯虚天殿鬼雾关只炼化小半即寝食难安）。元瑶将啼魂兽后续祭炼之术（绿色玉简）赠予韩立，但认为韩立一人难以完成。

## 转场：玄阴小岛
- 三日后，二人抵达乱星海内海一座荒芜无名小岛。此岛古称『妖兽墓穴』——人类立足乱星海前，附近低阶妖兽大限将至便来此等待消亡；虽妖兽早被剿尽，多年沉积使此岛阴气仍骇人，加之偏离正常岛屿航线而罕有人知。
- 元瑶以法决揭开谷前红光幻象，露出十余丈、阵中套阵、符文深奥的复杂还魂术法阵；阵中横置一具寒玉所制的白玉棺（长约七尺、高约三尺），即为妍丽所备躯体。还魂术对躯体要求极苛（灵根属性、诞生时辰、原主死期、有无冤气孽气等需掐准）。元瑶取出黑木盒置于棺盖，后退数步，准备施法。
