# 凡人修仙传 第2252-2254章

## 第2252章 傀儡大军
- **击毁顶阶傀儡**：泣灵老祖留下的赤红巨型魔晶傀儡能召出似牛似虎的魔兽虚影、被斩后以晶丝自愈。韩立先以青元剑光腰斩，再以亩许金灿灿光手镇压爆碎，最后凭空摄回其核心——一颗表面遍布符文与孔洞、轻飘飘的金色圆球，判断傀儡奥妙全在此物。
- **傀儡大军**：火海被银色火鸟（银焰所化巨鸟，可吞火涨身）吸空后，露出方圆千亩赤红晶石高台，四周凹地深坑中密布近十万具魔晶傀儡。大多为结丹、元婴等阶，化神、炼虚约占十分之一，另有数十头合体初中期的顶阶傀儡。
- **身份揭示**：韩立由此判定泣灵老祖（已于第2227章死亡）是炼制魔晶傀儡的大宗师。
- **收取**：韩立与银月各放出数十枚储物镯，霞光扫过即收走傀儡。数十头合体顶阶傀儡有抵抗摄取之意，被银月以法决压服后收取。傀儡收尽后巨台自行下陷百余丈，露出青色石门——泣灵真正藏宝处。
- **藏宝大殿**：石墩上摆满外界令人疯狂的顶阶宝物，正中三座高台分置三样东西：一柄锈迹斑斑的青色古戈、一个淡黄色葫芦、一条半尺长的漆黑飞舟。

## 第2253章 返回
- **分宝**：青色古戈为‘溶入玄天残片的圣器’，韩立认为更适合银月，赠予她；淡黄色葫芦韩立略检后意外收下；漆黑飞舟即传闻中的‘墨灵圣舟’。
- **墨灵圣舟（魔灵圣舟）**：被催动后疯涨为体长逾千丈、上下十几层、通体乌黑遍布灵纹的擎天巨舟，前后竖立银色巨幡、绘异兽奇禽，各甲板立满魔晶傀儡卫士。具空间属性神通。
- **返回灵界**：两月后巨舟自黑色空间漩涡中现身灵界一处无名山脉，比预想更接近人族。韩立称此行入魔界不虚此行，决定返回人妖交界的圣岛（莫简离、敖啸在此等候）。
- **圣岛会合**：半年后到圣岛主殿，敖啸老祖（银发青年外形）与莫简离正担心韩立。敖啸有意撮合孙女玲儿（即银月）与韩立结为伴侣，以使人妖两族更亲；莫简离提及韩立人界已有感情深厚的双修伴侣。韩立现身后受十余名圣岛长老见礼。
- **隐瞒**：韩立将螟虫之母被灭推到宝花与上古封印身上，对始魔之地隐匿的上古真仙一事只字未提，敖啸、莫简离未生疑。

## 第2254章 三清雷霄符
- **敖啸天劫将至**：莫简离透露敖啸下一次天劫没几年就要降临。敖啸坦言天劫本该千余年前降临，他以秘术拖延至今（为等玲儿进阶合体、兼防魔界入侵），代价是天劫更猛烈，现渡过几率不足半成。
- **银月身份**：玲儿/玲珑仙子即银月，敖啸之孙女，已进阶合体期（据世界状态须知，银月与玲珑仙子为同一人）。
- **度劫之物**：韩立的极山已炼化无法外借；其隐雷根弟子（海大少）非大乘对天劫无用。莫简离提出关键线索——**三清雷霄符**。
- **三清雷霄符**：道家无上秘符、对抵挡大乘天劫有奇效的消耗类秘宝，传说仅仙界真仙能炼制，各界久无人亲见。消息源自云岗族大乘石心老祖（随队入始魔之地、已陨落于上古封印）。此符现在灵族中，且不止一张。
- **持符者灵王**：符在灵族灵王手中；石心老祖曾与符主达成交易准备换取。韩立联想到此前在魔界洗灵池前灵王借白戚之身现身之事。灵王为灵族大乘，气息面貌百万年来始终如一，疑为活了百万年以上的老怪物（一说天劫间隔以十万年计，一说实已换代）。敖啸、莫简离对其极为忌惮，但因灵族繁衍极难、灵王深居简出、出手寥寥，故只能维持现规模。

## 关键转折
- 韩立收获傀儡大军＋泣灵秘藏（含墨灵圣舟），实力与底蕴大涨。
- 引出后续主线：为敖啸度劫寻取三清雷霄符，矛头指向神秘莫测的灵族灵王。

## 一、灵族之行的谋划（第2255章）
- 韩立向莫简离追问灵王与**石心老祖**的交易：石心老祖以**三清雷霄符**这等秘宝为报酬，受灵王之托去做一件极危险之事；莫简离只知大概，具体内容不明。
- **莫简离**提议：在魔界损了元气后本就要去灵族一趟，若得韩立（气息比入魔界前更深不可测）联手，灵王多半不敢生歹意，可老实交易讨回数张三清雷霄符。莫简离坦言自己渡下次天劫亦无把握，极想得到此符，对韩立未来天劫保命也有大用。
- **敖啸老祖**同样在魔界损耗、不久要度劫，莫简离力劝其勿涉险。
- 关键数据：得一张三清雷霄符，敖啸渡劫几率约增**一成半**；敖啸刻意拖延天劫**最多十年**。

## 二、韩立赠丹（第2255章）
- 韩立赠莫简离一瓶**茴焚丹**（大补元气损耗，服后闭关一年即可大致恢复）。
- 又看出敖啸除元气损耗外**神魂亦损**（昔被封印中遭厉害怪虫以神念秘术偷袭）；韩立赠数瓶丹药治神魂亏损。
- 约定：一年后莫简离元气恢复，二人即赴灵族讨符；敖啸留族中休养备劫。两族现仅三名大乘，任一出意外都是重大损失。
- 韩立自言当年大道未成时多蒙二人照拂，此为略偿旧恩。

## 三、圣岛见弟子与昊阴之石（第2256-2257章）
- 韩立于圣岛洞府见海大少、器灵子、白果儿、朱果儿等门人。
- **白果儿**（冰髓之体）短时修至**化神后期**；**器灵子**近化神后期大成瓶颈。韩立令二人赴**蛮荒世界**历练（不经风雨难成长，且唯蛮荒可寻自身机缘），各赐一储物镯保命之宝，二人欲尽快经天渊城出发。
- **海大少**（化神初期）请同往被拒：修为太低难发挥宝物，且隐雷根之事虽暂压，仍恐被老怪当人形抵劫秘宝掳去；命其暂代器灵子门下弟子之权以磨炼浮躁心性。
- **朱果儿**修为有长进，韩立约其次日午时单独问话。
- 冰凤师姑（冰凤之体、炼虚高阶）外出蛮荒久无消息，神魂牌无事，韩立疑其被困。
- 单独交代**白果儿**：寻找**昊阴之石**——常处冰川下数万丈、为万丈寒气掩饰，神念难觉，唯寒魄神通者可凭体内寒魄遥感；需求量大，令其历练时顺带搜集，事成重赏。白果儿暗下决心全力寻找。

## 四、元合五极山的拼图（第2257章）
- 韩立得元合五极山（后天玄天之宝）炼制法后，机缘所至已炼成：**元磁极山、太乙青光极山、阴阳大五行极山**。
- 若白果儿寻得足量昊阴之石，可炼成**昊阴寒魄极山**，即凑四座；仅余**北极元光极山**（线索在人界，欲借破界至宝化一缕神念下界取，风险大需谋划）。
- 五山齐成、炼成元合五极山，则数次大天劫无忧，乃至飞升之劫亦有一线机会。

## 五、蛮荒旧人：六翼与冰凤（第2256-2257章）
- 天元大陆某水域上空，**六翼**（面带金银花纹、隐约几分韩立面容的白袍青年，背生六枚近透明蝉翼、本体雪白六翼蜈蚣）以闪现穿插+晶寒之力冰封并吞噬一群化神级火狮，吸其烈阳之力与自身冰属性真元阴阳交汇炼化。
- **冰凤**（晶莹银袍女子、背现白凤虚影、不食血食、炼虚等阶近合体一步）被六翼裹挟同行。
- 六翼身世与动机：原为韩立灵宠，因韩立以**血祭之法**种下主仆印记；今已修成大乘并化人身，得知原主人韩立亦进阶大乘，遂势在必行赴**雷霆大陆**，欲借**素阴一族**（能沟通阴辰之星、凝练素阴星气）之素阴星气，以逆灵之法修成**真灵之身**，再斩杀韩立以彻底摆脱主仆印记。
- 六翼并欲强逼同属至阴生灵、一雌一雄的冰凤日后做双修伴侣；冰凤恼怒却自知非其敌手，被迫随行。二人欲借余族与雷霆大陆某大族的跨大陆超级传送法阵前往。
