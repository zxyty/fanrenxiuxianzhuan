# 第2397-2399章 祸水东移 / 失踪 / 明尊之议

第十一卷《真仙降世》。

## 一、祸水东移（第2397章）
- 六翼与冰凤是被真仙**马良**（绝世凶魔）从**血天大陆**一路追杀至其他大陆的逃亡者。起因疑为二人在血天大陆联合血道宗门埋伏马良，激其大怒。
- 六翼真元大损、危在旦夕。冰凤自陈本体为**五彩天凤嫡系后裔**，喷出含拇指大五彩圆珠的寒气，召出冰凤虚影，将自身**本源之力**直接灌注给六翼。因二人同为奇寒之体、本源神通近似方可施为。
- 六翼借此短时恢复，提出久蓄的『**祸水东移**』之计：赶往最近传送阵，经传送速往**角蚩族**领地——角蚩族是**雷鸣大陆**数一数二的超级势力，大乘众多，可望缠住马良。
- 一个月后，角蚩族主城之一**盖灵城**被滚滚血河淹没。瘦弱青年（马良）悬浮血河上空，旁有上百具角蚩族高阶残尸，其中一具天灵盖被打开、元婴被掏空（即镇城大乘**天象**）。
- 一尊千丈绿色魔像（头生独角、体覆绿魔眼、托金色宝塔）现身质问，被马良瞬间击杀、神魂被擒入光球后打入血河。马良出示六翼与冰凤的画像追问其下落，扬言三日内交不出人便逐城血祭整个角蚩族。
- 同一时间，化身两名普通角蚩族人的六翼、冰凤正飞速逃往另一座角蚩主城。

## 二、失踪（第2398章）
- 二人对话揭明：他们用**完整寄附神魂**之术把神魂寄附在两名角蚩族修炼者身上，暂时摆脱了马良监控。此术来自六翼当年在蛮荒异族洞府所得的**金阙玉书**一页所载仙家秘术，须仙人之体方能完整施展，六翼只能勉强修炼其中寄附一项；施法极危险（七八分把握），施展后本体陷入沉睡、神通封印大半，且只能支撑半日。能遮蔽追踪疑因其本身亦属仙界秘术。
- 马良识破寄附手段，但因嗜血祭、自大，乐得借口直接血祭。
- 冰凤提议实在不行可去找**韩立**（曾斩杀螟虫之母）求解神魂印记，六翼大怒拒绝（『宁愿陨落』）。
- 半月后，马良追至另一角蚩主城，遭早已埋伏的**六名角蚩族大乘老祖**联手并催动厉害禁制围攻。马良未用解锁法则之链的神秘符箓，仅以自身神通（受限大半）配合数件仙界宝物对抗，旗鼓相当。激战大半日，结果**四名大乘陨落、两名重伤遁逃**；战末角蚩族自爆数件玄天残宝击伤马良。
- 马良大怒，连续血祭三座角蚩大城。角蚩族被逼借**上古契约**召唤出三头供奉的上古真灵『**蹄龙**』『**泰雀**』『**阳鹿**』再战数日，波及方圆百万里。
- 战后三头上古真灵与马良一同凭空消失，无法联系，角蚩族高层面面相觑。马良在血天的所为传回雷鸣大陆，各族谈『魔』色变。
- 六翼、冰凤闻马良失踪大喜，月余后借他族跨大陆传送阵返回**风元大陆**。
- 同时，韩立带**南宫婉**等人传送出现于风元大陆某巨阵大殿；守卫甲士奉命接待，引韩立去见等候多时的**总执事**。

## 三、明尊之议（第2399章）
- 总执事为一名赤红头发老者——**明尊**。他坦言此前曾将韩立消息泄露给已陨落的**碧影**，韩立不以为意。
- 明尊以玉简告知马良在雷鸣大陆的事：斩杀四名角蚩大乘、与三头上古真灵大战、随后离奇失踪。本盟之人查到失踪处有微弱空间波动与破开虚空痕迹，无法继续追踪，判断马良借空间之力暂避而非陨落。
- 明尊出示画像，韩立认出**冰凤**（与己有私人渊源、非人族），并疑似认出另一男子为故人——本体六翼巨大蜈蚣（**六翼蜈蚣**）。明尊称二人疑似激怒马良并将其引向角蚩族，且很可能回风元找韩立。
- 明尊先祖为仙人嫡系后人，凭家传仙界功法记载断定马良**九成九是真仙**。
- 明尊提议：风元大陆顶尖强者联手、本盟另召强大真灵参战，以绝对力量逼退或镇压马良，否则散开必被逐一击破（援引血天血道宗门联手失败之例为戒）。已有大半道友表示愿在马良踏足风元时参战。
- 韩立暂时答应，并声明若届时人手不足便立即离场；明尊交予一张淡银色**传讯符**，约定有马良消息即传讯聚集。

## 备注
- 南宫婉在本卷已复生，以韩立道侣（『婉儿』『夫君』）身份随行、活跃行事；本片段中她明确在世（世界状态须知所记其于第745章之死为更早事件）。

## 第2400章 法则之链

- 韩立辞别石屋中的明道友，乘黑色巨舟离去；舟上南宫婉关切询问，韩立称回船舱再细说。
- **神秘空间内的博弈**：一名黑袍青年被紫金色法则之链缠身，三头真灵所化巨兽（鹿首熊身的「阳鹿」、灰色巨雀、双首牛兽「蹄龙」）各咬一根锁链另一端。三兽由角蚩族借玄天之宝之力以上古契约供奉。
- 此前有人（鹿首兽称『金龙王大人』所授）以半知半解之法激发法则之链反噬黑袍青年，黑袍青年反应极快，以空间秘宝将三兽一同扯入共担反噬。
- 黑袍青年自恃仙人之身＋仙家秘宝护身，反噬于他只需多休养，而下等真灵会飞灰湮灭。他以「收为灵仆带去真仙界」「半日内赐真魂丹成就真仙之魂、并斩断与角蚩族及此界一切联系」为饵分化三兽。
- 反噬加速，三兽心慌；阳鹿借故靠近反被警觉。阳鹿实已暗中投靠黑袍青年并对法则之链动了手脚——他一震挣脱双首牛兽并以血色巨印将其当场击杀。
- 真正的黑袍青年现身（原被缚者只是分身/虚影），逼阳鹿杀掉灰色巨雀以换取仅有的一枚真魂丹与唯一的归仙界名额。

## 第2401章 小瓶再现

- 阳鹿与灰色巨雀生死相搏，黑袍青年携血色巨印遁走。
- 黑袍青年现身雷鸣大陆，托起一只墨绿色小瓶，确认无人后遁往一座小山。
- 倒转小瓶放出重伤的阳鹿——果然是阳鹿胜出活下。阳鹿臣服参拜。
- 黑袍青年取出**仙灵宝鉴（金色书卷）**，命阳鹿吐出部分主魂订立**灵仆契约**（可不受界面之力影响进入仙界；契约非永久、危及性命之命令可付代价不执行，留有解约条件）。
- 契约订立后，黑袍青年赐下鸡蛋大小、金芒檀香的**真魂丹**。阳鹿听劝选择返仙界前再服（提前服会招此界法则排斥、削弱实力），以紫红木盒收好。
- 黑袍青年遁入墨绿小瓶疗伤；阳鹿守于瓶外山腹。
- **瓶中空间**：黑袍青年自述此次下界先被角蚩族法宝自爆伤元气、又受法则之力折腾，本需数年疗伤；幸得祖师爷除符箓外还赐下此**绿瓶仿制之品**（据说不足正品万分之一玄妙），疗伤时间可缩短十倍以上。他施法令瓶中遍生灵花异草虚影，再以小瓶虚影化巨型瓶口降下绿色灵雨，催灵草飞速凝实近实物，引药香青气入体疗养。

## 第2402章 人族再聚

- 半年后黑色巨舟抵天渊城，人妖两族沸腾。韩立召见合体之辈，留下愿留的小灵天人族修士，等候人族大乘莫简离。
- 韩立携南宫婉迎接莫简离。韩立介绍南宫婉为其在人界寻回的双修伴侣（曾被困小灵天），二人以平辈相交。
- 莫简离赠南宫婉一套**玄女碎阴针**（共108根、表面晶光透寒的黑色细针，顶阶宝物）。
- **好消息**：韩立此行除救回南宫婉外，还救出被困血天大陆多年的人族「冰魄仙子」（离族时已合体、亦为飞升仙人，敖啸曾提及），其已进阶大乘，但因境界未稳须在血天大陆再待数年。人族由此或同时拥有三名以上大乘，莫简离欲借势扩张领地。
- **坏消息**：韩立经赫连商盟得知血天大陆因一名疑似降临的真仙引发血祭而人心惶惶；韩立已暂答应明尊的联手提议，若有机会则出手镇压真仙，以免人族被真仙找上而独木难支。莫简离震惊，需再思量。
- 南宫婉亲沏小灵天灵茶待客。莫简离思忖后表示尚有一事不明，欲再问韩立（章末悬念）。
