# 第787章 吞魔啖鬼

第五卷 名震一方。地点延续韩立与阴罗宗黑袍青年的对决战场（乱星海一带）。

## 战斗经过

### 1. 剑丝失效，辟邪神雷受阻
- 黑袍青年祭出**鬼罗幡**（阴罗宗镇宗之宝），放出无数白森森骷髅鬼头扑向韩立。
- 韩立的剑丝（青竹蜂云剑金丝）掠过后，鬼头表面虽现裂痕却绿光一闪弥合如初——这些鬼头已非血肉之躯，被炼成无形阴鬼之身，不受刀剑之害。
- 韩立改激发飞剑中的**辟邪神雷**，化金网罩下。为首弯角鬼头厉啸，众鬼头喷碧绿鬼火硬顶住金网下坠之势。普通鬼火本应被辟邪神雷瞬灭，但鬼罗幡魔头之鬼火竟能勉强抵住电弧，令韩立意外。
- 黑袍青年见状大松一口气：此幡自上次大战后未及吸取阴魂修复、魔头元气未满，原本他对辟邪神雷颇为忐忑。担心尽去后，他催动法决将幡面孔洞从数尺涨至丈许，放出更多骷髅头围攻。

### 2. 啼魂兽出场吞魔
- 韩立拍灵兽袋放出**啼魂兽**（通体乌黑小猴）。鬼头一见此兽猛然震慑、盘旋不前、鬼啸大减，露出畏缩之色。
- 黑袍青年情急自断五指化五股血雾喂给为首弯角鬼头，鬼头吸入后鬼火大涨，威逼众鬼头再向啼魂兽喷鬼火。
- 啼魂兽锤胸，体形涨至数丈，从鼻中喷出**黄霞**：鬼火接触即被卷入，霞光冲入鬼头群，所过之处骷髅头毫无反抗被吸入。牛角主魔头凝绿烟化形为身高两丈、头生弯角的披发恶鬼，放黝黑雷火攻黄霞，雷火入霞光无声无息消失；恶鬼欲逃，被黄霞罩落吞入。
- 啼魂兽几次吞吐，空中骷髅头被一扫而空。黑袍青年又惊又怒——他早知韩立有妖异巨猿能吞噬炼尸尸气（阴罗宗铜甲尸大半葬于此猿口中），却没想到此兽竟能直接**吞魔啖鬼**，连主魔头都无力抵抗。此神通对阴罗宗的威胁比金雷竹更大，他悔意大生（若早知绝不会自告奋勇独身前来）。

### 3. 剑阵碎尸，金网困元婴
- 黑袍青年掷出七八件宝物欲突围，全被剑阵金丝切成碎屑。
- 他人幡合一，化阴森绿雾朝天空破空逃遁，欲借鬼罗幡护体抵挡剑阵。
- 韩立口喷金色电球，于高空爆裂成巨大金网（辟邪神雷），罩住方圆数十丈。绿雾被剑阵金丝困在边缘，护体雾气被削去大半，成困兽之斗。
- 青年厉喝以**封魂咒解封之法**要挟（事关解救韩立双修伴侣），韩立答"自会从你元神中知道想知道的一切"，欲用**搜魂术**。
- 护身阴雾被削尽，青年身躯被金丝碎尸万段，仅一团绿光裹**元婴**爆射而出，却撞入早布下的辟邪神雷金网，左冲右突被电弧击得惨叫。

### 4. 自爆要挟与银月擒拿
- 青年元婴威胁"若用搜魂术便自保（自爆）元婴"，韩立闻"自爆元婴"皱眉，停下金网收缩。
- 元婴提条件：放其元婴离去，便告知解除封魂咒之法以解救双修伴侣，否则宁死不交。
- 韩立目光闪烁权衡利弊之际，元婴上空紫网撒下罩住元婴并收紧；**银月**身影无声浮现于元婴后方，喷粉红香雾、十指连弹射出十余道银光。

## 关键看点
- 啼魂兽"吞魔啖鬼"神通首次明确展现，可吞噬阴罗幡元阴魔头乃至主魔头。
- 韩立"双修伴侣"被封魂咒所困，解法掌握在黑袍青年手中——成为韩立未直接搜魂、犹豫的关键，亦为后续悬念。
- 银月关键时刻出手擒住元婴，避免元婴自爆毁掉情报。

（注：据世界状态须知，「黑袍青年」于本章/第787章死亡（肉身被碎尸，元婴被擒）。）

## 背景
第五卷《名震一方》第788-789章。承接第787章韩立与偷袭落云宗的魔修斗法（魔修以临时传送阵将战场带至数百里外小岛），本段为收尾及后续规划。

## 第788章 路遇
- **擒杀魔修**：黑袍青年元婴被银月香雾笼罩、银光（针）扎中后神通尽失、面无人色。银月透露：她在原地只留一道幻影，真身早潜至一旁，趁对方只顾逃命时出手。韩立传音指挥、出手时机极佳。
- 韩立收回笼罩的金网（化金球入手）、大庚剑阵还原为三十六口飞剑缩小入袖。命银月带元婴入树林，搜寻解除封魂咒之法（林中冒出大片粉红香雾）。
- **落云宗援兵**：数百里外，六道惊天长虹（六名元婴初期修士）向小岛飞来。为首者为落云宗银发老者（程道友/程师兄），靠追踪标记感应韩立位置。同行有酒糟鼻胖老者（陈某）、貌若童子的火龙童子、灰袍尤姓老者、相貌堂堂的威严大汉等。
- 尤姓老者携至宝**迷仙钟**（黄濛濛小钟）出山相助，言当年程兄曾对其有大恩；众人因魔修敢深入**溪国**伤人而出手。
- 韩立青虹遁光赶到，告知已独力灭杀魔修、化掉元婴，并点明对方并非阴罗宗宗主，而是宗内一位长老。众人骇然——能击伤落云宗元婴初期修士又扬长而去的魔修竟被韩立一人斩杀。
- 韩立掷出**阴罗宗长老令牌**（乌黑、密布符箓咒文、中刻古文‘罗’、颇具威力的古宝）作证，众人再无怀疑，皆生结交之意。韩立辞别众人，与银发老者同回云蒙山落云宗。

## 第789章 极西之地
- **解咒难题**：议事大殿偏厅，韩立坦言以搜魂术得知——解除封魂咒的法决只有阴罗宗宗主与大长老掌握（牵涉宗内一种厉害秘术，他人无权修炼），所杀长老只会下咒不会解咒。银发老者忧及‘南宫妹子’。
- **替代之法**：韩立从魔修神识得知数种借外物强行破印之法，所需皆稀世之物（如万年玄金鸟蛋壳、菩提花果实，疑已绝迹）。其中**上古火妖兽内丹**他知下落——**坠魔谷**（天南第一凶地）中恰有一只**上古火蟾兽**，妖丹条件正合。借外物破印约一半成功率，不成也可大减毒咒效力，为赴晋国夺取解咒法决争取时间。韩立婉拒宗门人多相助（‘此事不是人多能办到的’）。
- **南宫婉冰壁**：密室冰壁中冰封着秀美无邪的女童（南宫婉），曾借生死一线机会修炼**轮回功**中最危险的‘**姹女天月决**’（置死地而后生之秘术）。韩立隔冰抚其面孔，自语担忧并立志取得解咒之法。
- **极西之行计划**（与银月对话于子母峰洞府）：
  - 入坠魔谷前先赴**极西之地**，取**大衍决**最后两层；并往**千竹教**（以傀儡术起家、独占极西之地）求**结丹期傀儡**炼制法。此前换得的媲美元婴修士的上古傀儡之法，部分辅料过于稀少、主料万年铁木需大量绿液催熟，赶不及坠魔谷之行；多得结丹期傀儡可在入谷前提升实力并充当探路者。
  - **路途艰险**：极西之地虽紧邻天南，却需经正道盟诸国并徒步穿越绵延万里、上空常年刮飓风的大沙漠（无法御器飞遁），来回顺利也需约一年；该地仅天南两中等国大小、资源贫乏，故正魔两道无意，千竹教独占。
  - **离去前杂务**（交银月协理）：将**玄天仙藤**交银月按时以绿液灌注、看能否救活（救不活只作炼器材料）；韩立要种下**第二元婴**（用**至木灵婴**）；并以阴罗宗长老储物袋中所得**祭炼魔尸法决**处置困于云梦山山腹的**尸魈**祸患。
  - **联手未定**：紫灵仙子与南陇侯同时邀其入坠魔谷，按苍坤上人路线图走、不贪心，韩立认为入谷不致迷失，与谁联手待后再说。
- 章末银月欲向韩立询问一桩搁在心里许久之事（悬念）。

## 关键决策 / 后续线索
1. 数年后坠魔谷之行势在必行（取火蟾妖丹）。
2. 入谷前先走极西之地（大衍决+千竹教傀儡法），来回约一年。
3. 后续需赴晋国夺取阴罗宗解咒法决。
4. 待办：催养玄天仙藤、种第二元婴、祭炼尸魈。

## 第790章 侵占

### 银月问器灵
银月问韩立为何从未让她以器灵形态加持飞剑威力、也未以器灵身份被唤出对敌。韩立答：召唤器灵化形虽能令法宝威力大涨，但消耗器灵精元，催使过频甚至会令器灵自行溃散；而以妖狐之体出现的银月对他帮助极大，不到万不得已不会将她化为器灵应敌。

### 第二元婴
- 韩立首枚元婴苦修二百余年练成，本章经数年培炼终于真正凝固，元婴出窍练飞行、施展瞬移神通后归位。
- 确认元婴凝固、修炼第二元婴无后顾之忧后，韩立取出「玄牡化婴」玉简参悟神通。
- 取出封印的至木灵婴金球（此前以玄阴经中的离魂术、配合离针法器持续削弱其本源神识）。如今趁其神识极弱，凭不下于元婴后期的强大神识强行抹去其最后灵识，注入自身部分神识，以秘法收入体内慢慢同化。
- 同化过程因人而异，约三五年至十余年；韩立不指望第二元婴在坠魔谷之行中派上用场。

## 第791章 天绝魔尸

- 韩立携十几枚可变大小的银色巨钉法器，前往抓雪云狐的沼泽山腹石屋，尸魈安然躺在石台上。
- 放出啼魂兽戒备；发现其背后血色鬼影图比前清晰、鬼影微微凸起，似将再有奇异变化。
- 炼尸法来源：韩立搜完黑袍青年神识后，从其头颅发髻取得玉签，内载十余种炼尸法（阴罗宗自旁门明抢暗夺，黑袍青年复制并有独到改进，秘不示人）。其中顶阶炼尸「天绝魔尸」以通灵厉害行尸抹去神识炼制，威力视行尸种类与原修为而定；尸魈这等天地灵尸为最佳选择之一。
- 抹除主元神：以青色霞光逼落玉盒金符，盒中涌出尸魈主元神所化绿雾鬼脸（与银月妖狐面容一致，证实银月妖狐化形本由尸魈控制变出）。鬼脸欲附身韩立，被啼魂兽黄霞（专克元神妖魂）卷扯；鬼脸自断逃回尸魈躯体，恰被缠金弧的银色巨钉钉中要害击出，逼出尸魈体内潜藏的一大一小两道分神，终被啼魂兽尽数吞噬。
- 银链古宝：束缚尸魈的银链刻满肉眼难辨的细小符文，掺庚精的青竹蜂云剑只能斩出米粒豁口；韩立对同一处连斩多次方将其斩断，收起拟日后修复试威力。

## 第792章 远行之前

- 检查尸魈躯体：除缺一臂外用以炼尸近乎完美，身躯强横、体内孕育尸火尸毒（修士沾染即毙命）。
- 以三色噬金虫为残臂凝聚补出一只三色手臂（除颜色、无绿毛外与原臂相同），并在石室布下玄奥法阵；此役耗时两月余，韩立元气大亏。
- 灵虫灵草近况：
  - 六翼霜蚣（柳玉所交记名弟子之灵虫）已互噬进阶一次，寒气大胜从前，产下24粒蜈蚣卵；韩立通知记名弟子来取走灵虫。韩立融其寒气修炼的紫罗天火威力亦有增。
  - 噬金虫聚成金球倒吊灵木，再进化一次即成熟体，即便有霓裳草催化仍历时漫长。
  - 玄天仙藤根须经绿液无效后，改用万年灵液加醇液浇灌转为碧绿，但无发芽生叶、生死难辨；暂搁置，待极西之地归来再说。
- 远行：韩立向程师兄（银发老者）告辞、封闭洞府带走重要之物，带银月乘御风车赴极西之地（料无高阶修士、无大危险，惟耗时久），须三年内赶回以免错过进入坠魔谷的最佳时机。
- 慕沛灵以侍妾身份、自言修炼遇瓶颈，请求随行，韩立应允并允诺路上指点其修炼（盼其早入结丹）。
- 对外口风：落云宗放出消息，称韩长老与魔修对决伤了元气、闭关数载养伤、概不会客；因其所杀为晋国阴罗宗元婴中期顶阶长老，众宗门不疑。落云宗声名一日千里，隐隐成溪国第一宗门，韩立声望无二。
