# 第511-516章 风起海外：奇渊岛黑石城与雾中岛立府

## 背景
韩立此阶段以「曲」姓散修身份现身乱星海奇渊岛，外貌为病怏怏的中年人（变化之术伪装），众人称其曲兄/曲道友/曲某/曲前辈。本段无具体被冒名对象，仅为化名伪装。

## 第511章 海渊妖兽
- 由许云引领，韩立与锦衣大汉（易兄）等新到修士进入黑石城**明珠轩**。主持人**明夫人**修媚术，验过众人定力后发牌引入拍卖区。
- 明珠轩分鉴宝厅与竞宝厅，专收购修士海外收获换灵石或参与拍卖；城中类似场所数处，每个背后有一位元婴期修士撑腰。
- **奇渊岛内情**：是乱星海唯一由众多势力联手控制的妖兽岛，联手后连星宫也不敢轻易得罪，**四大商盟**无法插足。岛名「奇渊」源于附近一座深不可测的**妖兽海渊**，常出七级乃至传闻八级以上高阶妖兽；此岛真实情况严禁外泄内星海。
- 竞宝厅拍卖蓝光石（水属性法器材料）、土属性法器等，无结丹期高阶修士在场。

## 第512章 炉鼎
- 许云解释奇渊岛与其他妖兽岛之别：发现灵石矿与原料产地后可自给自足，势力正将人手从内星海转移，凡人与炼气期修士拖家带口聚居，黑石城估计数万年后或成第二个内星海。
- 拍卖品：**妖兽幼卵**（大部分幼卵无法通过传送阵，故在奇渊岛便宜）。**紫云鹰**幼卵——成年为四级土属性妖兽，钢羽坚硬、飞速极快、利爪可破筑基护罩，以三千三灵石成交。
- 另拍卖**炉鼎**（年轻女修，需买家筑基中期以上），此处公开拍卖女修炉鼎令韩立等吃惊。

## 第513章 寸金阁
- 炉鼎因修为太浅流拍。韩立厌倦拍卖，借口离队，于一条街道在**寸金阁**前停步独自脱身；许云与易兄被韩立眼中寒芒所慑，疤脸修士传音警告易兄此人「行事高深莫测、修为远超你我」，勿牵扯议论。
- 寸金阁为城内最大原料商铺，号称只要说得出名字就拿得出。韩立略放结丹期修为，被引入贵宾室，见管事**离寻**（结丹初期）。离寻豪爽允诺相助，并很快查出韩立今日刚到黑石城的来历。

## 第514章 伴妖草
- 离寻召炼气期儒生**古玉**（精研上古丹方灵药）。古玉前两年得上古典籍、复制入玉简，记有韩立所寻的**千叶露**。
- 韩立查证后揭示：**千叶露其实是「伴妖草的汁液」，且伴妖草必须生长在八级妖兽身畔才有效**，棘手非常。
- 韩立以一瓶提升修为丹药＋一张符宝（黄色小剑符箓），从古玉手中换下整枚玉简（实因玉简另有要图，见516章）。
- 韩立又从寸金阁购得一支**玛瑙兽独角（玛瑙角）**，半尺长、尖锥螺旋状、浮蓝光，以五千进价付中阶灵石。
- 离寻试图以「介绍同道一齐出海」拉拢韩立，被婉拒。韩立走后，屋内山水画中飘出一**青光罩面的白衣人**：二人合谋拉拢，白衣人称交谈中莫名觉得被韩立监视，怀疑其修为远不止结丹初期或有特殊功法/宝物。

## 第515章 雾中岛
- 韩立在杂货铺购得海域图与妖兽情报：奇渊岛危险区甚多，海渊全速南飞半月可达、纵横千万里、深不可测。
- 韩立决意远离海渊另立洞府，向北飞十余日（北面修士稀少），再转西寻带小型灵脉的荒岛。
- 半月后选定一座约五六十里的**雾中岛**：身处由附近激流漩涡涌出的海雾中，雾只向四周扩散、岛上空无雾。其因岛底无数空洞栖息成千上万**吞海鱼**（一级下阶群居妖兽，无鳞、银色骨刺、爱浮海吞吐海水喷白雾）所致。韩立原是追逐一只四级妖兽**银翼鸟**误入雾海才发现此岛。
- 该岛灵脉中上等、却无人入住，环境隐秘。韩立在矮微山脉底部开凿洞府，布阵造幻象（山脉化作树林），又炼十几套简陋阵盘埋于全岛涌雾，将整岛彻底遮蔽。

## 第516章 幻焰蛾
- 韩立细读古玉玉简：内有数幅无文字说明的古怪图画，其一为一团五色光中的拳头大圆珠四周浮十余颗小珠——大珠正是从**虚天鼎**飞出的**补天丹**；他正为此画不惜多赠符宝换玉简。
- 韩立取出白玉匣中**五色小圆珠**（炼化**极炫**骸骨提炼而来；极炫为玄骨老魔逆徒，拥有血玉蜘蛛，疑其当年也入虚天鼎服过补天丹）。韩立推测这些圆珠或才是真正的补天丹（药性未炼化需重凝，类比黑煞教以人身为炉的「血凝五行丹」），不敢贸然服用。
- 韩立的虫室已扩为五间：两间分养两批**噬金虫**（数万只吞铁火蚁后又开始进化繁殖、互相吞噬；另近千精选虫子），其余分置**血玉蜘蛛**、终日酣睡的**啼魂兽**（韩立持鸣魂珠仍不敢轻易炼化）、以及虚天殿所收的**金丝蚕**。
- **金丝蚕**为奇虫榜尾的二次进化妖兽，足够时间可进化为奇虫排名第十七、天生具幻化能力的**幻焰蛾**；但金丝蚕被万天明抛弃时已元气大损、濒死。韩立遂以两粒五色圆珠分喂一对最壮的雌雄金丝蚕试药。
- 韩立将**九曲灵参**本体移植药园中央、以阵旗围困（凝结元婴的重要依仗），其白兔化身自投回本体；暂未用神秘瓶绿液测试灵参，而将所产绿液全用于催熟三大神木之一的**养魂木**根部。
- 末了韩立取出玄骨老魔一截在修罗圣火下安然无恙的肋骨，起了研究兴趣。
