# 第943-945章 敲诈·寒髓·回阳水与太阳精火

## 第943章 敲诈
- 地点：大晋晋京城内，临近大拍卖会。韩立以化名「厉师兄」、海外散修身份行走（实为借已故好友厉飞雨之名，起于第162章）。
- 元婴初期散修**肖老儿**（小老头/敲诈小老头）凭一只**变异蠕虫兽**（状如放大碧绿蜗牛，能看破幻术）看破韩立先前与阴罗宗对峙时把**竹筒幻化成长刀**的障眼法，以此要挟，索要三四件古宝或大笔灵石「破财消灾」，并称已派弟子尾随阴罗宗与天澜圣女，一旦出事便揭发。
- 肖老儿祭出**木奎盾**（黄色木盾古宝，防御惊人）护身。
- 韩立佯装妥协，抛出伪装成「云宵珠」的金银光团——实为所剩不多的**三色噬金虫**。肖老儿注入法力后光球爆裂，噬金虫扑咬其面门；韩立以神识冷哼重创其神识，再以**紫焰缠绕的青竹蜂云剑**（数尺长金剑）一斩破其木奎盾、剖其躯体，以**辟邪神雷**（金弧电网）连同紫冰彻底灭其元婴。**肖老儿于本章死亡**。韩立夺其储物袋，火球焚尸后遁走。
- 一刻钟后，**林银屏、葛天豪**等及肖老儿的结丹期弟子赶到，发现血迹，断定肖老儿被「姓厉的家伙」灭口。

## 第944章 寒髓
- 追兵议事：韩立（『姓厉家伙』）此前已**斩杀阴罗宗四长老、夺走镇宗之宝鬼罗幡**，神通不下元婴后期。
- **林银屏**身为天澜草原圣女（突兀圣女/天澜圣女），为追杀韩立滞留大晋数年；与**阴罗宗葛天豪**达成交易：圣殿原条件加成、宝物各取，阴罗宗倾力围捕。葛天豪令传音召集晋京所有阴罗宗长老弟子，引韩立入法阵合围，并知会闭关炼宝的**房宗主**（阴罗宗宗主）。
- 另一边，韩立在二十余里外街道清点肖老儿储物袋，发现一**六七寸碧绿玉瓶**，触之奇寒、神识难渗，内有银色液体。
- 回道观布阵后开瓶：寒光暴射、满室结冰，其极寒近乎乾蓝冰焰；倒出一滴化为米粒大银珠。**大衍神君**激动指认此为**万年冰玉（万年玄玉）中孕育的寒髓**，乃北夜小极宫世代相传的镇山之宝，瓶底雪花标记即北冥岛标记。

## 第945章 回阳水与太阳精火
- 大衍神君详解寒髓：
  - 对人类修士为炼器炼丹顶阶辅料；对妖族则是渡**化形雷劫**的绝佳保命之物（炼化后可大减雷电伤害）。
  - 北冥冰岛（北冥岛）位于天元阴脉寒眼，方圆万里酷寒，北夜小极宫传上古大神通、布有上古禁断大阵；历代高阶妖兽（万妖谷老妖牵头）来攻抢夺寒髓却从未攻克。
  - **核心秘闻**：寒髓真正主用是炼制上古秘药**回阳真水（回阳水）**的主原料，另须万丈熔岩下的**太阳精火**淬炼以成阴阳调剂。
  - 回阳水功效：白骨生肉、修补躯体，传言起死回生；公认延寿四分之一——可助结丹后期进阶元婴、元婴后期进阶化神。
- 大衍神君当年以搜魂术从北夜小极宫副宫主处得回阳水配方，此章以**淡黄玉简**赠韩立留存。太阳精火非本界可捕，故配方虽得，炼成无望。
- 韩立此行拍卖会本为给**傀儡**凑材料；大衍神君允诺危急时破例施**大衍诀『惊神刺』**神通相助。
- 大衍神君提点：韩立的**雪晶珠（雪晶球）**亦万年玄玉所炼，掺数滴寒髓可大增灵性威力。
- 韩立另起意：试将寒髓以**紫罗极火**吞炼融合——以紫焰光幕封瓶、引出寒气包裹成火球吞入腹中，于丹田内由元婴掐诀，令紫焰与银色寒气交织融合，本体入定。

## 关键转折
- 身份暴露：韩立的幻术化形被变异灵兽看破，「姓厉散修」身份在晋京曝光，遭阴罗宗与天澜圣殿联手追杀。
- 重宝入手：意外获得寒髓与回阳水配方，为日后进阶化神埋下伏笔。

## 第946章 韩长老

### 韩立重炼雪晶珠
韩立在某道观闭关一日一夜。发现**寒髓**的寒气丝毫无法被**紫罗极火**融合，叹其现今只能当材料，遂取出雪晶珠（雪晶球），滴入一滴寒髓银珠，以一缕青色婴火（紫罗极火）包裹凝练，欲在大拍卖会前重炼完毕。

### 晋京叶家「韩长老」
- 晋京北部皇城叶家府邸，一名青衫年轻人，相貌与韩立有**七分相似**，唯一双毫无感情的冷冰冰眼珠；动怒时**瞳孔深处泛黑、紫两色异芒**，整个人妖异鬼魅。被叶家尊为「韩长老」。
- 自称韩立诛杀其**主魂**，致其无法恢复全部修为，故视同名的韩立为大仇家。手中有一口被黑丝缠缚、灵性十足的金光小剑——乃韩立用**金雷竹**炼成的人界飞剑（青竹蜂云剑之一），他喷黑血压制其灵性，欲灭韩立后顺势收服无主之剑。
- 为叶家效力二十年，条件是叶家替其搜集**魔器**（伤人伤己的法器，已得一件）。叶家二长老决定：将追杀同名韩立之事交韩长老全权处理，只要不干扰大拍卖会即可。

### 叶家密议（云逸·二长老·美艳妇人）
- 乌冠老者**云逸**回报：阴罗宗（鬼罗宗）修士在京城坊市暗查一名与韩长老同名同貌之人；阴罗宗元婴长老忽然齐聚晋京。
- 那「另一个韩立」此前曾混入皇族修士栖身的**皇清观**，目的不明，美艳妇人疑其针对叶家。
- 方脸**二长老**（二叔）定调：韩长老修为不弱、通晓上古秘闻，是数年后「取宝之行」的关键助力，须全力笼络但也戒备（疑其隐瞒真实实力）；并设法协助铲除那同名之人。核心计划只对叶家本族核心长老透露。

## 第947章 大拍卖会

- 叶家炼成的**平山印**已在三皇观（三观）完成最后融合。太一门、天魔宗分别派**玄青**、**七妙真人**入宫，欲迫叶家出售平山印。二长老打算以平山印为明处诱饵，借两家相争浑水摸鱼，遮掩真正计划；嘱知情长老外出结伴，防七妙真人秘术暗算套取神识。
- **阴罗宗**遍寻韩立未果（韩立一直在道观炼器未出）。
- 拍卖会当日，韩立化为**黑脸大汉**、将竹筒幻化长剑，炼气收息压修为至**结丹期**入晋西坊市。入口处**葛天豪**等黑袍阴罗宗修士以神识扫视陌生散修打草惊蛇；并向天魔宗借得**妙音宝镜**，祭炼两三日后可破形，届时拿下韩立。天澜圣女未现身。
- 因参拍者激增三成，**元婴级拍卖会移至向天机阁临时借来的「九霄殿」**（悬空白石单层大殿，入场一千灵石），**结丹级仍在宝光殿**（一百灵石）。韩立恢复元婴气息，掷十块中阶灵石入场，得带号玉牌，飞入九霄殿。

## 第948章 墨金

- 主持为一元婴中期顶峰皂袍修士。先拍**姜黄晶**（南疆西部大沙漠产，炼土御性法宝材料，一万二千灵石成交）等不温不火之物。
- 殿中有三名佛宗大和尚（枯瘦、肥头、慈眉白须），中间老僧神识灵敏，曾感应韩立目光——韩立首见佛宗高阶修士。
- 后期出现**墨金**（重三斤四两，炼器稀有材料，底价三万）。韩立志在必得，与一名**黑气罩体老者**竞价，最终以**十二万灵石**拍得（远超常价）。
- 黑气老者转而以**二十余万灵石**拍得更稀的**离火沙**。其实为叶家**云逸**雇佣的**高姓老者**：事后于晋京民宅将离火沙交予云逸，云逸付灵丹（事成翻倍）、灵石及一份明日竞拍材料情报；并告知数日后地下交易会另有墨金可换。
- 首日拍卖结束，阴罗宗修士已撤离坊市，韩立从容离去。

## 关键看点
- 「韩长老」是与韩立同名同貌、瞳现黑紫异芒、自称主魂被韩立所杀的神秘强者，已夺得韩立一柄金雷竹飞剑，构成新威胁线。
- 叶家以平山印为饵、暗中囤积炼器材料（墨金、离火沙等）以备「数年后取宝之行」，并雇散修竞拍、事后灭口。
- 韩立购得苦寻已久的墨金，但高价与阴罗宗妙音宝镜破形之谋，使其身份面临暴露风险。
