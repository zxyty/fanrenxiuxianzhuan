# 第1377-1379章 真龙之魄 / 血晶摩诃剑 / 赤鬼

## 一、真龙之魄现身（1377）
- 天空金灿龙首被叶楚识破：体内灵力仅合体中期妖修水准，非真灵级，乃**强行催动真龙之血幻化的『真龙之魄』**。
- 韩立靠**大衍决**流转挣脱真龙之威的震慑。
- 白袍少女（叶家少主，下文『叶颖』）判定是**陇家**冲她身上的**天凤灵血**而来；陇家少主仅化神修士，单凭己力催不动真龙之血，**多半另有炼虚级修士随行**。
- 真灵世家有约：严禁互夺真灵血脉，违者遭诸世家联手剿灭；陇家不敢在人族势力范围动手，故借故跟至此地突袭。
- 叶家、陇家皆上古真灵家族，灵血深植血脉万年，按理无法夺取——叶楚疑陇家已有手段。

## 二、天凤之魄对决真龙之魄（1377）
- 叶颖布紫色法阵，吐出五团光球——内含**箫、琴、琵琶、刀、环**五件迷你仿制灵宝（黄球琵琶即韩立见过的**诧灵琵琶**）。
- 叶颖吞五色灵丹（彩凤所化），叶楚喷木灵气相助，头顶现青色光晕、白火、人形虚影。
- 叶颖灌入天凤之血，化为百丈**五色彩凤（天凤之魄）**；对面真龙之躯长二百余丈。
- 彩凤一翅生青色狂风、一翅生白色火海；金龙召银色电蛟群。三力交织肉搏。

## 三、阴阳化极决与血晶摩诃剑（1378）
- 彩凤施**阴阳化极决（阴阳两极决）**：『五行转换，众元归一』，将风之力化为蓝色冰莲极寒之力，与白焰相辅，压制雷电。
- 金龙大笑称『终于等到』，竟知此功法；随即化血红，龙口喷血柱凝成十余丈**血晶摩诃剑**（龙首凤身、血腥灵宝）。
- 陇家少主缩身化血光没入剑中，一斩将**血凤虚影从彩凤体内斩出并卷回剑内**，夺走叶颖大半天凤之血；彩凤溃散，血光坠回叶颖（叶颖睁眼，惊恐）。
- 叶楚指出：天凤之血只是暂困剑中，对方未真正炼化，夺下此剑仍可夺回甚至反抢真龙血脉。叶颖许诺韩立：助夺回灵血，叶家愿替他做一件力所能及之事。
- 三人升空夺剑，途中**陇羽、陇鳞**现身：陇家兄弟，千余年前炼虚初期、今炼虚中期，擅**养鬼驱魔术**。点破叶楚乃叶家派往木族的卧底（昔年威名正盛时销声匿迹）。
- 叶颖追问『阴阳两极决是否你们故意卖给我』（否则天凤之血不会被轻易分离），二陇拒答。陇家放出血书赤红魔影、漆黑画轴黑风。

## 四、赤鬼与筱虹（1379）
- 叶颖祭金刀（百丈刀光）、翠绿圆环；陇家以白色光幕护剑。一陇撕袍露出身上七八个赤红鬼头，放出五六丈**赤发红鬼（赤鬼）**，化巨剑挡下擎天刀光。
- 叶颖以箫琴琵琶三宝合三色光霞攻击，仍被红鬼瞬移偷袭，五件灵宝与七只红鬼缠斗，无法分心夺剑。
- 叶楚放十几口青黄飞剑，陷于陇家『披发魔首』瘴气阵中。
- **黑凤族筱虹**（黑裙少妇、妖修）牵制韩立，索要**天凤之翎**：妖族卧底早传回木族藏三支天凤浴火重生之翎，叶楚潜伏多年、叶少主亲自接应即为此物。
- 韩立摊牌『让还是不让』遭拒，遂出手：
  - 以**罗烟步+疾风九变**鬼魅近身，弹指射金丝剑气逼退筱虹（斩其裙角）。
  - 释**元磁神光**灰光卷击；袖手虚按召五颗白骨骷髅喷**五色极寒之焰**；背生**风雷翅**电闪瞬移至筱虹头顶，洒数十道金色剑光。
  - 筱虹以黑焰、墨色古镜乌光、白丝网招架；韩立催剑诀令数十剑光化数百铺天斩下，筱虹脸色一白。

## 五、势力/血脉对照
- 叶家＝天凤血脉真灵世家；陇家＝真龙血脉真灵世家；二家皆属真灵世家联盟。
- 真龙天凤等『逆天级』存在，典籍称不惧真仙界真仙，韩立由此战推断或确有其事。

## 背景
第九卷 灵界百族，第1380-1382章。多方混战：韩立（叶家阵营）对阵陇家双修（陇东等），战场另有叶楚、白袍少女、黑凤族妖女筱虹等。

## 第1380章 力压黑凤
- 韩立以元磁神光、青竹蜂云剑剑光、五色光焰、灰色霞光层层压制黑凤族少妇筱虹，其白色丝网、黑焰、五色光焰、古镜乌光尽数被破。
- 筱虹化十余丈黑凤本相，施展**空间神通**遁入空间裂缝。韩立浮现第三只妖目，施**破灭法目**（射出纤细黑丝）破其空间术，将黑凤逼出现形。
- 韩立以风雷翅瞬移、虚天鼎（青色小鼎）喷青丝捆缚，化漆黑大手（手臂脸颊浮金色鳞片）扼黑凤脖颈，借风雷翅之力将其如流星砸入地面，砸出十余丈巨坑。
- 黑凤昏厥。韩立摄走其储物镯，打入数张符箓，召出**元磁神山**（黑色小山涨数十丈）将筱虹镇压禁制于山下。
- **未杀的原因**：有他人在场，不愿真与七大妖族结生死大仇；他不像叶家、陇家是真灵世家。出手太快、神通诡异，筱虹（黑凤族嫡系妖修）出其不意被轻易拿下。
- 陇家两名修士目睹韩立以化神中期击败后期对手，惊讶。其一发血光（被叶楚青色闪电打成暴虐鬼头反扑叶楚）。
- 韩立趁机欺近空中**雪晶摩诃剑**（悬浮、血光、白色光幕护持）。一陇家修士拍出乳白色光手（炼虚秘术），韩立以肉身金光、黑白双拳硬撼击碎之。又以两颗雷珠攻光幕，引出黑色鬼臂格挡。金色剑气攻光幕时被十几道黑影击散。

## 第1381章 啼魂变身
- 黑影来源为陇家豢养千年的**无相鬼王**：身高十余丈、生双角、黑鳞、肉翅、奇长精铁手臂、面部扁平无脸孔、身躯千疮百孔却能吸黑色阴气瞬间复原。神通不下炼虚级。
- 鬼王凝聚百丈黑色鬼爪压向韩立。韩立放出**啼魂兽**（半尺黑色小猴），啼魂一击胸膛进阶变身为三十余丈巨猿，鼻喷青霞（摄魂神光）消解鬼爪、吞噬阴气；鬼王五道黑芒射入巨猿身体毫无效果。
- 鬼王再变：青焰战甲、白骨大锤（妖鬼头骨炼制，一黄风一黑雾），以黑色霞光击散青霞——**摄魂神光对鬼物失效**。
- 啼魂二次进阶变身：毛发变红、生三弯角、眉间裂开血红妖目、獠牙、背生三根黑骨刺，化狰狞恶鬼，阴气更胜无相鬼王。妖目喷出**血色电弧**，鬼王白骨锤、青焰战甲遇之如遇克星尽数溃散，电弧贯穿鬼王胸腔、缠绕缩小，被啼魂一口吞噬。
- 陇家双修、叶楚、白袍女子皆目瞪口呆。
- 韩立瞬移至白色光幕，召七十二道金光聚成十丈金色巨剑（青竹蜂云剑）一斩，光幕碎裂，雪晶摩诃剑裸露。金剑斩血剑剑身被反弹，雷珠亦无效。
- 陇家修士得意：**血晶摩诃剑乃混沌万灵榜上之宝**，凭韩立本事无法毁掉。
- 韩立将金巨剑分解回七十二口小剑、再幻化数百道金色剑光，掐诀布下**大庚剑阵**，以血剑为中心，欲逼出陇东元神或毁掉灵宝。

## 第1382章 剑阵与血剑
- 大庚剑阵布成（直径百余丈，无数金丝向中心靠拢）。陇家催血书射十道赤红魔影，被韩立**辟邪神雷**（金色电弧）克制灭杀——魔影带浓厚魔气，被神雷克制；韩立修为大进，辟邪神雷比人界时强出何止倍许。
- 啼魂青霞反袭七只赤发恶鬼，吞三只、逼退四只，白袍少女得以脱身，加入叶楚一方，五宝合青光压制陇家紫色瘴气。陇家再祭红色小鼓（赤红鬼蜂）、蓝色铁尺（八爪章鱼魔物）勉强稳住。
- 无陇家干扰，大庚剑阵全力运转，金丝切割血剑发金属摩擦声。血剑欲冲天逃出剑阵被反弹回，无人主持下灵光渐黯、哀鸣。
- 陇东传音：先威胁（毁此剑便与陇家不死不休、叶家庇护不久），后利诱（翻倍好处），韩立皆不理。
- 血剑灵光被磨尽、剑身现密布裂痕。韩立猛然醒悟（脸色大变）化青虹逃出——血剑**自爆**：炙白骄阳爆发，狂涨数百丈，金丝白芒接触即碎。顶阶通天灵宝自爆，合体期修士也只能暂避。
- 韩立提前以明清灵目（蓝芒）看破，险逃；仍有十几口飞剑被波及损伤灵性（因青竹蜂云剑掺炼晶等珍稀材料方未剑毁灵散）。
- 韩立以明清灵目看清骄阳中异状：一条五爪**血龙**口咬残缺**血凤**，借自爆掩护欲隐匿逃走。韩立瞬移至其上方，弹出五只骨戒喷出**五色寒焰（乾蓝冰焰）**将血龙困住，动作放慢十倍余。
- 韩立收回五色寒焰裹住血龙，再放出虚天宝鼎（虚天鼎），催通宝决涨数尺，喷无数青丝围杀下方血龙。（章节续）

## 要点
- 韩立战力凸显：化神中期跨级压制后期妖修与炼虚级鬼王、硬撼炼虚秘术。
- 啼魂兽设定：可两度进阶变身（巨猿→恶鬼），血色电弧专克鬼物，可吞噬炼虚级鬼王。
- 灵宝信息：雪晶/血晶摩诃剑=混沌万灵榜·顶阶通天灵宝，自爆威力惊人。
